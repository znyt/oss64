　　萧逸背靠着寒晶冰棺，静静地坐着，沉默着。
<br/>　　一动不动，一连坐了数天。
<br/>　　双眸，盯着洞穴之外的无数罡风。
<br/>　　思绪，却比这些罡风，更为凌乱。
<br/>　　原本清澈的眸子，渐渐泛出了血丝。
<br/>　　煞白的脸色，竟是比这遍地的白骨，更无生气。
<br/>　　连裂天剑姬那等至强者都无能为力。
<br/>　　连裂天剑姬那活了数百年的见识和手段，都没有丝毫办法。
<br/>　　连冰心圣果这等九品天材地宝，都没有任何效果。
<br/>　　他，能有什么办法？
<br/>　　……
<br/>　　而同一时间，裂天剑宗一行人，已回到了宗门。
<br/>　　冰缘大会，已结束半月有余了。
<br/>　　在雪翼雕一族那位极境强者的帮助下，白冰雪等人，很轻易地便离开了极寒之地。
<br/>　　而出了极寒之地，回到炎武王国境内，一行人御空飞行并非难事。
<br/>　　回到王都自然也不必耗时太久了。
<br/>　　言归正传。
<br/>　　白冰雪等人，一回到宗门，自是立刻前往竹林。
<br/>　　竹林，凉亭内，一白衣女子凭栏而坐，惬意假寐着。
<br/>　　不时抚琴一番，看一眼斑驳竹林间隙下照样而来的微暖阳光。
<br/>　　那样子，慵懒至极，丝毫不像一位武道至强者。
<br/>　　“嗯？”女子忽然睁开了眼睛，缓缓一笑，“回来了吗？”
<br/>　　“终于又有人给我抚琴了。”
<br/>　　女子自顾说着，慵懒地伸了个懒腰。
<br/>　　不多时，白冰雪、钟无忧等人，来到了凉亭前。
<br/>　　“师尊…剑姬前辈。”
<br/>　　众人恭敬行礼。
<br/>　　“嗯。”女子点了点头，随即轻咦一声，“怎么就你们四人？”
<br/>　　“萧逸小子呢？又自个儿跑没影了？”
<br/>　　钟无忧直起身，撇嘴道，“这次他不仅人没影了，恐怕渣都不剩了，宗主那群王八蛋…”
<br/>　　钟无忧愤愤地说着。
<br/>　　恰在此时，十道身影，从竹林之外飞跃而进。
<br/>　　不多时，来到了凉亭前。
<br/>　　正是宗主以及九位长老。
<br/>　　女子见状，眉头微微皱了起来，一股不妙的念头，涌上心头。
<br/>　　“怎么回事？冰雪，你说。”女子看向白冰雪。
<br/>　　“是。”白冰雪点点头，随即将冰缘大会所发生之事，原原本本说了一遍。
<br/>　　白冰雪话音刚落下。
<br/>　　女子已勃然大怒，素手一拍桌案，桌上器皿，尽皆粉碎。
<br/>　　“混账。”
<br/>　　女子怒喝一声，冰冷的目光，看向宗主等人。
<br/>　　“你们可知自己做了什么蠢事？”
<br/>　　宗主等人，包括白冰雪在内，瞬间一惊。
<br/>　　真正熟知女子的人都知道，女子向来恬静淡然。
<br/>　　这般发怒，还是第一次。
<br/>　　一众长老，不禁咽了口口水，心悸地低下了头。
<br/>　　唯有宗主，咬了咬牙，踏前一步，直视女子。
<br/>　　“蠢事？我倒觉得我们只是在弥补前辈您之前的愚蠢之举。”
<br/>　　“你说什么？”女子双眼一眯，皓齿下，泛着冰冷，“你再说一遍。”
<br/>　　女子的威压，显然非常人能承受。
<br/>　　宗主这般修为，竟不自觉地打了个冷颤。
<br/>　　但半晌，宗主还是咬牙硬声道，“难道不是吗？”
<br/>　　“当日王都大战，前辈保下萧逸，赶走墨寒，不是愚蠢是什么？”
<br/>　　“论资质，墨寒身怀武道冰晶武魂。”
<br/>　　“论威望，墨寒如日中天，受所有人支持。”
<br/>　　“国主、药王谷、血雾谷、王都九成以上九成势力，还有我们宗门本身的一众长老。”
<br/>　　“而萧逸呢，他算什么？”
<br/>　　“他只不过是北山郡那等小地方冒出来的忽然崛起的天才罢了。”
<br/>　　“墨寒万众归心，前辈反而一意孤行，将他赶走。”
<br/>　　“让得我们裂天剑宗，站到了整个炎武王国的对立面。”
<br/>　　“这不是愚蠢是什么？”
<br/>　　宗主一句又一句，语气，也愈加激动，乃至冰冷。
<br/>　　“放肆。”一旁大长老冷喝一声。
<br/>　　“宗主，你可知道你自己现在在说什么？”
<br/>　　“何时轮到你质疑前辈的决定，你又有什么资格用这样的语气在前辈面前放肆。”
<br/>　　“我清楚得很。”宗主激动地吼了一句。
<br/>　　“你清楚些什么？”女子再难掩心头怒火，一巴掌凌空扇出。
<br/>　　宗主直接被轰飞，口吐鲜血，倒于地面。
<br/>　　“你若真的清楚，岂会干这种蠢事，萧逸…”女子冷声说着。
<br/>　　宗主，却状若疯狂地打断道，“休要再给我提那些虚无缥缈的武道真意。”
<br/>　　“武道真意，是天极境的范畴。”
<br/>　　“可是，天极境，谁能真的达到？”
<br/>　　“我裂天剑宗，传承无数年，惊才绝艳的先辈数之不尽，谁曾达到过？”
<br/>　　宗主双眼锱铢欲裂，直视女子，“天极境啊，天极境啊。”
<br/>　　“老宗主达到了吗？裂天剑魔前辈达到了吗？”
<br/>　　“哪怕是前辈您，您又有那个自信说必能达到吗？”
<br/>　　“你…”女子一时语塞。
<br/>　　宗主激动道，“连你都不敢说必定能达到，你又凭什么说萧逸必能达到？”
<br/>　　“说到底，墨寒才是宗门的最大希望。”
<br/>　　“前辈真想让宗门在你手上毁于一旦吗？”
<br/>　　“愚蠢的，错的，是你。”
<br/>　　宗主颤抖着手指，指向女子。
<br/>　　女子沉默着，脸上的冰冷，化为了复杂。
<br/>　　“错了吗？”女子喃喃自语。
<br/>　　“错了，确实是我错了。”
<br/>　　女子蓦地叹了口气。
<br/>　　“师尊。”一旁白冰雪迟疑着想说些什么。
<br/>　　女子摆摆手，落寞道，“从15年前起，我便错了。”
<br/>　　女子摇了摇头，站了起来，转身离去。
<br/>　　宗主冷笑一声。
<br/>　　女子蓦地转过头，“你今日这些话，是为宗门着想，还是为了白墨寒着想，你清楚得很。”
<br/>　　宗主冷笑道，“是，我是为了墨寒。”
<br/>　　“我是他的师尊，自该给他最好的，自该一切为了他。”
<br/>　　“可那又怎样？”
<br/>　　“萧逸，已经死了，死了啊。”
<br/>　　“前辈你没得选择，日后，宗门始终会交到墨寒手上。”
<br/>　　“待得墨寒回归，宗门会给他最好的，整个炎武王国会给他最好的。”
<br/>　　“要不了多久，就连冰武王国那位大皇子，都会被他踩在脚下。”
<br/>　　“墨寒，会成为整个炎龙大陆的传奇。”
<br/>　　女子闻言，摇了摇头，自顾离去。
<br/>　　那张绝美的容颜下，泛着冰冷，心灰意冷的冷。
<br/>　　后方，宗主等人肆意大笑，“传令下去，举宗同贺。”
<br/>　　“宣告全王都，迎接墨寒回宗。”
<br/>　　“是。”一众长老，应答一声。
<br/>　　钟无忧等人的脸色，难看到极点。
<br/>　　……
<br/>　　入夜，月明星稀。
<br/>　　竹林深处的宗门祠堂内。
<br/>　　女子负手而立，看着前方无数灵位，陷入了沉思，神色落寞至极。
<br/>　　“我答应过师尊，会保住宗门。”
<br/>　　“可现在，宗门却要毁在我手上吗？”
<br/>　　“若是15年前，我心硬些，便不会有今日之祸事。”
<br/>　　“若是…”
<br/>　　女子落寞地自语着，又摇了摇头，她知道，没有若是。
<br/>　　女子蓦地转过头，看向宗门祠堂之外的远方天际。
<br/>　　“若是…你能回来，我许你一切。”
