　　洛前辈已离去。
<br/>　　原地空间封锁，也已消散。
<br/>　　“两位前辈可有大碍？”
<br/>　　萧逸急急问了一声，刚要放出感知，云渊长老二人反倒闪身来到他身旁。
<br/>　　“我们无碍。”云渊长老擦了擦嘴角鲜血，道。
<br/>　　“倒是你…”
<br/>　　“我没事。”萧逸摇了摇头。
<br/>　　他乖乖交出剑帝本源，洛前辈自没有对付他。
<br/>　　二人闻言，松了口气。
<br/>　　“可恶。”云渊长老霎时暴怒，“没想到老院长竟如此不要脸，万里迢迢特地来此，就为了抢一个小辈的东西。”
<br/>　　“林夜那老怪物也是，平时神出鬼没，真地关键时刻却没人影。”
<br/>　　讳执事摇了摇头，“老院长显然是有备而来。”
<br/>　　“否则，他不会特地等在万剑碑林外。”
<br/>　　“剑帝本源在剑帝碑内的话，天下无人能夺。”
<br/>　　“可萧逸小子将剑帝本源带了出来，那就不一样了。”
<br/>　　“老院长是看准了剑帝碑关闭，林夜不会出来，更不会知晓这里的情况。”
<br/>　　“有备而来？”云渊长老眉头一皱，“他就这么确定萧逸小子能得剑帝本源？”
<br/>　　讳执事肯定地点了点头，“我说了，副院长与我说过这些事。”
<br/>　　“老院长，早便知晓萧逸小子比谁都出色，也远比莫悠出色。”
<br/>　　“而且，老院长曾经亲自指导过萧逸小子，以他的手段和眼力，自然知晓萧逸的本事。”
<br/>　　“他得知萧逸小子来参加剑域盛事，已经猜到萧逸小子有很大的机会能得到剑帝本源。”
<br/>　　不错，洛前辈是有备而来。
<br/>　　而且，也已猜到萧逸有很大的几率能得到剑帝本源。
<br/>　　否则，他不会特地来此，还特地等候在万剑碑林之外半月时间。
<br/>　　甚至于，还把学教至宝破晓钟也一并带来了。
<br/>　　剑帝本源，正常人，根本无法携带。
<br/>　　即便是萧逸，也须得碑灵为他在手腕上开辟一处空间。
<br/>　　若洛前辈不是对萧逸极有把握，根本不会把破晓钟也一并带来。
<br/>　　云渊长老，猛地盯着萧逸，“那你也不能乖乖交出剑帝本源。”
<br/>　　“好歹战一番，我和无视定当拼命护你，为你争得一丝逃遁机会也不一定。”
<br/>　　“不行。”两道声音，忽然异口同声而出。
<br/>　　说话的，是萧逸和讳执事。
<br/>　　二人对视了一眼，笑笑。
<br/>　　“什么？”云渊长老，再次眉头紧皱。
<br/>　　“杀意。”萧逸率先张开了嘴巴。
<br/>　　“杀意？”云渊长老面露疑惑。
<br/>　　“不错。”讳执事沉声道，“云渊，你没发现，老院长眉宇间已露出杀意。”
<br/>　　“虽然这抹杀意隐藏得极深，却没瞒过我。”
<br/>　　“我敢确定，若萧逸小子不交出剑帝本源，他会毫不犹豫地杀了萧逸小子。”
<br/>　　“当然。”讳执事满意地看向萧逸，“这小子也发现了，而且也处理得很好。”
<br/>　　萧逸苦笑一声。
<br/>　　洛前辈这等人物，若真的下杀手，怕是他和云渊长老、讳执事三人，休想活命。
<br/>　　讳执事继续道，“这小子不着痕迹地带出‘无拖无欠’这个理由，倒是让老院长没了理由杀他。”
<br/>　　“我又不是傻瓜。”萧逸撇撇嘴。
<br/>　　“而且，不知道为什么，我见过的传奇强者，已然不少。”
<br/>　　“可洛前辈给我的感觉，比谁都可怕。”
<br/>　　“多可怕？”讳执事问道。
<br/>　　“不知怎么说。”萧逸摇了摇头，“心里发颤吧。”
<br/>　　对，萧逸见过的传奇强者，如风刹殿总殿主、修罗殿总殿主、天藏学宫老院长等等。
<br/>　　他当然也无法准确知晓他们的境界实力。
<br/>　　但却大概有个感觉。
<br/>　　可他们给萧逸的感觉，都仅是深不可测。
<br/>　　可洛前辈，却给萧逸一种不由来的可怖，甚至是心神发颤。
<br/>　　这种感觉，并非是实力上的感觉。
<br/>　　反倒是更像是一种莫名的气息。
<br/>　　如果说修罗总殿主的强大，是如万仞高山地高大，或如大地般的厚实。
<br/>　　那么，洛前辈，则似极了一个深渊，轻易让人万劫不复。
<br/>　　讳执事和云渊长老闻言，同时看向萧逸，眼中闪过一丝异色。
<br/>　　“小子，我只能告诉你，你的直觉，果真很准。”讳执事沉声道。
<br/>　　萧逸皱了皱眉，他总觉得，讳执事二人似乎知晓什么秘辛，他却欲言又止。
<br/>　　“好了，别多说了。”云渊长老快速道。
<br/>　　“无视，你我立刻赶回学教。”
<br/>　　“若是能联合副院长与一众学教老师，说不定能让老院长改变主意，在莫悠吸收之前，拿回剑帝本源。”
<br/>　　“没用的。”讳执事摇了摇头。
<br/>　　“怎么没用。”云渊长老急道，“惊风长老、天赤长老等一众长老，始终认萧逸这个弟子。”
<br/>　　“副院长就更不必说了。”
<br/>　　“还有一众学教弟子。”
<br/>　　“难不成老院长还能一手遮天，不顾学教所有人反对不成？”
<br/>　　云渊长老，显然是急坏了。
<br/>　　堂堂巅峰剑修，一身傲气，也只能想出个联合其他学教老师，联手反对洛前辈的办法。
<br/>　　“时间不够。”洛前辈言简意赅。
<br/>　　“以老院长的速度，怕是无数几个时辰就能回到黑云学教。”
<br/>　　“笑话。”云渊长老冷笑一声，“莫悠即便有老院长辅助引导，也不可能在短时间内吸收殆尽。”
<br/>　　“剑帝本源，可是剑帝的本源力量，若想悉数吸收，成长到他那个高度，起码…”
<br/>　　对，碑灵曾说，若萧逸自己一个人吸收，怕是不知何年何月才能悉数吸收殆尽。
<br/>　　即便萧逸在剑帝碑中，碑灵替他引导，也需要数十年时间。
<br/>　　数十年时间，对比那些动辄过万年、数十万年，乃至更长的岁月，已经算是很短了。
<br/>　　讳执事冷声打断道，“若是老院长非是让莫悠吸收炼化呢？”
<br/>　　“你是说…”云渊长老脸色一变。
<br/>　　“不错。”讳执事沉声道，“老院长必定会用于莫悠的武魂吸收。”
<br/>　　“莫悠的武魂，年幼觉醒时便是黑色阶品。”
<br/>　　“前些天待得老院长全力为他觉醒，他的武魂，已俨然是当世最强的黑色武魂。”
<br/>　　“武魂，乃天地规则所化，剑帝本源融入武魂，等同融入契合度最高的天地规则中，根本就是轻松之事。”
<br/>　　“再加上老院长的全力引导，要不了多久就能将整份剑帝本源融入莫悠的武魂中。”
