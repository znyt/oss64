　　像是现在一些小机遇，北邙学院的高层当然看不上，不屑于去争抢。
<br/>　　但是楚风眠身上的机遇，人皇罐，战龙决，祖龙至尊剑，这种东西要是一旦现世。
<br/>　　只怕是北邙学院的那些老古董们，都没有任何一人会能够抵挡得住其中的诱惑。
<br/>　　楚风眠要是真的展现出来这些，才是真正的找死。
<br/>　　楚风眠可不相信，这些老古董可能抵挡得住诱惑。
<br/>　　前世的时候，剑道之主的剑道，令七大宗门都不惜一切代价想要得到，围杀了剑道之主。
<br/>　　七大宗门尚且如此，北邙学院也是一样。
<br/>　　要是真正的大机遇，像是人皇罐，这些老古董都会疯狂的去争抢，不会在乎什么所谓的脸面。
<br/>　　起码在楚风眠度过生死大劫之前，这些秘密，还都全部不能暴露出来。
<br/>　　“走吧，老夫现在就带你们二人，前往北邙圣域。”
<br/>　　太上长老大手一挥，直接席卷着楚风眠跟白萱的身躯。
<br/>　　楚风眠也不抵抗，任由太上长老，抓着楚风眠，一跃踏入到了空间之中。
<br/>　　这空间周围，极为扭曲，这种扭曲，已经不是先天出现的了。
<br/>　　而是一些大能，以无上手段，强行进行的扭曲，让人们踏入其中，都无法察觉到下一步的道路。
<br/>　　太上长老，就在这扭曲空间之中，带着楚风眠跟白萱，连连穿行，最后便是来到了一处特殊的空间之中。
<br/>　　这空间的周围，都被扭曲，常人看见，都不敢触碰。
<br/>　　不过太上长老却是带着楚风眠跟白萱，直接踏入到了这扭曲空间之中。
<br/>　　只见周围的空间，天翻地覆，太上长老带着楚风眠跟白萱，一下子来到了另外一处极为古老的空间之中。
<br/>　　这空间，古老到极致的地步，仿佛是远古时代的遗留。
<br/>　　楚风眠感觉到，在这圣域之中，其中的种种法则，都是跟外面，极为不同，是一处截然不同，玄妙无数倍。
<br/>　　在其中修炼，一日时间都比起外面十日时间，还有有效率的多。
<br/>　　尤其是这里可是远古大陆遗留，其中蕴含的种种远古法则，对武者来说，也有着极大的好处，足以参悟。
<br/>　　“在外面，我可以随意打破空间，就算是在北邙学院中，我现在都可以穿梭空间，可是这里，这周围的空间，却是如同铜墙铁壁一般。”
<br/>　　楚风眠打量了周围一眼，不由的感叹了一声。
<br/>　　他的实力，就连生死境高阶的强者，都可以直接捏死，可是在这里，却是无法打破空间。
<br/>　　远古时代，种种天地神兽，异兽，先天强者，在其中疯狂战斗。
<br/>　　要是法则不坚固，只怕是远古大陆都不知道被打碎多少次了，能够承受住这种传说中的战斗，远古大陆的法则，自然是坚硬无比。
<br/>　　像是现在的九域，只是曾经远古大陆的一部分，演化而来的，其中的法则还很薄弱，神力境就可以打破空间。
<br/>　　而在这里，生死境高阶，甚至是生死境巅峰的人物，都无法打破空间。
<br/>　　“这里，便是我们北邙学院的圣域！”
<br/>　　太上长老开口道，语气之中，也蕴含着十足的骄傲。
<br/>　　“这北邙圣域，是远古大陆的遗留，其中的种种神奇，等你们到了便知，现在我就带你们前去参加考核。”
<br/>　　“你们二人，虽然的被学院关注，拥有了晋升的资格，不过若是无法通过考核，也依然无法真正晋升为种子弟子。”
<br/>　　太上长老冷冷的开口道。
<br/>　　“这考核，可不容易，只有五成几率可以通过，你们二人还要全力才行。”
<br/>　　“哦？”
<br/>　　楚风眠也有些没想到，这北邙学院，晋升种子弟子，居然是需要如此麻烦。
<br/>　　单单是其中一个资格，便是难以获取，得到资格后，还需要另外进行考核才行。
<br/>　　千挑万选，才能晋升成为种子弟子。
<br/>　　也怪不得诺大的北邙学院，种子弟子最多才不到一百人，单单是如此严厉的考核，就很难通过了。
<br/>　　不过楚风眠倒是自信满满，他现在的实力，被说是一个种子弟子，甚至就是核心弟子，那夜天君，楚风眠都有着一战的自信。
<br/>　　这考核，想要通过，简直是轻而易举。
<br/>　　太上长老，带着楚风眠跟白萱，来到了一处广场之上。
<br/>　　这广场之中，有着一道森严无比的阵法，留下了一道门，这考核，便是就在这阵法中进行。
<br/>　　“这位是考核长老，你们的考核，便是由此他进行。”
<br/>　　太上长老将楚风眠跟白萱，带过来，便是站在一边去了。
<br/>　　“你们两个就是楚风眠，白萱？这次来参加考核的人？”
<br/>　　考核长老，坐在一张椅子上，瞥了一眼楚风眠跟白萱，头也不抬的说道。
<br/>　　“你们两个，过来，进去，这阵法里面，有着一尊生死境六重的魔头，你们两个人斩杀了他，便是就算是通过了这次考核。”
<br/>　　“什么？生死境六重？”
<br/>　　白萱听到考核长老的话，脸色猛然一变道。
<br/>　　“这考核，不是只需要斩杀生死境五重的魔头，便是算通过吗？”
<br/>　　这种子弟子考核的消息，白萱也了解了不少，毕竟以她的资质，就算是没有碰到奇遇，将来也是很有可能会直接晋升种子弟子的。
<br/>　　这考核，应该是只需要斩杀一尊生死境五重的魔头，便是就算是胜利的。
<br/>　　可是如今，这考核长老所说的，可是生死境六重。
<br/>　　生死境五重，跟生死境六重之间，虽然只差一重境界，但是生死境六重，却已经是成为了生死境高阶。
<br/>　　只差一重境界，其中的力量，却是相差数倍，甚至是十倍的差距。
<br/>　　这种考核，简直就是故意刁难。
<br/>　　以神力境极限的实力，击败生死境高阶，简直是不可能做到的一件事。
<br/>　　考核长老目光闪烁，冷笑的开口道。
<br/>　　“那是平时，你们二人如今一起晋升，难度自然加大一点，你们二人联手，只要是能将这魔头斩杀，便是就算你们二人一起通过了，怎么，难道不应该吗？”
<br/>　　“那还请长老允许，我们二人分开考核，每个人斩杀生死境五重的魔头。”
<br/>　　白萱压抑了下心中的怒火，平静的开口道。
<br/>　　“这样如何？”
