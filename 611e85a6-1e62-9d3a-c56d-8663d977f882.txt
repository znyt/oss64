　　第329章
<br/>　　“打仗么，死人的事情是经常发生的。”
<br/>　　“可是大家要想一想为何而打仗，如果是为了自己保护自己的父母妻儿，那么死也无憾。但如果是为了杀死别人的父母妻儿，大家就要想一想，这仗该不该打。”
<br/>　　俘虏们被押在演武场，徐神机在点将台上慷慨激昂。
<br/>　　“我们蛤蟆寨的规矩大家是知道的，如果想回家的，我们发粮食发路费。如果想留在这里干的，大家以后就都是兄弟，大碗喝酒，大碗吃肉。”
<br/>　　“也有原先留在这里的兄弟，可能你们以前也认识，大家可以问问他们，在我们蛤蟆寨怎么样。有没有饿着过他们，有没有打仗的时候送他们去当炮灰，有没有自己不冲锋，却让大家去送死。”
<br/>　　兵败如山倒，程大雷除了缴获大量物资外，便是擒获了超过一千五百兵重甲兵俘虏。这些人身强力壮，如果能收为己用，将是战场上一股重要力量。
<br/>　　程大雷是有留下的意思，所以让徐神机忽悠，当然也可以说是感化他们。
<br/>　　这些人曾经都是莫鸣米的手下，老实说，在当老大这一点上，莫鸣米做得有些差劲。而蛤蟆寨优待俘虏的事情早已传开，这次一见局势不利，这些人便举手投降，不得不说有这方面的原因。
<br/>　　“如果想要留下的，可以在这里登记。想走的，当然也可以，只要在留个名字就行。”
<br/>　　“真的肯放我们走么？”有一名俘虏大着胆子问。
<br/>　　“当然……”徐神机略顿一顿：“只不过走后，还去帮着莫鸣米打仗的，下次再见面，我们可就不是兄弟，而是敌人了。”
<br/>　　这些人开始陆陆续续登记，有些人拿着路费离开，也有些人选择留下。最后一千五百名俘虏，留下的大概占三分之二，也就是一千人左右。
<br/>　　程大雷对这个结果是相当满意的，只不过这一千人暂时还不能投入战场。程大雷的法子是将他们送到落玉寨，派刘悲过去，给他们做思想感化工作。
<br/>　　同时，程大雷也开始令人修复因战斗而被破坏的蛤蟆寨，好在有固防和建筑精通的双成作用下，这个进程是很快的。
<br/>　　此时，这次战斗的战利品数目也已被清点出来。大概数目是：盔甲兵器若干，战马三百匹，工程车投石机若干，帐篷若干，粮草不计其数。
<br/>　　程大雷隐隐有种暴富的感觉，以战养战果然是发家致富最快的办法。
<br/>　　大概是十天后，战后工作完成，这一日程大雷把山寨的中层干部聚在聚义厅。
<br/>　　“我就说么，那姓莫的想和程当家作对，根本就是找死。”云中龙。
<br/>　　“还用你说，我第一眼就看出来了，姓莫的是秋后的蚂蚱，蹦跶不了多久。”乔鹤。
<br/>　　“看以后姓莫的还敢不敢猖狂，这块地方，可不是他说了算的。”游九楼。
<br/>　　云中龙三个毕竟背叛过程大雷，虽然最后迷途知返，当然，在他想来是迷途知返。这件事既然已经发生过，想要再获得程大雷的信任可就不容易了。
<br/>　　所以三人话里话外，都捧着程大雷聊，表面上笑呵呵，内心却十分忐忑。
<br/>　　“我知道打赢了这一仗，大家都很兴奋，但今天不是庆功的时候，真正的庆功宴，等我们平了琴川关再说。”程大雷。
<br/>　　云中龙心中一惊，难以置信道：“程当家，你要攻打琴川关？”
<br/>　　程大雷点点头：“怎么了？”
<br/>　　“啊，没什么，没什么。”云中龙赶忙说了一句，内心却震惊无比。
<br/>　　琴川关墙高兵多，想要打下来谈何容易。况且莫鸣米在琴川作威作福这么多年，渐渐大家已将他的存在视为一种定律，谁也没有取而代之的想法。
<br/>　　却没想到，程大雷要的不仅是打败莫鸣米，而是要攻占琴川关，取其而代之。
<br/>　　这件事把云中龙吓了一跳。
<br/>　　程大雷要的不是和莫鸣米在琴川关平起平坐，既然已经在正面战场上击败莫鸣米。此刻他刚刚被击败，损兵折将，士气低沉，正是痛打落水狗的好时候。
<br/>　　趁你病要你命才是程大雷奉行的不二法门，接下来，会议议题，就是如何攻打琴川关。
<br/>　　这也是为何今天把云中龙三人叫到这里的原因，对于琴川关的情况，他们三人是了解的。
<br/>　　这次行动的主要困难，就是如何进入琴川关，琴川关算是一座大城，靠程大雷目前手上的人去做攻城战，几乎相当于去送死。
<br/>　　会议讨论到很晚，却也没讨论出什么合适的法子。程大雷听着大家的法子，心中忽然一动：
<br/>　　自己在琴川关还有一只『眼睛』，怎么差点把这件事忘了。
<br/>　　胜利之后，蛤蟆寨洋溢着一种喜悦的气氛。大战之时有多紧张，如今就有多松弛。程大雷虽然说不许庆功，但不免还有许多人喝起酒来。
<br/>　　这也是没有办法的事。
<br/>　　夜到三更，月上中天，在许多人都喝得烂醉如泥的时候，有两个人睁开眼睛，悄悄潜出屋子。
<br/>　　明月照大地，今天不是做坏事的时候，二人对视一眼，都冲彼此眼中看到了紧张。
<br/>　　幸运的是，一路上没有碰到警戒哨，只要打昏门口的守卫，二人就可以潜出蛤蟆寨。
<br/>　　“要走啦？”
<br/>　　忽然想起的声音，吓了二人一跳。等二人回过神来，看到山寨的大门前坐着一人，在明月下独自饮酒。
<br/>　　二人一楞，随即默默低下头去。
<br/>　　“大当家。”
<br/>　　这二人不是旁个，正是林少羽和林冲。
<br/>　　“要走了也不说一声，这样做人，有些不地道啊。”程大雷没有看二人，只是一边饮酒，一边轻轻叹口气。
<br/>　　“大战结束，蛤蟆寨已经用不着我们了，我们……还是走吧。”林少羽。
<br/>　　“怎么，没有你们，这次我就过不去了么？”程大雷漠然道。
<br/>　　林少羽沉默良久，忽然道：“大当家，生为人子，杀父之仇不可不报呀。”
<br/>　　程大雷看看林少羽再看看林冲，这两个里面如果有一个明白人话也好说，可是一个明白人没有，就怎么讲也讲不明白了。
<br/>　　他无可奈何的叹口气，问：“想好去哪儿了么？”
<br/>(本章完)
