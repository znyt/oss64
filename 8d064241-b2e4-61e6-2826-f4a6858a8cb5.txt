        &lt;h3&gt;第三卷第二百八十八章  手术室外&lt;/h3&gt;
<br/>
<br/>        潘青松被陈元良赶出了病房，他确实有很多事情要做，陈子澜还在病床上躺着，手头上的事情一件又一件，整风肃纪是他的责任和义务，虽然他人微言轻，但是能拨拉几棵老鼠屎出来也是好的。
<br/>
<br/>        岳父大人明事理，这是他们这些小辈的福份，他和陈子澜恩爱一生，也和陈家的家教有很大的关系，陈子澜为人正直，孝顺父母，且陈家老两口其实都是他们的榜样。
<br/>
<br/>        他们也想要和陈家老两口一样，执子之手与子偕老，暮日苍苍携手同往。
<br/>
<br/>        大家在门口站了一会，陈子昂出来，而陈元良也要进手术室了，大家的心提起来了，心里都在默默的祈祷，祈祷一切安康。
<br/>
<br/>        陈元良看了一眼大家，冲大家挥了挥手，笑了笑。
<br/>
<br/>        虽然陈元良让大家都走，但是没有一个人离开，在手术室外，大家坐的坐站的站，悄悄的等待着，偶尔小声的说着话。
<br/>
<br/>        潘青松觉得有些事情应该让陈子昂知道，他走到陈子昂的身边低声说，：“子昂，我有些事情想给你说。”
<br/>
<br/>        “三姐夫，你说。”陈子昂并不知道潘青松知道了夏颜的事情。
<br/>
<br/>        “你没有上大学是有原因的，还有最近有些人专门针对你。”
<br/>
<br/>        “谁告诉你的？”陈子昂叮咛过陈子霖不让陈子霖说出去。
<br/>
<br/>        “张函！”
<br/>
<br/>        陈子昂沉默了，张函此时告诉潘青松是什么意思？
<br/>
<br/>        “这些事情你就不要管了，已经过去了那么久，我早都知道了，我已经不计较，不但不计较我还感激她，要不然我也到不了今天这种地步。”陈子昂怕三姐夫搅入这些事情对他不利。
<br/>
<br/>        他们陈家势单力薄，无背景无靠山，潘青松能到这个份上全靠自己的努力，陈子昂只想她的亲人们平平安安的度过一生，潘青松能不接触的尽量不要去接触，而且她还怕影响到潘禺的前程。
<br/>
<br/>        “子昂，你说的是什么话？如果你一味纵容和姑息就是助纣为虐，你看看现在的西城，信息口上乌七八糟的，成何体统？”
<br/>
<br/>        “姐夫，你想要做什么？”
<br/>
<br/>        “我要严查到底！一网打尽！”潘青松恨恨的说。
<br/>
<br/>        “你的想法是好的，但是现在还不是时机，小鱼小虾可以捞一捞，大鱼就会被你惊走。目前你的身份只能是给夏颜搔搔痒。”陈子昂微微一笑，目前她还不想动夏颜。
<br/>
<br/>        张函想让潘青松动夏颜，她就偏不如了张函的愿，这个时候动夏颜，无疑打草惊蛇，还有，她想让陈子乔慢慢的淡出来，按照陈子乔所做的事情，没有什么好下场，但是最起码能轻一点是一点，私心每个人都有，陈子昂早都不是圣母了，她的甸城之行还没有开始，甸城那边没有行动，这边是无论如何不能有动静的。
<br/>
<br/>        “你一个生意人，想那么多干什么？”潘青松听了陈子昂的话微微一愣。
<br/>
<br/>        “为了我的亲人们我不得不想，三姐夫，不是不报时候未到，夏颜现在势头正盛，你找她的麻烦不是自寻烦恼？”
<br/>
<br/>        “可是我的心里不舒服，我的工作性质也不容许我放任不管，这个倒不是因为你是我的家人。”潘青松嫉恶如仇，这事情让他知道了他必定要一查到底。
<br/>
<br/>        “这样吧，私人的事情你就不管了，省得人家说你是公报私仇，你呢让人查查洛一院和洛二院的事情，看看有什么情况，我给你提供一些线索。”陈子昂轻声说。
<br/>
<br/>        “你有线索？”
<br/>
<br/>        “你忘了我是干什么的？那个项目我们是有参加的。”陈子昂笑着说。
<br/>
<br/>        隔靴搔痒，夏颜一定会很开心的。
<br/>
<br/>        “那你把线索给我。”
<br/>
<br/>        “那你答应我不要再去调查我上学的事情了。”陈子昂盯着潘青松，她和潘青松讨价还价起来了。
<br/>
<br/>        “行，公事大于私事，你不计较我也懒得去计较了。”潘青松心里想着，反正从公事上也能动了夏颜。
<br/>
<br/>        “说好了，到时候我给你资料，但是你不能亲自动手。”陈子昂叮咛潘青松，“三姐夫，我三姐还在病床上躺着，你要留点精力来照顾我三姐。”
<br/>
<br/>        “好，我知道了。”潘青松想起陈子澜，心里也多了一丝权衡。
<br/>
<br/>        陈子昂和潘青松达成了一致，张函想要潘青松在西城搅起浑水的愿望落空了，但是他却不知道陈子昂已经开始布局了，他要陈子昂作为导火索，陈子昂却想要保护她的亲人们，陈子昂将导火索引到了洛城。
<br/>
<br/>        陈元良的手术进行了俩个小时，等医生出来的时候，大家用期待的眼神看着医生。
<br/>
<br/>        “手术很成功，后期就要看病人自己的意志了，尽量让他心情开朗。”医生看着陈子昂点点头。
<br/>
<br/>        大家都明白医生的话，有一些癌症病人手术过后是能存活很多年的，有些人就是几个月，最重要的和病人自己的求生意志有很大的关系。
<br/>
<br/>        “大家都回去吧，子建和我留在医院，你们该干什么就干什么去。”冉灵卉看着自己的丈夫被推到icu病房，她也只能是远远的看着，大家呆在这里也没有什么事情。
<br/>
<br/>        “妈，爸让你和我一起回去呢。”严慧芝对冉灵卉说。
<br/>
<br/>        “我不回去，我就在医院等着，他醒来就能看到我了。”冉灵卉固执的说。
<br/>
<br/>        “那大家就都散了吧，我陪妈妈在医院。”陈子建最近一直陪着俩个老人。
<br/>
<br/>        “二哥，你休息两天，我来陪妈妈吧。”陈子乔对陈子建说。
<br/>
<br/>        “别和我争了，我又没有事情可干，你们都得上班，你们让我闲下来干什么啊？”陈子建如今是完全担负起了照顾俩个老人的工作。
<br/>
<br/>        他电脑里的资料都没有了，原本想着看看冯睿有没有动他的电脑，可是那天听了夏颜的话，他简直是如万箭穿心，要不是有父母亲要他照顾，他怕是也会倒下来，现在照顾父母亲成了他支撑下去的信念。
<br/>
<br/>        二秀今天会给他送电脑到医院，但是电脑送过来又有什么用呢？
<br/>
<br/>        大家看着二哥有点急，便都听从了二哥的话，各自散去。
<br/>
<br/>        陈子悦对陈子昂说：“子昂，你还要在西城呆几天？”
<br/>
<br/>        “等爸醒过来我就走。”陈子昂要回北京，这边的事情都在按照她的计划有条不紊的进行着。
<br/>
<br/>        耗材上市的日期也越来越近，她得回北京。
<br/>
<br/>        “那子昂你能不能去看看蔷悦，蔷悦很想你。”陈子昂回来都没有去陈子悦的家里。
<br/>
<br/>        他们都是在医院见面，陈子悦有话想给陈子昂说。
<br/>
<br/>        陈子昂看了一样陈子悦：“行，我现在和你一起过去。”
<br/>
<br/>        时间如此匆忙，对陈子昂来说，每一分一秒都是重要的，她在和时间赛跑。
