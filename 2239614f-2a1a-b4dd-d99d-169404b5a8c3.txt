<br/>　　顷刻间，慕清霄睁开星眸，视线朝四周环顾一圈，不知何时，他已经出现在一片森林的上空。
<br/>　　感受着空气中灵气的流动，灵气浓郁程度堪比中千世界，但又有所不如，确实是斗破苍穹位面。
<br/>　　时隔已久，他终于回来了。
<br/>　　慕清霄身处的地方似乎是一片山脉中，站在高空，他能清晰的感觉到，远处有两道气息正在产生碰撞。
<br/>　　气息堪比化神期修士，也就是斗破位面中的斗皇强者。
<br/>　　“是魔兽山脉吗？”
<br/>　　慕清霄神识扩散，将周围笼罩其中，随后嘴角牵起一抹莫名的笑意。
<br/>　　“真是巧，没想到会在这里遇见。”
<br/>　　不多时，伴随着兽吼声，风暴与紫色光束齐聚，两道庞大身影以风驰电掣一般的速度席卷而来。
<br/>　　只见一直足足七八米巨大，浑身覆盖着一层紫色结晶体的魔兽出现在视线中，日光照耀，光华四射，颇为刺眼。
<br/>　　狰狞硕大的狮头，血红中泛着奇异紫光的兽瞳，布满獠牙的巨嘴，最引人注目的就是它头顶的断角。
<br/>　　似乎，曾经被锋利的东西所削断。
<br/>　　这只六阶魔兽，正是慕清霄当初在魔兽山脉中碰到的紫晶翼狮王。
<br/>　　只见紫晶翼狮王怒瞪着对面长约数丈，浑身银色毛发的巨大狼形魔兽。
<br/>　　“苍狼王，你莫以为本皇是真的怕了你吗？”
<br/>　　“哼，紫晶翼狮王，本王不就是借你点紫灵晶用用，何必苦苦相逼？”
<br/>　　整整被紫晶翼狮王追了数千里，苍狼王也有些恼怒。
<br/>　　“苍狼王，若是你不将紫灵晶还给本王，本王定与你不死不休！”

<br/>　　想到这里，紫晶翼狮王就恼怒无比，甚至有将苍狼王抽筋扒皮的冲动。

<br/>　　“你是在说我吗？”
<br/>　　就在紫晶翼狮王与苍狼王对峙的时候，凭空一道清朗的声音在虚空中响起。
<br/>　　“谁！”
<br/>　　紫晶翼狮王与苍狼王皆是一惊，几乎是异口同声的开口喝道。
<br/>　　旋即，空间轻轻波动了一下，只见一名身穿白袍，面如冠玉，风姿卓越的青年踏着虚空出现在两者视线中。

<br/>　　“不错。”
<br/>　　慕清霄微微一笑，道：“没想到这么巧，会在这里遇见你。”

<br/>　　话语刚落，苍狼王就化作一道飙风，直接向慕清霄杀去。
<br/>　　慕清霄瞥了眼苍狼王，星眸一眯，冷哼道：“舌燥！”
