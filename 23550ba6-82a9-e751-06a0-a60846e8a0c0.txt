　　当时《僵尸道长》热播，算是80后一代的回忆，就算没看过，也没有人没听过这部电视剧，叶少阳当年十几岁，是在网吧里看的这部电视剧，虽然里面有些僵尸之类的邪物演的不符合真实，不过叶少阳研究过里头的灵符，除了因为复杂而简化了不少程序之外，这些灵符大部竟然是真的。
<br/>　　当时叶少阳还问过青云子，法术界曾经有没有过毛小方这样一号人物。青云子表示自己也不清楚，不过道门典籍里没有记载过这样一号人物，叶少阳也就当成是虚构的了，没有想到……世上真的有毛小方这号人，而且，居然让自己给遇到了！
<br/>　　自己曾经以为燕赤霞也是假的，后来经广宗天师的口中得知，真的有过这号人，当时自己就足够震惊的了，如果才知道毛小方也真存在，而且被自己当面遇到了……叶少阳此时的心情，激动的不行，一个劲地盯着毛小方看，发现别说，这毛小方虽然跟他的扮演者林正英虽然看着就长的不一样，不过特征上还真有点相似，毛小方嘴上也有一撮小胡子，神态也有几分相似，更是感到惊奇，很怀疑里林正英或者那部剧的导演之类的，是真的知道毛小方这号人物，不然也不会用这名字，至于相貌，估计也是毛小方留有照片或者画像之类，于是林正英就照着毛小方的样子来化妆。
<br/>　　能跟这位电视里的人物处在一个时代、并且坐在一张桌子上吃饭，这种感觉……叶少阳不知道怎么形容。
<br/>　　叶少阳总算是冷静下来，苦笑着，对毛小方说道：“我何止听过道兄的名字，我还看过……”
<br/>　　“看过？”毛小方不解，皱眉道：“叶道兄与我之前见过。”
<br/>　　叶少阳随口敷衍了过去，他没有办法跟他解释真相，除非他想让别人把自己当成是疯子。
<br/>　　“毛道长，你是不是很擅长捉僵尸？”叶少阳忍不住问道。
<br/>　　毛小方有些吃惊，“这件事，叶道长居然也知道。不错，我天道宗修炼的主要便是灭尸之术，传闻，先祖师曾擒拿封印尸王女魃。”
<br/>　　叶少阳一听，浑身一怔，敢情……女魃是被天道宗的人封印的？
<br/>　　叶少阳陡然想起一件关于天道宗的传闻，说道：“你们天道宗是不是有个周真人，当年率领法术界大战无极鬼王的事，你听说过没有？”
<br/>　　毛小方一惊，放下筷子，说道：“道兄你说的是发生在明末的法术界之殇？”
<br/>　　叶少阳耸了耸肩，道：“我不知道，我只是听我师父提过这件事，也只是一笔带过。”
<br/>　　毛小方道：“不知令师是茅山上哪位宗师？是云秋祖师还是伏明天师？”
<br/>　　叶少阳苦笑起来，犹豫了一下，决定还是实话实说，不然云秋生是自己太师祖，伏明子是自己师祖，实在不好冒认，而说别的又不可能。
<br/>　　“我师父，是茅山的青云子。”
<br/>　　毛小方当场怔住，看了他半天，眼神也开始带着一点狐疑，说道：“恕我直言，茅山当代掌门与二代弟子中，我只听说过刚那二位，据说茅山一向是单传，我可没听说过令师的名号……”
<br/>　　叶少阳笑了笑，“没听过是正常的，家师……嗯，有些难言之隐，请原谅我无法明说。”
<br/>　　毛小方听他这么说，知道有难言之隐，心中想到了一种可能，他口中的青云子，莫非……是茅山的弃徒？所以不在名册之中，而叶少阳的“没法明说”，更是加深了他这一方面的猜测。
<br/>　　“了然！”关于别人的出身背景，江湖上的规矩是不能多打听的，毛小方也没有再问下去，接着叶少阳的话题说起那一场被称为“法术界之殇”的战争，虽然时隔几百年，但毕竟是天道宗门派历史上最为光耀的事情，因此有比较详细的记录。
<br/>　　毛小方看叶少阳很喜欢听这个，于是给他讲述了一遍那场战斗的大致过程，天道宗，本来是全真的一个不大不小的流派，几百年内，只出了周真人一个逆天的人物，干过这么一件名扬天下的大事，那一战之后，法术界死伤惨重，元气大伤，周真人羽化，天道宗从此一蹶不振，再也没有了周真人在世时候的荣光。
<br/>　　“无极鬼王，究竟有多强？”叶少阳听的十分动容。
<br/>　　毛小方深吸了一口气，道：“当时那一场大战，虽然是我先祖周真人主持，不过其中真正的强者不是他，也不是茅山和龙虎山的天师，而是从青冥界和修罗界来的几位，其中有两位仙人……”
<br/>　　“仙人？”
<br/>　　毛小方点点头，“一位是道家斩三尸证混元大道的大罗金仙，一位是佛门修成罗汉金身的尊者。”
<br/>　　叶少阳嘴巴张成了一个“O”，道：“是那两位仙人？”
<br/>　　“不知道，据说这两位仙人不愿意透露姓名，只说是帮助人间渡劫，这两位仙人法术极强，根本无法想象，另外还有法术公会的极为灵仙和上仙……而对方，不过是无极鬼王在人间的一抹投影，只有他真实的最多一两成的实力，以寡敌众，居然差点灭了所有人，据说当时法术界为了保护人间，联合起来，开辟了虚空裂缝，将战场引到空界。
<br/>　　无极鬼王本来是可以取胜的，不过他当时似乎发现了东岳大帝当年封禅天下的封禅台，想要开辟封禅台，想要强开封禅台，封禅天下，结果为此耗费了大量的修为，这才被法术界完成反杀……”
<br/>　　叶少阳听得惊心不已，忍不住问道：“怎么样封禅天下，会有什么后果？”
<br/>　　毛小方笑着摇了摇头：“没有人知道，在那一战之后，据说封禅台又继续隐藏起来，没有人知道在什么地方。这一战之后，无极鬼王的一律神念留了下来，无法消灭炼化，被几大宗师封印在蓬莱岛，用人家法术渡化，最后反而成就了一位大才……便是悬空观的无极天师。”
<br/>　　叶少阳点头说道：“这个我也听说了。”
<br/>(本章完)
