        &lt;h3&gt;第三卷第二百七十七章  郝景文在西城&lt;/h3&gt;
<br/>
<br/>        郝景文和寒伯安通过电话之后也陷入了沉思，江俞轩和陈子昂这一对搭档了很多年的老搭档出了什么问题？
<br/>
<br/>        为什么俩个人之前都没有任何征兆？
<br/>
<br/>        傅海生和从哲坐在旁边看着他们年轻的上司不说话，他们也沉默下来了。
<br/>
<br/>        “你们说陈子昂和江俞轩有没有将来？”郝景文突然就问傅海生和从哲。
<br/>
<br/>        俩个人被郝景文问得一愣，尤其是傅海生，嘴巴张了张，没有说出话来。
<br/>
<br/>        郝景文好奇的看着傅海生，傅海生似乎很尴尬？
<br/>
<br/>        “大家都看着俩个人般配，但是到底怎么样只有俩个人心里明白，他们在一起搭档也很多年了，如果真的有将来，还用等到现在？情深缘浅而已。”从哲叹了口气，要不是阴差阳错遇见陈子昂也没有他从哲的今天。
<br/>
<br/>        “怎么说？”郝景文问好奇的问从哲。
<br/>
<br/>        “他们俩个人都是单身吧？俩个人在一起共事了这么多年，要真的是相爱不就早在一起了，可是你看看他们，谁也不肯朝对方前进一步，那不就是直接等着错过？”
<br/>
<br/>        “不是，我觉得是江总的原因，他摇摆不定，不知道自己需要什么。”郝景文突然就很庆幸自己和陈思宇之间的感情，他除了陈思宇，从来都不会对另外的一个女人另眼相看，也不会去和另外的女人暖昧不清。
<br/>
<br/>        陈子昂的情况特殊，陈子昂是不会向江俞轩主动的，所以在俩个人的角逐中，如果江俞轩不主动的话，陈子昂是永远都不会对江俞轩表达她的情感，她永远都会将她和江俞轩的感情放在同事和朋友之间。
<br/>
<br/>        现在最主要的一点是，张倩楠和江俞轩已经结婚了，陈子昂不会和张函接近更不会和江俞轩接近，按照陈子昂的个性，怕是宁愿一辈子单身，也不会再向俩个人靠近。
<br/>
<br/>        他们已经错过了，从哲说的很对。
<br/>
<br/>        傅海生一直不说话，郝景文不知道傅海生和陈子昂的关系，所以他也只是好奇一下就放过了傅海生，虽然郝景文也有八卦心，但是工作还是很重要。
<br/>
<br/>        “傅总，你来说说后期你们的运作？”郝景文和傅海生接触之后，发现傅海生确实是个人才。
<br/>
<br/>        傅海生的领导力在从哲之上，而且思维也比从哲活跃很多，如果可以将傅海生调到西城帮从哲共同承担。
<br/>
<br/>        “我只能对泉城的发展做一个简要的概括。泉城是山城，地方不大，却地广人多，在物流这一块前景还是蛮好的，但是租赁运营可能不是很好，毕竟人流量在哪里放着，也有旅游业的兴起，但不是主力，主力是当地人的消费水平，还有，去泉城旅游的人大多数是西城的人，现在高速公路一通，去泉城能当天来回，或者是隔天回，大部分人是自驾游。”傅海生侃侃而谈。
<br/>
<br/>        “难道泉城就没有市场了？”郝景文问。
<br/>
<br/>        “主力在省会城市，周边可以辐射，我们不是有app吗？可以将app装在别的车上，我们不用投资运营，只收取部分费用，一举两得。”傅海生将自己心中的想法说出来了。
<br/>
<br/>        郝景文紧紧的盯着傅海生，他倒是提了一个好的运营方案，只要有人用他们的系统和app，那么对方就是他们公司的一部分，运营费用进行合理分配就可以了，他们还能占领市场。只是这个利润的分配需要财务核算，才能确定怎么分成。
<br/>
<br/>        “你的提议很好，你能不能拿出一套整体的方案，包括利润的分配问题？”郝景文抓住傅海生不放。
<br/>
<br/>        “那我回去做一个方案吧。”傅海生微微一笑，他就是替周宏生来开个会，就被从哲和郝景文给抓住了。
<br/>
<br/>        “傅总真是人才，能不能到西城或者是北京上班？”郝景文觉得像傅海生这样的人才就不应该埋没在泉城那个小地方。
<br/>
<br/>        “海生才疏学浅怕是不能担当大任，况且我还有年迈的父母和幼小的孩子要照顾，还是窝在小县城的好。”泉城是傅海生的家乡啊，从小生长的地方，哪里山清水秀，安宁惬意。
<br/>
<br/>        傅海生是不会到西城的，夏颜如今在西城红红火火，一手遮天，他好歹背着夏颜丈夫之名，这让他无颜立足。
<br/>
<br/>        北京，是有些向往，只是北京也不会是他的福地，如果去了，怕也是自寻难堪。他傅海生之前的雄心壮志早就丢弃在了生活的犄角旮旯里。
<br/>
<br/>        他无力去反驳生活给予他的痛苦折磨，他只能视而不见听而不闻，装作一切都是很好的样子，可是只有当夜深人静的时候，痛苦就撕扯着他的心，他常常被噩梦惊醒，一边是夏颜疯狂的大笑，一边是父亲痛苦流涕，还有陈子昂冷冰冰决绝的脸庞。他不知道为什么他一个上帝的宠儿将生活过成了这个样子，一手好牌被他打得稀巴烂。
<br/>
<br/>        “哈哈哈哈，那就随你，我们公司的大门永远都向你敞开着，只要你愿意，我们随时欢迎你，对了，你的专业是电子信息化，陈子昂公司也需要你这样的人才，我可以推荐你到她的公司，她那个人啊，求才若渴，如果你去了她的公司就有用武之地了，还有，他们公司研发的东西那才叫高大上。啧啧啧。”郝景文觉得傅海生不应该被埋没了，西城他可以不呆，但是北京可以啊，在陈子昂的公司上班，不就有用武之地了？
<br/>
<br/>        “我还是将你交给我的事情干完再说，周宏生最近喝酒出车祸了，在床上躺着呢，他公司的一切现在都是我在打理着，我一时半会儿也离不开。”傅海生没有想到郝景文直接就提到了陈子昂，这让他有点不知所措。
<br/>
<br/>        “行，你先做好手头上的事情！从哲，那西城的运营你就要开始加快步伐，车辆已经投放完毕，人员也完全到位，我希望在这个季度我们西城的效益翻一翻。”郝景文对傅海生和从哲说。
<br/>
<br/>        “按照昨天运营的数据显示，我们不仅仅是只会翻一番，我们带动了西城的面貌，尤其是出租行业的革新，昨天仅仅一天，我们占领了十分之一的市场，假以时日，我们将会是这个行业的龙头老大。”从哲满怀信心。
<br/>
<br/>        “我会将我们的app加以推广，吸引一部分的出租车或者是私家车加入我们的行业，这个方案我会用两天的时间给你。”傅海生也笑着说。
<br/>
<br/>        “好，看来我们西城的事业要上一个新高度，哈哈哈，加油，老铁们！”郝景文觉得很开心，每一次开会他都会有一个新的锲机，有一个新的想法萌生。
<br/>
<br/>        俗话说三个臭皮匠顶个诸葛亮，真的是不错。
