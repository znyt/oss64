　　现在，五颗灵珠缺且二，水灵珠与风灵珠。
<br/>　　仅仅见识过地仙的威压，慕清霄更不敢托大，以他现在的能力想去神界夺取风灵珠与神树，无异于虎口拔牙。
<br/>　　至于景天手中的魔剑，不急一时。
<br/>　　水灵珠就在灵溪谷的紫萱手中，近在眼前，慕清霄没有放过的理由。
<br/>　　不过，紫萱的实力肯定不弱，还有一名不知深浅的圣姑，慕清霄依旧需要谨慎，他可不想阴沟里翻船。
<br/>　　待湖水平息，慕清霄将剑意收敛后，便转身离去天璇宫，化作一道剑光，眨眼间就出现在了天枢宫外。
<br/>　　“请进。”
<br/>　　和煦的声音在耳边响起，慕清霄微微一笑，缓步进入天枢宫中。
<br/>　　进入天枢宫，清微依旧盘坐于蒲团上，脸上一直都是挂着温和的笑容。
<br/>　　“已经想好了？”
<br/>　　慕清霄沉吟片刻，道：“不错，蜀山虽是福地，但瓶颈并不是灵气充沛就能冲破的，我打算下山出去走走。”
<br/>　　“嗯，也好，打算何时回来？”
<br/>　　“短则两三年，长则五六年吧。”
<br/>　　其实，关于这个问题，慕清霄早就已经计划过了。
<br/>　　他打算先去灵溪谷找到水灵珠，然后就直接离开仙剑三位面，在这里虽能成仙，但需要时间，他需要快的方式。
<br/>　　他可不想在其中修炼个十年百年，修成地仙后在去神界取灵珠和神树，简直是浪费生命。
<br/>　　最重要的是，他急需将体内的红莲业火处理掉，一直将能够威胁自己生命的东西留在体内，任谁都会觉得不愉快。
<br/>　　反正这里的任务没有完成，他终究是会回来的。
<br/>　　“草谷那丫头平时古灵精怪，好在心地善良，也是我唯一的弟子，我不在的时候，希望掌门帮忙照顾一二。”
<br/>　　“在蜀山山门中，无人能够欺负她，此事你无需担心。”
<br/>　　闻言，慕清霄微微点头，清微老头子也算是仗义，自己也算是蜀山的一员，下次回来的时候，或许可以顺手将邪剑仙灭了。
<br/>　　毕竟，这是他的任务，也算是帮蜀山清除一害。
<br/>　　商量过后，慕清霄就转身离开，这时，清微却徐徐开口道：“等等，老夫有一问题需要你解惑……”
<br/>　　“掌门请问。”
<br/>　　慕清霄脸色有些古怪，但心里却隐隐猜到了清微想问些什么。
<br/>　　只见清微双眼盯着慕清霄，笑眯眯的说道：“你小子曾经说自己去过古藤林，其中的土灵珠是你拿走的吧？”
<br/>　　“……不错，是我拿走的。”
<br/>　　慕清霄并没有否认，因为已经没有否认的必要，后者已经确定是自己拿走的灵珠。
<br/>　　清微并不觉的意外，但眼中却有些困惑，道：“你小子要灵珠何用，你可知灵珠对景天他们的重要性？”
<br/>　　“灵珠可封印锁妖塔。”
<br/>　　话语刚落，清微微微有些吃惊。
<br/>　　“看来，你小子知道的挺多，老夫很好奇，凭你的实力，根本不可能拿走鬼界中的火灵珠，你是如何进入鬼界的？”
<br/>　　“鬼界虽隐秘，但也并非完全进不去，景天他们都能进去，我为何进不去？”
<br/>　　慕清霄的一番话语，令清微无言以对。
<br/>　　但情况完全不同，景天和徐长卿他们能进入鬼界，是因为他的帮助。
<br/>　　否则，想在鬼界中横行根本不可能，但慕清霄靠的是什么？
<br/>　　“罢了，其他的事情老夫也不想知道，灵珠能封印锁妖塔，不容有失，除非你另有办法，否则老夫可不会任你乱来。”
<br/>　　“掌门，难道你不知道桥到船头自然直的道理？”
<br/>　　说罢，慕清霄转身离去。
<br/>　　“灵珠对我来说至关重要，如果景天他们无法解决邪剑仙，到时候我会出手，至于锁妖塔中的东西，区区一群魑魅魍魉，既然你们能关进去，难道还能上天不成？”
<br/>　　清微抬起头来，慕清霄身影已经消失，无奈之下，心中复杂的情绪化作一身叹息。
<br/>　　“希望，你别让老夫失望才好。”
<br/>　　离开天枢宫后，慕清霄径直前往了一趟医药堂，只是站在远处看了一眼草谷，见小丫头正在学习，并没有出声打扰她。
<br/>　　离开医药堂，慕清霄并没有去后山的药园。
<br/>　　他的储存空间中，高阶灵药不知凡几，等下次回来的时候，在去药园也一样。
<br/>　　回头看了一眼天璇宫，慕清霄不再犹豫，转身化作一道剑光，直接向东边方向疾射而去，目标正是灵溪谷。
<br/>　　清微并没有告诉他明确的位置，只是告诉他，灵溪谷在向东两百里的地方。
<br/>　　范围有些广，不过知道方位后，慕清霄找起来也容易许多，两百里的距离，剑光数息后就已经抵达。
<br/>　　站在上空，脚下是一片郁郁葱葱的森林，没有城镇，视线四周倒是有些山谷。
<br/>　　清微老头子知道他在找灵珠，没必要在这种事情上忽悠他，所以灵溪谷肯定就在附近数十里内的地方，否则方向就错了。
<br/>　　慕清霄视线环顾四周，目光最后落在了四面八方的山谷中。
<br/>　　紫萱与圣姑总不可能生活在地下，山清水秀的山谷倒是一处清秀的好地方。
<br/>　　不知何时，慕清霄已经出现在一座山谷上空，此处有山有河，鸟语花香，但其中并没有人居住。
<br/>　　他已经是大乘九重巅峰修士，灵魂力量根本不输给渡劫期修士，如果有阵法或者结界，根本就瞒不过他的双眼。
<br/>　　随意扫了一眼，慕清霄身影转眼再次消失。
<br/>　　方圆几十里内，总有就只有两座山谷，其余都是一些小山坡。
<br/>　　来到另外一座山谷上空，慕清霄微微一凝，此处果然被布下了空间结界，结界很强，估计是渡劫期修士布下的。
<br/>　　“看来，这里就是灵溪谷了。”
<br/>　　与此同时，灵溪谷内一座竹楼中，两名女子坐在桌前。
<br/>　　陡然，其中一名身穿长袍的女子目光一凝，道：“紫萱，有客人来了。”
<br/>　　“圣姑，来者是何人？”
<br/>　　“从未见过，但此人修为不弱，既然能找到灵溪谷，想必与蜀山的清微掌门有关。”
<br/>　　说着，圣姑没好气的看了紫萱一眼，道：“此人不是徐长卿，我劝你死心吧，你们是不会有结果的，别忘了你的职责。”
