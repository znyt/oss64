        &lt;h3&gt;第三卷第二十章  心思各异&lt;/h3&gt;
<br/>
<br/>        江俞轩拉开傅海生，一个大男人哭成这样，不丢人么？
<br/>
<br/>        傅海生确实醉了，他嘴里还在喊着“子昂”。
<br/>
<br/>        从傅海生的话里，陈子昂知道了很多信息，傅海生一直都在给她写信，只是这些信都如沉大海，他嘴里的“他”是张函吗？
<br/>
<br/>        不过她看到的信也就十来封。
<br/>
<br/>        陈子昂看着江俞轩和陈子霖扶着傅海生远去，心里百感交集，五味杂陈。
<br/>
<br/>        她和傅海生注定有缘无分吧？傅妈妈是不是也会嫌弃她是农村的孩子呢？不过傅妈妈人很好的样子，他们都好吗？
<br/>
<br/>        月光洒在地上，托着陈子昂纤细修长的声影，有些寂寥和孤单，陈子昂抬起脚踢着自己的影子：“陈子昂，这是你嘛？”
<br/>
<br/>        影子跟着她起起伏伏，蹦蹦跳跳，摇曳了一地的月光，破碎了一团团光影，夜微凉，人影单，那一些青葱的岁月不复返。
<br/>
<br/>        江俞轩和陈子霖出来的时候就看到了蹦蹦跳跳的陈子昂，她踏着月光，就像一只精灵，在月光下舞蹈，扎起的马尾飞舞在月色下，那一身洁白的裙子随着她翩翩起舞。
<br/>
<br/>        她轻轻的哼着“因为爱情”，悠扬的歌声透过月光，透过层层空气，静静的落在他们的心田。
<br/>
<br/>        “子昂。”陈子霖看着时间不早了，便打断了陈子昂。
<br/>
<br/>        “哥，俞轩。”
<br/>
<br/>        “来，我们回去了。”江俞轩拉起陈子昂的手，替她擦去脸上的泪痕。
<br/>
<br/>        陈子昂心里微暖，无论如何，她身边不是还站着江俞轩和她七哥吗？还有身后的那么多同事。
<br/>
<br/>        “以后，还是别喝酒了。”陈子霖。
<br/>
<br/>        “这不是开心吗，见了四姐还有老同学。”
<br/>
<br/>        “你就是一只小蜜蜂，我到处给你掐花。”江俞轩幽怨的说，他不但是她工作上的助手，他还得帮她灭火，灭那么多被她点燃的星星之火。
<br/>
<br/>        看着傅海生抱着陈子昂大哭，他就恨不得将傅海生踹上几脚，和陈子昂一起工作了这么多年，他都没有拥抱过陈子昂，他傅海生凭什么，就凭他喜欢过她？
<br/>
<br/>        没见过这么怂的人，都一起相处了好多年，现在才抱着哭，早些时候干嘛去了？所以他和陈子霖将傅海生送回家，往沙发上一扔扭头就走。
<br/>
<br/>        “就这样走了？”陈子霖看着倒在沙发上的傅海生。
<br/>
<br/>        “不走？你陪他睡？”江俞轩冰冷的盯了一眼陈子霖。
<br/>
<br/>        陈子霖有点不理解江俞轩明明是翩翩玉公子，说翻脸就翻脸。他不知道的是江俞轩现在看到陈子昂身边的烂桃花他就烦，更烦的是，张函还要他当卧底。
<br/>
<br/>        张函也是没胆的人，明明爱的死去活来，还不敢说，明明都是孩子他爸了，还只能偷偷摸摸。大不了什么都不要，和陈子昂过一个小日子不是也很幸福吗？搞不懂这些男人为什么就抽了筋，他也抽筋了，陪着陈子昂一起疯。
<br/>
<br/>        他倒是很欣赏陆宇霆，和陈子昂一心一意的谈了两年恋爱，说放手就放手，不拖泥带水，不给陈子昂希望也不给陈子昂伤害。
<br/>
<br/>        如今，陆宇霆过得也很幸福，两口子算是恩恩爱爱，平步青云。
<br/>
<br/>        看看傅海生，过了多少年了，才抱着哭？
<br/>
<br/>        看看张函，结婚这么多了，硬是搞得一个闺门怨男，我呸，还天天让给他给他汇报陈子昂的行踪。
<br/>
<br/>        “俞轩，想啥呢？”透过月光，陈子昂看着江俞轩一脸的忿忿不平。
<br/>
<br/>        “陈子昂，你赶紧找个人嫁了吧，省得我天天忧心。我特么的又当爹又当妈。”江俞轩心里有些烦躁。
<br/>
<br/>        “不嫁，就要你又当爹又当妈。”陈子昂咯咯的笑出了声，头一歪靠在他肩膀睡着了。
<br/>
<br/>        “陈子霖，你们家咋养了这么一个没心没肺的人出来了？”一会满脸泪痕，一会晴空万里，转眼到头就睡，全然无视那些个为她哭天喊地的人。
<br/>
<br/>        “我们陈家的人都是没心没肺的，没有那么多弯弯绕绕的坏心眼，你要是不习惯你可以离开。”陈子霖对江俞轩的态度不太好，一个助手还敢嫌弃他的妹妹。
<br/>
<br/>        “你那个四姐弯弯绕绕的不是很多吗？”
<br/>
<br/>        “你？”
<br/>
<br/>        “我看啊，你们陈家就陈子昂一个人是个木头，你们这些个当哥的当姐的肯定从来都没有管过她。”
<br/>
<br/>        陈子霖沉默不语，陈子昂是他们家最小，其实按说最小的是家里最疼爱的，而他们家恰恰相反，临到最小的资源是最缺乏的。
<br/>
<br/>        “没话说了吧，我就知道你们就是这样的，要不然陈子昂也不会这么辛苦。哼！”
<br/>
<br/>        “我们家孩子多，我们确实没有照顾好子昂。”
<br/>
<br/>        “说什么呢？”陈子昂抬起头。
<br/>
<br/>        “说明天的事情。”江俞轩
<br/>
<br/>        “俞轩，我突然就不想做这个项目了。”
<br/>
<br/>        “因为傅海生？”
<br/>
<br/>        陈子昂不语，迷瞪了会，感觉舒坦多了。
<br/>
<br/>        “傅海生虽然对你有情，但是我觉得在工作上他还是不错的，要不然不会做到现在的这个位置上。”
<br/>
<br/>        “不是，我怕水太深。”
<br/>
<br/>        “子昂，你要是觉得不愿意做了我们明天就回。”陈子霖对四姐现在的状态又担忧又不满。
<br/>
<br/>        他们从小就被教育着要堂堂正正做人，爸爸妈妈虽然大字不识一个，但是俩个老人家却是相濡以沫，陈家家风极严，记得小时候，他偷偷的摘了邻居家的一个苹果，爸爸将他好一顿打，把点豆子的豆铲都打断了，他在床上爬了好几天，等他好一点了，周末将大家都叫回来，陪着他一起罚站。
<br/>
<br/>        “人穷志不短，你们要给我记住，不吃嗟来之食，不偷不抢，要靠自己的本事生活，你有多大的本事你就吃多少饭，是你的你拿，不是你的坚决不能拿，做人要头顶天脚踏地，堂堂正正。要知恩图报，要亲善待人。如果以后再发生这样的事情，你们弟兄姐妹一起挨打。”
<br/>
<br/>        父亲将打断的豆铲扔在他们的面前。
<br/>
<br/>        陈家兄妹怕是只有四姐一个人，腰板是挺不直了。
<br/>
<br/>        “明天看看情况吧，先了解一下再说。对了哥，我还想回去看看爸妈，这都到洛城了，回去也不远了。”
<br/>
<br/>        “好，那咱俩回去看爸妈，让江俞轩明天联系傅海生。”陈子霖不想和江俞轩在一起。
<br/>
<br/>        他也不想看傅海生见着妹妹那深情款款的让人起鸡皮疙瘩的眼神。
<br/>
<br/>        都什么事情嘛！
<br/>
<br/>        “为什么我去？我也要回去看爸妈？”
<br/>
<br/>        “得了，你的爸妈还不知道在哪里呢？”想轻轻松松的认亲？没门！不就一个助理嘛！陈子霖恶狠狠的对江俞轩说。
<br/>
<br/>        “俞轩，你明天就在这里看看情况，我和我哥回去老家一趟，下午就可以回来了。”
<br/>
<br/>        “好吧，子昂，我也很想看你出生的地方呢。”
<br/>
<br/>        “改时间，有机会了我专门邀请你过去。”
<br/>
<br/>        到了酒店，江俞轩想扶着陈子昂，被陈子霖挤到了旁边。
