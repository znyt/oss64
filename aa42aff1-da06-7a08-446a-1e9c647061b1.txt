        瓶口还未离开她的嘴边，何明旭就将她手里的这瓶矿泉水给夺了过去，并大口大口地吞咽了起来。
<br/>
<br/>        童筱颖不得不拿起另一瓶矿泉水，扭开瓶盖来假装喝水，其实她是把嘴里含着的那一小口水给吐掉。
<br/>
<br/>        何明旭的酸辣粉只吃到了一半，突然就端着碗筷不动了。
<br/>
<br/>        童筱颖这才放下自己手里酸辣粉，抬起手来，在何明旭的眼前摇了摇。
<br/>
<br/>        她就知道他对她起了疑心，还好她刚刚拿水的时候，故意拿了被她下了药的那瓶。
<br/>
<br/>        “何明旭，把桌子收拾一下！”童筱颖命令道。
<br/>
<br/>        何明旭果真行动了起来，把桌子上的外卖，全部塞入了垃圾桶中。
<br/>
<br/>        童筱颖接着掏出手机，对着何明旭录像，她一边录，一边问道：“马智杰是不是你买凶杀的？”
<br/>
<br/>        何明旭就这么呆呆地立在童筱颖的面前，双眼发直，一句话也不说。
<br/>
<br/>        童筱颖见他不说话，犹豫了一下，又从自己的包包里拿出纸和笔，递给何明旭，换了一种问话方式：“何明旭，你听好了，我现在命令你，把杀害马智杰的凶手名字写出来！”
<br/>
<br/>        何明旭接过纸和笔后，直接趴在桌子上写了起来。
<br/>
<br/>        他一边写，她就在一旁拿着手机一边给他录像。
<br/>
<br/>        她要将他的犯罪证据录下来！
<br/>
<br/>        当何明旭写完第一个字“林”的最后一笔画，童筱颖的脑海里立马闪过“林若琴”的名字。
<br/>
<br/>        果不其然，何明旭将“林若琴”这三个字，写在了白纸上。
<br/>
<br/>        那么，问题就来了……
<br/>
<br/>        林若琴为什么要杀马智杰？
<br/>
<br/>        “我命令你，把帮凶的名字也一起写出来！”童筱颖又命令道。
<br/>
<br/>        何明旭又开始写，他写了一大堆的名字，其中有他自己的名字，也有她童筱颖以前的曾用名“诺筱颖”。
<br/>
<br/>        她什么时候成了杀害马智杰的帮凶了？！
<br/>
<br/>        童筱颖有点儿恼火，却无意间看到手机上的时间，发现时间上已经差不多了。连忙合上手机，拿走了纸条，去卫生间里换了衣服。
<br/>
<br/>        她从卫生间里出来后，打开了自己手机里的变音功能，然后又从何明旭的裤兜里掏出手机，接着从包包里翻出那张小黄卡，给上面的中介人打了通电话。
<br/>
<br/>        童筱颖报了宾馆的地址和名字以及房门号后，说了一句：“房门是虚掩的，直接进来就是。”
<br/>
<br/>        挂了电话后，童筱颖离开前，给何明旭下了最后一道命令：“我出去后，下一个进来的女人，你可以随便上！”
<br/>
<br/>        童筱颖说完，将房门虚掩着，抱着包包躲在了楼道里，直到她看到有个打扮得花枝招展的女人，进了何明旭的房间，她便立即打电话报警，顺便还通知了当地的媒体热线。
<br/>
<br/>        这是他何明旭自愿跟她打赌的，赌输了，不能愿她。
<br/>
<br/>        离开宾馆后，童筱颖并未走太远，她就在这附近找了个小吃摊坐下，等候警察上门抓人。
<br/>
<br/>        结果，不出五分钟，就有几个素衣男子冲进了那家宾馆内。
<br/>
<br/>        随后的没过多久，媒体记者也跟着过来了。
<br/>
<br/>        童筱颖原本以为，警察只会抓何明旭和那名“小姐”，却没想到，这家宾馆里竟然“卧虎藏龙”。
<br/>
<br/>        警察居然抓了一窝人出来。
<br/>
<br/>        但是令童筱颖感到纳闷的是，她并未在那群被抓的男人里面看到何明旭的身影。
<br/>
<br/>        这到底是怎么回事？
<br/>
<br/>        因为随后赶到的警车，鸣笛声吸引了周遭群众的围观。
<br/>
<br/>        童筱颖也站在了这围观的群众里，她的双目扫过一个个被抓的男人的脸，她还真就没有看到何明旭的人影。
<br/>
<br/>        怎么会这样？
<br/>
<br/>        是她哪里出错了吗？
<br/>
<br/>        童筱颖不禁握紧了双拳。
<br/>
<br/>        如果，抓不到何明旭，她这一来二去的计划，岂不是全部泡汤了。
<br/>
<br/>        直到警车开走，围观的群众也跟着散去，童筱颖仍旧想不出个所以然来。
<br/>
<br/>        要不要再回去看看？
<br/>
<br/>        算了！
<br/>
<br/>        万一回去，反被何明旭设局，她岂不是自投罗网。
<br/>
<br/>        只是，童筱颖心里有点儿不甘心。
<br/>
<br/>        就在她黯然神伤地转过身去，无意间地抬眸，看到不远处出现在自己面前的男人，刹那间惊愕了。
<br/>
<br/>        傅夜沉身姿英挺，穿着黑色的西装恍若与这夜色融为了一体。
<br/>
<br/>        他踱步向她走了过来，冷冽的眸子里有一丝血丝，整个人显得有些疲惫，但是却无损他高贵的气质，如精雕细琢般的精致五官这个时候显得更为立体，额头上滚落着豆大的汗珠。

<br/>        “大、大哥？！”童筱颖木讷地开口。
<br/>
<br/>        “我的话，你为什么不听？”傅夜沉疾言厉色地问。
