　　第513章
<br/>　　这白衣尼当是程大雷所遇到的最难缠的对手。一杆拂尘打开，如一柄密不透风的伞，任程大雷的攻击如狂风暴雨，但她却能做到风雨不透。
<br/>　　防守并不见如何吃力，却能做到没有破绽，随手一招，看上去毫不费力，却能将程大雷的攻击挡开。相反，她漫不经心攻出的一招，却可以给程大雷带来巨大的压力。
<br/>　　不过，相比聂隐娘，她却是圆润有余，杀机不足。
<br/>　　与程大雷过招的白衣尼同样不轻松，甚至忍不住去想，单枪匹马来对付程大雷是不是一计错着。到她这种境界，早已修炼到以劲化气的先天地步。只要一口真元不泄，便有源源不断的体力。如程大雷和聂隐娘，连这种境界的门槛都摸不到。
<br/>　　可白衣尼毕竟老了，老人在技术和经验方面，自然远强于年轻人，可同样，老人也没有年轻人锐意进取的精气神。
<br/>　　程大雷一柄剑直来直往，如一柄重锤不停敲击在白衣尼的防御罩上，只要能敲开一丝缝隙，程大雷便能再进一步，斩掉白衣老尼的脑袋。
<br/>　　“大当家，退了。”
<br/>　　这时候，秦蛮带来蛮字军赶到，布好了包围圈，上千人手里都握着强弓劲弩，只要程大雷与白衣尼拉开距离，到时候便万箭齐发，将对方打成筛子。
<br/>　　程大雷也未必要与白衣尼决个胜负，千金之子坐不垂堂，自己什么身份，怎么可以和一个无明白的老尼拼命。自己只要把她留在此地，自然有人可要她性命。
<br/>　　听到秦蛮的高语，程大雷立刻攻出一剑，逼的白衣尼姑后退防御，然后身子跃起，施展出轻身功夫，想要退回本方大本营。
<br/>　　“想走，留下吧。”
<br/>　　万没想到，白衣尼姑在力穷之境地，又凭空生出一股力气。在真气灌输下，拂尘笔直如枪，如枪又如剑，打向程大雷胸口。
<br/>　　虽然是软拂尘，但若是被打中，程大雷非被打出个血窟窿不可。程大雷激灵出了一身冷汗，在这一瞬间，他忍不住去想，面对这神秘的白衣尼姑，自己是否太托大了。
<br/>　　也是绝处逢生，在没有办法的情况下，程大雷使出千斤坠的功夫，身子硬生生落在地上，险之又险的躲过这招铁拂尘。然后随意挥出一剑，身子在地上打了个滚，立刻向外退去。
<br/>　　程大雷也只是随意挥出一剑，根本无奢望能伤到白衣尼，白衣尼确实也没想到，但这无意的一剑却割破白衣尼半幅白袍，划中了她的小腿。
<br/>　　以无心算有心，往往有出人预料的收获。
<br/>　　程大雷就地一滚，以极其狼狈的姿势逃回了己方大本营。白衣尼姑自然不能容他这样走脱，迈步要追杀过来，迎面撞上的却是一个个拉开硬弓的蛤蟆城战士。
<br/>　　人人表情肃穆，目光如铁，一股乱军丛中厮杀出来的血勇之气不胫而走。
<br/>　　“放！”
<br/>　　前队同时放箭，箭雨袭向白衣尼姑，在第一轮箭雨过后，前排退后，后排向前，衔接无比默契，两轮攻击没有任何时间差。
<br/>　　这顿时给白衣尼姑带来巨大的压力，手中的拂尘打开，将一枚枚羽箭挥落，却不能有一刻松懈。
<br/>　　程大雷此刻已经退回到己方阵地，加紧时间恢复体力，目光得空看着被围攻的白衣尼。
<br/>　　“能拿得下么？”程大雷问身边的秦蛮。
<br/>　　“大当家放心，她走不了。”秦蛮冷静道：“要生擒么？”
<br/>　　“生擒什么，直接杀掉。”
<br/>　　秦蛮点点头，示意自己明白。他举起手，拳头攥紧又松开，然后重重挥下。
<br/>　　攻击出现了暂停，这当然不代表秦蛮准备放过白衣尼，他们只是在换武器。
<br/>　　箭法什么的，本就不是蛤蟆城擅长的，也并不是没有专门训练过，不过他们有更强的武器。
<br/>　　在整个天下而言，蛤蟆城很显然是唯一将火器发挥到极致的势力。
<br/>　　放弃弓箭后，转而用火雷攻击，点燃后迅速投出，袭向中央的白衣尼。只听得耳边轰隆隆炸响，尘土荡开，黄土被整个掀开一层。
<br/>　　百里胜和钟伟虎脸色顿时变得苍白如纸，他们在草原上都见识过程大雷使用火器的景象，就是那一次，将帝国军杀破了胆。如今这一幕重现，二人都想起了不好的回忆。
<br/>　　程大雷紧盯着白衣尼的动作，她不停躲避，现在整个人狼狈无比，僧袍变得破烂不堪，脸上蒙了一层黑灰，但不断爆炸的火雷，还没能要了她的性命。
<br/>　　她在不断躲避的过程中，手中拂尘一甩，将几枚火雷反打下包围圈中。
<br/>　　秦蛮手下人的反应能力不慢，立刻躲避，但终究又几人躲避不及，被爆炸殃及。也就是趁着这个空隙，白衣尼飞身向外，有阻拦的人，被她轻而易举的击杀。
<br/>　　秦蛮脸色铁青得难看，没想到在这样的境地，还是被白衣尼突围成功。
<br/>　　正这时候，他感觉身边一道人影窜了出去，直奔那白衣尼姑逃走的方向。
<br/>　　这正是程大雷，他已看得仔细，这白衣尼显然已经负伤，如今是她最虚弱的时候。今天若能杀了她，当是最好，今天若杀不了，怕会引来更大的祸患。
<br/>　　白衣尼很是狼狈，脚尖点地，一跃数丈，向远处逃去。急急如丧家之犬，现在已没有任何高人风范。
<br/>　　程大雷当然不会放过他，提着剑缀在其身后，想这白衣尼已经负伤，当是跑不太远。如今他杀心已起，今天非宰了这老女人，完成任务不可。
<br/>　　蛤蟆城外，百里胜和钟伟虎还有些摸不着头脑，就见眨眼之间，程大雷和那白衣尼姑就一溜烟跑得不见了踪影。
<br/>　　“程当家这是……”钟伟虎睁大眼睛道。
<br/>　　“喔，没事，只不过有人要杀大当家而已。石高于岸，浪必催之，像我们大当家这么浪的，经常会遇到这种事。”
<br/>　　“那个……程贤弟不会有什么危险吧？”百里胜问，心道，若是这白衣尼姑能杀了程大雷也是好事一件。
<br/>　　“不会，不会，往往是找他麻烦的人比较危险。”
<br/>(本章完)
