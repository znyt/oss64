　　“死了没有？”

<br/>　　这可是一尊超越生死的大人物出手，含怒之下，斩杀楚风眠，楚风眠居然是足以在这样的攻击下，没死？
<br/>　　别说是狼狈不堪，就算是只剩下一口气在，都是极为不可思议的一件事了。
<br/>　　而且现在楚风眠的样子，虽然看起来狼狈，但是实力仍在，没有受到太重的伤，这简直是不可思议的一件事。
<br/>　　“此人，真的还是人？”
<br/>　　“这种刀锋，让任何妖兽来，都要被撕裂吧。”
<br/>　　若不是亲眼所见，谁又能相信。
<br/>　　一个神力境武者，居然足以从一尊超越生死的大人物手中，抗下如此可怕的刀锋。
<br/>　　别说是让一尊神力境武者前来，让任何一位生死境的天才，都做不到这一点来。
<br/>　　这已经不是人了。

<br/>　　一尊武者，不由的喃喃自语起来。
<br/>　　还未度过生死大劫，楚风眠便是就拥有如此可怕的实力，要是将来度过生死大劫，岂不是足以斩杀超越生死的大人物？
<br/>　　这个想法，足以称得上是疯狂。
<br/>　　但是如今众人看向楚风眠，却都是心中相信这一点。
<br/>　　“可恶！”
<br/>　　在这虚空之中，那尊出手的大人物，看到楚风眠，居然是没有死在这刀锋手中，似乎是更加震怒了。
<br/>　　以他的身份，对这样一个小辈出手，居然都没有斩杀。
<br/>　　这简直就是一种奇耻大辱。

<br/>　　又是一股雄厚的力量波动，就在这虚空之中，又是一道刀锋，打破虚空，再度向着楚风眠斩杀而来。
<br/>　　似乎是今日不将楚风眠斩杀，誓不罢休。

<br/>　　就算是这大人物的身份在尊贵，通天阁也不可能忍受这样一而再再而三的挑衅。

<br/>　　这大人物的声音，霸道无比，就算是硬闯武斗大会，打破规矩，也丝毫不惧，一声声中，尽是威胁。
<br/>　　“夜牧老祖，这里是武斗大会，你从这里出手杀人，未免不将我们通天阁放在眼中了吧。”
