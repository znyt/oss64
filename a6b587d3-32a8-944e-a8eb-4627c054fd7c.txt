　　圣品灵丹的材料，虽然都极为难以寻找，但是若是有着无穷无尽的财富的话，倒是也有可能全部收集到。
<br/>　　楚风眠心中暗自想到。
<br/>　　有钱能使鬼推磨，只要有着足够的财富，楚风眠也有自信，可以将这圣品灵丹所需要的材料，全部收集齐。
<br/>　　当然在北域那样的地方，就算是有着财富，也做不到这一切。
<br/>　　不过这里是中域，收集一套炼制圣品灵丹的材料，应该不是没有可能。
<br/>　　楚风眠一边想着，已经是在心中制定这计划。
<br/>　　炼制圣品灵丹的材料，自然不是在寻常地方可以寻找得到，至少要前去一些极为繁华的商业城池才行，在一些大商会的手中，很有可能有着保留。
<br/>　　楚风眠一边思考着计划，一边在中域飞行，他环视四周，便是发现，在前方突然有着无数魔气，冒了出来。
<br/>　　在楚风眠的前方，是一座巨大的山谷，这一股股的魔气，正是从这山谷之中冒出来的，将这正片山谷，都渲染成一幅像是魔谷。
<br/>　　“这魔谷里面，居然有着一棵圣药？”
<br/>　　楚风眠在这无数魔气之中，突然分辨出了一种独特的清香，这种清香，只是属于圣药的清香。
<br/>　　圣药，便是炼制圣品灵丹的必备之物，一枚圣品灵丹，至少需要数十种圣药，才可以炼制。
<br/>　　没想到他刚刚想要计划炼制圣品灵丹，这圣药便是送上门来了。
<br/>　　“走！”
<br/>　　楚风眠没有任何犹豫，便是身躯一动，向着这魔谷的方向，飞了出去，在靠近魔谷附近的时候，楚风眠的身形突然是从空中，停了下来。
<br/>　　“在里面，有人战斗？”
<br/>　　楚风眠刚刚靠近这魔谷附近，便是感觉到了，从这魔谷之中传来的一阵阵灵力波动。
<br/>　　这显然是有这人，在里面战斗，而且这灵力波动，传递的如此之远，其中战斗的武者，实力绝对不弱。
<br/>　　“看看去。”
<br/>　　楚风眠沉吟了一阵，便是身形一动，直接向着魔谷的方向飞了过去，以他的实力，倒是不怕碰到什么危险，而且对着魔谷之中的圣药，楚风眠是势在必得。
<br/>　　“这是，阵法？”
<br/>　　楚风眠靠近这魔谷的一刻，周围的魔气便是察觉到了楚风眠的存在一样，全部向着楚风眠袭击过来。
<br/>　　这些魔气，层层叠叠，如同一条条锁链一般，全部向着楚风眠袭来，似乎是要将楚风眠给封锁起来。
<br/>　　楚风眠这才注意到，这一片魔谷，实际上都是因为这其中一道巨大无比的阵法笼罩起来的，才硬生生将这一片山谷，化为了一片魔谷。
<br/>　　“看来这又是那个小魔头做的了。”
<br/>　　楚风眠看到这一幕，轻笑开口道，面对这无数魔气的袭来，他手指轻轻一点，在楚风眠面前阻拦的那些魔气，全部破碎，楚风眠的身躯一闪，直接冲了进去。
<br/>　　进入这魔谷的瞬间，两道身影，便是出现在了楚风眠的眼中。
<br/>　　这两道身影，一男一女，其中那男子，一身黑衣，身上还沾满了无数血迹，一身魔意，看起来像是这魔谷的主人，境界在生死境九重左右。
<br/>　　而那女子，却是看起来年纪不大，只有二十岁出头的样子，不过境界却也达到了生死境八重的地步，算得上是一位天才了。
<br/>　　不过，这女子的实力，显然要稍逊一筹，而且这男子还是在主场之中，几乎已经是一边倒的局面。
<br/>　　男子连续进攻，而这女子，却只能够是一直逃避防守，可是却也无法从中冲出去。
<br/>　　如此一来，可以说不用多久的时间，这女子便是必定陨落了，没有悬念。
<br/>　　“古家二小姐，没想到你居然真的如此愚蠢，还敢来这魔谷之中，你以为你真的可以杀了我？可笑，古家的人，本尊还没有杀过，正好今日就以你的鲜血，来祭炼我的灵器！”
<br/>　　这男子占据上风，大笑起来，从他的手中，一把白骨灵剑突然出现，猛然一剑，便是斩了过去。
<br/>　　这一剑之中蕴含有着极为恐怖的剑意，滔天魔意也是全部凝聚其中，要准备在这一剑之下，这将女子，彻底斩杀。
<br/>　　“不好！”
<br/>　　这女子的脸上，露出了几分绝望，尤其是感觉到这剑锋的可怕的时候，她的脸色已经是苍白无比。
<br/>　　她本以为可以追杀这男子，才误入到了这魔谷之中，却没想到一切都是计谋，将她彻底的算计了，令她现在想逃都没有办法。
<br/>　　“死吧！”
<br/>　　男子阴冷一笑，剑锋落下，这恐怖的剑锋之下，女子身上凝聚的灵力，几乎是在瞬息之间，都被打破，恐怖的剑锋已经是席卷这女子的身边。
<br/>　　在下一刻，这女子的身躯，便是就会被剑锋，无情的撕碎。
<br/>　　男子的脸上，都露出了几分胜利的得意。
<br/>　　但是就在这个时候，突生异变。
<br/>　　一道巨大无比的手掌，突然从天而降，出现在了这女子的面前，硬生生的将这剑锋，抓在了手中，将其剑锋都给直接捏碎。
<br/>　　这出手之人，正是楚风眠。
<br/>　　“九魔城的人，没想到中域也有你们这些小魔头。”
<br/>　　“什么人？”
<br/>　　这男子也是吃了一惊，似乎是没有想到，居然在这个时候居然会有人出手，挡下了他的一剑。
<br/>　　而且在这魔谷之中，他已经是布置出了层层禁制，可现在居然还是被人闯了进来。
<br/>　　“什么啊，原来又来了一个送死的。”
<br/>　　不过当这男子，看到楚风眠，感觉到楚风眠身上气息的一刻，他的脸色便是逐渐的平息了下来，生死境巅峰的境界，跟他也是在伯仲之间，就算是能够闯入到了这魔谷之中，也没有什么。
<br/>　　在这魔谷之中，只要不是超越生死的人物，他都不会怕。
<br/>　　“正好，又来一个送死的，那你们两个就一起死吧！多管闲事的废物！”
<br/>　　男子大喝一声，突然之间又是一剑，直接向着楚风眠斩了过来。
<br/>　　这一剑凶狠无比，那是绝对是杀招，血染剑锋，一股股的魔意，都是化为了一尊尊魔神，加持在了剑锋之上，万魔之剑，要将人直接斩杀。
