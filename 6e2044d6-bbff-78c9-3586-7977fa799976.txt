　　眼前的这位苏凌长老，比起楚风眠这一世见到过的任何人，都要强大的多。
<br/>　　楚风眠虽然见到过像是夺花尊者，虚家老祖这等已经达到生死境实力的强者。
<br/>　　但是却远远无法跟这苏凌长老抗衡，只怕是苏凌长老挥手之间，都足以轰杀了像是虚家老祖这样的生死境强者。
<br/>　　生死境，每一重境界，都有着极大的差距，不是那么好逾越的。
<br/>　　北邙学院会让这苏凌长老前来主持这一届的考核，也不是没有这秀肌肉展现实力的意思。
<br/>　　这主持考核，可不是一件容易的事情，在场的百万武者，几乎每一个都是资质极高的天才，极为狂傲。
<br/>　　其中的人，也不免有着像是燕古这等，实力已经逼近到生死境的强者。
<br/>　　要是这些考核弟子，引发冲突，暴动起来，没有足够的实力，也休想可以将这些人压住。
<br/>　　这苏凌长老，境界达到生死境五重的地步，实力更是强大的惊人，也只有这样的人物，才能压制住在场这些心高气傲的天才。
<br/>　　“你们这次来参加北邙学院的考核，规矩你们应该也知道，先将北邙令牌交出来，会有人来检查，只有拥有北邙令牌的人，才能来参加考核。”
<br/>　　苏凌长老，冷冷的开口道。
<br/>　　随着他开口，便是有着数千尊武者，纷纷出现。
<br/>　　这些武者，皆是身着蓝衣，都是北邙学院的弟子，一个个开始，认真的检查起北邙令牌来。
<br/>　　“你，没有令牌，没有资格参加这一届的考核，滚！”
<br/>　　在前方，突然一阵动静，一名武者，似乎是没有北邙令牌，被其中一尊北邙学院的弟子，直接给当众扔了出去。
<br/>　　丝毫不停任何解释。
<br/>　　北邙学院行事，便是一直都如此霸道无比。
<br/>　　等到楚风眠的时候，楚风眠跟寒月漓便是就将准备好的北邙令牌递了过去，那名北邙学院的弟子便是看了一眼，就还给了楚风眠。
<br/>　　不过还回来的令牌上，却是背面突然多了两个数字。
<br/>　　寒月漓的是一百零三，楚风眠的则是九百六十一。
<br/>　　“所有考核弟子，按照顺序，一个个走入到里面去，一百人一组，参加考核！”
<br/>　　苏凌长老，在上方大声喊道。
<br/>　　突然间，下方的一些武者，手中的北邙令牌，便是开始闪烁起来，他们上面的数字便是一。
<br/>　　一共一百人，直接一齐走了出去，向着宫殿内部走去。
<br/>　　第一组，第二组，第三组。
<br/>　　这上面的号码，便是分组。
<br/>　　这武者离开的速度很快，一组离开，令一组便是也被叫了出去。
<br/>　　“下一组。”
<br/>　　随着苏凌长老再次开口。
<br/>　　寒月漓手中的令牌，亮了起来。
<br/>　　“到我了。”
<br/>　　寒月漓回头看了楚风眠一眼，便是直接走了出去，离开了。
<br/>　　这北邙学院的考核，虽然困难无比，但是对楚风眠来说，他倒是并不担心寒月漓会有什么问题。
<br/>　　毕竟如今这一届的新人中，楚风眠看见的，能够跟寒月漓抗衡的人，寥寥无几。
<br/>　　在她的这一组中，她想要考核成功，轻而易举。
<br/>　　宫殿中的武者，越来越少，逐渐离开，终于楚风眠手中的令牌，也是亮了起来。
<br/>　　楚风眠从人群中走出，便是跟其他人，一起走入到了宫殿内部。
<br/>　　这宫殿内部，随着人的指引，来到了一处巨大无比的擂台上。
<br/>　　擂台混战。
<br/>　　这北邙学院的考核，简单无比，便是擂台混战。
<br/>　　谁能够在百人中，战胜所有人，便是就可以直接加入到北邙学院中去。
<br/>　　在这里，一切的种种都没有任何的意义，唯一可以依靠的，便是实力。
<br/>　　“规则只有一个，你们站在擂台上去，只有一个胜者，可以加入到北邙学院，剩下人，全部被淘汰。”
<br/>　　“你们可以选择认输，只要喊出认输，便是有一股力量可以带你们离开，但是若是不喊，死死抗争的话，死也赖不得别人。”
<br/>　　一尊北邙学院的弟子，在一旁，讲解着规则。
<br/>　　楚风眠这一组人中，有着不少人，都是第一次来参加这北邙学院的考核，听到这如此残酷的规则，都是脸色微微有些紧张。
<br/>　　不过很快，众人的脸色都是平静了下来。
<br/>　　敢来参加北邙学院，都是一方小有名气的天才，规则虽然残酷，却没有人会惧怕。
<br/>　　“哼，一群废物，本少劝你们现在乖乖认输放弃吧，免得到了台上，就连认输的机会都没有。”
<br/>　　一尊武者，冷声开口道，言语中，尽是嚣张之意。
<br/>　　这是一尊年轻男子，他的一双眼睛，扫过楚风眠一群人，便尽是不屑。
<br/>　　看起来也是一尊狂傲到极点的人物。
<br/>　　“屠雄，居然是他，完了。”
<br/>　　一些武者，似乎是认出这年轻男子的身份，一个个脸色苍白起来。
<br/>　　不过更多的人，却是似乎不清楚这男子身份，纷纷询问道。
<br/>　　“此人是谁？看起来实力似乎极为强大。”
<br/>　　“这人是屠雄，虽然不是什么大家族出身的弟子，但是据说得到过不少奇遇，实力极强。”
<br/>　　“他上一届便是已经参加，但是当时运气不好，在小组中遇到了一尊实力逼近生死境的超级强者，抗衡了许久才落败的，他的实力，足以轰杀巅峰圣者。”
<br/>　　“什么，足以跟生死境强者抗衡的人？”
<br/>　　听到那人的话，在场不少的脸色都是变的苍白了许多。
<br/>　　跟这样一尊凶人分到一组，也是绝对的不幸了。
<br/>　　在场的许多人，都是实力也就神力境一重，二重左右的，怎么可能跟如此凶人抗衡。
<br/>　　屠雄，听到众人的议论，冷哼一声道。
<br/>　　“哼，一群废物，不想死的都滚吧，免得来浪费本少时间。”
<br/>　　其中几尊知晓这屠雄实力的武者，听到这话，脸色虽然极为不甘，但是却也只能直接举起手来，选择认输离开。
<br/>　　上了台上，可真的是生死有命了，激烈的战斗中，能不能喊出认输，都不一定。
<br/>　　喊不住来，可就只有死了。
<br/>　　“我认输。”
<br/>　　“我也认输。”
<br/>　　大约有着八人，直接便是认输，离开了。
<br/>　　他们都是上一届参加过的武者，自知不敌，直接离开。
