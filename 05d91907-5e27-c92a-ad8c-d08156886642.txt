　　第176章
<br/>　　程大雷皱眉看着城下的杜茂，心道这老小子果然没死，可他怎么投了戎族。
<br/>　　“杜老大，我敬你是条汉子，怎么与戎族为虎作伥，不怕祖坟被人刨了么。快快放下兵器，与我一起共抗蛮人！”
<br/>　　“姓程的，你休要废话，快快下来，让我一棒敲了你的脑袋。”
<br/>　　“执迷不悔。”程大雷一挥手，冲城下秦蛮道：“干掉他。”
<br/>　　秦蛮一手提枪，一手拍马照着杜老大就冲了过去。
<br/>　　“狗奸贼，受死吧！”
<br/>　　“来得好！”
<br/>　　一枪直接扎了过去，杜茂矮身躲过，一棒向秦蛮座下马砸了过去。枪往下走，去扎杜茂背心，双方插招换式，打得又险又急。
<br/>　　观战的诸人皆惊起一身冷汗，这样的杀招才是真正的战场厮杀之技，招招要的都是对方性命。即便程大雷不在战场，也能感到凛然杀气，如果换成他自己，是绝对不敢置身其中的。这样的对阵，稍微拙一拙，就是人头落地，横死疆场。
<br/>　　赤眉认得这人，知道这是投靠小山部的武人，他的主人正是忽律雄，只是赤眉没想到，此人竟如此了得。
<br/>　　原来帝国也有这样的好汉，丝毫不弱部落的高手。
<br/>　　可惜，这样的高手竟是忽律雄的手下。
<br/>　　眨眼之间，双方已经交手数个回合，最后两匹马转圈厮杀。众人耳边只感觉眼花缭乱，根本看不清二人动手的详细过程。
<br/>　　铛！
<br/>　　重重一声响，铁枪和狼牙棒交在一起。双方都是臂力惊人的猛将，此刻捉对角力，脸上青筋暴露，额头豆大的汗珠掉落。
<br/>　　程大雷拿着弓箭在城垛晃呀晃，想要瞄准杜茂。
<br/>　　“大当家，放暗箭不地道吧。”徐神机。
<br/>　　“对待敌人要像秋风扫落叶般冷酷，有什么不地道的。”程大雷瞄了半天，最后摇了摇头：“没办法，离得太远了。”
<br/>　　“大当家，你说秦蛮能打赢么？”
<br/>　　“秦蛮和杜茂的本事应在伯仲之间，想要赢，估计得出奇招。”
<br/>　　铛！
<br/>　　徐神机看得仔细，就见秦蛮身子一顿，竟被杜茂的狼牙棒压了过来。
<br/>　　“大当家，不好。”
<br/>　　“别急，慢瞧。”
<br/>　　秦蛮被压在马鞍桥上，杜茂挥舞着大棒向他抡了过来。忽然，马鞍上的秦蛮一挺身，枪尖扎在了马头上。
<br/>　　烈马吃痛昂头，直接将杜茂掀翻在地。此刻的秦蛮握紧长枪，向地上的杜茂扎来。
<br/>　　“狗贼，死吧！”
<br/>　　抬头只看见铁枪扎来，生死关头的杜茂猛地一个翻滚，爬起来便逃，秦蛮拍马追赶，此时从北蛮部大营过来几匹快马要接杜茂。
<br/>　　“秦蛮，穷寇莫追，回来换人！”
<br/>　　秦蛮好恼恨没有杀了杜茂，只是追得太远容易中戎族的暗箭。他恶狠狠啐了一口，拨转马头进了城门。
<br/>　　“戎族的渣渣们，知道厉害了么。现在下马投降，跪下磕头，本将军传你们一招两招，也够你们一辈子受用了。”
<br/>　　“帝国人打帝国人，有什么好得意的，你们擅长的不就是狗咬狗内斗么。”
<br/>　　程大雷眉毛跳动两下，恶狠狠道：“再来！”
<br/>　　赵子龙长枪白马出了城门。
<br/>　　“哈，你们当真是无人可用了，竟让一个娃娃出来送死。”
<br/>　　“娃娃，快回去吧，吃几年奶再回来上阵。”
<br/>　　“好啦，好啦。”赤眉挥挥手：“谁过去杀了这娃娃。”
<br/>　　“我去，我去。”
<br/>　　“都莫要与我抢，这次由我亲自上阵！”
<br/>　　“您老都六十了，还是歇歇吧。”
<br/>　　忽地一匹马冲了出去，马上一人披头散发，手持两把弯刀。
<br/>　　“嘿，怎么让他抢了先！”
<br/>　　两马相错，这戎人被赵子龙一枪挑落马下。
<br/>　　戎族这边都睁大了眼睛，张大的嘴巴无法合拢。
<br/>　　“谁？”
<br/>　　赵子龙挑枪，被他枪尖扫过的众人，心底忽地冒出一阵寒意。
<br/>　　赵子龙连战七人，连战连胜。
<br/>　　蛤蟆寨这边趾高气扬，赤眉这边却一个个如霜打的茄子般抬不起头来。
<br/>　　这娃娃竟如此厉害！
<br/>　　“四爷，回来吧。”
<br/>　　看得赵子龙已然力竭，再打下去便有危险，程大雷急忙招他回来。
<br/>　　他在城头得意洋洋的笑着：“渣渣就是渣渣，不堪一击，忘了说一句，你们戎族那个呼尔勒小王子什么的，也是他杀的。”
<br/>　　“呼尔勒是他杀的！”
<br/>　　赤眉心中惊喜交加。呼尔勒是北蛮王最看重的孩子，在北蛮部威望极高，如果他还活着，赤眉这种人根本没有出头之日。也就是呼尔勒死了，诸王子才有争权夺势的机会。
<br/>　　这是赤眉心中的喜，惊的是呼尔勒那样的猛人都死在这娃娃手中，这娃娃果然了得。
<br/>　　现在士气低迷，显然是受了赵子龙的影响，必须要有一场迅速的胜利，方可提升士气。
<br/>　　赤眉拍马向前，在射程范围外停下，高声喊道：
<br/>　　“姓程的，你可敢出城一战，若你能胜了我下一员将，我立刻退兵。”
<br/>　　“呵，姓赤的，我是指挥者，不逞匹夫之勇，这样艰深的道理，你们戎族的渣渣是不会懂的。”程大雷也不傻，自己手下能用的不过秦蛮赵子龙二人，可赤眉手下却是高手如云，他可不愿意去送死。
<br/>　　“莫非你是怕了不成！若是惧怕我等，速速开门投降。我免你一死，给你一个牵马执鞍的机会。”赤眉在城下不足挑衅。程大雷出来不出来都没关系，只要他们怯战，也可以提升手下士气。
<br/>　　程大雷根本不搭理他。
<br/>　　“大当家，不如你下去和他打一场。”徐神机道。
<br/>　　“咦，你是有多恨我不死。”程大雷。
<br/>　　“现在敌人在下面骂阵，于士气不利，若大当家亲自上阵，必能提升我军士气。”
<br/>　　“这……”程大雷想了想，自己摆了蛤蟆寨的山贼一道，虽然他们尚未公开反对自己，但山寨内的确对自己有怨言。
<br/>　　若有这场胜利，或者只是出阵打一打，也许就能平息他们但怨气，也可以提高自己在山寨的威信。
<br/>　　“秦大哥，子龙弟，你们要保护我，一看我不成，随时出阵救我。”
<br/>　　程大雷骑马提斧，不忘嘱咐秦蛮二人。
<br/>　　见程大雷出来，赤眉冷哼一声，挥手道：“把那怪物放出来！”
<br/>　　紧跟着，程大雷看到一个奴隶模样的汉子从戎族营地中走出。赤眉丢给他一杆枪，道：“姓林的，杀了他，我放你离开。”
<br/>
