　　此刻，世界都好似静止了一般。
<br/>　　所有的声音，所有的动作，所有的一切都不复存在。
<br/>　　只有蒋天庆失去思考能力的呆滞表情，他一动不动的站在不远处的楼层里，望着下方道路上那丧尸头领。
<br/>　　它的身躯受到了强大冲力，在往前方倾斜，而它的后背处则是有着一个弹孔，以及一股溢出的黑血。
<br/>　　丧尸头领中弹了！
<br/>　　那是一发绝命狙击枪的子弹！
<br/>　　在这个紧要关头能开枪，并且能一枪就击中丧尸头领的人，除了陈少爷还有谁？！
<br/>　　而有如此强大的威力能冲破丧尸头领恐怖防御力，也只有经过了三次改造，已经是接近热武器中巅峰存在的绝命狙击枪！
<br/>　　蒋天庆的情绪即将爆炸，他从窗户探出半个身子，朝着陈少爷所在的那处高楼望去。
<br/>　　那座高楼距离这里很远，蒋天庆也是用上了望远镜才看到高楼顶层的场景，这会儿的陈少爷正一只脚踏在楼顶最边缘，一只脚稳住身躯踩在后方，整个人有小半个身子都在半空，相当极限也危险的射击姿势。
<br/>　　但他的手很稳，他的表情很坚定，他就这么站在那里，说开枪就开枪！
<br/>　　这时候的蒋天庆，才猛然惊觉一个微不可查的细节，那就是楚涵将丧尸头领拦截的时候，恰好挡住了陈少爷的射击角度。
<br/>　　因为那高楼在他们后方！
<br/>　　当时并不知道丧尸头领会出现在哪里，而射击手进入状态之前需要准备，蒋天庆理所当然的将陈少爷带到了这片地区最佳的射击地点。
<br/>　　但这却与丧尸头领出现的位置隔得太远，甚至超出了陈少爷的射击范围！
<br/>　　楚涵却是从另一个侧面方向追来，他们没有时间和机会来选择与丧尸头领碰面的地点，唯一的可能就是先行拦截丧尸头领，然后在战斗中不断朝着陈少爷靠近。
<br/>　　但谁也没有想到的是，丧尸头领根本没打算与楚涵拖延时间，而是采取了速战速决的方式，并且在下落的刹那丧尸头领还自作聪明的一个侧移，瞬间与楚涵调换了位置！
<br/>　　原本楚涵处在丧尸头领与陈少爷之间的状况，便一下子成为了楚涵和陈少爷将丧尸头领夹击。
<br/>　　而更没有人能想到的是，已经超出射击范围的陈少爷，竟然毫不犹豫的选择了开枪，甚至在如此恶劣的射击环境之中，还命中了这只丧尸头领！
<br/>　　如果不是这些不利条件，刚才的那一发子弹恐怕就不是命中后背，应该是头颅、脖颈、耳道等一切致命部位！
<br/>　　谁都清楚射击范围有多重要，同样也是一名射击手的极限能力。
<br/>　　超过了射击范围当然也能开枪，但子弹会出现在哪里则就不可控制，尤其在狼牙各种战役之中，常常会出现敌军与我军人员交杂的情况。
<br/>　　而在射击范围内，高于一定命中率的射击手，就有能力控制不伤及我军人员，并将敌军一击毙命。
<br/>　　当下的科技尤其是狼牙科技部的科技，早已将发达到能制造出超远距离的枪支。
<br/>　　绝命狙击枪就是其中之一！
<br/>　　一切都需要基于太多的基础条件，而之前丧尸头领与楚涵近战，超出射击范围的陈少爷哪怕身为神之狙击，也断然无法扣下扳机，那是不可控制的结果，子弹会射·入丧尸头领还是楚涵身上，谁也不知道。
<br/>　　所以在有狙击手存在的时候，那只丧尸头领忽然的改变位置，哪怕就此露出了最大破绽，陈少爷也无法开枪。
<br/>　　但是，谁能想到楚涵毫不犹豫的朝前狂奔？
<br/>　　甚至还一口气跑了二十米远！
<br/>　　这个让蒋天庆深度怀疑的反常举动，让丧尸头领都呆住的愚蠢行为，却是在给陈少爷制造机会！
<br/>　　靠太近，陈少爷无法开枪，因为楚涵是他唯一的心里障碍。
<br/>　　而楚涵跑出了二十米远，这就超出了绝命狙击枪的影响范围，这时候哪怕陈少爷没命中，但根据其逆天能力而射·出的子弹方向，也不会给楚涵带来危险。
<br/>　　只有两种可能，要么命中丧尸头领，要么命中地面！
<br/>　　但神之狙击陈少爷在有了楚涵制造的机会之后，怎么可能失误？
<br/>　　至于楚涵的行为，反常？愚蠢？
<br/>　　当枪声响起的那一刻，楚涵的妖孽程度也终将暴露！
<br/>　　他毫不犹豫的行为制造机会，普通狙击手无法与之配合，更没有胆子扣下扳机，也无法做到他想要的结果。
<br/>　　但楚涵知道，陈少爷做得到！
<br/>　　他和其他所有的射击手都不同，有关一切射击的那些理论知识他当然熟悉的倒背如流，但这都是他已经有了神之狙击这一称号之后，被楚涵跟何峰强行命令去补习的资料。
<br/>　　因为他不仅是神之狙击，还是战团射击队的队长，是狼牙中将，是楚涵最强力的左膀右臂！
<br/>　　那些理论对于陈少爷来说，只是学懂了之后再结合他自己开枪时的感受，用来教导射击队成员。
<br/>　　在此之前的陈少爷开枪，全凭直觉！
<br/>　　这两年来他的每一次命中，都不是靠的理论而是直觉，亦或者再加上经验和不断训练后增强的体能。
<br/>　　哪怕知道当下已有的最全面射击技巧，哪怕背熟了无数本理论知识，但这些都只是他身为射击队队长的职责。
<br/>　　别人开枪需要看环境，要预估目标状态，要计算无数数据，要看风看雨看地表看湿度看空气……
<br/>　　但陈少爷，他不需要。
<br/>　　神之狙击之所以被如此评价，就是因为他的能力包含了一切理论，甚至还有更多还没被发掘和认知的内容。
<br/>　　这所有的一切都转化为一种暗示，也就是陈少爷自己称之为的直觉。
<br/>　　只要体能强大到任何后坐力都不是问题，只要精力充足到可以继续维持下去，只要子弹的速度追的上目标。
<br/>　　他就能命中！
<br/>　　什么时候开枪，枪口应该对准哪个方向，该开几枪……
<br/>　　所有的一切都成为了陈少爷的下意识，这就是这世上独一无二的能力，并不被列在射击或命中之中，而是傲世独立存在的一个。
<br/>　　它名字就叫：神之狙击。
<br/>　　这也为从开始到现在一直是最逆天的一种，从楚涵的上一世到现在的末世三年，以至于将来的末世十年，永远都是！
