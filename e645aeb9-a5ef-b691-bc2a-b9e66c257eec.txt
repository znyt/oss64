　　“这是什么武技？本源！这是龙族的本源！龙族的正统，这种武技不是只有始祖可以掌控的吗？你到底是什么人？”
<br/>　　龙摩罗死死的盯着楚风眠，大声的咆哮道。
<br/>　　他可是一尊九劫古帝巅峰的魔龙，在刚刚看见楚风眠施展出的始祖第二式的一刻。
<br/>　　他就立刻察觉出了其中的玄妙来，这一招几乎是代表着龙族正统。
<br/>　　是龙族之中，最为古老，最为玄妙的武技。
<br/>　　在许多古老的传闻之中，甚至是就连踏入天人境的龙帝，都未必足以掌控的武技。
<br/>　　现在却是出现在了一个名不经传的小人物手中。
<br/>　　龙摩罗连连出手，打出无数杀招，打算抵挡楚风眠的攻势。
<br/>　　但随着楚风眠的身形一闪，一步踏出，从他的背后那龙皇虚影，咆哮一声。
<br/>　　龙摩罗打出的无数杀招，纷纷破碎开来，楚风眠身形向着龙摩罗冲了过去。
<br/>　　“不好，此人实在是不好对付。”
<br/>　　龙摩罗退后连连，也知晓这一次他是撞到了铁板之上了。
<br/>　　他本来是想要对付楚风眠，夺取了楚风眠的血脉之力，来增添他的实力。
<br/>　　所以便是以无上手段，将他跟楚风眠一同拉扯到了虚空深处来。
<br/>　　一旦是陷入在这虚空深处之中，想要冲出去，几乎是难上加难。
<br/>　　这一招，本来是他想要对付楚风眠，让楚风眠没有机会可以离开，才施展的招式。
<br/>　　如今却算是搬起石头砸了自己的脚，在这虚空深处之中，就算他现在想要离开，都并不容易。
<br/>　　更何况楚风眠，还在背后一直追杀他，根本不可能会放过他。
<br/>　　“魔龙真身！”
<br/>　　龙摩罗大声的咆哮一声，只见在他身躯之上的龙族血脉之力，全然爆发而出。
<br/>　　他的身躯，也是在虚空之中变幻这，化为了一尊足有数百丈之长的漆黑魔龙，两道龙爪对着楚风眠当即轰击而来。
<br/>　　“不用挣扎了，一条小小的魔龙，除非是你踏入半步天人境的境界，不然在我的血脉面前，你任何的挣扎，都没有任何的意义！”
<br/>　　楚风眠看着变化出真身的龙摩罗，也是冷笑一声。
<br/>　　从楚风眠的身躯之上，龙皇血脉之力翻腾而出，化为了滚滚龙意，笼罩在了这龙摩罗的身上。
<br/>　　这龙摩罗的血脉，最多也只是远古战龙血脉，楚风眠现在的血脉，却是蕴含着几分远古龙皇的血脉。
<br/>　　血脉的尊卑，刻画在这每一尊龙族的身躯之中去。
<br/>　　面对更加尊贵的血脉，会受到压制。
<br/>　　如果是对付其他尊贵血脉的魔族，像是一些魔族之中的王族，甚至是传说之中的魔族皇族血脉，几乎不亚于魔龙实力的那些魔族强者。
<br/>　　楚风眠想要对付起来，还要棘手几分。
<br/>　　但是这龙摩罗，却是一尊魔龙，龙族的力量，在他的血脉之中，根本是无法抹除。
<br/>　　这龙摩罗身上的力量，在这血脉的威势之下，都是足足减少了三成。
<br/>　　让他本来招架楚风眠攻势的时候，都是无比的艰难，现在更是雪上加霜。
<br/>　　接连不断的攻击，已经是轰击在了龙摩罗的身上，每一次轰击，都打碎了他身上不少的鳞片。
<br/>　　“可恶！可恶！此人的血脉怎么会如此厉害！”
<br/>　　龙摩罗疯狂的咆哮道，他的力量已经是被楚风眠死死的压制住了，只能够是匆忙应对楚风眠的进攻，就连反击都做不到。
<br/>　　他看的出来，楚风眠的力量虽然远超八劫古帝，但是跟他相比，最多也就是在伯仲之间。
<br/>　　如果硬拼力量，可能他的力量，还要在楚风眠的力量之上。
<br/>　　但是现在，因为他的力量受限于血脉之力的压制，在楚风眠的面前，只能够是被动挨打。
<br/>　　连续不断的轰击，这龙摩罗的身躯都要被打碎。
<br/>　　“老家伙，念在同为龙族的份上，我给你一个机会，臣服与我，我可以不杀你，甚至是未来，给你机缘，让你踏入半步天人境！”
<br/>　　楚风眠盯着龙摩罗，冷声说道。
<br/>　　这龙摩罗，是楚风眠所见到过的最为强横的一尊九劫古帝。
<br/>　　楚风眠虽然不知道此人距离半步天人境，还有多久，但是如此实力，哪怕是五名，六名寻常的九级古帝围攻他，都要被他轰杀。
<br/>　　楚风眠要是没有血脉之力的压制，对付这龙摩罗的胜算，也只有五成。
<br/>　　一旦是降伏了此人的话，楚风眠的目的可以达到，还可以给剑道门，留下一个王牌，有这样一尊强者坐镇。
<br/>　　除非是有半步天人境的强者，去攻打剑道门，不然的话，剑道门遇到再大的危险，便是无忧。
<br/>　　“笑话，就你一个小小的八劫古帝也想要降伏本帝？少帝都不能降伏我，你算是什么！不是血脉之力的压制，我岂能怕你！”
<br/>　　龙摩罗听到楚风眠的话，疯狂的咆哮道，眼神之中无比疯狂。
<br/>　　“今日本帝就算是栽到了你的手里，你也休想要占到一点便宜！”
<br/>　　“末日升龙！”
<br/>　　轰隆！
<br/>　　从龙摩罗的背后，一尊金色的龙影，突然之间飞了出来，从天而起。
<br/>　　随着这一尊金色龙影现身的一刻，龙摩罗身上的力量，也是达到了巅峰。
<br/>　　他一身的力量，全部凝聚在了这一尊金色龙影之上，蕴含毁灭之力，彻底的拼命了。
<br/>　　末日升龙！
<br/>　　楚风眠在见到这金色龙影的一刻，眼神之中都是无比的忌惮。
<br/>　　他拥有龙族血脉，自然知晓这一招，便是龙族绝对的杀招，一旦释放，便是唯有身死的下场。
<br/>　　是宁为玉碎不为瓦全的一招，所有龙族的最后一招。
<br/>　　这一招的恐怖，楚风眠曾经在实力弱小的时候，便是见到过一尊蛟龙，施展出的这一招，威力恐怖到了极点。
<br/>　　眼前这龙摩罗，可是纯粹的战龙血脉，施展出的末日升龙，威力强横到了极点。
<br/>　　周围的虚空，都在震动。
<br/>　　这龙摩罗，现在被楚风眠逼入到了绝境，落到楚风眠的手里，宁死也不愿为楚风眠所降服，这便是龙族血脉的荣耀。
<br/>　　“可惜，不能降服此人。”
<br/>　　楚风眠见到这末日升龙这一招打出的一刻，也是自言自语了一声。
<br/>　　这一招打出，代表了龙摩罗，已经打算是要跟楚风眠同归于尽。
