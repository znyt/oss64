　　金光险地，入夜时分。
<br/>　　深处。
<br/>　　轰…轰…轰…
<br/>　　一声声连绵不断的剧烈轰鸣，骇人之极。
<br/>　　一道单薄的身影，正与一头庞然大物，鏖战不停。
<br/>　　剑影，巨力下的气流冲击，充斥着周遭。
<br/>　　战斗双方，自然是萧逸，以及那头金刚白猿。
<br/>　　不错，萧逸再次找上了这头金刚白猿。
<br/>　　这头金刚白猿吞食金光的实力，绝对达到了圣皇境巅峰。
<br/>　　这等妖兽，对萧逸而言，自然是历练效果极大。
<br/>　　锵…剑气激扬，星光长河延绵而出。
<br/>　　轰…巨大的拳头，无差别般密集轰出，澎湃的气流，不过片刻便将周遭轰成废墟。
<br/>　　萧逸与金刚白猿的战斗，除了激烈二字外，再难形容。
<br/>　　不过，十数分钟后。
<br/>　　剧烈的轰鸣声，澎湃的气流，仍旧不断出现。
<br/>　　倒是剑气与星光长河，已然消散。
<br/>　　萧逸的剑，并未再与金刚白猿硬撼，而是身影闪烁间，躲避金刚白猿恐怖的拳头。
<br/>　　又是十数分钟后，萧逸的动作，再次变了。
<br/>　　或闪或避的动作，再度变回剑气飞扬，星光长河碾压而下。
<br/>　　又是十数分钟后…
<br/>　　时间，渐渐过去。
<br/>　　萧逸与金刚白猿的战斗，便在这般情况下，不断变幻着。
<br/>　　剑气激扬，星光长河碾压而下时，便是他元力充沛时。
<br/>　　或闪或避，便是他在磨砺着他的步伐、反应能力。
<br/>　　这般不断转换，对萧逸的反应能力，应变能力，以及掌控元力，有着极高的考验与磨砺效果。
<br/>　　整整大半晚后，萧逸满头大汗，这才停下。
<br/>　　嗖…
<br/>　　身影一闪，瞬间远遁。
<br/>　　剩下的半晚时间，萧逸继续游走于金光险地内，不过，不再是磨砺，而是猎杀。
<br/>　　一头头妖兽，被他快速斩于剑下。
<br/>　　收了妖兽精血与内丹，并将内丹炼制成金光丹。
<br/>　　直到初晨时分，夜色褪去，萧逸才停下猎杀，开始一日的打坐。
<br/>　　……
<br/>　　翌日，入夜。
<br/>　　金光险地深处。
<br/>　　锵…一道剑气疾速打出，重重地达到了一头浑身白色皮毛的巨兽之上。
<br/>　　吼…金刚白猿吃痛，怒吼一声。
<br/>　　萧逸凭空而现，轻笑一声，“我又来了，战吧。”
<br/>　　吼…随着金刚白猿的一声怒吼。
<br/>　　双方的战斗，再次打响。
<br/>　　又是大半夜过去，萧逸远遁而离。
<br/>　　剩下的半晚时间，再次陷入了猎妖当中。
<br/>　　……
<br/>　　日复一日，一月时间，转瞬便过。
<br/>　　萧逸的时间，几乎都在有规律地循环度过着。
<br/>　　白日，打坐修炼，并做着感悟。
<br/>　　入夜，一半时间与金刚白猿鏖战；一半时间，横扫妖兽森林，一路猎妖，收集妖兽精血与内丹。
<br/>　　今日。
<br/>　　某棵参天之树上，萧逸盘膝而坐。
<br/>　　不过片刻，一股气势的爆发，在他身上涌现。
<br/>　　又是片刻，气势再度爆发。
<br/>　　这时，萧逸才缓缓睁开了眼睛，眼中尽是喜意。
<br/>　　“圣王境八重。”
<br/>　　在这金光险地历练一月，萧逸特地选在今日将修为突破了两重，踏入圣王境八重。
<br/>　　当然，他的那些灵脉，仍旧未动。
<br/>　　那是他留着冲击圣皇境的。
<br/>　　突破这两重修为的修炼之物，来自于他这一月的历练。
<br/>　　虽然大部分时间都在猎妖，但游走于这金光险地，一路横扫，自也不时会遇到珍稀的天材地宝。
<br/>　　算上他自己身上的一些修炼之物，终于在今日选择了突破，且是连跳两重。
<br/>　　不过，这两重的修为提升，对他的实力倒无太大增加。
<br/>　　毕竟，他现今的最强战力，乃是来自于星光降临这一手段。
<br/>　　他能拥有圣皇境八重以上战力，是因为八万星光之力降临。
<br/>　　他能拥有力敌圣皇境巅峰的实力，是因为破晓钟上那一剑出20剑齐至的手段。
<br/>　　八万星光，20道连绵而出，才达到了惊人的威力。
<br/>　　也就是说，这些实力的来源，最主要还是源于星光之力，而与他本身的修为并无太大关系。
<br/>　　与修为有关的，仅仅是修为的高低，让得他感悟并操控星光降临的难易与否。
<br/>　　要提升星幻剑道的威力，唯一的办法，只有掌握更多的星光降临。
<br/>　　也就是说，他的武道境界层次，继续提高。
<br/>　　故，他现今提升的这两重修为，对于八万星光加身下的最强战力，并无半分提升。
<br/>　　当然，一月磨砺，也并非是没有收获。
<br/>　　与金刚白猿激战，无时无刻不在磨砺着他对星光的掌控。
<br/>　　一月激战，倒是让他感悟颇多，且星光的掌控，增加了5000道。
<br/>　　也就是说，他现今能掌控降临的星光，达到八万五千道。
<br/>　　待得他能掌控9万星光降临时，便单凭星光加身就有圣皇境九重战力；若再加上的破晓钟上手段，足以轻松对付圣皇境巅峰了。
<br/>　　另外，一月磨砺，破晓钟上的手段，他同样有所收获。
<br/>　　原本，他只能一剑出，20剑齐至。
<br/>　　如今，已能一剑出，30剑齐至。
<br/>　　这两者，对于现今萧逸的最强战力提升，倒是极大。
<br/>　　而提升的这两重修为，唯一给予萧逸的帮助，便是萧逸使用其他武技时威力更强。
<br/>　　如寂空剑道上的武技，血界斩等等。
<br/>　　若想真正有质的飞跃，还是需要修为上的大幅提升，毕竟修为才是武者的基础。
<br/>　　在萧逸现今的计划中，便是先以星幻剑道作为实力。
<br/>　　随后在磨砺中，不断提升修为。
<br/>　　他的修为，绝不会止步于圣王境八重。
<br/>　　待得修为突破圣皇境，甚至达到圣皇境后期时，许多底牌，便能发挥出应有的绝对威力了。
<br/>　　而这，需要时间。
<br/>　　萧逸，亦极限地极快着历练的脚步。
<br/>　　“今晚，便是最后一晚吧。”萧逸自语一声。
<br/>　　在金光险地历练一月，也该离去了。
<br/>　　一月时间，他收获了大量妖兽精血，也炼制了大量金光丹。
<br/>　　再继续待在这里，虽然仍旧有历练效果，但却比之以前缓慢了。
<br/>　　……
<br/>　　入夜，萧逸一如既往地找上了金刚白猿。
<br/>　　吼…金刚白猿怒吼一声，那双泛着凶光的眼眸中，带着极其人性化的仇视，以及一丝无奈。
<br/>　　在它眼中，面前这个年轻的人类武者，一月来，每到入夜时分，便会准时现身。
<br/>　　每次的激战，总让它烦不胜烦。
<br/>　　但今晚，这些日子，便要结束了。
<br/>　　金刚白猿的眼眸上，凶光凛然。
<br/>　　身上气息，正在疾速飙升。
<br/>　　“吞食金光吗？”萧逸淡淡一笑。
<br/>　　但这丝笑容，不多时便戛然而止。
<br/>　　“到达圣皇境巅峰了，嗯？还未停止，气息仍旧在飙升。”
<br/>　　萧逸双眼一眯。
<br/>　　金刚白猿，则露出了极其人性化的一声狞笑。
