        童筱颖也是看得一脸惊呆。
<br/>
<br/>        大狼狗亚兹见童筱颖没动静，于是趴在地上打了个滚，又向童筱颖咧着嘴吐出了粉色的大舌头。
<br/>
<br/>        童筱颖怔愣了一下，这才明白过来，于是伸出手来，试探性地搭在了大狼狗亚兹的头顶，轻轻地抚摸着。
<br/>
<br/>        这一瞬间，所有人都看得目瞪口呆。
<br/>
<br/>        “亚兹，你的名字好好听！亚兹，你好！我叫筱颖。我们是好朋友啦！”童筱颖终于放宽了心态，抬起双手捧住了亚兹的狗脸，并温柔地亲了亲亚兹的眉心。
<br/>
<br/>        亚兹非常乖顺地任由童筱颖摆布，哪怕童筱颖的手上有肉汁的味道，它也不去舔舐童筱颖的手。
<br/>
<br/>        “筱颖？你真的是筱颖吗？”突然，一个娇柔的女音，从玄关口传来。
<br/>
<br/>        童筱颖循声望去，只见玄关口处，走出来一个学生打扮的女生。
<br/>
<br/>        “雅姗？！”童筱颖怔愣了。
<br/>
<br/>        季雅姗疾步走了过来，蹲下身去时，一手拍了拍大狼狗亚兹的头顶，一手拿掉了童筱颖脸上的白羽假面眼镜。
<br/>
<br/>        “真的是你啊！”季雅姗大吃一惊。
<br/>
<br/>        她刚走到玄关口，就听到有人在对亚兹说话。
<br/>
<br/>        她听到她说“我叫筱颖”的时候，她还在怀疑是不是自己听错了。
<br/>
<br/>        “你怎么会在这儿？”两人几乎异口同声。
<br/>
<br/>        而与此同时，十爷看到童筱颖的那张脸时，瞬间瞪大了眼睛。
<br/>
<br/>        “你的脸怎么呢？谁打的？”季雅姗看到童筱颖的脸颊红得发肿，有些心疼地摸了摸她的脸颊。
<br/>
<br/>        童筱颖却只能苦笑。
<br/>
<br/>        她打了人家一巴掌，人家还她两巴掌，她还能说什么呢？
<br/>
<br/>        季雅姗眼珠子一转，似乎是明白了什么，从童筱颖面前站起身来，朝十爷走了过去。
<br/>
<br/>        就在童筱颖好奇季雅姗为什么会出现在这里的时候，季雅姗突然往十爷的大腿上一坐，双手环住了十爷的脖子，声音软糯地撒娇道：“阿十，筱颖是我的好朋友。你怎么能派人打她呢？”
<br/>
<br/>        “你不是说，你从来就不交朋友的吗？”十爷的目光从童筱颖的身上收回来后，伸手揽住季雅姗的腰肢，将她往自己怀里一靠。
<br/>
<br/>        “那是因为，没有人愿意当我的好朋友呀！这不，筱颖她愿意啊！”季雅姗俏皮地眨了眨眼睛。
<br/>
<br/>        十爷眸光微沉。
<br/>
<br/>        季雅姗连忙在十爷的脸上亲了一口，继续用她那娇嫩欲滴的音调，软绵绵地接着说道：“阿十，你就放过筱颖，好不好嘛？还有筱颖的朋友！筱颖是我的朋友，而筱颖的朋友，那也是我的朋友！筱颖若是受伤了，我会心痛的，筱颖的朋友若是受伤了，筱颖也会心痛。筱颖心痛，我也会不开心的啊！”
<br/>
<br/>        季雅姗这话一落，童筱颖瞬间难以置信地看向了季雅姗。
<br/>
<br/>        十爷仍旧不吭声，只是抬了抬手。
<br/>
<br/>        那名牵狗的保镖，又将大狼狗亚兹给牵回了内屋。
<br/>
<br/>        季雅姗见状，打铁趁热地继续说道：“你看，就连平日里从来不待见陌生人的亚兹，都那么喜欢筱颖了！阿十若是还不放过筱颖，只怕，亚兹也会不开心。”
<br/>
<br/>        “今天可不是周末！”十爷答非所问。
<br/>
<br/>        季雅姗自然是知他这话的另外一层意思。
<br/>
<br/>        没错，他每次只有周末的时候才会让她陪着他。
<br/>
<br/>        而她现在突然出现在这里，而且还这么晚，很明显是受人之托，被人给叫过来的。
<br/>
<br/>        季雅姗当然不会出卖那个叫她过来的人，只是更加亲昵地挽着十爷的脖子，将脑袋轻轻地埋在十爷的肩窝上，微微一笑，然后柔柔地撒着娇，说道：“我想你了！想得睡不着！所以就想来这里碰碰运气，看看你在不在！”
<br/>
<br/>        “好！”十爷冷淡地开口，大手在季雅姗的腰肢上用力握了握，“看样子，你是被我给养胖了点，该做做运动，减减肥了！今晚，你就留在这里陪我过夜！”
<br/>
<br/>        刹那间，季雅姗的脸部紧绷，笑容渐敛。
<br/>
<br/>        童筱颖还不明是什么情况，就被这位十爷身边的保镖，连带着秋容一起给推出了门外。
<br/>
<br/>        保镖们都守在了门外，房间内就只剩下季雅姗和十爷，还有那只大狼狗亚兹。
<br/>
<br/>        忽然间，童筱颖感到心痛了。
<br/>
<br/>        刚刚……
<br/>
<br/>        倘若她没看错的话……
<br/>
<br/>        雅姗是不愿意留下来过夜的吧！
<br/>
<br/>        童筱颖低垂着头，想哭却哭不出来。
<br/>
<br/>        起初被扇了两耳光，那么痛，她都没想过要哭，可是刚刚，看到季雅姗那不情愿的眼神时，她的心突然间被刺痛了。
<br/>
<br/>        此时，秋容醒了过来，看到搀扶着自己的童筱颖，连忙拉起她的手，细细地打量着：“小谢，你的手，你的手……”
<br/>
<br/>        “我的手没事！”童筱颖低沉地应了声。
<br/>
<br/>        秋容这才将目光往童筱颖的脸上挪去，顿时身心一怔。
<br/>
<br/>        像！
<br/>
<br/>        这张脸……
<br/>
<br/>        和婴宁有那么几分相像！
<br/>
<br/>        她就知道，如果十爷看到小谢的这张脸，一定会放过小谢。
<br/>
<br/>        童筱颖见秋容盯着自己看，下意识地抬起手来，摸了摸自己的脸颊，问道：“是不是被打肿呢？”
<br/>
<br/>        “对不起！”秋容鞠了个九十度的躬，再次向童筱颖道歉。
<br/>
<br/>        童筱颖慌忙将秋容扶起来，心里总觉得是自己的问题。
<br/>
<br/>        她若是没有认错人，不给十爷那一巴掌，或许就不会……
<br/>
<br/>        而且，也不会害了雅姗……
<br/>
<br/>        童筱颖再次想到季雅姗的时候，一转身刚准备敲门，就被守在门口的两个保镖给拦住。
<br/>
<br/>        她把她当朋友，而她却什么事都为她做不了。
<br/>
<br/>        秋容见童筱颖还想去“招惹”十爷，立即上前阻止，并拉着她往电梯口那边走。
<br/>
<br/>        “小谢，不要再去招惹十爷了！十爷真的不是我们能惹得起的！”秋容紧紧地挽住童筱颖的手，一边拉拽着她的步伐跟着自己一起挪动，一边深沉地劝说道。
<br/>
<br/>        其实，刚刚她是假装被吓晕了过去。
<br/>
<br/>        因为，那样的情形，她害怕，甚至不敢去面对。
<br/>
<br/>        万一小谢的手被那只大狼狗亚兹咬得血肉模糊，事后，她大可以说自己什么都不知道，或者装疯卖傻来糊弄过去。
<br/>
<br/>        她只要她自己还好好的活着，比什么都重要。
