　　轰！
<br/>　　随着叶尘这道声音落下，身上的气息，竟然再次暴涨！
<br/>　　“逆生百转，第二转！第三转！第四转！”
<br/>　　此时已经处于暴走状态下的叶尘，仅存那么一丝理智，竟然一口气将逆生百转，运转到第四转的境界！
<br/>　　不但眼眸越发的赤红，身上青筋也根根暴起，似乎随时都要炸裂开来一般。
<br/>　　一般人若是面对如此强大的痛苦，只怕早就承受不住，直接晕厥过去。
<br/>　　然而，这种痛苦，相比于痛苦之源给叶尘造成的精神痛苦来说，根本微不足道。
<br/>　　甚至，肉体上的痛苦，反而让叶尘的精神缓和了几分，甚至有一种畅汗淋漓的感觉。
<br/>　　而现在，他的脑海之中，只剩下一道念头，那就是杀光眼前的所有人！
<br/>　　与此同时，
<br/>　　上千名药王宗弟子，功力聚合在一处，所形成的那道巨掌，也已经朝叶尘狠狠拍落。
<br/>　　“哼！”
<br/>　　叶尘冷哼一声，直接一拳迎了上去。
<br/>　　轰隆隆！
<br/>　　巨掌和巨拳相撞在一处，周围空间顿时一阵剧烈晃动，甚至已经发现了扭曲，出现了道道裂痕。
<br/>　　蹬蹬蹬！
<br/>　　撞击过后，叶尘顿时倒退了七八步。
<br/>　　纵然如今他的功力，比之前提升了七八倍，可是以一人之力，正面对抗上千名药王宗弟子之合力，终究还是要逊色几分。
<br/>　　倘若是理智未失的情况下，叶尘肯定不会选择正面对抗，而是会先寻找对方的破绽，从弱点入手。
<br/>　　可是现在，他满腔的杀意怒火，根本不可能会去理智思考。
<br/>　　被众人合力击退之后，叶尘再次暴喝一声，继续朝下方的众人冲了过去。
<br/>　　轰隆！
<br/>　　双方再次交手，叶尘也再次被震退。
<br/>　　被药王宗众人保护在中央的云千秋，顿时露出一抹得意的笑容，
<br/>　　“我就不信了，难道你还真能一以敌千不成？”
<br/>　　然而，
<br/>　　还没等云千秋彻底放松下来，接下来的一幕，顿时让他脸上的笑容，瞬间凝固。
<br/>　　“逆生百转，第五转！”
<br/>　　叶尘在暴怒之下，竟然不顾后果，再次强行提升实力！
<br/>　　轰！
<br/>　　施展出第五转之后，叶尘的功力，比正常状态下，足足提升了十几倍！
<br/>　　嘭嘭嘭！
<br/>　　叶尘的肉身再也承受不住，身上数根经脉，竟然直接破碎！
<br/>　　噗！
<br/>　　强大的反噬之力，让叶尘也狂喷了一口鲜血。
<br/>　　“死！”
<br/>　　处于癫狂状态中的叶尘，仿佛感觉不到痛苦一般，再次找下方冲杀了过去。
<br/>　　轰隆！
<br/>　　千名弟子合力形成的巨掌，和叶尘的拳头再次狠狠相撞！
<br/>　　顿时发出一道璀璨的光芒，继而连那空间，都被生生砸出一片黑洞！
<br/>　　这一次，
<br/>　　叶尘没有倒退半步，
<br/>　　而下方，
<br/>　　那上千名弟子，竟然齐齐狂喷了一口鲜血，瞬间死伤一片，哀嚎声不绝于耳，变成了一盘散沙。
<br/>　　“什么！”
<br/>　　看到这一幕，不但云千秋懵了，就连其他四大宗门的高手们，也个个满脸骇然之色。
<br/>　　这家伙，竟然真的以一人之力，击败了上千名药王宗弟子！
<br/>　　而且还是正面硬碰硬的对抗！
<br/>　　即便是化神天君，也不一定可以做到如此程度吧？
<br/>　　此人的实力，怎么会强大到如此地步！
<br/>　　怪物！
<br/>　　简直就是怪物啊！
<br/>　　叶尘一拳破了药王宗的护山大阵，两眼死死锁定云千秋，再次冲了过去，
<br/>　　“死吧！”
<br/>　　云千秋顿时吓得浑身一个激灵，几乎没有任何犹豫，转身便逃，同时还不忘，向其他四大宗门的高手求救。
<br/>　　“诸位！这小子中了痛苦之源，已经疯了，你们还不出手，是要等他把我们一个个全部杀掉吗？”
<br/>　　其余四大宗门的高手，不由得对视了一眼，但是终究没敢轻举妄动。
<br/>　　云千秋顿时急了，眼看后方叶尘已经追了上来，再次大吼道：
<br/>　　“这小子施展某种秘术，不可能一直这么强，我们只要联手拖住他，必定可以将其击败，然后用搜魂之术，夺取他的秘术功法，大家人人有份，如何？”
<br/>　　云千秋这话一出，众人顿时全都两眼一亮，眼眸中纷纷露出一抹炙热之色。
<br/>　　不得不说，云千秋的这个提议，实在是有诱惑力。
<br/>　　就算是傻子也能看出来，眼前这个世俗界的少年，无论是修炼的功法，还是所使用的兵器，以及如今所施展的秘术，都是他们生平仅见，如果说不动心，那是绝对不可能的。
<br/>　　“诸位，云老前辈言之有理，大家一起动手！”
<br/>　　与药王宗向来交好的天雷宗宗主上官云阙。率先站了出去。
<br/>　　一旦有人带了头，其他人众人自然也不甘其后，顿时也纷纷出手。
<br/>　　四大宗门，足足三十多名元婴境修士，其中还不乏一些元婴境巅峰，完全可以在天榜跻身前十的存在！
<br/>　　下方的祝明飞三人，看到这一幕，才刚刚从之前的喜悦中回过神来，顿时全都脸色大变，
<br/>　　“卑鄙！”
<br/>　　“堂堂五大宗门，竟然联手对付一名少年，真是把我们蓬莱仙界的脸，全都给丢尽了！”
<br/>　　祝明飞忍不住痛骂了一声，转而又望向一旁的苏蔓，道：
<br/>　　“苏蔓姑娘，我们先带你们离开此地，然后再回来助叶兄弟一臂之力！”
<br/>　　说着，祝明飞便要拉着苏蔓离开。
<br/>　　然而没想到，苏蔓却坚定的摇了摇头，
<br/>　　“不行！我不能丢下小尘不管，而且我相信，小尘一定不会有事的！”
<br/>　　祝明飞顿时眉头一皱，长叹了一口气道：
<br/>　　“苏蔓姑娘，我希望你要理智一些，这些人可都是蓬莱仙界顶级强者，天榜上一大半的人都在这里！”
<br/>　　“即便是化神天君，面对如此阵容，也要退避三舍！”
<br/>　　“更何况，叶兄弟如今已经有些神志不清，怎么可能会是这些人的对手？”
<br/>　　“我们还是趁着没有人注意我们，先离开这里再说！”
<br/>　　然而，
<br/>　　面对祝明飞的劝说，苏蔓却丝毫不为所动。
<br/>　　“要走你们走！反正我是不会走的！”
<br/>　　“唉！”
<br/>　　祝明飞见苏蔓如此固执，不由得再次叹了一口气。
<br/>　　而此时，九天之上，
<br/>　　三十多名元婴境高手，已经开始纷纷施展出各种的神通和法宝，将叶尘围困在了中央。
<br/>　　纵然叶尘如今施展秘术，提升了十几倍的功力，可是整个人处于癫狂状态，在五大宗门众强者的围攻游斗之下，竟然也无法占据上风……
