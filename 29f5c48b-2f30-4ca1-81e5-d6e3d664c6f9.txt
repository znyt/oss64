　　虽然他们都是法师，也都是刚从法术公会出来，入世不深，对金钱没有什么偏执的爱好，但他们也是人，对世俗的世界也有一些需求，此刻见到这么多黄澄澄的金子，怎么能够不心动！
<br/>　　妙心的反应还好，想了一下说道：“这些金子，显然没有被用上。上面还有着驱除鞑虏的话，显然针对的不是满清，而是元朝当局，现在清朝都完了。”
<br/>　　叶少阳明白她的意思，接过话头说道：“我现在总算知道为什么石门上有锁孔了，这是修墓的人留下的，专门为了等他的后人进入，来拿这些金子。”
<br/>　　众人恍然。
<br/>　　“你是说，钥匙掌握在修墓的人后代的手中。”毛小方道，“那为什么没有来拿走这些财宝？”
<br/>　　“这就不知道了。可能中间出了什么意外，反正修墓人的计划，并没有成功。”
<br/>　　“倒是坑苦了我们。还死了一个同伴。”卢晓清叹了口气，看了一眼那些装着金元宝的坛子，道：“还好有这些金元宝，对我们来说，也算是一个补偿了。”
<br/>　　大伙都没有异议。这些金元宝的主人，死了都几百年了，财宝而今是无主之物，也算是取之有道，再说就算他们不拿，总有一天这古墓也会被被人发现，到时候这些财宝也都是人家的。这个时代还没有什么上交郭嘉的说法，无主之物，谁发现就是谁的。
<br/>　　叶少阳是最无所谓的，他不是这个时代的人，早晚是要回去的，之前赚的那几根金条已经足够花了，要太多钱也没用，但也不反对别人分钱。
<br/>　　妙心说道：“你们想要分钱我不反对，不过现在可是不方便拿，等我们办完正事再来这里拿走不迟。”
<br/>　　反正金子放在这里也不会飞走，大伙也没意见，于是打算先离去，继续下墓，走之前，在这间密室里再观察了一遍，这时候梧桐意外地发现，房间最里侧的一个夹角，岩石有些异样，伸手掰了一下，居然把石头掰下来，里面是空心的，惊讶之下，立刻招呼大伙过去。
<br/>　　把鲛油灯伸进去，黑洞洞的，竟然是一个更大的空间。
<br/>　　密室？
<br/>　　大伙愣了一会，尝试去掰附近的岩石，这才发现，墙角上下的岩石都是交叉着堆上去的，乍看上去，跟别的地方一样，因为是交叉码上去的，虽然时间久了有些松动，但如果没有外力触碰的话，自己是不会倒下来的。因此之前大伙并没有发现这里有异样。
<br/>　　把所有石头扒开之后，暗道也显示了出来，叶少阳举着鲛油灯，第一个走了进去。
<br/>　　在鲛油灯的照明之下，可以看到这是一间比外面稍微小一点的房间，里面的摆设……简直让人叹为观止：
<br/>　　靠墙摆放着两排货架，上面全是各种小玩意，一眼扫去，有珊瑚树、琉璃塔、各种造型别致的古玩玉器之类，灯光照上去，反射出五颜六色的暗光，简直亮瞎人的眼。
<br/>　　卧槽……
<br/>　　叶少阳暗自感慨了一下，走了过去，用鲛油灯一件件照过去，但无论是色泽还是造型，看上去都美轮美奂，对这些古玩玉器他虽然不懂，但也看得出这些都是珍品。
<br/>　　边上，毛小方拿起了一个玉笋般的笔筒，在手里把玩起来，说道：“这个应该是象牙的，看这工艺，至少价值几百两黄金。”
<br/>　　在另外一个架子上，摆放的全是玉印坯子，各种造型、各种材质的都有，摆满了整整一架子。大伙看得真个是目瞪口呆。
<br/>　　“这才是真正的藏宝室！”卢晓清有些激动地说道，“这里每一件都是珍品，相比之下，外面那十坛金子真的不算什么。”
<br/>　　没有人怀疑他的话。古玩玉器这些，只要是珍品，本来就是无价之宝，更何况这里还有这么多……
<br/>　　“那为什么要弄个密室出来，不把这些东西摆在一起呢？”叶少阳蓦然想起这个问题，提了出来。
<br/>　　大伙面面相觑，谁也不明白墓主人为什么要多此一举。
<br/>　　“你们看这里！”梧桐站在两个货架的中间，招呼大家过去。大伙走过去一看，在中间那一块墙壁上，被人掏出了一个石槽，石槽不算太平整，开口有点向下，但里面是平的，一开始大伙不明白为什么这样，用鲛油灯照进去，里面是一卷卷的东西。
<br/>　　随便抽了一个出来，是一个长条状的东西，外面是防水的油纸，打开之后，里面赫然竟是一幅画卷。
<br/>　　“这……这是步辇图？”妙心看到落款，声音有些颤抖。“没错，这是阎立本的《步辇图》，传说中失传的一幅名画，我见过临摹的，如果这个是真迹，那可真是价值连城！”
<br/>　　地师家族，一向都是跟古墓之类的打交道，古墓之中，又多有古董字画之类的藏品，因此妙心在这方面有些造诣，并不奇怪。至于其余人，也就是跟叶少阳差不多，一门心事都在修行方面，对这些不大懂，也就是跟着看个热闹了。
<br/>　　妙心又打开了第二个油纸包，里面是一幅字帖，妙心快速地扫了一眼，吸着气道：“这个更了不起，王羲之的《蜀都帖》，如果是真的……”她已经想不到词来形容了。
<br/>　　书法方面，叶少阳也听懂的，顿时有些兴趣，凑上去看了一会，上面的字，的确是神韵无限，看上去不像是临摹的。
<br/>　　油纸包一共有十来个，大伙逐一取出来，打开之后，都是字画，大部分妙心也没见过，但看署名都是历史上的大家手笔。
<br/>　　“如果那些古玩玉器的价值，还不足以换到十坛金子，这里的字画，绝对是这些金子的十倍价值！”
<br/>　　“我知道这个槽口为什么是向下的了。”叶少阳道，“因为岩石会渗水，如果不朝下开的话，时间一长可能会有水流进去，就算有油纸包着，也未必就不会被水泡。”
<br/>　　梧桐朝石槽里看去，发现在最深处还有一个油纸包，跟外面那些竖着排列的不同，这一个油纸包是横着摆放在里面的。
