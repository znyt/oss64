　　楚风眠也知道这事不能急于一时，以古家家主的人脉，他去寻找，肯定是比起楚风眠要快得多。
<br/>　　“锦儿，楚公子这几日就住在我王家之中，你好好招待的，不能怠慢了楚公子。”
<br/>　　古家家主向着虚空之中开口道，不一会那古锦，便是凝聚虚空隧道，直接踏了进来。
<br/>　　“是，楚前辈请随我来。”
<br/>　　楚风眠点了点头，便是脚下一动，跟随古锦，直接离开。
<br/>　　离开了这虚空隧道，两人便是来到了真正的古家之中。
<br/>　　这古家，占据了宁州古城一大块区域，其中族人极多，有着数万人，奴仆足足有着百万，不过这数万人中，只有寥寥无几，才是真正古家嫡系之人。
<br/>　　这古家二小姐古锦，便是其中嫡系家族的成员之一，在古家之中地位超然。
<br/>　　在古家中，宫廷林立，足足有着千万座宫殿，位于其中。
<br/>　　两人来到古家之中，一边行走，一边谈论着。
<br/>　　“不知楚前辈收获如何？”
<br/>　　古锦开口道。
<br/>　　她如今自然也清楚，楚风眠到底是为何而来。
<br/>　　他们古家虽然不是商会，但是却也经常坐着一些生意，有着自己的宝库，楚风眠跟古家家主交谈，正是为了生意。
<br/>　　“收获不小。”
<br/>　　楚风眠笑了笑开口道。
<br/>　　这次救了古锦，倒是还算运气不错，不然楚风眠来到这宁州古城，也不好寻找门路。
<br/>　　“对了，古小姐，最近中域是发生了什么事情了？关于九魔城，什么魔头叛乱？”
<br/>　　楚风眠突然开口道。
<br/>　　他突然想起来，这之前那三位监察使所说的话。
<br/>　　九魔城，现在是七大宗门之一，虽然跟其他六个宗门相比，九魔城还算是另立独行，不过也不至于闹得生死仇敌的份。
<br/>　　而且那血剑魔尊，不好好在九魔城所在的魔域待着，怎么会跑到了中域来了？
<br/>　　这其中肯定是发生了什么变故才对。
<br/>　　“楚前辈不知道？”
<br/>　　古锦表情微微有些惊讶，这件事在中域之中弄的人心惶惶，人尽皆知，楚风眠居然不清楚，不过一想到楚风眠是从北域而来，古锦心中倒是理解了不少，缓缓开口道。
<br/>　　“就在半个月前，九魔城之中，发生了一次动乱，其中接近半数的九魔城弟子，突然背叛开来，这些九魔城弟子，再也不顾七大宗门的条约，开始屠杀武者，修炼魔功。”
<br/>　　“九域之中的大部分地区，都已经是受到了影响，我们中域也不例外，这里是中域边境，所以经常会有一些九魔城弟子，在这附近袭击武者。”
<br/>　　“那血剑魔尊，便是其中之一，时常躲在这宁州古城之外，他不敢进入宁州古城来，却是等待武者离开，一旦有武者离开，他便是出手斩杀，夺取财富，难缠无比。”
<br/>　　“这些九魔城叛乱的弟子，大部分都是来到了中域之中，可以说现在整个中域，都没有安宁之日。”
<br/>　　说道这里，古锦的脸色都有些担心起来。
<br/>　　九魔城弟子，一向都是凶残无比，不过之前有着七大宗门之间的制约，所以九魔城弟子也少有外出的，也都还克制的。
<br/>　　可是这一次大批九魔城弟子叛乱，那些叛乱的九魔城弟子，便是不顾一切，大肆屠杀武者，掀起了一阵血雨腥风。
<br/>　　尤其是九魔城的势力，更是在七大宗门之中，都足以排列第一，就算是半数弟子叛乱，都是极为可怕的一件事。
<br/>　　“九魔城居然内乱了？”
<br/>　　楚风眠脸色微变，这倒是他没有想到的，九魔城的弟子，修炼魔功，一个个性格都极为怪异，所以内部一直都是一片混乱。
<br/>　　只是像是如此之大的叛乱情况，楚风眠倒是从未听到过的，不知道其中到底是发生了什么，才会发生这种叛乱。
<br/>　　这叛乱，对楚风眠来说，倒是没什么坏处，如今九魔城弟子叛乱，足以将七大宗门的压力都给牵制过去，对楚风眠来说倒是可以轻松一些了。
<br/>　　毕竟跟九魔城叛乱相比，一个楚风眠，倒是不算什么了。
<br/>　　不过楚风眠的心中，倒是还有些不安，九魔城，在九域之中一直都是一个不安定因素，他们这次叛乱，也许是足以将整个九域震动，倒时候楚风眠也会受到波及。
<br/>　　“实力！只有足够的实力，才足以在乱世之中，存活下去！”
<br/>　　楚风眠目光闪烁。
<br/>　　想要在这九域之中立足，最为需要的，便是实力，没有实力，便是没有任何的资本。
<br/>　　楚风眠必须要尽快提升他的实力才行。
<br/>　　“走吧。”
<br/>　　九魔城叛乱的消息，对现在楚风眠来说，倒是没有太多影响，反正楚风眠也是孤家寡人一个，没有什么需要牵挂的。
<br/>　　一般的九魔城弟子，越不可能是楚风眠的对手，他也不需要太过担心，只是需要注意一些。
<br/>　　“嗯，我古家之中好像是出事了。”
<br/>　　两人一边走着，突然古锦身形停了下来，开口道。
<br/>　　“楚前辈，我需要先去处理一些事情，还请楚前稍等片刻。”
<br/>　　“哦？无妨，我跟你一起前去吧。”
<br/>　　楚风眠随意开口道。
<br/>　　他现在也没什么事做，不如去凑凑热闹。
<br/>　　“那好。”
<br/>　　古锦脸色似乎是有些焦急，也没有再说什么，直接身躯一动，化为一道遁光，从这古家中穿行起来。
<br/>　　几个呼吸间，他们二人便是来到了一处宽阔的广场之上。
<br/>　　在这广场周围，围着不少人，都是古家的弟子，或者奴仆。
<br/>　　在这广场中心，一名男子，正躺在地上，他的胸口，都被硬生生的撕开，伤势极为严重，若非是这男子的境界已经是达到了生死境，不然的话，这样的伤势，已经是足以将他致命的。
<br/>　　不过就算是这样，这男子现在也是虚弱无比。
<br/>　　楚风眠一眼望过去，便是看出了这男子的蹊跷，按理说一名生死境的武者，碰到这样的伤势，也应该很快便是足以恢复过来。
<br/>　　可是这男子身上的伤势，却是似乎被一种奇特的灵力所伤，令他迟迟无法恢复伤口，以至于到现在，越来越虚弱。
