　　妙心突然从背包里拿出一根看上去像是雪茄烟的东西，是一根粗香，点燃之后，立刻冒出青色的浓烟，在妙心作法之下，青烟反方向朝着楼梯下方飘去。
<br/>　　妙心站着不动，从叶少阳手中取过三色莲花灯，自己对着楼梯上方照亮，等了不一会，只见一股青色的烟顺着楼梯上方飘了下来。
<br/>　　人在楼下，烟是朝楼下飞的，结果居然从头顶上方飘下来，虽然这是破解悬魂梯的不二法门，不过亲身经历这个过程，内心还是有些不能接受这逻辑……
<br/>　　妙心让叶少阳捧着香，自己走到烟气交结的地方，蹲在地上，从背包里摸出一只石笔，在地上画了个奇怪的好像五角星一样的符号，然后在最中间，竖下了一根拉蜡烛，点燃之后，让大伙跟她一起往反方向走。
<br/>　　转过了几个弯，再一次回到了远处，视野之中出现了那支插在地上的蜡烛。
<br/>　　妙心微微一笑，走到跟前，双手叠起来，念了一遍咒语，然后睁开眼，用力吹灭了蜡烛。
<br/>　　在蜡烛被吹灭的一瞬间，大伙都感到了身边产生了一种奇异的变化，好像是空气中产生了某种看不见的震动，然后……一切恢复如常。
<br/>　　“没事了？”叶少阳问，环顾左右，跟之前看上去没什么区别。
<br/>　　“应该是没事了。”妙心说道，招呼大家往下走，还是跟之前一样的转弯。大伙不免有些提心吊胆，结果转到到第四个弯的时候，前方出现了一条长长的甬道，大伙这才长出了一口气，总算是走出了悬魂梯。
<br/>　　“等一下！”叶少阳叫住打算走上甬道的几个人，指了指自己手中的三色莲花灯，三色莲花灯现在成了真正的三色：红，绿，黑。而且风明明是从甬道另一头吹过来的，但是三色莲花灯的灯火，却是往甬道的方向倒去，似乎那里有什么东西在吸引着火苗。
<br/>　　“有鬼，有妖，还有僵尸……”在场的都是法师，看到三色莲花灯呈现出这个样子，顿时就明白了。毛小方深吸了一口气，喃喃道：“这古墓里有僵尸是正常的，有鬼也可以理解，怎么还有妖来着？”
<br/>　　谁也回答不上来这个问题。
<br/>　　“前面应该有危险了，大家小心一点！僵尸什么的还好对付，万一有针对盗墓贼的机关，我们也是肉身凡胎，可别着了道。”
<br/>　　妙心让大伙排成一字长队，这样可以尽可能地少跟地面接触，以免触发某种机关，然后自己走在最前面，叶少阳跟在她伸手，为她掌灯照亮，然后步步为营地往甬道里走去，每一步都注意着脚下。
<br/>　　甬道很长，上下左右都是用墓砖砌成的，以拱形呈现出来，四周没有任何装饰，砖石看上去也很牢固，走在甬道里，叶少阳想到这甬道是一砖一瓦盖出来的，心中有些感概，不要说别的地方，单说这么长的一条墓道，和之前走过的楼梯，就不知道要多少人用多少时间才能修得成。
<br/>　　而这，从目前来看，这最多只是古墓的一小部分……真正的威胁，还在古墓的深处。
<br/>　　甬道走到头，两边各自出现了一条路，呈现出一个“丁”字的形状。
<br/>　　在路口对面的墙壁上，有一尊雕像，是一尊佛，坐在莲台上，一只手拈花，另一只手竖在胸前，头上戴着三藏法冠，神色静穆，叶少阳特意观察了嘴角，发现这尊佛脸上没带笑。
<br/>　　梧桐走到佛像前，躬身拜了一拜。
<br/>　　叶少阳问道：“这是什么佛？”
<br/>　　“这是文殊菩萨，不过是藏传佛教的形象，跟汉传佛教的菩萨形象不同，我在一些典籍里见过藏传佛教的佛像，很多记得不清楚，但是几位菩萨还是能认出来的。”
<br/>　　妙心听她这么说，道：“那果然是跟苯教有关了，苯教为西域****，在佛教传入之后，吸收了很多佛教里的东西，也供奉佛门的主神。”
<br/>　　梧桐道：“这个可以说得通，只是，我还是想不明白，你不是说这是赵禥子孙的墓吗，为什么会供奉苯教的神？”
<br/>　　“我也不知道，看来只有先下墓再说了。”
<br/>　　叶少阳用三色莲花灯分别照了照两边，都是一样的通道，于是问妙心应该走哪一边。
<br/>　　妙心感受了一下，风是从左边那条道上吹出来的，于是提议先走左边。跟之前一样，她让叶少阳拿着三色莲花灯，然后盯着脚下，小心地迈开步子，突然发现地板上有些印记，低头看去，地上居然有很多刻痕，在青石板上蜿蜒盘旋……
<br/>　　妙心低头看了一会，感觉有些不对劲，于是招呼叶少阳从包里拿出几根蜡烛，点燃之后，摆在通道两边，然后人往后退了十几步，对着烛光打量过去，大致看清了地上的刻痕，是一副图案，模样有点像是蜘蛛，但是抽象的很，线条繁复但是不乱，大伙一起观察了半天，还是看不出这究竟是什么东西……
<br/>　　妙心猜测，这可能是苯教的某种图腾，至于作用是什么，一时间也猜不透，大伙只好暂时放弃，继续往甬道的一头走，没走多远，借着三色莲花灯的烛光，能够看到墓道的两边，有一些拱形的门洞，一个一个的挨着，中间隔着十来米的距离。
<br/>　　“看着怎么跟防空洞似的。”叶少阳脱口而出，想到了防空洞里的那些壁洞。
<br/>　　“什么是防空洞？”妙心问了一句。
<br/>　　“就是防止飞机轰炸的地方……”
<br/>　　看着妙心和别人离奇的目光，叶少阳猛然想起，这个年代还没有防空洞，只好讪讪一笑，“我随口一说，你们当没听见就好。”
<br/>　　这个年代还没有飞机轰炸，自然不可能有防空洞，而且这里是古墓，这些一个一个的墓室里，不知道都有什么，叶少阳提着三色莲花灯，对着一个墓室先走了进去。
<br/>　　墓室很小，只有一般房屋的卫生间大小，里面只有一个东西：棺材。
<br/>　　石头打造的棺材，大头朝内，小头朝着外面，正好镶嵌在墓室里，四周已经没有什么空间。
