全网都陷入了一种沉默，整座海城都因为夏夕绾而安静了下来。
<br/>
<br/>        厉老夫人又被送入了重症监护病房，夏夕绾走出了医护大楼，来到了外面，研究院所有的人都出来了，站在不近不远的距离看着她。
<br/>
<br/>        这时周萍追了出来，她脸色变幻未定的看着夏夕绾，“夏夕绾，我没有想到你竟然身怀如此惊艳的医术，我现在才算明白为什么院长会连夜打电话给我，让你空降研究院。”
<br/>
<br/>        说着周萍又道，“夏夕绾，我要跟你郑重的道歉，说实话，开始的时候我就对你有偏见，所以你跟夏妍妍争执的时候，我错误的相信了她，原本夏妍妍的一切荣誉都该是你的，今天院长就回来了，我会跟院长坦白一切，引咎辞职，同时我也希望你可以回来研究院，研究院需要你这样出色的医学人才。”
<br/>
<br/>        周萍对工作相当的负责，也敢作敢当，她现在道歉并且承担一切后果，但她恳请夏夕绾可以回来。
<br/>
<br/>        是她将夏夕绾赶出去的，现在她想把夏夕绾请回来。
<br/>
<br/>        夏夕绾澄亮的翦瞳看着周萍，勾唇一笑，“周主任，我并不怪你，但是关于我回研究院的事情，我还要考虑一下。”
<br/>
<br/>        说完，夏夕绾转身离开了。
<br/>
<br/>        她并没有立刻答应回研究院。
<br/>
<br/>        周萍还有所有人都看着夏夕绾离开的俏影，现在夕阳的余晖洒满了大地，女孩脚步轻盈的走入了那金黄的光辉里，清风拂动着她身上的裙摆和长发，这么一刻她显出几分惑人的瑰丽。
<br/>
<br/>        夏妍妍也来了，她垂在身侧的两只手狠狠的拽成了拳，她知道从这一刻起，夏夕绾已经走到了璀璨的灯光下，成为了焦点。
<br/>
<br/>        这时周萍转身走了过来，她看着夏妍妍的目光十分的严肃，“夏妍妍，自从你进入研究院，我一直对你抱以厚望，但是你忘记了，医术第二，人品第一，这一次你恶意抢夺救人的荣耀，错误引导舆论风向，差点害死了厉老夫人，我要对你进行全院通报批评，并且从此刻起将你停职，等院长回来亲自对你进行处理！”
<br/>
<br/>        夏妍妍的脑袋“轰”一声，她知道她完了，这一次是真的完了，她挪动红唇想要求情，“周主任，我…”
<br/>
<br/>        周萍不想听她再多说一个字，直接离开了。
<br/>
<br/>        周萍走了，夏妍妍僵在这里，她感觉大家的目光都聚集了过来，对着她指指点点，鄙夷，同情，厌恶，看好戏…什么都有。
<br/>
<br/>        这时赵立英匆匆的跑了过来，“妍妍，究竟发生了什么事情，救老夫人的真是夏夕绾？”
<br/>
<br/>        赵立英到现在都不相信这件事是真的，他最瞧不起最讨厌的夏夕绾竟然是真正的天才少女，所以他跑过来亲口问夏妍妍。
<br/>
<br/>        “妍妍，你自己去看看微博吧，现在微博都炸开了，海城各大周刊都在报导你假天才少女的事情，刚才研究院的通报批评也第一时间公布了，那个众筹为你偿还十二个亿的账号当即关闭了，很多家主流媒体纷纷下场点名批评你，更直言你在医术造假，欺骗大众的背后更是人格的缺陷，言辞相当犀利。”
