　　枯朽天，沧澜霸，战魔尊者的一身血气，如今全部被楚风眠吞噬。
<br/>　　他们三人身上，可不止是有着血气，更是有着从颜古龙帝洞府中得到的无数宝物。
<br/>　　这些宝物，他们虽然吞噬，却都没有来得及完全炼化。
<br/>　　如今其中的灵力，便是全部便宜了楚风眠。
<br/>　　“统统炼化！”
<br/>　　楚风眠催动战龙决的力量，将其中的无数灵力，全部炼化，凝聚成为远古战龙精血。
<br/>　　楚风眠所修炼的战龙决，本身就算是一个大胃王，但是一旦是有着足够的灵力支撑，在遭遇瓶颈之前，都可以迅速突破。
<br/>　　这三人的血气，如今被楚风眠吞噬炼化，令楚风眠的实力，顿时再进一步。
<br/>　　不过却也还远远达不到令楚风眠境界再进一步的地步。
<br/>　　“想要令境界再进一步，至少也要在吞噬十个像是落长恨这样武者才行。”
<br/>　　楚风眠心中暗叹一声，他的战龙决，没有足够的奇遇，想要支撑修炼起来，也是困难无比的一件事。
<br/>　　“境界突破极难，不过眼下，却还是有着另外一样可以提升我实力之物。”
<br/>　　楚风眠扫视周围一圈，他的视线，顿时落在了那祖龙至尊剑上。
<br/>　　突然，楚风眠身躯一动，猛然之间，便是冲着那祖龙至尊剑飞了过去。
<br/>　　“他要做什么？”
<br/>　　“莫非他是打算要收取这把天级灵器？”
<br/>　　一些北域武者，看着楚风眠连连发威，已经的惊骇无比，看这楚风眠接下来的举动，一个个皆是连连后退，不敢上前。
<br/>　　这祖龙至尊剑上的禁制，可以远远要超越这密室中所有宝物的禁制，一旦是触碰，很有可能遭到反噬。
<br/>　　楚风眠身形一动，瞬间来到这祖龙至尊剑的面前，他毫不犹豫，顿时将战龙决的力量催动到了极致。
<br/>　　顿时楚风眠的右臂，便是化为了一条龙爪的样子，猛然之间，对准那祖龙至尊剑抓了过去。
<br/>　　“给我收取！我为战龙！这祖龙至尊剑正好契合我的血脉！”
<br/>　　楚风眠长啸一声，将身上的战龙气息发挥到了极致。
<br/>　　祖龙至尊剑上的禁制，在感觉到了这战龙气息后，缓缓的全部消散开来，令楚风眠一把就将这祖龙至尊剑拿到了其中。
<br/>　　“好！”
<br/>　　一把收取这祖龙至尊剑，握在手中，楚风眠才能够真正感觉得到这祖龙至尊剑的力量，到底是有着多么强大。
<br/>　　一把天级灵器的力量，比起地级极品灵器要强大数十倍，更何况这祖龙至尊剑，更是龙族的至宝。
<br/>　　在天级灵器之中，都称得上是极品的存在。
<br/>　　手握祖龙至尊剑，楚风眠感觉得到，他身躯之中远古战龙血脉的力量，都被完全的激发了出来，两股力量互相交融，令楚风眠身上的力量气息，都在膨胀不止。
<br/>　　“好，这祖龙至尊剑果然是龙族至宝，上面的力量足以是跟我的血脉完全契合，寻常武者得到这祖龙至尊剑，也只能发挥出十分之一的力量，但是我拥有战龙血脉，却是足以将其中的所有力量都给发挥出来！”
<br/>　　楚风眠心中大喜道，这祖龙至尊剑，绝对是最为适合楚风眠的灵器。
<br/>　　“不好！此人将那天级灵器收取了！”
<br/>　　“好强大的力量！天级灵器！若是我可以得到，足以让我的实力增进一步都要多！”
<br/>　　一些北域武者，看着楚风眠手拿祖龙至尊剑，简直是一个个眼神中嫉妒无比。
<br/>　　“杀！这天级灵器我们也要得到！杀了他！”
<br/>　　“对！我们只要是能够杀了他，得到了这把天级灵器，出去之后，我们就是无敌的存在！”
<br/>　　“我就不信，此人真的是无敌不成！我们这么多人出手，肯定会杀了他！”
<br/>　　就在楚风眠破开禁制，拿到祖龙至尊剑的一刻，在外边，无数灵力，却是凝聚。
<br/>　　在场的百位北域武者，几乎是同时出手，瞬息之间足足上百道武技，凌空打出，全部冲着楚风眠轰击而来。
<br/>　　顷刻之间，无数灵力已经是融为一体，化为了浩瀚光芒，冲着楚风眠，直接冲刷而来。
<br/>　　在这光芒之中，所蕴含的，足足是有着百位实力达到巅峰圣者的武技！
<br/>　　“不好！”
<br/>　　楚风眠脸色上都微微露出几分紧张，这一股浩瀚光芒的实力，比起落长恨四人联手时的，都要强大。
<br/>　　这些武者的实力，虽然每一尊的实力远不如落长恨等人，但是数量繁多。
<br/>　　蚁多咬死象，在宝藏面前，这些人已经是不顾一切，联手也要杀了楚风眠，来夺取这祖龙至尊剑，爆发出的力量，不可小视。
<br/>　　“找死！”
<br/>　　楚风眠眼神中杀意浓郁到了极致，这些人如今都想要趁乱打劫，之前若不是楚风眠出手，他们就连一点宝藏都得不到。
<br/>　　如今却反倒是过来对付楚风眠。
<br/>　　“看来你们统统都要死！”
<br/>　　楚风眠长啸一声，当机立断，一身灵力已经是涌现爆发到了极致。
<br/>　　这些人的举动，彻底激怒了楚风眠。
<br/>　　“轰隆！”
<br/>　　从楚风眠身上，一股股澎湃灵力猛然爆发而出，巨大的力量，横扫整个密室之中。
<br/>　　“不要怕！此人的实力再强，难道抵得过我们这么多人的联手？”
<br/>　　一尊北域武者抬着头，看向楚风眠大声吼道。
<br/>　　“小子，将你从密室中得到的宝物全部交出来，我们可以放你安然无恙的离开！”
<br/>　　“不然今日，在我们这么多人的联手之下，你想要活着离开，根本没有可能！”
<br/>　　“不错，楚风眠，你实在是太贪心了。”
<br/>　　又一尊北域武者，站了出来，看向楚风眠冷冷的开口道。
<br/>　　“这颜古龙帝洞府中的宝物，不是你一个人可以占据的，将其中的宝物交出来吧。”
<br/>　　“交出宝物？贪心？”
<br/>　　楚风眠听到这几人的话，心中的杀意已经是浓郁到了极致。
<br/>　　“这宝物，都是各自得到的，若不是我，你们谁又可以进入到这密室之中？”
<br/>　　“现在反咬一口，让我交出宝物？”
<br/>　　“我能杀了落长恨他们，能杀了荒古门弟子，你们以为就凭你们这些蝼蚁，合起力量来，足以对抗我？”
