<br/>　　她出自绝情居，修炼的功法比较特殊，绝对不能对男人，有一丝一毫的情愫！
<br/>　　“我……我没事！”
<br/>　　洛冰玄暗暗稳定了心神之后，又恢复了之前的冷漠，
<br/>　　“你为何要救我？”
<br/>　　叶尘知道洛冰玄的脾气，也不以为意，微微一笑，道：
<br/>　　“因为你之前帮过我挡过一枪，所以这一次，算我还你的！”
<br/>　　洛冰玄听到这话，顿时秀眉一蹙，她也不知道是怎么了，一看到叶尘那淡然温和的笑容，就莫名的觉得心跳加速，这种感觉让她很不舒服，
<br/>　　“我之前说过了，我从来没有想过要帮你！所以你不欠我什么！这是我和云破天之间的恩怨，你……你现在可以走了！”
<br/>　　叶尘顿时乐了，
<br/>　　“大姐，你确定要我走？”
<br/>　　虽然他知道洛冰玄的性格，一向喜欢拒人于千里之外，可是却也没有想到，这女人竟然不近人情到这个地步，确实有些出乎他的意料之外。
<br/>　　洛冰玄的美眸之中，顿时闪过一抹犹疑之色，不过最终还是一咬牙，决然的点了点头，
<br/>　　“我确定！”
<br/>　　洛冰玄之所以这么‘不近人情’，自然是有原因的。
<br/>　　她所修炼的功法比较独特，必须要断情断欲才能达到极致。
<br/>　　而眼前这个叫叶尘的家伙，总给她一种说不出来的感觉……
<br/>　　甚至她有一种预感，如果不尽快断绝和此人的关系，今后必定会严重影响她的心境！
<br/>　　所以，明知道当前形势对她十分不利，她还是毅然决然的，要和叶尘划清界限。
<br/>　　叶尘又哪里知道，洛冰玄此时的想法，听到她这话，整个人都懵了，
<br/>　　“冰玄，莫非你是担心的安危，所以才故意这么说的吗？那你尽可以方向，就这几个废物，我分分钟就能把他们解决了！”
<br/>　　“你……”
<br/>　　洛冰玄听到叶尘这话，又是好气，又是好笑，正打算继续恶语相向，逼迫叶尘离开。
<br/>　　不料，另外一边的云破天，听到叶尘的这番话，但是气得肺都要炸开了，两眼红的像兔子一样，声音几乎是从牙缝中挤出来的一般。
<br/>　　“小子！你说谁——是——废——物？”
<br/>　　他堂堂无极宗的千年不世出的天才，百年天骄榜上第三的存在，竟然被人说是废物！
<br/>　　这让云破天如何能忍？
<br/>　　听到云破天的咆哮声，叶尘却连头都没有回，依旧用后背对着他，嘴角微微一扬，冷冷道：
<br/>　　“一个只会用卑鄙手段出手暗算他人，而且还喜欢以多欺少！云破天，你不但是废物，还很下作！！”
<br/>　　轰！
<br/>　　叶尘轻飘飘的一句话，却犹如一把利刃，狠狠刺中了云破天内心深处，最敏感的那根神经！
<br/>　　他作为天之骄子，何曾被人如此羞辱过？
<br/>　　云破天只感觉，一股怒气，几乎瞬间填满了他整个胸腔，继而直冲头顶，连头发也顿时根根倒竖起来，
<br/>　　“你！找死！”
<br/>　　气到几乎要暴走的云破天，当即脚下猛然一蹬，
<br/>　　轰！
<br/>　　原地顿时被云破天踏出了一个深坑！
<br/>　　下一刻，
<br/>　　云破天整个人，犹如离弦之箭一般，朝叶尘爆射而去！
<br/>　　“百杀神拳！！”
<br/>　　轰隆！
<br/>　　这云破天不愧是一代绝世天骄，虽然修为被大幅度压制，如今只有区区筑基境，可是这一拳轰出，威力之大依然让人惊骇，便是寻常金丹境，恐怕也会被直接打爆。

<br/>　　“那小子肯定死定了！”
<br/>　　一旁另外两名无极宗的天才，看到这一幕，顿时纷纷欢呼振奋起来。
<br/>　　然而，另外一边，
<br/>　　叶尘依旧背对着云破天，就仿佛没有看到后方的云破天动手一般。
<br/>　　倒是洛玄冰脸色大变，忍不住惊呼提醒，道：
<br/>　　“小心！快躲开！”
<br/>　　叶尘却依旧不闻不问，冲着洛玄冰微微一笑，调侃道：
<br/>　　“原来玄冰仙子这么在乎我的死活啊？可真是让人感动啊！”
<br/>　　说出这话的同时，云破天也已经冲到叶尘背后三米之内，当他看到叶尘居然不闪不避，用后背对着他，先是微微一愣，继而越发愤怒，
<br/>　　“小子！你竟然敢小瞧我！那就，死去吧！”
<br/>　　云破天将所有真元都灌入自己的拳头之中，狠狠砸落在叶尘的后背之上，满脸的狰狞之色！
<br/>　　他甚至几乎已经看到，下一刻，叶尘被自己一拳打爆的场景。
<br/>　　然而，
<br/>　　就在他的拳头，将要落在叶尘身上的那一刹那，
<br/>　　轰！
<br/>　　叶尘周身上下，竟然升起了一道璀璨的金色光芒！
<br/>　　铿！
<br/>　　云破天这一拳，仿佛砸在了一面铜墙铁壁之上，继而一股强大的反弹之力，从他的手臂上传来，更是将他震得连连倒退！

<br/>　　云破天实在是无法接受眼前的这个事实，对于一向自命不凡的他来说，这个打击实在太大了！
<br/>　　“废物！现在认清楚自己的位置了？那，去死了！”
<br/>　　就在云破天还处于震惊中难以自拔之时，耳边陡然响起了叶尘，那冰冷中充满嘲讽的声音！
