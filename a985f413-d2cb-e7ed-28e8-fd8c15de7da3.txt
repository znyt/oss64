        &lt;h3&gt;第三卷第一百五十四章  冯睿的算计2&lt;/h3&gt;
<br/>
<br/>        当张函在陈子昂家里陪着俩个孩子的时候，冯睿拿着呼延临庆买的东西到北京给蔡少芬和弥青云辞年。
<br/>
<br/>        这几年，张函和陆玉的关系缓和很多，而且最近这两年陆玉又去了甸城，所以俩个老太太的心思也就不再在探听陈子昂的消息上了，不过就是这么多年以来，见了冯睿都成了习惯了。偶尔不听听冯睿叨叨几句似乎总感觉缺点啥。
<br/>
<br/>        蔡少芬还是那样无所谓的讽刺几句冯睿，弥青云都懒得问陈子昂的消息了，她的儿子现在忙着事业，现在的身份地位比她这个当妈的都高很多。
<br/>
<br/>        这么多年以来，他们都知道了陈子昂在北京，而张函也没有找过陈子昂，所以他们都不在乎这件事情了，况且一个陈子昂也翻不起什么浪。
<br/>
<br/>        但是，她们还是很好奇陈子昂有没有结婚，找了一个什么样的人。
<br/>
<br/>        当冯睿拿出来在商场买的东西给她们时，她们还是有些奇怪，为什么冯睿这一次改变套路了。
<br/>
<br/>        不过，女人嘛，无论是年龄大的还是年轻人，都喜欢美丽的物件，冯睿这一次拿的东西虽然没有之前的东西值钱，但是胜在新颖。
<br/>
<br/>        所以俩个人倒是没有多问什么。
<br/>
<br/>        “最近都还好吧？”弥青云淡淡的问冯睿。
<br/>
<br/>        “都好呢，家庭工作都是老样子，一成不变多少年了。”冯睿叹口气，她可不想说她离婚了。
<br/>
<br/>        “都一大把年纪了，你还想改变什么啊？莫非你还想和你家陈子建离婚，重新二春？”蔡少芬开了一个玩笑。
<br/>
<br/>        “哈哈，都多大年纪了啊，我们都快要退休了。”弥青云也被蔡少芬逗笑了。
<br/>
<br/>        “是啊，这么大年龄了，也没什么可折腾的了。”冯睿也附和着，她离婚了，但是她没有想过要二春，如果陈子建愿意向她道歉的话，她愿意和他在一起。
<br/>
<br/>        “你儿子最近怎么样？”弥青云还是不忘关心一下冯睿。
<br/>
<br/>        “还是要谢谢蔡姐姐呢，孩子工作还不错，后期就看他自己发展，哎，我这个儿子啊，还没有我那干女儿让人省心。”冯睿叹了口气。
<br/>
<br/>        “你还有干女儿？”蔡少芬和弥青云好奇起来了。
<br/>
<br/>        “有呢，怎么没有，从上高中、大学到工作都是我帮忙折腾的，现在已经是西城信息部门的老总了。”冯睿自豪的说，仿佛那个干女儿就是她亲生的女儿一样。
<br/>
<br/>        “哦，那是不错哦，现在的信息化很吃香，现在西城的老总是老马吧？”蔡少芬问，他们陆家的人陆陆续续回来了。
<br/>
<br/>        “是的。”
<br/>
<br/>        “那你的这个干女儿还是很能干的嘛。多大年龄了？”弥青云也很好奇。
<br/>
<br/>        “和我们家子昂是同学，人也长得漂亮，很能干。”冯睿自然而然的带出了陈子昂。
<br/>
<br/>        “你们家陈子昂现在还是单身吗？”蔡少芬问，她家陆玉现在也是一个地方的副总的，主管信息化建设。
<br/>
<br/>        “唉，不怕俩个姐姐笑话，我一直以为子昂是单身，你都不知道人家做些啥事情，唉、唉、别提了。”冯睿连声叹气。
<br/>
<br/>        “怎么了？”蔡少芬和弥青云坐直了身体。
<br/>
<br/>        “人家到北京的那一年就和江家大少勾搭上了，未婚先孕，生了一对双胞胎。而且到现在都不结婚。俩个人一直不清不楚的。”冯睿摇摇头，满脸的疼惜。
<br/>
<br/>        “你说什么？怎么可能？”弥青云惊呼。
<br/>
<br/>        “不是，你们家倩楠不是都要和江家大少结婚了吗？”蔡少芬也吃惊了，这个可是爆炸性新闻啊。
<br/>
<br/>        “是啊，本来十一要结婚的，江家说是有大项目要做，推迟了婚期。”弥青云感觉蹊跷。
<br/>
<br/>        “冯睿这样一说，那个推迟婚期可是个借口啊。”蔡少芬分析着。
<br/>
<br/>        “哎呀，我们家倩楠还蒙在鼓里。”弥青云心里很是生气：“这个陈子昂就是一个祸害，怎么就和我们张家就扯不清呢？”
<br/>
<br/>        “俩个孩子多大了？”蔡少芬问。
<br/>
<br/>        “这个我还不清楚，当时生气，也就不想问，不过听说陈子昂到北京的那一年就和江家大少在一起了，这都多少年了。孩子怕也是个小学生了吧。”冯睿不知道陈子昂的孩子多大了。
<br/>
<br/>        弥青云听到小学生三个字，眼前突然就晃过那一次碰见的俩个小孩子，哪两个孩子真是可爱。唉，人家陈子昂的孩子都上小学了，她家张函还没有个音讯。
<br/>
<br/>        弥青云的心情突然就不好了。
<br/>
<br/>        “好了，不说了，我要回去找我们倩楠，将这件事情给她说了，不能让这对渣男渣女骗了。”弥青云狠狠的说。
<br/>
<br/>        “生啥气，你让张函安排人调查一下不就行了，真是事实，那就赶紧让倩楠离开江家大少不就行了。”蔡少芬说。
<br/>
<br/>        “能让张函调查吗？”弥青云不确定的问。
<br/>
<br/>        蔡少芬：“怎么不能，让张函看清楚陈子昂是一个什么样的人不是更好。”。
<br/>
<br/>        “我觉得这么多年了，小函估计都忘记陈子昂了，如果让小函去调查陈子昂怕是不妥。”弥青云担心的说。
<br/>
<br/>        “有什么不妥的，刚好可以知道小函是不是真的忘记陈子昂了，要是没有忘记，陈子昂这样，小函不就看到了陈子昂的嘴脸了？”蔡少芬可是知道张函和她女儿一直没有在一起。
<br/>
<br/>        “唉，我也是没有办法的事情，也是不久前才知道。要不然我就早早的给你说了。”冯睿叹口气。
<br/>
<br/>        “好啦，你不要说啦，以后有什么情况随时说。”弥青云烦躁的摆摆手。
<br/>
<br/>        冯睿听了弥青云的话，心里暗暗开心，这还是需要她给她们提供情况，她就纳闷了，这俩个女人咋还这么关心陈子昂。
<br/>
<br/>        弥青云也没有心思再呆下去了，饭都没有吃完，她们就散了。
<br/>
<br/>        冯睿看着满桌子的菜，重新坐下来，慢条斯理的吃着。
<br/>
<br/>        如今和陈子建离婚了，陈子昂的消息从哪里来？她想了想，陈家兄弟姐妹们，也就是陈子澜好说话一点，听说陈子澜出车祸了，在西城骨科医院住着，估计要住一段时间，她可以借口去探望陈子澜，从侧面打听一下陈子昂的事情。
<br/>
<br/>        另外她还是要和陈子乔把关系搞好，毕竟她在陈家呆了那么多年，还给陈家生了个孩子，没有功劳也有苦劳。
<br/>
<br/>        陈子乔现在在西城说话也是响当当的人，以后有个什么事情也能帮上忙。
<br/>
<br/>        她决定回去就联系陈子乔一起去医院看看陈子澜。
<br/>
<br/>        冯睿慢条斯理的吃着桌子上的菜，想着后续的事情，突然就感觉很心酸，她这样的辛苦也不知道为了什么，她已经快要从位置上退下来了，上头说要让年轻人有锻炼的机会，如今她也没有什么前途了。
<br/>
<br/>        归根结底怕是想找个靠山吧，可是蔡少芬和弥青云能靠得住吗？
