　　轰…轰…轰…
<br/>　　高空中，几乎每时每刻都爆发出惊天轰鸣。
<br/>　　火焰流光的速度，快得惊人。
<br/>　　寻常武者，肉眼难视之，也根本跟不上流光速度。
<br/>　　但那头庞大的蓝色雷龙，如此庞大的身躯下，却极容易看出变化。
<br/>　　明显可以看到，火焰流光的每次冲撞，那庞大身躯之下，必是一阵凹陷。
<br/>　　单从此观之，落于下风的，反倒是这头雷电滋鸣，狂猛异常的庞然大物。
<br/>　　但，别忘了，与萧逸对战的，可不止这头巨大雷龙。
<br/>　　嗖…
<br/>　　龙啸的身影，瞬间动了。
<br/>　　他的拳头，同样狂猛。
<br/>　　他的速度，同样惊人。
<br/>　　他那壮实的身躯，充斥在一片雷光之下。
<br/>　　他的拳头，被一层滋鸣着的雷电包裹。
<br/>　　轰…
<br/>　　龙啸，拦下了疾速轰击着巨大雷龙的萧逸。
<br/>　　两拳对轰，整个高空，一大片范围成了一片宛若虚无的真空地带。
<br/>　　二人交手的瞬间，仅仅是余威，已让得方圆百里之内，一切一切，尽数湮灭，包括空气。
<br/>　　二人拳头的碰撞，更是一瞬间宛若超越了时间的限制，如同定格。
<br/>　　下方，东方铁云暴喝一声，“龙域主、易霄分殿主，是否有何误会？”
<br/>　　“不妨暂时停手，说个清楚。”
<br/>　　“没什么好说的。”龙啸咬牙切齿，“易霄此等凶徒，本域主今日必杀之。”
<br/>　　萧逸同样眼眸冰冷，“龙啸，看来是易某在你六龙地域内，太过客气，太过好脾气了。”
<br/>　　“你那等威胁，在我眼中，不过是个笑话。”
<br/>　　“你找死。”龙啸拳头握得更紧，拳上雷电，滋鸣愈响。
<br/>　　萧逸的拳头，瞬间被震开。
<br/>　　萧逸的身影，亦被震飞百米之外。
<br/>　　嗖…萧逸身影一动，再次出手。
<br/>　　嗖…龙啸，同样狂猛而出。
<br/>　　轰…轰…轰…
<br/>　　这下子，成了二人在高空中的鏖战；其宛若山崩地裂般的恐怖气势，更甚当日萧逸与毒袍鏖战之时。
<br/>　　一分钟后。
<br/>　　“嗯？”萧逸不禁皱了皱眉。
<br/>　　他发现，体内的元力，正以极快的速度流逝着。
<br/>　　而同时，他身上的灼热感，也愈发惊人，渐渐让他产生了一丝难受感。
<br/>　　“是龙炎。”萧逸心头暗道一声。
<br/>　　龙炎，绝非人力所能完全驾驭的。
<br/>　　他凭着九阳轮回诀，使用一段时间也就罢了，时间长了，他的肉体，根本吃不消。
<br/>　　再者，与他战斗着的，可不仅是龙啸，还有那头巨大的雷龙。
<br/>　　简单来说，龙啸身后，有着这条惊人的雷煞作为后盾。
<br/>　　有此雷煞踏于脚下，龙啸几乎立于不败之地。
<br/>　　观这条雷煞的庞大度，其孕育时间，起码在数十万年以上，蕴含的雷电力量，相当惊人可怕。
<br/>　　也就是说，这条雷煞，足够给予龙啸耗之不尽的力量，一直战斗下去。
<br/>　　即便萧逸的身体能完全承受龙炎，怕也会被龙啸活活耗死。
<br/>　　难怪龙啸在六龙地域之内，会等同无敌。
<br/>　　凭此雷煞，借地域之力，谁能是他的对手？
<br/>　　“得速战速决。”萧逸心头瞬间有了想法。
<br/>　　轰…轰…
<br/>　　萧逸全力与龙啸对轰了两拳，随后身影借力而退。
<br/>　　身影刚退，已双手齐出，道道火焰游龙凭空而现。
<br/>　　不过片刻，已是数百条六色火焰游龙翱翔天际。
<br/>　　“噬。”萧逸暴喝一声。
<br/>　　数百条六色游龙，狂猛朝龙啸吞噬而去。
<br/>　　“嗯？”龙啸身体一稳，脸色怡然不惧，一拳轰出。
<br/>　　雷电力量的增幅下，龙啸的拳头，拳拳惊人。
<br/>　　不过，当他的拳头与一条火焰游龙接触时，却瞬间被震退数十步。
<br/>　　“怎么可能，好狂猛的火焰游龙。”龙啸吃了一惊。
<br/>　　事实上，单凭六色火焰游龙，根本奈何不了龙啸。
<br/>　　不过，若是六色游龙之内，加入了一丝龙炎呢？
<br/>　　龙炎的狂猛，岂是龙啸能挡的。
<br/>　　“哼。”龙啸一拳吃亏，冷哼了一身，身影一闪，瞬间躲过了其它袭来的游龙。
<br/>　　萧逸眼眸一冷，双手齐出。
<br/>　　兽手之下，他的控火能力，强悍到极点。
<br/>　　即便是数百头威力惊人的火焰游龙，仍旧在他手中如臂指使，操控完美。
<br/>　　数百条游龙，几乎瞬间封锁了龙啸的所有退路。
<br/>　　一条条宛若蕴含灵智的游龙，霸绝惊人的眸子，齐齐凝视着龙啸。
<br/>　　有那么一瞬间，龙啸不自觉地打了个冷颤。
<br/>　　在他眼中，那似乎是真正的炎龙，真正的炎龙大陆第一兽。
<br/>　　“哼，装神弄鬼罢了。”龙啸瞬间稳住了心神，同样双手齐出。
<br/>　　龙啸脚踏于巨大雷龙之上，脚间，源源不断地雷电力量涌入自身。
<br/>　　吼…吼…吼…
<br/>　　同样是顷刻间，数百条小型雷电游龙，狂猛而出。
<br/>　　火焰游龙、雷电游龙，在这一瞬间，剧烈交锋。
<br/>　　萧逸的控火能力，自是惊人，数百条火焰游龙，操控得宛若完美。
<br/>　　而龙啸的龙家功法，本就是操控地域之煞，也就是这条雷煞，自然的，同样操控得完美惊人。
<br/>　　偌大个高空，瞬间成了游龙狂猛，交缠噬咬之地。
<br/>　　高空百里，化作了一片火焰与雷电的肆虐之地。
<br/>　　“好强。”下方，东方铁云震惊地看着高空中千龙齐舞的景象。
<br/>　　那火焰与雷电的肆虐交锋，哪怕是丝毫威能，都足以让他心惊胆战。
<br/>　　“龙啸仗着龙家雷煞狂猛也就罢了，易霄分殿主的实力，竟也这般可怕？”
<br/>　　“我猎妖殿，竟出了这般了不起的天骄。”
<br/>　　……
<br/>　　高空中，战斗激烈万分。
<br/>　　但要说谁占上风，却暂时未分出。
<br/>　　萧逸，此刻仍旧奈何不得龙啸。
<br/>　　“不能再拖了。”萧逸双手操控游龙间，咬了咬牙。
<br/>　　若是在六龙地域之外，他有把握胜龙啸。
<br/>　　可在六龙地域内，怕是只有传奇以上，方能奈何这龙啸。
<br/>　　再者，龙炎的反噬，自己也承受不了多久。
<br/>　　嘭…
<br/>　　萧逸忽然大手一挥。
<br/>　　高空中，数百条火焰游龙，瞬间爆炸。
<br/>　　与之交锋的数百头雷龙，亦在爆炸中湮灭殆尽。
<br/>　　“哼，没辙了吗？”龙啸冷笑一声，冷笑中，杀意凛然。
<br/>　　“天火流星。”萧逸猛地暴喝一声。
<br/>　　双手齐出下，无数火焰，从天而降。
<br/>　　那密集的火焰，数以万计，一颗颗，如若流星陨石坠落。
<br/>　　流星之火内，蓦地发出道道怒吼。
<br/>　　看真切些，从天而降的密集火焰流星，竟如一道道龙炎吐息。
