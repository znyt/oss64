　　第529章
<br/>　　鹰嘴岭一共九位当家，山寨三五百小喽啰，在兖州和青州交界的这块地盘，已经算一股不小的势力。
<br/>　　毕竟，不可能人人都如程大雷一般，手中有三万余精兵强将，虽然还顶着一个山贼的名头，但早就是无人不忌惮的一方枭雄。
<br/>　　今夜，鹰嘴岭剩下的八位当家，除老九白面鬼张猛外出不在山寨，其他七位当家都聚在山寨的聚义厅。
<br/>　　“咱们找那姓程的，为大当家报仇！”
<br/>　　“报仇？怎么报仇。咱们这百十口子，还不是去送死。要知道，那位可是姓程……”
<br/>　　“姓程又怎样，姓程的莫非就三头六臂，刀枪不入。我不管你们怎么想，大哥不能这样白死！你们不管，我带人去找老大报仇。”
<br/>　　“快去快回，去得早，老大和你的葬礼一块办。”
<br/>　　“你……”
<br/>　　聚义厅上吵得不可开交，无非一方要打，一方要忍。说来说去，却也都是忌惮那个人的可怕。
<br/>　　朱刚摆摆手，压下众人的争执，道：“诸位兄弟，且先静一静，听我说一句。大哥的仇一定要报，咱们不能让大哥白死。”
<br/>　　“三哥这话有礼，我赞同他，咱们兄弟怕过谁。”
<br/>　　“诸位兄弟稍安勿躁，先等我把话说完。”饿死鬼朱刚道：“可那位毕竟姓程，确实也不能小觑。如果明刀明枪报仇，怕是不行。究竟该如何杀了姓程的为大哥报仇，咱们还得好好想想。”
<br/>　　“三弟这话有理，你给大家说说，究竟该怎么办？”二当家长须鬼，名叫孙山起，生得身材修长，长须垂胸，倒是个俊品人物。
<br/>　　不过，人不可貌相，外表堂堂的孙山起倒是没什么脑子，反而面目狰狞的老三饿死鬼有些急智。
<br/>　　“我们山寨的弟兄都算上，也未必能拿住程大雷。不过这程大雷，已是帝国通缉的要犯。要杀他的可不止一家两家，朝廷要杀他，江湖绿林也要杀他，只要咱们把他出现在兖州的消息放出去……”朱刚说着说着，嘴角挂起一抹冷笑：“他既然敢来兖州，就别想再活着出去。”
<br/>　　孙山起睁大眼睛：“三弟，妙啊！”
<br/>　　饿死鬼朱刚叹了口气，道：“唉，若是九弟在山寨，又何必如此麻烦，凭九弟的武艺，本也不用怕他。”
<br/>　　“对了，九弟去兖州城，什么时候能回来？”
<br/>　　“谁知兖州的事情办得怎样，路上再耽搁些日子，更说不好何时能回来。”
<br/>　　正此时，一喽啰急急忙忙跑进聚义厅，口中道：“几位当家的，九爷……九爷他回来了。”
<br/>　　正此时，一人踱步走入大厅，口中道：“诸位哥哥，可是让我好想。大哥呢，我路上抢了个婆娘，正要送给大哥尝鲜。”
<br/>　　孙山起几人面面相觑，老三朱刚道：“九弟你先坐下，有件事我慢慢和你说。”
<br/>　　张猛困惑的坐下，听罢之后，瞬间气得三尸神暴跳，五灵豪气腾空，口中哇呀呀叫道：“好一个程大雷，实在欺人太甚，我必亲手砍下他的脑袋，去给大哥报仇。”
<br/>　　……
<br/>　　大约天快亮的时候，程大雷和刘发财一行人到了黄土镇。
<br/>　　“爹，爹……”
<br/>　　黄为善刚迈入院子，便呼喝出声。程大雷跟在他们后面，见黄家好气派的院子，一片大庄园，虽然有些土财主的傻气，但也算得上豪奢。
<br/>　　半夜的院子被吵醒，檐下的灯笼挂了起来。
<br/>　　一帮下人前呼后拥，口中喊着：“少爷回来了，少爷回来了。”
<br/>　　一个老者拄着拐杖从堂屋出来，拐杖敲着地面邦邦作响。
<br/>　　“不孝逆子，回来便回来，大呼小叫做什么。”
<br/>　　黄为善赶忙跪下行礼，黄老汉目光扫到程大雷和刘发财，道：“这两位看着有些眼生，是你路上结交的朋友么？”
<br/>　　“还没与爹爹说，我路上遇到这强人，多亏这两位好汉出手相助。”
<br/>　　黄老汉迈步下了台阶，弯腰向程大雷与刘发财行礼。
<br/>　　“小儿年幼，多亏了两位壮士。这就命厨房准备饭菜，好好招待一下两位好汉。”
<br/>　　“嗳，饭就不着急吃了。”程大雷摆摆手：“有洗澡水么，干净屋子收拾出来一间，先让我们好好歇息一下。”
<br/>　　“喔喔，是老汉疏忽了，来人，先带两位好汉下去休息。”
<br/>　　黄家的招待很是周到，命下人收拾出来一间干净屋子，又备好了热水。等程大雷和刘发财分别洗了澡，换好干净衣服回到房内，看房内已经摆着几样贴心的酒菜。
<br/>　　“好哇！”刘发财哈哈大笑：“看来黄家很懂事么，懂得报恩。”
<br/>　　伺候的丫头撇撇嘴，道：“当然，我家老爷是十里八乡有名的善人，但凡没活路的叫花子求到家门前，都是给钱米给粮，灾年也常常放粥。”
<br/>　　“我们可不是叫花子，我们是你们的救命恩人。”刘发财。
<br/>　　如此一来，就看出程大雷和刘发财的区别。刘发财坐在那里大块朵颐，盯着那丫头身子口水都快流出来了。可程大雷毕竟是一方枭雄，手下有帮服侍的人，对于黄家的招待，他顶多能说句凑活。
<br/>　　程大雷摆摆手，把那丫头赶了出去。他用银子试了试饭菜，确认无毒后，才吃了些裹腹。
<br/>　　“嗨，我说兄弟，你用得着这么小心么。”
<br/>　　“篱牢犬不入，小心才能走得万年船。”
<br/>　　程大雷冷冰冰的说了一句话，现在他已经有些累了，不愿意搭理他。而程大雷在蛤蟆城，毕竟是一言定人生死的人物，当他不故意表演的时候，那种说一不二的气概就出来了。
<br/>　　真货和假货毕竟不一样，刘发财大概也看了出来：程大雷这个人不简单。他之前对自己说的，或许是假冒的。
<br/>　　当然，他琢磨不透程大雷的身份，也就猜不透程大雷的用意。无非就是心理多留了个心眼，对程大雷说话谨慎了很多。
<br/>　　程大雷躺在炕上，心里存着提防，他也不是要提防谁，只不过是下意识的生活习惯。
<br/>　　而刘发财也不敢睡熟，他心里也存着提防，他提防的人是程大雷。
<br/>(本章完)
