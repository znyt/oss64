　　目标写字楼附近的道路全都已经被K国政府派出的快速反应部队封锁，几处制高点也安排了狙击手，阳光下，狙击步枪上的狙击镜闪着刺眼的银光。
<br/>　　七人座MPV通过警戒线后，便是停在了马路边，车门拉开，几个人走下汽车。
<br/>　　慕容飞身后背着狙击步枪枪袋，抬头环顾了一下四周，空气中那一闪一闪的狙击镜反光，全都落入了他的眼中，摇了摇头，说道：“啧啧啧，实在是太业余了，就这么明晃晃的暴露着，他们到底是狙人呢，还是被人狙？”
<br/>　　“的确是太业余了。”李毅峰也是跟着摇头，说道：“位置太明显，伪装也太业余，这根本就不是狙击手，而是被狙击目标。”
<br/>　　慕容飞和李毅峰两个人几句话，就把K国快速反应部队的布置，批评的一文不值。可是在他们两个人眼里，这真的是太业余，太业余了。
<br/>　　罗昊对着慕容飞和李毅峰两个人做了个手势，慕容飞和李毅峰两个人会意的点了点头，背着狙击步枪枪袋离开了。他们之前已经勘查好了合适的狙击位置，而他们选择的狙击位置比起K国快速反应部队狙击手选择的位置就要专业多了，攻守兼备。
<br/>　　“狂龙，狐狸就位，完毕！”
<br/>　　李毅峰架起狙击步枪，眼睛盯着狙击步枪上的白光瞄准镜，观察着写字楼内的情况。这栋写字楼内绝对隐藏着高手，这点毋庸置疑，所以必须小心提防。
<br/>　　“无常就位，完毕！”
<br/>　　慕容飞也是进入了自己的狙击位置，架好了狙击步枪。这时，慕容飞透过狙击镜看到了一个人影，慕容飞调整了一下狙击镜的倍率，确认情况后，按住耳麦，说道：“我看到吴云那崽子了，在十五层楼东侧玻璃墙背后！”
<br/>　　罗昊闻言，立刻拿起望远镜，朝着慕容飞所说的位置看去，如果是看到了藏在玻璃墙背后，握着狙击步枪的吴云。果然，智天使这边也是有所准备的。
<br/>　　吴云潜伏在那个位置显然是有一段时间，而在此之前，K国快速反应部队的狙击手居然没有发现他的存在，只能说吴云选择的位置非常专业。
<br/>　　吴云潜伏的狙击位置正对着写字楼前面的马路，他那个位置可以很好的监控到写字楼前方的情况，虽然这个时间点对于吴云那个狙击位置是面光，狙击镜在空气中形成的折射会暴露他，但是写字楼的玻璃墙却是给他提供了最好的掩护，阳光落在写字楼玻璃墙上所形成的反光，将狙击镜那星点反光很好的掩盖了起来。
<br/>　　写字楼整栋楼高二十五层，吴云选择了十五层这个位置，不仅使自己具有良好的狙击视野便于进攻，在关键时刻也方便他撤退，不管是往楼顶的直升机停机坪跑，还是往楼下的地下停车库跑，都非常方便。
<br/>　　吴云这家伙进入“暗月”这么长时间，果然还是学到了不少的东西，同样的，这混蛋手上沾的鲜血恐怕也不少。
<br/>　　罗昊心中暗暗嘀咕。
<br/>　　“发现智天使没有？”罗昊按住耳麦问道。
<br/>　　“暂时还没有发现。”慕容飞回答道。
<br/>　　“子午教官准备进入写字楼，提高警惕！”罗昊吩咐道：“无常，给我盯死吴云！”
<br/>　　“明白！”
<br/>　　“收到！”
<br/>　　慕容飞把注意力主要集中在吴云身上，而吴云显然也是知道了自己被盯上，但是他却并没有做出什么表现，还是待在他现在的狙击位置没有移动，甚至还很挑衅的对着慕容飞竖起了一根中指。
<br/>　　“妈的，敢对老子竖中指。”慕容飞通过狙击镜自然是看到了吴云左手竖起的中指，骂咧道：“有机会老子分分钟教你做人！”
<br/>　　慕容飞话虽然是这么说，但是他真的有把握能赢得了吴云吗？慕容飞自己也不敢肯定。以前在吴云参加龙隐部队选拔的时候，他的确是吴云的教官，而那个时候的吴云也的的确确是个彻头彻尾的菜鸟，他想怎么虐就怎么虐；但是现在时间过去这么长，谁都知道现在的吴云已经不再是当年那个毛头小子了，这家伙就是在“暗月”中也有了一些名堂，现在知道“吴云”这个名字的人或许不多，但是知道“暗天使”这三个字的可不少。
<br/>　　马路上，子午抬头看了眼碧蓝的天空，深吸了一口这异国他乡的空气，迈步朝着马路对面的写字楼走去，站在写字楼门口，具有光影感应功能的玻璃门自动打开，子午迈步走进了写字楼。
<br/>　　偌大的底楼大厅中除了子午之外，没有其他人，光洁的大理石地面上倒映出子午的影子。
<br/>　　“GHOST，你来了。”
<br/>　　大厅广播中传来了影天使的声音，这次他说话的声音没有经过技术处理，跟之前在视频直播中的说话声音有明显的不同。
<br/>　　子午环视了一下大厅，看到了安装在大厅几个角落处的监控探头，盯着一个探头说道：“是的，我来了，我们之间的游戏可以开始了。”
<br/>　　“游戏？”影天使轻声重复了一句，说道：“你说得不错，这就是一场游戏，只是这场游戏玩得很大，你输得起吗？”
<br/>　　“你又不是GM，你怎么知道自己一定会赢？”子午不屑地勾了勾嘴角，说道：“而且你跟我交手这么多次，你哪次赢过我？”
<br/>　　“这就跟赌博一样，最后一把SHOWHAND之前输了多少无所谓，只要最后一把SHOWHAND的能赢就行了。”影天使说道：“坐电梯上顶楼，我在那里等你！”
<br/>　　子午迈步朝着电梯走去，刚按下按钮，电梯门就打开，走进电梯，按下数字键“25”；电梯门关闭，电梯带着子午迅速往上，显示屏上，代表着楼层的数字飞快跳动着。
<br/>　　“叮！”
<br/>　　电梯中传来一声轻响，电梯停在了25层，电梯门向两边缓缓打开。
<br/>　　电梯门刚打开，子午就看到了站在电梯外的影天使和智天使。智天使，子午看过照片，而影天使也就是SHADOW，子午却是第一次见。
