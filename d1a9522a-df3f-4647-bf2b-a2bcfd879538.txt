　　翌日清晨，苏莫与聂乾和韩盈一起，三人联袂离开了火元峰。
<br/>　　苏莫并未知会火元圣师，就这样走了。
<br/>　　说来也是奇怪，火元圣师虽然收留了苏莫，但一个多了，却没有任何的指示与教导。
<br/>　　不说武道上的教导，就算是炼器之术，都没有任何教授，就这样放任他自己独自修炼。
<br/>　　对此，苏莫也有猜测，可能是他以前连番拒绝火元圣师，让对方有些气恼了，想将他在此地晾一晾。
<br/>　　不过，苏莫也不在意，这样也无所谓，虽然可能有些损失，但换来的是他的自由与时间。
<br/>　　他还真的有些担心，火元圣师会逼着他学习炼器呢！
<br/>　　所以，他离开之时，也没有知会火元圣师。
<br/>　　没过多久，三人便离开了神武学府，飞离了学府所在的陨落星辰。
<br/>　　没有了陨落星辰上巨大的重力影响，三人的速度立刻暴增十倍，如同三道闪电，消失在了天际之中。
<br/>　　……
<br/>　　南苑，精美的院落内，雅致的凉亭中。
<br/>　　尤天韩和太史轩相对而坐，两人的身前，凉亭之外，还站着一名蓝衣青年，正是上次引领苏莫来此的人。
<br/>　　“尤大哥、太史大哥，那苏莫离开了学府！同行的还有聂乾和韩盈！”蓝衣青年向尤天韩和太史轩抱拳说道。
<br/>　　“离开了？”尤天韩闻言，顿时轻笑了起来。
<br/>　　太史轩同样轻笑了起来，眸中寒光一闪，笑道：“苏莫啊苏莫，你敢耍我们，那就别怪我们无情了！”
<br/>　　太史轩虽然在笑，但心中的杀机，却是凌厉无比。
<br/>　　上次，苏莫说要考虑三天，结果当天就住进了内府火元峰，还将聂乾和韩盈也带到了火元峰，这让太史轩几乎气的吐血。
<br/>　　这明显是在耍他们！
<br/>　　故意打他们的脸！
<br/>　　他们怎么可能忍下这口气！
<br/>　　“太史轩，你要亲自动手？”尤天韩轻声问道。
<br/>　　太史轩闻言，略一沉吟，摇头道：“三个蝼蚁而已，何需我亲自动手，让赵城和郭剑青去吧！”
<br/>　　太史轩才懒得亲自动手，他可是武帝境五重武者，何需动手去杀三个蝼蚁！
<br/>　　让两个武帝境四重巅峰的人出马，便足以斩杀苏莫三人了。
<br/>　　太史轩的修为是武帝境五重，他之所以能和尤天韩这种强者平齐平坐，靠的不是实力，而是身份。
<br/>　　因为他是太史世家的核心族人，身份地位远不是尤天韩等人可比。
<br/>　　尤天韩等人，不过是加入了太史世家的派系，将来投靠太史世家而已，身份地位差距巨大。
<br/>　　所以，太史轩和尤天韩，虽然修为差距极大，却能平起平坐。
<br/>　　“那就让赵城和郭建青两人去吧！”尤天韩点了点头说道，显然对于这两人的实力颇为信任。
<br/>　　“是！我现在就去安排！”凉亭外的蓝衣青年点了点头，便转身前去安排。
<br/>　　“等等！”
<br/>　　这时，太史轩突然开口，喊住了蓝衣青年，叮嘱道：“让他们做的干净点，不要留下什么线索！”
<br/>　　太史轩为人谨慎，他可不想为了苏莫等三位蝼蚁，而得罪火元圣师。
<br/>　　虽然他背景深厚，但他也算代表了太史世家一脉，若是得罪了火元圣师，难免日后会有不必要的麻烦。
<br/>　　“太史轩大哥放心，以赵城和郭建青的实力，肯定不会留下什么线索的！”蓝衣青年说道，言罢，他便大步离去了。
<br/>　　……
<br/>　　无尽高空之上，茫茫云海之中，苏莫和聂钱、韩盈，三人速度如电，一路疾行。
<br/>　　“苏兄，我知道最快到达天冥星的传送路线，而且不需要花费什么灵晶！”韩盈一边飞行，一边对苏莫说道。
<br/>　　“哦？还有这么好的事情？”苏莫闻言惊讶，他身上可是半块灵晶都没有了，本来还想乘用传送阵之时，想向两人借点灵晶用呢！
<br/>　　“当然，这条传送路线上，大多数的星空传送阵，都是由神武学府背后的十三个大势力掌控的，只要有神武学府的身份令牌，便无需支付任何费用！”韩盈说道。
<br/>　　“哈哈！那真是太好了！”苏莫闻言大喜，看来这神武学府弟子的身份，好处还不少。
<br/>　　而且，他更加清晰的感觉到，神武学府背后十三个大势力的强大。
<br/>　　“昨晚我仔细算了一下，我们只需三个月出头的时间，便能到达天冥星。”韩盈道。
<br/>　　“嗯，如此最好！”苏莫点了点头。
<br/>　　“苏兄，天冥星是巫族的地盘，巫族势大，我们到时还需小心谨慎，不要得罪了巫族之人！”聂乾说道。
<br/>　　苏莫闻言，心中苦笑，他此行的目的就是杀巫族，怎么可能不得罪巫族之人。
<br/>　　若是让两人知道了他此行的目的，不知有何感想！
<br/>　　不过，苏莫也没有告诉两人他此行的目的，准备到达天冥星之后，再如实告诉两人。
<br/>　　“到时候我会小心的！”苏莫点头说道。
<br/>　　就这样，三人一边闲聊，一边赶路，倒也不觉得枯燥。
<br/>　　时间飞逝，很快三人便飞行了五日，距离布置星空传送阵的地方，还有两日的路程了。
<br/>　　这一日，晴空万里。
<br/>　　三人正在全力赶路，如同三道流光，身形所过之处，将空间都撕裂出长长的裂缝。
<br/>　　“嗯？”就在此时，苏莫眉头一皱，眸中闪过一道精光。
<br/>　　因为他感觉到，身后居然有人在追踪他们，而且还是有两人。
<br/>　　而此刻，来人距离他们只有三万里了。
<br/>　　苏莫的感知力何等敏锐，突破到武帝境之后，感知力更是暴增，轻易的便发现了身后之人。
<br/>　　这追踪而来的两人，速度比他们稍快一些，最多一刻钟的时间，便能追上他们了。
<br/>　　苏莫的脸色沉了下来，是什么人要对付他们？还是他们三人一路飞行，引起了歹人的注意，想要杀人夺宝！
<br/>　　苏莫第一个便想到了太史世家一脉，毕竟他得罪了太史轩，而这段时间，对方并没有什么动作，或许就在等待他出来。
<br/>　　不过，来人是否是太史世家一脉的人，现在还不清楚！
<br/>　　“两位，我们有麻烦了！”苏莫沉声说道。
<br/>　　“怎么了苏兄？”聂乾面带不解之色，他并没有察觉到危险临近。
<br/>　　这时，韩盈的脸色也沉了下来，她也察觉到了有人追了上来。
<br/>　　她的战魂是‘洞听耳’，感知范围非常的广，只不过她之前警惕性不高，才没有注意到而已。
<br/>(本章完)
