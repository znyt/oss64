<br/>　　毛小方张大嘴，怔怔地看了他半天，摇了摇头。
<br/>　　“那这是怎么回事呢？梧桐姑娘，峨眉山俗家弟子，她是唯一大的门派中被法术公会选中并且带走培养的，法术界的人都知道她，这不可能是假的。”

<br/>　　叶少阳猛然站起来，咬牙说道：“我不知道为什么会这样，但我相信，她跟冷玉绝对有着某种关联，我要搞清楚怎么回事！”
<br/>　　“怎么搞清楚？”
<br/>　　叶少阳一下子就不知道所什么了，是啊，怎么弄清楚呢？
<br/>　　毛小方也在帮忙思考这件怪事，想了想说道：“你说，少阳子，这个梧桐，会不会是你未婚妻的姥姥或者奶奶之类的，因为血脉的关系，所以长得比较像？”

<br/>　　叶少阳猛然一拍大腿，“我想到一个办法，能确认她是不是冷玉！”
<br/>　　毛小方道：“刚咱们讨论，不是都觉得不可能是一个人吗？”
<br/>　　“那只是推论出来的，梧桐是很有名，但是你见过梧桐吗？你不也是第一次见她，你怎么知道她长什么样子？”
<br/>　　毛小方怔了一下，“可是……妙心他们见过啊。”
<br/>　　“我知道，我只是假设一下，我也觉得，她们不可能是同一个人，不过我想验证一下她们之间有没有关系，我知道冷玉身上有个胎记，我可以去看看梧桐身上有没有。”
<br/>　　毛小方道：“八成是没有，不可能有。”
<br/>　　“没有的话，是正常的，如果有的话，那两人之前，肯定是有某种关联，我无论如何得求证一下。”
<br/>　　“哦，那这倒是个好办法，你可以去问问梧桐。”
<br/>　　“光问不行，万一她出于什么目的撒谎呢，我得亲眼看看到底有没有。”
<br/>　　“说的也是，你未婚妻的胎记，长在什么地方？”
<br/>　　“在这里，有一个黑痣。”叶少阳指了指自己的胸口。毛小方看了一眼，并在内心换了一下身份，想象着是梧桐，顿时苦笑起来，嘴角抽了抽，说道：“少阳子，你还是换个办法吧，这办法行不通。”
<br/>　　“什么意思？”
<br/>　　“你自己看看，那是什么部位？”
<br/>　　叶少阳低头看了一下自己手指的地方，离胸前重要的那个点，只有不到几厘米，顿时明白了……自己居然没有考虑到男女有别，这么敏感的位置，自己如果提出看一眼，估计会被当成色狼打死……
<br/>　　“这个……得想个稳妥的办法，不过，我一定要亲眼看看，确认一下！”
<br/>　　叶少阳下定了决心，再度走到窗前，望着外面发起呆来。
<br/>　　就在叶少阳做出这个决定的时候，在他的隔壁，妙心和梧桐待在房间里，也在进行着有关的讨论。
<br/>　　“梧桐，你对很多人都是冷冰冰的，但是今天看到那个叶少阳，为什么说了这么多的话，连我都感觉到很诧异。”
<br/>　　梧桐瞥了她一眼，道：“你以为是什么？”
<br/>　　妙心笑了笑道：“你们身份差距太大，他只是一个茅山外门弟子，配不上你的，我可不敢乱猜。”
<br/>　　“别胡说。”梧桐瞪了她一眼，目光移到窗外，说道，“在看到他的一瞬间，我有一种似曾相似的感觉，好像在哪里见过……”
<br/>　　妙心并没有铛回事，说道：“他也是法师嘛，而且好歹也是个天师，说不定在什么行动中见过。”
<br/>　　“不，你不明白我的意思，不是见过这么简单。他……妙心，你有没有过这种感觉，看到一个没见过的人，你也知道没见过，但是感觉却很熟悉，似乎在哪里见过……”
<br/>　　妙心歪着头想了一会，摇了摇头，“没有。”
<br/>　　“我以前也没有过，但是见到他，我真的会有这种感觉……我也不知道为什么？”
<br/>　　“一见倾心？”妙心说完自己笑起来。梧桐瞪了她一眼。
<br/>　　妙心走到她跟前，抓着她的双手，表情略微严肃起来，说道：“不过，你可不要随便动心哦，你是张师兄的人……”
<br/>　　梧桐横了她一眼，道：“都说我跟他是金童玉女，天生一对，你也这么觉得？”
<br/>　　“难道不是吗？你们很般配。”
<br/>　　梧桐微微低头，说道：“他哪里都很好，但是有一些地方，我并不喜欢。”
<br/>　　“什么地方？”妙心很好奇。
<br/>　　“说不好，就是感觉跟他在一起，少了点东西。”梧桐自己也歪头想了一会，道：“各方面都不坏，但没有怦然心动的感觉。”
<br/>　　妙心一脸的不解，在她的理解中，并不知道怦然心动是一种什么样的感觉。
<br/>　　梧桐不知道为什么，又想起来刚见过一面的叶少阳，心中有一种很奇怪的感觉，起身走到窗前，朝远处皑皑青山望去。
<br/>　　在她房的隔壁，叶少阳也在坐着一样的事情。
<br/>　　两个人，却是有着不一样的心事。
