　　猎者联盟一夜之间毁于一旦的消息在华夏爆发，到处都是有关这次事件的消息流走，从段氏基地周边的针对性谋杀，到这次事发的始末，不少人都从中道出了种种阴谋。
<br/>　　上京基地的众人也不例外，虽说猎者联盟在华夏不属于任何一方基地，更与大多数上将不相熟，可谁也无法忽略这个看似与世无争的组织。
<br/>　　除了惯有的神秘之外，便是其内猎者数量惊人，几乎覆盖了整个华夏的地域，并且很多高级猎者都只有代号流传在外，其真实身份没有人知道。
<br/>　　他们或许是某个基地内的将军，也可能是某个成名已久的高阶新人类，甚至是上将也说不定。
<br/>　　而在神秘家族的成员攻入猎者联盟失败，最终灰头土脸的离开后，猎者联盟的名气便在世界上空前之高，到达了某种神圣的地步。
<br/>　　所以当总部被毁，有人刻意针对的事件爆发之后，很多人都不难想到这其内的种种诡计。
<br/>　　各大基地的首领更是同一时间想到了一个点，那就是神秘家族！
<br/>　　于是在第二天，一场极其隐蔽的会议便就此召开，地点出乎意外的设立在了南部联盟军的路途之中，时间为深夜。
<br/>　　数十名上将及中将乘坐直升机连夜赶来，抵达了文奇胜所在的营地，这片地带被大批的士兵严加把守，那些将领也统统是伪装而来，看上去就像是各基地最普通的一次物资投放。
<br/>　　而这次秘密会议中，牧司令和洛冥更是双双到场，可见其重视程度。
<br/>　　“各位请自便。”文奇胜作为这次会议的主场人，在得到消息之后还来不及震撼，便立即需要面对如此多的将领前来。
<br/>　　“紧急事态，不得不特殊处理，倒是委屈诸位深夜赶来了。”牧司令说着，微表歉意。
<br/>　　段江伟摆摆手，道：“都可以立即，这次事件若不赶紧处理，我段氏基地都要背黑锅了。”
<br/>　　洛冥一愣：“怎么说？”
<br/>　　段江伟皱眉道：“虽然略加分析之后都能猜到是神秘家族的手笔，并且基本上都锁定了白家，但猎者联盟那里可并不完全信任我们。”
<br/>　　此话一出，上官荣没好气的说道：“猎王不至于那么笨吧？这都看不出来？”
<br/>　　“他不是笨，他是太聪明了，聪明的有些过头。”段江伟说着，神色严肃：“我不知道这次事件的爆发对他来说到底是不是打击，我们如此紧急的开会，对方心中考虑的或许不仅仅是向神秘家族讨回说法，甚至还有将猎者联盟彻底与我们这些基地势力分离的态度。”
<br/>　　“什么意思？”上官荣愣住，显然没听懂。
<br/>　　“这些都是我根据他的反应所猜测。”段江伟有些惆怅，苦笑道：“似乎除了狼牙的楚涵，他根本不信任我们任何一人，而你们也知道，楚涵现在根本无法出现，我们各基地与猎者联盟之间的友谊，就有些微妙了。”
<br/>　　“不满诸位。”一名上将忽然出声，有些欲言又止的说道：“我有好几名部下都在这次事件爆发后情绪激动，恨不得立马冲到猎者联盟总部去，经过我几番询问才知道，他们都有一重猎者的身份。”
<br/>　　此话一出，会议室里的众人都安静的有些诡异。
<br/>　　紧接着金泽宇也起身开口：“实不相瞒，我……也是一名猎者，当然代号就不说了，在我父亲身亡，天阳基地最困难的时候，我带着一批精英找到了猎王，想要接一些任务周转资金，你们也知道那时候天阳基地危机重重，猎王很义气，不但替我们保密，还让很多猎者在天阳基地周边聚集，一下子让天阳基地快要崩溃的经济链重新运作了起来。”
<br/>　　有着金泽宇的一番话，不少中将也纷纷出声，说出了曾经受到猎者联盟一臂之力相助的事情。
<br/>　　牧司令听着众人的话，有些心惊，他忽然意识道猎者联盟已经渗透到了整个华夏，根深蒂固！
<br/>　　“这就……有些危险了吧？”洛冥眯起了眼睛：“这个组织，结交能力强的过分啊！”
<br/>　　——————
<br/>　　在华夏秘密会议展开之际，凌晨的丛林小道上，白允儿和白羽一前一后，两人保持着一定的距离，只是走在前面的白允儿气定神闲，跟在她身后的白羽欲言又止。
<br/>　　“想说什么就说吧。”白允儿忽然开口。
<br/>　　白羽快步走上前，道：“你为什么不告诉他们猎者联盟的总部是被异种毁掉的？还是异种王出的手。”
<br/>　　白允儿笑道：“告诉了又怎样？”
<br/>　　白羽着急道：“告诉了之后，他们就不会这样针对神秘家族，也不会像眼下这般针锋相对。”
<br/>　　白允儿再次一笑：“所以呢？你觉得有人会对白家造成威胁？”
<br/>　　“那倒不至于。”白羽摇着头，分析道：“眼下他们都以为这次事件是白家指使，虽然我们白家有千年底蕴，但华夏深处的那些人有没有在这些年里准备什么，谁也说不准，总有那么几处地方，我们白家从来都没有渗透成功过，我担心的是分裂围剿。”
<br/>　　“你是说狼牙战团惯用的战术？”白允儿有些意味不明的笑出声：“普通人再如何进攻，都对神秘家族的成员无用吧？”
<br/>　　白羽皱眉：“但我总觉得，他们有所隐瞒。”
<br/>　　“当然有所隐瞒。”白允儿忽然停下脚步，转身直视白羽：“但那又如何？”
<br/>　　白羽一惊：“姐姐你就不担心白家出事？或者说华夏的人族势力出事？这两方之中总有你在意的一方吧？”
<br/>　　白允儿笑了起来，笑容异常冰冷：“关我何事？”
<br/>　　白羽有些答不上来，他知道白允儿一向清冷，但这置身事外的样子，似乎也有点过了吧？
<br/>　　“那……那楚涵呢？”白羽最终说出这个名字。
<br/>　　白允儿果然神色变了变，而后开口道：“对他造成不了什么影响，人类和神秘家族早晚会兵刃相向，提前几年也无所谓，牧司令和洛冥不至于什么准备都没有，白家也太过于小看普通人类的势力，至于狼牙那里，他们专心对付的是埋骨之地，正好当不了前锋。”
<br/>　　白羽惊了一下，呐呐道：“姐姐你是故意的？故意让他们开战？”
<br/>　　白允儿嘴角轻勾：“有何不可？打起来，打的你死我活，才好。”
