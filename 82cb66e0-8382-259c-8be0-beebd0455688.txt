　　西方灵山。
<br/>　　“混账，原始那厮，简直欺人太甚！洪荒之中，谁不知道师兄你修炼的就是因果法则，他元始竟然以此成圣，那岂不是断了师兄你的圣路吗？正是混账！”准提道人大声怒骂道，语气之中，充满气急败坏。
<br/>　　在他面前，接引道人，闭上双眼，双手合什，默念道经，不为所动。
<br/>　　“师兄，你倒是说句话啊？”准提见此，急声道。
<br/>　　听此，接引睁开双眼，面色平淡，完全看不出丝毫的生气。他深吸一口气，对着准提淡淡道：
<br/>　　“师弟莫急，此路断了，再修他路便是，吾还有他路可走，大道三千，条条可证圣人。”
<br/>　　“师兄你倒是好心态，你就不生气？”准提道人诧异道。
<br/>　　“师弟，你道心不稳了，还是好好静下心来静颂《黄庭经》吧。”
<br/>　　“师兄，你——”准提大急，接着气急败坏道：“也罢，既然师兄你也不急，那我急什么？”
<br/>　　语罢，准提道人便不再多言，回到密室，静心闭关，争取早日成圣。三清都成圣了，他们若是不再努力，那岂不是让人笑话了吗。
<br/>　　在他走后，接引道人睁开眼睛，看了一眼准提，之后又将目光投向东方大地，愣愣出神，那里正是昆仑山方向。
<br/>　　良久，他才收回目光，心中默默道：“元始，三清，因果成圣吗？东方大地，人杰地灵，真是令人羡慕啊。不知何时我西方会出现这等人物？哎！”
<br/>　　不在多语，接引道人闭上双眼，身上玄光闪耀，金光弥漫，一股强烈的佛性禅意在他身边弥散开来。更深处，一股强烈的寂灭之意，在孕化之中。
<br/>　　显然，接引道人准备以寂灭之道成圣。说起来，元始亦会寂灭之道。只不过，元始的寂灭与接引的寂灭不同。
<br/>　　寂灭之道乃大道，寂灭之下，又分为好几个方向，谁也不能规定，想要领悟寂灭，必须从一个方向出发。元始是以至冷至寒之中领悟到的大枯寂，寒冰冷到极致，万物都将归于寂灭。而接引道人领悟的寂灭显然不是与元始在同一方向上。接引领悟的是佛道的寂灭之意。佛家的寂灭之境倒是与道家的清净境有些相似。
<br/>　　而除了这两者之外，寂灭之道还有黑暗寂灭、魔道寂灭、圣光寂灭等。所谓万道殊途而同归，不管哪一道，到最后都会同归。
<br/>　　就如剑道分为截剑道、空间剑道、时间剑道、毁灭剑道等。寂灭之道同样如此。
<br/>　　娲皇宫中。
<br/>　　“元始，哼！”女娲冷眼看着昆仑山方向，娇声冷喝道。
<br/>　　在他身旁，伏羲苦笑，他对于他这个妹妹，可是十分了解的。他这个妹妹什么都好，就是争强好胜心太重了。如今三清中的玉清元始成圣了，以她的好胜心，真不知道要做些什么。
<br/>　　沉默了一会儿，伏羲对着女娲道：“妹妹，你可有感悟到成圣之机？”
<br/>　　“哥哥，我最近似有所悟，但不知为何，就是不能突破到最后一步，感悟到成圣之机？”犹豫了一会儿，女娲沉声道。
<br/>　　思索了一会儿，伏羲道：“对于成圣之事，我也不甚清楚，这需要妹妹你亲自去感悟，谁也帮不了你。不如，妹妹你到洪荒之中好好游历一番，说不定机缘一到，便能感悟呢？”
<br/>　　“对啊。”女娲眼神一亮，道：“好，我在这娲皇宫中待着也不是办法，我也好久没有到洪荒中走过了，今日就出去走走。”
<br/>　　“走，哥哥。”
<br/>　　接着，女娲便带着自己的哥哥，向着洪荒大地走去。
<br/>　　玉虚宫中，原始头顶盘古幡，玉虚琉璃灯与太上无极元始庆云纷纷沉下，落入泥丸宫，周边异象渐渐消散，恢复了正常。
<br/>　　此时，老子上前，拱手笑道：
<br/>　　“恭喜大兄悟透天机，证道成圣。”
<br/>　　“哈哈！恭喜大兄身成造化，证道成圣。”通天大笑，对着原始恭贺道。只是，他们尤其疑惑，为何今日的大兄，会给自己一种与往日不同的感觉，而且，为何他的头发是白的？
<br/>　　原始微笑颔首道：“两位兄弟不必多礼，你们迟早也会如此的。”
<br/>　　“大兄今日为何会有些不同？”不懂就问，这是通天一贯的性情。
<br/>　　“你是想说，为何我的头发变白了？是吗？”原始笑道。
<br/>　　“对啊，大兄，这是为何？”通天问道。
<br/>　　“这是因为我修炼一门道法所导致的，不过，已经不碍事了。”原始没有过多的解释，只是随便说了一句。
<br/>　　“哦。”通天亦没有再多问。
<br/>　　“咦！有趣！”原始突然惊异道。
<br/>　　自从他成圣后，他的元神便与鸿蒙紫气融合在了一起，他的元神变成了鸿蒙元神。鸿蒙元神是承载着圣道之基的地方，十分重要。此时在他的眉心深处的鸿蒙元神忽然诡异闪动，紧接着，在他的元神之中，凝聚出一道虚幻的漩涡。渐渐地，那漩涡形成实质，形成一个黑白双色的眼珠，同时，上面还有道道银光闪烁。
<br/>　　因果天罚眼！
<br/>　　天道因果之力与天罚之力的结合，在再以鸿蒙元神的特性，多方面因素的结合，形成的神眼。此神眼已经可以堪比一道削弱版的天道之眼了。
<br/>　　天道之眼能够监视天下，发出天罚，他这道神眼，一样可以。
<br/>　　同时，一旦他发动因果天罚神眼，他便可以看到洪荒之中，每个人身上的因果之力。他也可以通过他们身上的因果推算出事情的经过。
<br/>　　若是以后他这神眼进化到最后，完全可以替代天道之眼。
<br/>　　在此眼彻底凝聚成形后，他便可监视天下因果，执掌天下因果报应了。前世、今生与来世，三世因果，都逃不出他的掌控。
<br/>　　天网恢恢，因果轮回，报应不爽！
<br/>　　“哈哈！死吧！你的元神，我将用去点灯，你的躯体，我将用去祭旗。”这时，一道嚣张的声音在原始脑海中想起。
<br/>　　原始心念一动，脑海之中瞬间便出现了那声音中的画面。
<br/>　　那是一个满身杀气的魔头，将一个男子的元神抓在手中，他疯狂的大笑着。瞬间原始心中就出现了那男子一生所做之事，竟是一些虐杀生灵的画面。同时，他的脑海之中，也出现了那男子应有的报应。
<br/>　　原始心念一动，那正在大笑的魔头，瞬间便被九天之上落下的一道雷霆劈死，同时，他的魂魄被九天之上落下的太阳之火不断灼烧，直至化为灰烬，永世不得超生。
<br/>　　又一道信息传来，原始心念一动，顿时又出现另一道画面。
