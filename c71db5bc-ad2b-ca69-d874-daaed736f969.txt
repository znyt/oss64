　　在楚风眠得到这清仙藤之际，那一旁古锦的伤势，也逐渐的恢复了过来。

<br/>　　古锦身上的气息已经是平静了很多，向着楚风眠走了过来，急忙开口道。
<br/>　　“多谢楚前辈出手相助，不知楚前辈这次来到中域，是为何时？我古家在中域，也算是有些门道，也可以帮下楚前辈，以报答这次恩情。”
<br/>　　“哦？倒不是什么大事，我这次来到中域，一是为了参加九域天才战，二是就想要购买一些炼制灵丹的灵药。”
<br/>　　楚风眠开口道。
<br/>　　他这次的目的倒是很明确，也没什么不能说的。
<br/>　　九域天才战，每年参加的武者不止百万，楚风眠来参加，也不算是一件什么事。
<br/>　　“九域天才战？这九域天才战，倒是还有一个月的时间，不着急，至于买些灵药的话。”
<br/>　　古锦突然笑了笑，脸上露出几分自豪道。
<br/>　　“我们古家，是宁州古城最大的家族，大部分的商会，都跟我们有些联系，不如楚前辈随我前去宁州古城？我们古家可以来跟前辈做这个生意，至于九域天才战的事，我们宁州古城也有着可以直接前往远古角斗场的传送阵法，可以随时前去。”
<br/>　　“既然如此，倒是恭敬不如从命了。”
<br/>　　楚风眠点了点头道。
<br/>　　“那就请古小姐带路吧。”
<br/>　　楚风眠本来也就是要去宁州古城，如今跟这古锦结伴而行，倒也不错。
<br/>　　而且楚风眠要买的灵药，基本上大部分都是圣药，甚至是顶级的圣药，没有一些门路，想要买到也不容易。
<br/>　　有这古锦相助，也会方便许多，他也不用推辞。
<br/>　　九域天才战，还有接近一个月的时间，在这一个月的时间内，楚风眠倒是也正好多收集些圣药。
<br/>　　“好，楚前辈请随我来。”
<br/>　　古锦身形一动，便是从这魔谷之中化为一道遁光，直接飞了出去，而楚风眠也则是仅仅跟随其后。
<br/>　　楚风眠跟随这古锦，一路飞行，一路张望，两三个时辰后，在他的目光之中，突然一座庞大无比的城池，映入到了楚风眠的眼神之中。
<br/>　　说是一座巨大无比的城池，但是实际上这大小，却是比起无数国度还要庞大，乃是一个巨型国度一般，高大的城墙，足有千米之高，一眼望过去，不知道是有多少强者，都隐藏在其中。
<br/>　　宁州古城。
<br/>　　楚风眠对中域，还算是记得一些，这宁州古城，也算得上是中域的一方巨大的商业城池了，虽然在中域排不上号，可是也比起北域的北邙城要繁华的多。
<br/>　　进进出出的武者，都排成了长长的队伍，足有数万米之长，每一名进城的武者，都需要被检查身份。
<br/>　　若是想要排队进城，都至少需要数日的时间。
<br/>　　“我们直接进入其中，不用排队。”
<br/>　　古锦看到楚风眠的目光，笑着开口道，她的遁光一动，直接向着宁州古城中飞了过去，楚风眠则是紧跟其后。
<br/>　　“这是什么人？居然直接遁光飞到宁州古城中？不想活了吗？”
<br/>　　“凭什么我们要排队，这些人就可以直接飞过去！”
<br/>　　一些排队的武者，也看到这两道遁光飞了进去，有些不满的开口道。
<br/>　　“嘘，你们不要命了，能够直接飞到宁州古城里的，可都是大人物，小心被他们听见，白白葬送性命。”
<br/>　　旁边的一名武者急忙开口道，这才让那武者乖乖的闭嘴，不敢在说话。
<br/>　　这样的议论，还有很多很多。
<br/>　　“是谁？来者止步！”
<br/>　　楚风眠跟古锦的遁光，刚刚降落到宁州古城上空，两道身影便是直接飞了过来，这两道身影，都是生死境五重的武者，这样的人物，在一些小地方都是老祖宗一般的存在了。
<br/>　　不过在这宁州古城中，却也只能是担当巡视护卫，来巡查宁州古城的情况。
<br/>　　“是我，古锦，速速放行！”
<br/>　　古锦也见怪不怪了，平静的开口道。
<br/>　　“原来是古小姐。”
<br/>　　两名护卫看到古锦，急忙放开。
<br/>　　宁州古城中最大的家族，便是古家，他们可不敢阻拦古锦，便是将那让开，让楚风眠跟古锦的身形，直接是落了下去。
<br/>　　“锦儿？是锦儿回来了？”
<br/>　　楚风眠跟古锦，刚刚落入到宁州古城之中，突然一道着急的声音，突然响起。

<br/>　　楚风眠都感受到了这一股压力，不过他灵力一运转，身上的压力便是都消失不见。
<br/>　　不过楚风眠也能感觉到，这威压的主人，绝对是一尊大人物，生死台阶第六重，炼心之境的大人物。
<br/>　　距离称皇的第七重，天命之境，只差一步之遥的存在。
<br/>　　只有这种人物，才有这如此恐怖的威势。

<br/>　　“宁州古城的主人！”
<br/>　　周围一些武者，感受到这气息的一刻，都是有些激动的说道。

<br/>　　古锦看到这古家家主，低声说了一句。
<br/>　　她这次贸然离开，差一点闯了大祸，自身难保。
<br/>　　“没事就好，就算是那些人激你，你也不应该去讨伐血剑魔尊啊，血剑魔尊性格狡猾，极难对付，不过幸好，你还是平安归来了，没有遇到什么危险就好。”

<br/>　　“父亲，这位就是我的恩人，不是这位楚前辈出手相救，女儿就要死在血剑魔尊手中了，还是依仗楚前辈出手，轰杀了血剑魔尊。”
