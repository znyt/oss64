        &lt;h3&gt;第三卷第二百七十三章  相聚亦是分别&lt;/h3&gt;
<br/>
<br/>        张函和寒伯安聊得很投机，寒伯安问了很多张函风向上的问题，他们是生意人，有些事情他们也是需要了解的，作为张函来说，寒伯安和陈子昂也是朋友，只要是陈子昂的朋友，他都感觉亲切，只要是不涉及机密的问题，他都会耐心作答。
<br/>
<br/>        “一直疯传的落家和吴家的事情，你怎么看？”寒伯安问张函。
<br/>
<br/>        “你们寒家和落家关系很好，你们又是怎么看的？”张函反问寒伯安。
<br/>
<br/>        “落雪在欧洲事件可能不是简单的事情，有可能有人不愿意他们两家结合，谁都知道他们强强联手的后果，而且落雪排除万难，将吴彤身边的女人一个个都折杀了。”寒伯安敲着桌子，说落雪是为情，他实在是难以相信。
<br/>
<br/>        “你觉得现在谁最有能力和吴家抗衡？”张函问寒伯安。
<br/>
<br/>        “你是说？”寒伯安看着张函。
<br/>
<br/>        “所以，你们有可能都是他拉拢的对象，我希望你们都能保持初心，不要去趟这趟浑水，就算是落家介入了，你们也都要置身事外，尤其是不要和他们有任何瓜葛，要不然会万劫不复。”张函郑重的说。
<br/>
<br/>        “你不是......”
<br/>
<br/>        “我什么都不是，我只是一个安分守己的合法公民，对于腐朽我会坚决制止，对于不良风气我会严厉打击。”
<br/>
<br/>        “我明白了。”寒伯安看着张函那帅气的脸庞，虽然他的脸色苍白，他集忧郁悲伤痛苦勇敢乐观坚强于一体，但是掩盖不了他那种与生俱来的正直的气势。
<br/>
<br/>        张函给了寒伯安定心丸，他寒伯安也是一个合法公民，对于腐朽他会坚决制止，对于不良风气他会严厉打击，最起码他不会助纣为虐。
<br/>
<br/>        “我听说子昂要去甸城发展，你怎么看？”苍岚雪已经将手伸到他这里，像陈子昂这样的新秀有可能是他们要拉拢的对象。
<br/>
<br/>        “陈子昂要去甸城基本快谋划几年了，我相信她心里一定会有底的，她已经和那边的谷家联系上了。”张函并不打算隐瞒寒伯安。
<br/>
<br/>        “可是谷家？”寒伯安心里对谷家没有好感，沧岚雪竟然敢来威胁他，想到在美洲的遭遇心里就不爽。
<br/>
<br/>        “陈子昂很聪明，这么多年做事情从来不越规矩，所以她才能快速的发展，你不用担心她。”
<br/>
<br/>        “感觉你好了解她。”寒伯安轻笑一声，朝她们看过去，安男正拉着陈子昂的手，一脸温柔的看着陈子昂。
<br/>
<br/>        而江俞轩那种惊慌失措的样子，让他感觉有些好奇。
<br/>
<br/>        “感觉是俞轩做错了事情的样子。”寒伯安对张函说。
<br/>
<br/>        “江俞轩爱上了陈子昂，不过他自己一直不知道而已。”张函淡淡的说。
<br/>
<br/>        “哈！”寒伯安吃惊的看着张函，他竟然如此淡然的讨论别的男人爱着他心爱的女人。
<br/>
<br/>        “不是，他这么几年不是一直和倩楠在一起吗？而且他都和倩楠结婚了。”寒伯安也吃惊不小，他们之前也曾经以为江俞轩喜欢陈子昂，俩人相处的方式让人感觉很默契，后来江俞轩说张倩楠是他的女朋友，他们才相信江俞轩和陈子昂是朋友关系。
<br/>
<br/>        “他们都是自欺欺人的人，不像我敢爱敢恨。”张函还是淡淡的语气。
<br/>
<br/>        “你？你敢爱敢恨又能怎么样？现在还不是苦哈哈的坐这里和我聊天，陈子昂追到手了吗？”寒伯安嗤之以鼻，他寒伯安坐这里，别人都不要谈爱。
<br/>
<br/>        爱是需要宽容，需要呵护、需要追求、需要心灵相契，需要双方面共同进步，共同成长，而不是你追我赶，更不是拱手想让。
<br/>
<br/>        张函并没有回答寒伯安的话，他的眼神停留在陈子昂的身上，陈子昂那周身笼罩着悲伤的孤独的样子是他从来不曾见过的。
<br/>
<br/>        几个人围着她，但是她的身影却显得那么的孤单，那么的瘦弱，那么的孤独，张函轻轻的叹了口气，他毕生都在奋斗着想要向她靠近，但最近几次见面只是偶尔点点头，多的话都不曾说过。
<br/>
<br/>        他们还会有见面的机会，不过那个时候他将是以陆家女婿，陆玉的爱人和她见面，他和他的子昂怕是没有今生，如此不如让一个爱她的人陪着她可好？
<br/>
<br/>        张函站起来朝陈子昂走过去。
<br/>
<br/>        “子昂。”他轻轻的喊着陈子昂，大家见张函来了，便各自站起来相让，江俞轩不想动，却被张倩楠拉着手拽到了寒伯安的身边。
<br/>
<br/>        “......”陈子昂嘴唇动了动，但是什么话都没有说，她的心情很复杂。
<br/>
<br/>        “子昂，这会儿是不是特别后悔认识我？是我让你过着这样苦恼而纠结的生活？”张函在陈子昂的身边坐下。
<br/>
<br/>        “我只是觉得你后来不应该来打扰我的生活，哪怕我就那样的想念一辈子总比现在的这种状况要好，而且你们还利用江俞轩来探听我的一切。”
<br/>
<br/>        “你都知道了？我就想着聪明如你，肯定有一天会知道真相，没有想到你知道的这么早。”张函微微的叹了口气。
<br/>
<br/>        “我想知道的自然就会知道的。”
<br/>
<br/>        “那么你六哥你知道吗？”张函轻声问。
<br/>
<br/>        “你想说什么？”陈子昂抬起眼睛，和张函在一起呆了三年，她知道张函聪明睿智，她能查到的张函肯定也会查到，但是她现在不确定张函对她的爱有多深，在强大的利益面前一切都是未知数，何况张函还是陆家的女婿。
<br/>
<br/>        “我就问你，你知道你六哥多少？”张函再一次问陈子昂。
<br/>
<br/>        “他的一切我都知道了！”陈子昂闭上眼睛深深的吸了口气，然后挣开眼睛看着张函：“所以我们以后可能会站在对立面！”
<br/>
<br/>        “是的，我们会站在对立面，因为我是陆家的女婿，我和陆家是一体的！”张函看着陈子昂的眼睛，她的眼神坚定没有任何犹豫，脸上也没有任何惊讶。
<br/>
<br/>        “该来的终究是会来的，张总，和你作对手我很开心，我不会有一丝丝犹豫的！”陈子昂站起看着张函，这个男人给过她幸福，如今他们终于是要成为对手了，他曾经的豪言壮语、他曾经的美好愿望都被他丢弃在世俗的漩涡里。
<br/>
<br/>        再见，张函，再见，那个梦里出现的男人！
<br/>
<br/>        她的六哥在泥潭里挣扎了十几年，可是这些人却在糟蹋着她六哥的尊严和梦想，他们披着正义的面纱，干着龌龊的事情。
<br/>
<br/>        张函看着眼前的陈子昂，她就像一匹充满着昂扬斗志的猛虎，随时准备出击！
<br/>
<br/>        这才是他想要看到的陈子昂，而不是那个充满悲伤，毫无希望的陈子昂。
<br/>
<br/>        人，活着总得需要精神支撑！
<br/>
<br/>        陈子昂转身离开张函融于到大家的欢声笑语中，似乎一切都不曾改变，她和大家一起举杯，和大家一起说笑，他们还都是老朋友，只是这一次聚餐之后他们将各奔东西，不复往昔。
<br/>
<br/>        寒伯安依然是那个能掌控大局的人，他能感觉到整个气氛的微妙变化，但是他依然将大家凝聚在一起:“我们有幸相识，无论世态如何改变，我相信每个人心里都有一杆秤，对人对己，我也相信我们都是初心不改，不负韶华！前路如何艰险，我们依然会在一起！为了我们永远在一起干杯！”
<br/>
<br/>        今宵离别后，何日君再来？世事千万变，初心永不改，我们能不改初心吗？
