　　未能问出神界的入口所在，这令叶尘多少有些失望。
<br/>　　不过他也清楚，他先是杀了神界神子卡斯特，然后灭掉了教廷，神界定然不会坐视不理，对方早晚会找到他的头上，他和神界之间，终将会有一场大战。
<br/>　　……
<br/>　　而叶尘一脚踏灭西方第一大势力，这一消息很快传遍全球各国高层，不但各国高层全都为之颤抖，在全球武道界，也同样引起了巨大轰动。
<br/>　　之前叶尘虽然逼的米国投降，可是有不少人，却依旧心存希冀。
<br/>　　而如今，统治欧洲大陆上千年，甚至一度称霸全球的西方教廷，也被叶尘抹杀，让那些还心存侥幸之人，彻底的绝望了。
<br/>　　经此一战，叶尘方才真正站在了世界之巅，整个星球都被他踩在了脚下！
<br/>　　然而，
<br/>　　叶尘却清楚的很，在世俗界之外，那神、仙、妖、魔四界，绝对不会善罢甘休。
<br/>　　在不久的将来，他将会面临一场，更大的危机。
<br/>　　……
<br/>　　神界，神王殿，
<br/>　　“区区一个世俗界小子，竟然敢杀我爱徒，毁我神界传承！简直不可饶恕！我要以神的名义，让你接受这世上最残酷的惩罚！”
<br/>　　一道充满愤怒的咆哮之声，自神王殿中发出，整个神王殿都跟着一阵乱晃，似乎随时都要崩塌一般。
<br/>　　而周围神王殿中的奴仆们，纷纷跪俯在地上，一个个瑟瑟发抖。
<br/>　　“神王大人，此事万万不可能冲动啊！”
<br/>　　下方一名老者，忽然站了出来，一脸凝重道：
<br/>　　“据我们在仙界的眼线汇报，那小子在不久前，曾大闹蓬莱仙界，不但灭了仙界五大宗门之一的药王宗，还杀了数十米元婴境高手，此子的实力，实在是深不可测啊！”
<br/>　　这老者一出口，整个神王殿，顿时炸开了锅，
<br/>　　“此事难道真的？这怎么可能！”
<br/>　　“世俗界之中，竟然有如此人物？”
<br/>　　“听说此人还是一名不满二十的少年，实力怎会如此逆天？难道是某位夺舍重生的老怪物不成？”
<br/>　　……
<br/>　　众人议论纷纷，连坐在王座上的神王，也不由得眉头紧皱，冷声道：
<br/>　　“大祭司，那照你这意思，难道这个仇，我们神界就不报了不成？”
<br/>　　那被称作大祭司的老者，微微一笑，道：
<br/>　　“神王大人勿虑！据我所知，此子胆大妄为，不但同时得罪了我们神界和仙界，连妖、魔两界，也被此人搅得天翻地覆！”
<br/>　　“我们何不联合其他三界，共同围剿此人，此人同时得罪四界，纵然他真的是化神天君，也难逃一死！”
<br/>　　神王沉吟了片刻，
<br/>　　“如此确实最为保险！只不过，四界争斗数千年，如今却要联手对方一名世俗少年，且不说其他三界会不会同意，说出去也实在丢人！”
<br/>　　大祭司一脸严肃道：
<br/>　　“神王大人莫要小瞧此人！其实从当初柯莱特向我汇报此人的情况之后，我便一直在留意此子，根据我这段时间收集的情报来看，此子十有八九，应该就是某位大能者夺舍重生之人，甚至说不定，是从那个世界来的！”
<br/>　　神王听到这话，顿时两眼一瞪，
<br/>　　“你是说，那个世界……”
<br/>　　大祭司点了点头，
<br/>　　“此人一年之前，还只是一名华夏国的普通高中生，如今却已经具备化神天君的实力！除了是大能者夺舍重生，我实在找不到更好的解释！”
<br/>　　“而如今这个星球上，自从两千年前，进入末法时代开始，便再也没有出现过真正的化神天君，除了那个世界，我也实在想不到更为合理的原因！”
<br/>　　说到这里，大祭司微微一顿，
<br/>　　“这两千余年以来，虽然也有不少强者，踏入那通天仙路，进入那个世界，可是却无一人返回，没有人知道，那个世界到底是什么样子，难道神王大人您，还有其他三界的强者们，就不想了解一下吗？”
<br/>　　听到大祭司的这番话，神王的双眸之中，顿时闪过一抹浓浓的炙热之色，
<br/>　　“如何不想？本王困在半步化神境界，已经长达百年，始终无法真正迈入那一步！”
<br/>　　“而那通天仙路，又实在过于诡异莫测，一般也只有那些寿元将尽之人，才敢踏入其中，倘若此子真如大祭司所说，那倒是一个绝佳的机会！”
<br/>　　大祭司深吸了一口气，道：
<br/>　　“既然如此，那我亲自去三界一趟，定能促成此事！”
<br/>　　……
<br/>　　叶尘并不知道，神界居然打算联合其他三界，共同来对付他。
<br/>　　此时的他，早已经回到华夏多时。
<br/>　　蓬莱仙界那边始终没有任何动静，不过叶尘已经让华夏海军，密切监视东海上的动静，一旦出现任何异常，他便会第一时间得知。
<br/>　　而与此同时，
<br/>　　曦月的残魂，在各种丹药的滋养之下，已经凝聚出了元神，只是还尚不能完全脱离肉身而存在。
<br/>　　眼看蓬莱仙界那边迟迟没有反应，而萧若曦一直待在狂仙门中，也有些闷闷不乐，叶尘索性带着她，回到了天海大学。
<br/>　　反正以他如今的精神念力，即便人在天海市，也能随时洞察云州的情况，顷刻之间便能赶回。
<br/>　　……
<br/>　　重返校园，回到宿舍，
<br/>　　三名室友看到叶尘归来，均是十分欢喜。
<br/>　　“尘哥！这一个多月你都跑哪儿去了？想死兄弟我了！”
<br/>　　老四李虎第一冲了上来，直接和叶尘来了一个熊抱。
<br/>　　陈东和唐威两人，虽然不似李虎这么热情，不过同样满脸欣喜之色，同时眼眸深处还透着一抹敬畏。
<br/>　　要知道，那日叶尘离开之时，在帝豪大酒店前所发生的一切，他们直到现在还历历在目。
<br/>　　连天南省第一大家族的公子哥唐清山，都对叶尘毕恭毕敬，他们自然不敢再把叶尘当做普通人看待。
<br/>　　三人随便寒暄了一会，李虎忽然道：
<br/>　　“对了，尘哥，你不在的这段时间，那杨之凡可是嘚瑟的很！听说他老子向咱们生物学院，捐献了一瓶紫金仙泉，如今连院长都把他当做宝贝疙瘩，尾巴都快要翘到天上去了，根本不把我们这些人放在眼里！”
<br/>　　叶尘听到这话，不由得淡淡一笑，并没有多说什么。
<br/>　　区区一个杨之凡，在他眼中，连蝼蚁都算不上，他又岂会放在心上？
<br/>　　李虎见叶尘毫无反应，顿时有些气馁，只好话锋一转，道：
<br/>　　“为了庆祝尘哥回归，今天晚上我做东，咱们出去好好玩玩，如何？”
<br/>　　陈东和唐威两人纷纷表示赞同，叶尘自然也没有什么意见。
