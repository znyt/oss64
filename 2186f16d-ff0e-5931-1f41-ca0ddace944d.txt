　　回到义庄的时候，文才和九叔大概已经睡下，有些担心停尸间内那口棺材，刘哔没有先回房间，而是径直来到停尸间。
<br/>　　走的时候他记得自己锁过大门，但现在已经被人打开了，转了一圈，他才从一张屏风后面看到正在守夜的文才。
<br/>　　虽然是在守夜，但这家伙却睡的和死猪一样，撇了撇嘴，刘哔来到棺材旁边。
<br/>　　深夜，置身在一具棺材旁边，怎么都会让人觉得有些不舒服，周围正在燃烧的鲜红蜡烛，更是增添了一丝诡异的气氛。
<br/>　　搬了一把椅子坐下，刘哔静静的等候着，虽然明晚僵尸才会从棺中出来，但他也怕有什么意外发生。
<br/>　　大概过了一两个小时，天色已经微微泛起了一片鱼肚白，棺材内部突然传来一声细微的响动。
<br/>　　虽然这声响动并不明显，却让刘哔立即将目光投向了棺材，心中警觉了起来。
<br/>　　不出意外的话，僵尸现在应该已经复活了，虽然不知道能不能出来，但在这个时候，他不敢有丝毫的大意。
<br/>　　随后，棺材又没有了动静，刘哔没有放弃，两道淡红色的视线，依旧停留在身前的木棺之上。
<br/>　　几分钟之后，木棺中又是传来了一阵声响，这次刘哔听得清晰，心中微微有些紧张了起来。
<br/>　　响声越来越大，似乎是有什么人在里面翻来覆去一般，逐渐，木质的棺盖被抬起了一道口子，一只长满指甲的干枯手掌从里面伸了出来。
<br/>　　随之而来的，是一股浓郁到极致的腐臭味，这股味道，他再熟悉不过了。
<br/>　　随手从祭台上拿过九叔先前做法用的桃木剑，刘哔定了定神，缓缓将体内的天地灵气凝聚到桃木剑上，随后，他的两只眼睛也变得通红。
<br/>　　他已经进入了战斗状态，如果僵尸真的提前从棺中出来，他会毫不犹豫的给它雷霆一击。
<br/>　　那只已经有些腐烂的干枯手掌，慢慢的扒开棺盖，顺着边缘向外摸去，似乎是想要将棺盖打开一般。
<br/>　　但碰到三人之前用墨斗弹出的线网之后，又猛然缩了回去，像是碰到了什么极为克制它的东西。
<br/>　　从刘哔的视角看来，那些墨斗上面盘旋着一些金色灵气，就在和僵尸手掌相触的时候，那股金色灵气突然暴动，这才将僵尸给逼了回去。
<br/>　　但他心中清楚，光是靠这么一张不完整的灵气大网，是无法束缚住里面僵尸的。
<br/>　　手握桃木剑，他也不敢继续坐在原地了，而是从椅子上站了起来，如临大敌一般的盯着木棺。
<br/>　　门外，响起了一阵脚步声，心中微微一动，刘哔收起桃木剑上覆盖的灵气，眼中的猩红色也逐渐黯淡了下去。
<br/>　　另一边的小门中，传来了一声开门的声音，九叔一身白色睡衣，手中拿着一盏油灯，开门从外面走了进来。
<br/>　　“阿哔，你怎么在这？”来到木棺旁边，九叔看到了手握桃木剑的刘哔，有些狐疑的问道。
<br/>　　他方才正在睡觉，但体内的灵气突然躁动了一下，像是感应到了什么，他觉得有可能是任老爷子的尸体出了问题，方才过来查看一下。
<br/>　　“我从院子里撒尿，就听见这边有什么声音，怕发生意外，过来查看一下。”行了一礼，刘哔淡定的回答道。
<br/>　　点了点头，九叔也没有怀疑什么，而是绕着棺材看了一圈，敲敲打打了几下，才收回了双手。
<br/>　　“有发现什么不对吗？”看了一眼刘哔，九叔有些担忧：“算算时间，任老爷子也差不多要尸变了，可千万不要出什么事情啊。”
<br/>　　摇了摇头，刘哔将之前所见隐瞒了下来，在僵尸破棺而出的时候，他不想让九叔发现棺材里面的尸体已经变成僵尸。
<br/>　　如果被九叔知道了这个消息，难免其不会再布下几个防护措施，如果僵尸出不来，他也就没机会杀它来赚取积分了。
<br/>　　在这个世界，死者为大，大部分人也讲究入土为安，如果没有特殊原因，任家是不可能任由旁人破坏自家老爷子尸首的。
<br/>　　而九叔，也显然明白这个道理，所以明知道这具尸体会变成僵尸，依旧任由其放在这里。
<br/>　　“哐啷！”
<br/>　　就在两人对话的时候，不远处文才睡觉的地方，传来一声响动，像是有什么东西掉在地上一般。
<br/>　　刘哔和九叔皆是心中一惊，急急忙忙的走了过去，绕过了用来遮挡的屏风。
<br/>　　一张简陋的竹床上，文才抱着一只蛇形抱枕，睡得正香，旁边一个小型置物架倒在了地上，上面杂七杂八的物品撒乱了一地。
<br/>　　先前的声响，大概也是这个置物架倾倒在地才发出来的，刘哔舒了一口气，他现在才模糊想起，原著中好像是有这么一个剧情。
<br/>　　但是这个剧情并不重要，再加上时间过于久远，他一时间倒是没能想起来，反倒是真以为这边出了什么意外。
<br/>　　“唉，睡得和死猪似得，这样的人，最适合看义庄。”叹了口气，九叔走到床边，伸手替文才盖紧了被子。
<br/>　　天气马上入秋了，晚上还是比较冷的，再加上停尸间这种地方阴气过重，如果晚上睡觉不盖被子的话，难免会引起体内的一些疾病。
<br/>　　“好了，我回去睡觉了，这两天你辛苦一些，替文才守一下夜吧，等棺材重新入土，应该就不会有事了。”九叔打了个哈欠，开口吩咐道。
<br/>　　等到重新入土的时候，他会重新升坛做法，将任老爷子体内的尸气袪除，那样应该就不会有什么问题了。
<br/>　　应了一声，刘哔也没有拒绝，就算九叔不吩咐，这两天他都会盯住这里的，这可关系着一笔积分的进账问题。
<br/>　　反正以他丧尸的体质，就算一直不睡觉也不会有什么问题，借着这段时间，他还能修炼一下灵种，提升提升自己的实力。
<br/>　　见刘哔答应下来，九叔略微放了放心，哈欠连天的从小门中走了出去。
<br/>　　将九叔送至门口，刘哔关上了门，重新回到棺材旁边，盘坐在一张木椅上，进入了浅层次的修炼状态。
