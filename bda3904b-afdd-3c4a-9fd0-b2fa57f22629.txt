
<br/>　　<br/>　　    午桥和王生过完阴之后，老马在旁边基本上就已经了解的差不多了，但得到的信息并不全面，因为这个李杨所知道的并不多。<br/>　　<br/>　　    老马给陈重去了个电话，跟他说道：“跟我们之前了解的差不了多少，主要还是那个叫吴洋的人谋划的，你和小臣还有你老婆的生辰八字也是那个吴洋给的，我们抓到的这个其实就是个马仔和打手的角色，只知道该知道的”<br/>　　<br/>　　    陈重顿时叹了口气，说道：“那意思是，白审了一点干货都没有啊？”<br/>　　<br/>　　    “那都不是，至少知道吴洋是在哪落脚的，人就在川中呢，李杨是条小虾米，但吴洋肯定是条大鱼啊，再把他给抓了的话，那想知道的不就能知道了？”老马说道。<br/>　　<br/>　　    “行，今晚就办了他，人给我控制住……”陈重寻思了下，说道：“让齐泉带人跟你一起过去，听你的吩咐，北出马来的人也会去吧？”<br/>　　<br/>　　    老马点头说道：“肯定去的，价钱我让他们随便开了，他们做生意还是很有分寸的，不会半途而废的”<br/>　　<br/>　　    “那好，剩下的你安排吧，我就听结果”<br/>　　<br/>　　    老马跟陈重打完电话，午桥皱眉说道：“马爷，有件事我得事先跟你说一下”<br/>　　<br/>　　    “嗯，你说吧！”<br/>　　<br/>　　    “陈重的那块地皮是阎王坟场这个事，你怎么没有告诉我？”王生刚才过阴的时候，李杨除了提到吴洋这个人外，还说到了那块地是阎王坟场，午桥跟王生一听就知道这次来川中的事，简单不了了。<br/>　　<br/>　　    马爷摇头说道：“你也知道，我和陈重不懂这方面的事，什么阎王坟场之类的，你觉得我俩能知晓么？听名也不像什么好地方啊，陈重要那块地是准备做生意的，他要知道这块地会给他惹来这么多麻烦，我估计当初他连碰都不会碰，所以现在是骑虎难下了，被赶鸭子上架了，你说有谁愿意给自己身上招麻烦啊，对不？”<br/>　　<br/>　　    午桥一想也是，别说陈重了，这阎王坟场的风水格局他也就是听过而已，见都没有见到过，他觉得陈重和老马也自然不可能知晓了，确实应该不是有心瞒着他的。<br/>　　<br/>　　    午桥接着说道：“算了，我们既然赶了过来，也没有中途就撤出去的道理，送佛送到西吧”<br/>　　<br/>　　    老马拍了下他的胳膊，笑道：“放心，不会差事的！”<br/>　　<br/>　　    午桥挺认真的说了一句:“我们的事还真没人差过呢？”<br/>　　<br/>　　    缪斯酒吧里，王惊蛰又被菜刀文给拉了过来，他本来是不想来的，但这货神秘兮兮的说，这个酒吧是王惊蛰的福地，那里会诞生他的一世姻缘，你要是不去的话我就是绑也得给你绑过去，哥们千里抗猪槽全是为了你，你得理解我的一片苦心啊。<br/>　　<br/>　　    王惊蛰无奈，生拉硬拽的被菜刀文又给带到了酒吧去。<br/>　　<br/>　　    然后，他就又看到了那个一天前，一个人喝酒一个人唱歌的姑娘。<br/>　　<br/>　　    今晚的酒吧人比较多，可能姑娘来的有点晚，进来的时候都没有位置了，但她也没走，就一个人坐在候客区里拖着腮帮子等位，然后当她的眼神老是不经意间的飘过来的时候，王惊蛰就会发现她眼神中经常会闪烁出火花来，给他闪的一愣一愣的。<br/>　　<br/>　　    “看什么呢？”丁武过来，趴在王惊蛰后面顺着他的眼神看向了茅小草，然后笑道：“看见了一个孤单寂寞等待拯救的少女啊？我跟你说，能连续两天一个人来酒吧的女人，就只有两种可能”<br/>　　<br/>　　    “啥啊？”<br/>　　<br/>　　    “释放那无处安放的青春，或者是……”丁武想了想说道：“这里有什么她放不下的东西或者人”<br/>　　<br/>　　    王惊蛰翻了翻白眼，说道：“我看她啊就是闲的”<br/>　　<br/>　　    丁武无语的说道：“你就没看见，她老是眼带桃花的朝你放光呢么？”<br/>　　<br/>　　    王惊蛰无语的说道：“对，然后她放的都快要成斗鸡眼了，不是，我就奇怪了，你怎么老往我身上扯啊？这酒吧里的男人多了去了，你觉得我这一堆一块的放在这里，有什么吸引人的地方，然后人家非得来电我啊？”<br/>　　<br/>　　    丁武仔细想了想，点头说道：“那还真没有……”<br/>　　<br/>　　    “去去去，接客去吧，我再喝一会我就走了，我深深的觉得我的气质实在不适合这里”<br/>　　<br/>　　    丁武幽幽的说道：“那意思是我多管闲事了呗？”<br/>　　<br/>　　    “必须的！”<br/>　　<br/>　　    这时，丁武身上的电话忽然响了，他拿出手机看了下号码，说道：“喂，泉哥？”<br/>　　<br/>　　    “大老板有个事要办让我领队，要的都是会办事有分寸的自己人，边缘人一概都不掺和，是马爷给我打的电话，小武你来川中有一段时间了，一直都在酒吧里没有参与过老板的吩咐的事”<br/>　　<br/>　　    丁武一听，直接就明白咋回事了，点头说道：“明白了泉哥”<br/>　　<br/>　　    “嗯，一会我过去接你，十分钟后到门口我给你打电话，你自己出来就行了”<br/>　　<br/>　　    “好叻，我等你”丁武点头说道。<br/>　　<br/>　　    “有事了？”王惊蛰抬头问道。<br/>　　<br/>　　    “嗯，泉哥找我出去一趟，我得过去了，你们在这自己玩吧，我还不知道什么时候能回来呢，到时你俩自己回去吧”丁武觉得，要想融入一个集团，你得需要一个崭露头角的机会，而这个机会肯定得是能为老板办事做起的，所以跟着齐泉出去他没有任何拒绝的理由。<br/>　　<br/>　　    人的一生会有很多的机会，自然需要你去勇于把握，有的时候你可能错过了一件事，也许就是错过了你这辈子最应该珍惜的机会，丁武的理念就是这样的。<br/>　　<br/>　　    丁武走了以后，菜刀文说上一趟厕所就离开了座位，他去了卫生间一趟后，转头就去了候客区那边。<br/>　　<br/>　　    “你就打算一直在这等着啊？你就过去贴个桌不就得了？两个人的相识，不就是从第一句话开始的么，难不成你还老想着跟他眼神沟通啊？”菜刀文看着小草问道。<br/>　　<br/>　　    茅小草说道：“上赶子的都不是买卖，我的气质不允许我那么主动”<br/>　　<br/>　　    菜刀文顿时无语了：“大姐，你都主动连续两天来这守株待兔了，你还要啥气质啊？”<br/>　　<br/>　　    小草脸顿时就红了，呐呐的说道：“我，我不是没地方去么，闲的”
