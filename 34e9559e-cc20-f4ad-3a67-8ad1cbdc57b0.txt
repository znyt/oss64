夏夕绾当然知道自己在逆天而行，那些以命换命的传说未必是真的，她不信鬼神论，只信自己。
<br/>
<br/>        但曼陀罗花毒剧毒无比，第一次试毒就已经侵蚀了她一身宝贵的血液，来势相当的凶猛，二次试毒会怎么样，她也没有什么把握。
<br/>
<br/>        她不知道陆子羡曾经得到过最后一株曼陀罗花毒，他是帝都院士，手下是有自己的精英医研团队的，现在他很明确的告诉她，曼陀罗花毒会残留在她的体内，一个中了花毒的人，是活不长的。
<br/>
<br/>        其实这是一个死局，她跟陆寒霆，只能活一个！
<br/>
<br/>        陆子羡起身，他嗓音淡漠道，“你好好考虑一下吧。”
<br/>
<br/>        说完，他直接离开。
<br/>
<br/>        “陆二少！”夏夕绾迅速上前，一把拽住了陆子羡的衣袖，“你不要将这件事告诉你哥，这是我跟你哥之间的事情，你没有这个权利去告诉你哥！”
<br/>
<br/>        陆子羡停下了脚步，他垂眸，曜亮的黑眸落在了她的手上，现在她纤白的手指蜷着，用力而执着的拽住了他的衣袖。
<br/>
<br/>        就在这个时候，耳畔响起了“咔嚓咔嚓”的声音，大门被推开，一批媒体记者涌了进来。
<br/>
<br/>        他们手里都抓着照相机逮着这一幕就一阵乱拍，像打了鸡血一样的兴奋激动。
<br/>
<br/>        夏夕绾收回了自己的小手，她抬眸看向这一批突然涌入的记者们，夏妍妍没有来，记者倒来了，她想一想就知道这是夏妍妍玩的把戏。
<br/>
<br/>        夏妍妍是真的怕了，大概想趁现在她还是一个谜的时候将她斩草除根，所以她迫不及待的设计了她和陆子羡这一局。
<br/>
<br/>        这时记者们争先恐后的将话筒递了过来，
<br/>
<br/>        陆太太，请问你跟这个帅哥孤男寡女独处一室又拉拉扯扯的在干些什么，你是不是出轨了？
<br/>
<br/>        陆太太，陆总知道这件事吗，陆总这么宠你，你是不是偷偷的给陆总戴了一顶绿帽子？
<br/>
<br/>        陆太太，你金针封穴，刚坐上天才少女的位置，还没有坐热，又爆出出轨丑闻，你有想过后果吗？
<br/>
<br/>        这位帅哥，你看着很眼生啊，请问你是海城人吗，你知道夏夕绾是有夫之妇吗？
<br/>
<br/>        这位帅哥，请你讲两句，你跟陆太太之间是谁主动的，是她主动勾引你的吗？
<br/>
<br/>        陆子羡面对这些话筒和镜头神色淡淡的，俊美的五官没有丝毫的波动，不意外不惊慌，清冷而疏离。
<br/>
<br/>        夏夕绾也没有退缩，她挺直了纤柔的美背从容的看着这些人，什么话都没有说。
<br/>
<br/>        很快，喧嚣的人群就安静了下来，大家做了这么多年的采访，还真是第一次见到这么淡定的二人组。
<br/>
<br/>        夏夕绾和陆子羡淡淡的看着他们，那意思好像是我们静静看着，你们再卖力点。
<br/>
<br/>        这可是当场捉奸了，多大的爆料啊，这两个人真是一点面子都不给。
<br/>
<br/>        大家都静下来了，这时夏夕绾淡淡的勾了一下红唇，她清丽的嗓音如珠玉般清晰的传递到了每一个人的耳膜里，“你们在说什么，出轨？这俗话说得好，捉奸要在床，你们捉到了什么，捕风捉影？”
