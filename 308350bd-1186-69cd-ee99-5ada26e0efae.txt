　　第209章
<br/>　　都说官匪不同炉，其实很多时候，官就是匪，匪就是官。要不怎么说，圣人不死，大盗不止呢。一个地方，没有官，或者没有匪都是不行的。
<br/>　　程大雷怎样都算得上大盗，在长安城，如他这样的大盗还有三位。毫无例外，他上了通缉令，赏银高达一千两。
<br/>　　当然，捕快都是兄弟，官兵经常喝酒，可面子工作还是要做的，万一有哪个不开眼的新捕快，上赶着要抓程大雷，真被抓到了也麻烦不是。
<br/>　　所以每次出门前，程大雷都要做些伪装。将眉眼描长，嘴上粘满胡子，青衫改黑袍，眨眼间，程大雷就从一个肤色苍白的吸血鬼，变成一个长须垂胸的大汉。
<br/>　　四人走出地下城，外面阳光普照，四人大摇大摆走在街上。
<br/>　　“大当家之后有什么打算，是继续留在长安，还是拉一号人马，占山为王，找回曾经的兄弟。”徐神机。
<br/>　　“长安虽好，规矩却也多，那里比得上占山为王痛快。”程大雷叹口气道：“只不过，我在长安还有件事情要办。”
<br/>　　徐神机三人点点头，皆明白程大雷话中的意思。
<br/>　　程大雷如今看似潇洒，其实他只是某种规矩的执行者，江湖他要给面子，官府他要给面子，一手托两家哪里好做。当然，程大雷做得还不错，但也很累。
<br/>　　而程大雷沦落长安，山寨弟兄下落不明，这一切都是拜一个人所赐。
<br/>　　程大雷待在长安，要办的不是事，而是人。
<br/>　　如今武科举将至，长安城人满为患，街上到处都是带着兵器的习武之人。习武之人么，脾气火爆些也正常，一言不合就能动起手来。
<br/>　　“决斗，快走，前面有人决斗了。”
<br/>　　不知谁喊了一声，大街上的人呼啦压了过去，程大雷几人也顺着挤了过去。
<br/>　　只见前方的街道上，相对立着两人，一人身穿紫色长袍，腰间佩一柄长剑，一人穿粗布短衣，手里拎一柄柴刀。
<br/>　　“你来了？”紫袍人。
<br/>　　“我比你先来。”短刀男。
<br/>　　“你不该来的。”
<br/>　　“可我已经来了。”
<br/>　　“若你知道我的名字，可能就不会来了。”
<br/>　　“我知道你是一剑西风钟西风，我也来了。”
<br/>　　“我也知道你是山北樵夫周砍柴，只是你手中的刀，能砍柴也能杀人么？”
<br/>　　“砍柴是外行，杀人很内行。”
<br/>　　“杀人，杀什么人？”
<br/>　　“你。”
<br/>　　声音落地，宛若一字电闪，空中若闻啸声，长刀斩了出去。铛地一声，应是兵器碰撞之声，二人动作极快，眼花缭乱，围观诸人都看不真切。
<br/>　　再回过神来时，就见两人缠抱在一起，抓头发掏心脚都使了出来。这个抓住那个头发，那个卡住这个脖子。
<br/>　　“你放手！”
<br/>　　“你先放手！”
<br/>　　“你放我就放。”
<br/>　　“你不放手我也不放。”
<br/>　　“你不放我也不放。”
<br/>　　“嘘！”
<br/>　　围观的人群出发一阵嘘声，觉得好没意思，纷纷散去。
<br/>　　“干嘛呢，干嘛呢……”正在这时候，一队巡城的卫兵赶到，将俩人抓了起来：“长安城内不许持械私斗，一看就是外地人，抓起来，抓起来。”
<br/>　　长安城内的治安要比帝国其他地方更严些，而因为最近大批习武之人涌进城内，巡逻的卫兵任务更重。其实习武之人么，真看谁不顺眼，暗地里一刀捅了就是了，真闹到当街决斗的地步，要么是脑袋不清楚，要么是脑袋不清楚的高手。看二人的情况，估计是前者。
<br/>　　卫兵队长看见程大雷，将他认了出来：“这二位是您的朋友？”
<br/>　　互相倒也认识，如果程大雷点点头，卫兵也就把这两人放了。
<br/>　　“不认识。”程大雷大手一挥。
<br/>　　这样的事情，每天都要发生一两件，很多人已经见怪不怪。看徐神机三人的样子，应是第一次遇到，站在那里大眼瞪小眼。
<br/>　　程大雷扭头尴尬的笑笑：“长安嘛，和其他地方不一样的。”
<br/>　　正这时候，忽有一人在程大雷身后拍了拍，程大雷眼睛随即一寒，手已摁在剑柄上，猛地回头，当场就要摁剑出鞘。
<br/>　　待看清后面之人后，程大雷才无奈的松开剑柄，道：“干嘛呢，人吓人吓死人知道不。”
<br/>　　程大雷身后那人愣了愣：“你，你干嘛呢，刚才你的眼神好吓人呐，我不就是和你打个招呼么。喂，你怎么每次都是这个打扮，就不能换一套。”
<br/>　　徐神机三人的目光落在这人身上，穿一袭蓝袍，嘴上两撇胡须，可模样一看就知道，乃是女子乔装改扮。
<br/>　　“大当家，这位是？”
<br/>　　“喔，我的老大。”程大雷随口道。
<br/>　　“你上面还有老大？”
<br/>　　三人一楞，随即也认出了这女子是谁，正是曾经被扣进蛤蟆寨的李婉儿。只不过，许久未见，她出落得更加标致，如一朵鲜花已经盛放。
<br/>　　“我当然有老大，不然谁罩着我。”程大雷挥挥手道：“借一步说话吧。”
<br/>　　长安城四员大盗，各有一块地盘，头上也各自有人罩着，不然很多事情根本玩不转，要不怎么叫官匪勾结呢。而罩着程大雷那人，姓李名婉儿，正是那帝国明玉公主。
<br/>　　程大雷沦落到长安，阴差阳错遇见了李婉儿，互相也算旧相识，一来二去关系处得还不错。于是这李婉儿就成了程大雷的靠山。
<br/>　　几人来到一处茶楼，找了个雅间坐了。
<br/>　　程大雷看着男扮女装的李婉儿有些无语，道：“你怎么又从宫里跑出来了？老周知道了，又和我没完没了。”
<br/>　　老周是将李婉儿带大的太监，与程大雷也认识。
<br/>　　“程大雷，你说你多久没去找我了。”
<br/>　　“公主殿下，我很忙的好不好，你这董事长什么事都不做，活都是我这总经理来干。”程大雷给自己倒杯茶：“你这次找我干嘛？”
<br/>　　“程大雷，你带我私奔好不好？”
<br/>　　噗！
<br/>　　程大雷刚喝的茶吐了出来，徐神机三个也是一楞，看看程大雷，再看看李婉儿。眼里都写着一句话：
<br/>　　“有古怪！”
<br/>　　李婉儿叹口气：“你不带我私奔，父皇就要将我嫁人了。”
<br/>　　
<br/>(本章完)
