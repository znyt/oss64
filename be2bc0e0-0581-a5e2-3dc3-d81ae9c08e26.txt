　　陈晓宇冷冷说道：“好的很，反正你我有赌约在身，将来桃花山龙华会上，我要你当众磕头认错！”
<br/>　　叶少阳撇了撇嘴，自己的气已经出了，也懒得再跟他争论，转头对妙心说道：“你家祖上，是不是懂得此案天八卦？”
<br/>　　妙心犹豫了一下，答道：“我家祖上曾有机缘，参透过先天八卦中的两组易数……”
<br/>　　此言一出，在场几人都惊呆了，陈晓宇之前还怀疑过世上不存在先天八卦，听了妙心的话，顿时觉得脸上火辣辣的疼。他敢于怀疑叶少阳，但是不敢怀疑妙心说的话，当下缩到一边去不敢言语。
<br/>　　叶少阳点头说道：“这就对了，这轮盘禁制，正是根据先天八卦中的一组易数制定，如果你祖上不懂先天八卦，根本不可能打得开这道机关，除非打开机关的不是你祖先。”
<br/>　　妙心睁大眼睛，盯着他看了好一会，说道：“你怎么能看出先天八卦的易数？”
<br/>　　“这……”叶少阳为难地挠着头，笑了笑，说道：“如果我说我懂先天八卦，你信不信？”
<br/>　　“就你？”陈晓宇哈哈大笑起来，“叶少阳，你之前说的那些，还或许让人相信，但是……先天八卦，你一个天师，居然说什么先天八卦？你要是会先天八卦，你不早就一掌劈死我了？哈哈哈……”
<br/>　　叶少阳气定神闲地看着他，懒得跟他理会，转头看其余人，也都露出明显不信的神色，曹雨兴和卢晓清更是面带鄙夷，只是不像陈晓宇那么直接地表现出来。
<br/>　　叶少阳冲妙心耸了耸肩，说道：“我没法让你相信我会先天八卦，总之……你相信我有不能说的苦衷吧。”
<br/>　　妙心看了他一会，没有再说什么，转头看着打开半扇的石门说道：“既然门打开了，我们进去吧。”
<br/>　　说完，自己第一个闪身进了石门，叶少阳生怕里面有危险，紧随其后。
<br/>　　大伙紧接着鱼贯而入。
<br/>　　三色莲花灯的火光只能照到十米之内，远的地方就照不到了，只能隐约看出这是一个很大的空间。叶少阳打着三色莲花灯，顺着墙壁往里走，在墙上突然发现了一个镶嵌在墙上的烛台，用灯火一照，里面居然还有灯油，于是把三色莲花灯凑近了点亮。
<br/>　　“是鲛油。”叶少阳抽了抽鼻子，说道。
<br/>　　鲛油是非常特殊的一种油，用做长明灯的话，可以燃烧几百年，更不要说光放着不点了。叶少阳曾经也在古墓里搜罗过鲛油，不过自从收了橙子之后，对鲛油就有点反感，不再用这东西，不过眼下这鲛油既然是古墓里现成的，也就顾不上了。
<br/>　　点亮鲛油灯之后，叶少阳顺着墙壁继续往前走，没走多远，又发现了镶嵌在墙壁上的一座灯台，里面也有灯油，于是接着点亮。
<br/>　　就这样，叶少阳绕着墓室走了一圈，一共发现了七盏长明灯，逐一点亮，整个墓室就这样被照亮了。
<br/>　　大伙环顾左右，这才发现这是一间椭圆形的石室，中间空空荡荡的，什么都没有，只有……一口井。
<br/>　　大伙面面相觑，不知道这是怎么回事，走了过去，小心翼翼地趴在井研上，用灯照下去，能看到下面有火光的倒影，说明是有水的。
<br/>　　不过，三色莲花灯在这里似乎失去了作用，之前变幻的火苗也变得正常起来，这让大家百思不得其解。
<br/>　　“妙心姑娘，这是怎么回事？”有人问道。
<br/>　　妙心趴在井研边上，皱着眉头端详了半天，摇头说道：“我也从来没见过这么墓中有水井的……”
<br/>　　转头朝四周观察了半天，说道：“按照方位，这里应该是配享室，但是不知道为什么什么都没有。”
<br/>　　梧桐接过话头，说道：“什么都没有，只有水井，那这口水井一定有问题。”
<br/>　　大伙面面相觑，不知道是怎么一回事。
<br/>　　妙心说道：“但是我能感知到，之前中了我松针的那只鬼鱼，此刻就在这里。”
<br/>　　“在这口水井下面？”叶少阳怔了怔，说道，“会不会那些棺材下面的孔洞，是通到这口水井里的？毕竟两边都是有水。”
<br/>　　曹雨兴奸笑道：“你下去看看？”
<br/>　　叶少阳哼了一声，表示回应。眼下情况未明，鬼知道这口水井下面有什么，别说自己现在实力不济，就算自己是灵仙，就这么跳下去，在水里万一遇到什么邪物，八成也得挂在下面。
<br/>　　大伙望着这口水井——这是眼下唯一的线索，必须调查，但又不知道怎么调查，反正没人愿意跳下去。
<br/>　　卢晓清突然说道：“我有办法，虽然不算太好，但也是唯一的办法了，这水井下面，必然有古怪，我们撒一包朱砂和石灰下去，万一有僵尸或许什么妖精，一定会憋不住跳上来，到时候我们不就知道怎么回事了？”
<br/>　　大伙互相张望着，都在考虑卢晓清的计划。
<br/>　　“如果出来邪物，怎么办？”叶少阳问道。
<br/>　　卢晓清冷哼道：“如果有邪物出来，那就灭了它呗。我们这么多人，还怕它不成，我就不相信这里能有什么邪物我们这些人对付不了的，到时候……叶天师你离远点，免得被伤到。”
<br/>　　谈正事的时候，还不忘挖苦自己几句，这帮人，真的是……叶少阳恨不得把他们都暴打一顿。
<br/>　　大伙商量之后，觉得这是唯一的办法，不过妙心没有让他们急着尝试，而是顺着墙壁慢慢走过去，她不相信，这个墓室会是死胡同，结果绕了一圈之后，还真让她发现了一道暗门。
<br/>　　就在墓室的最里边，有一道紧贴在墙上的暗门，看上去更像是一块石板贴在墙上，四周严丝合缝，无法开启。
<br/>　　门上没有什么机关，也没有什么符印加持，大伙端着烛台上下检查之后，只发现了一个手指粗细的槽子。
<br/>　　“锁孔？”有人发出疑问。
<br/>　　妙心检查了半天，点头说道：“不错，就是锁孔。只有钥匙才能打开。”
<br/>　　大伙一听，当场就傻眼了，谁也没想到，这座古墓里的石门，居然是用这种最原始的方式镇守在这里。
