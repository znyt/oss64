　　二来他在这些兄弟们中间很有威信，小三知道自己一时间压不住人，当老大的话，很可能适得其反。
<br/>　　不过最最重要的原因，是他的自保的心理：有吴连长在前面顶着，将来就算万一出什么变故，第一个倒霉的都是他，自己可以跑，实在跑不掉，也可以改旗易帜，一般来说，师爷型的这种人才，在哪里都是受欢迎的。因此小三这么做，其实是明智之举。
<br/>　　叶少阳趁着他们喝血酒的工夫，凑到翠云的耳边，低声说了什么，翠云震惊地看着他，想要问什么，叶少阳点点头，压低声音说道：“什么都别问，相信我就好。”
<br/>　　十几个土匪喝完了血酒，像是做成了什么大事，个个都很兴奋，说了一些豪言壮语，恨不得这就下山，不过外面还在下雨，只能呆着。
<br/>　　一个士兵表示自己有点饿，于是找老万他们要东西吃。老万他们见证这群逃兵杀人和立志去当土匪的过程，早就吓破了胆，不敢有所隐藏，把背包里的食物都拿出来，给这些逃兵们分吃。
<br/>　　大伙吃着东西，小三凑到吴连长耳边，望着老万等人，说道：“大哥，我们这就要去做土匪，眼前就有一桩没本钱买卖，等着我们去做呢，也算是锻炼一下本领。”
<br/>　　吴连长犹豫了一下，脸色沉下来，走到老万五人面前，抱拳说道：“国家无道，我等也不想给督军卖命了，打算自立山头，缺些经费，还希望诸位老乡赞助则个。”
<br/>　　老万等人一听，顿时就傻眼了，互相望去，一时间说不出话来，这些逃兵，眨眼之间就变成土匪了……
<br/>　　吴连长笑道：“也不让诸位麻烦，大家只要打开包袱，让我等自己检查就好。奉劝老乡们，可别耍小聪明，我这口快枪可不是吃素的。”
<br/>　　然后招呼了两个人，拿着枪走到了老万面前去。
<br/>　　老万老泪纵横，也只好打开了背包，双手颤抖着将钱袋子取了出来。虽然拿他的钱，比割肉还难受，但毕竟还是性命更重要。
<br/>　　拿了老万的钱袋子之后，几个士兵一拥而上，把他的包袱行李翻了个底朝天，将值钱的东西洗劫而空。这才放过他，来到陈毛子面前，陈毛子见事已至此，也没什么好说的，只好把钱都送上。
<br/>　　之后轮到翠云和叶少阳，几个士兵见叶少阳躺在地上，一动不动，用脚踢了踢，说道：“怎么回事？”
<br/>　　“他，他吓昏了。”翠云低着头说道。
<br/>　　几个士兵哈哈大笑起来，也没在意，让翠云把钱拿出来。翠云犹豫着将一个鼓鼓的钱袋子拿出来，苦苦哀求给他们留下一些。
<br/>　　士兵们提着钱袋子，将里面的钱都倒在了地上，是一堆大洋，至少有几百个，几个人眼睛都看直了，小三也走过来，问翠云为什么有这么多钱。
<br/>　　翠云双手捂着脸，一边哽咽着，一边告诉他们，“这是我表弟在家里卖了地和房子，拿着钱想去江西奔亲戚谋个出路的，全在这里了，老总们多少给我们留一点啊……”
<br/>　　几个士兵当然不理，一个士兵指着躺在地上的叶少阳，笑道：“出路是有的嘛，可以跟我们一起做好汉，只不过你这男人胆子太小，那是不行的。”
<br/>　　将钱袋子装好，几个士兵最后来到二蛋面前，二蛋早已经将钱袋子翻出来，却只有寥寥几块大洋。
<br/>　　一个士兵立刻抬起枪，用枪口顶着他，说道：“你不老实！”
<br/>　　二蛋噗通一声跪在地上，苦苦哀求道：“老总们，我真的没钱了，我本来是出来找活干的，结果我老爹给的钱都让我在局子里输了，我这是实在没钱，才想到出去躲难……老总们明察，小人是真的没钱了。”
<br/>　　猛然间，二蛋想到什么，说道：“我跟你们上山，反正我也是烂命一条，我要入伙，跟你们一起上山，求老爷们成全。”
<br/>　　二蛋一边说一边对着面前几个士兵们磕头。
<br/>　　用枪顶着他的那个士兵，转头探寻地朝吴连长和小三望过去。
<br/>　　“不要这样没骨气的。”小三做了一个枪毙的手势。
<br/>　　二蛋看到之后，不等面前的士兵动手，哭着叫道：“老总们，我有线索提供啊，我边上这个姑娘，她身上有三根小黄鱼啊！”
<br/>　　士兵们一听，登时震住。
<br/>　　翠云瘫软在地上，低着头，浑身瑟瑟发抖。
<br/>　　小三看了她一眼，斥道：“扯淡，一个普通妇女，身上怎么会有三条小黄鱼！”
<br/>　　二蛋哭着说道：“是真的，老总，她路上跟她表弟说话，我听见了，她说她把三条小黄鱼缠在腰上了……总不能是吃的小黄鱼吧……”
<br/>　　“二蛋你不是人！”翠云盛怒之下，大声骂了起来。
<br/>　　“还真有货！”小三也感到吃惊，让大家闪开，自己走到翠云面前，盯着她臃肿的腰身说道：“自己拿出来吧，还等什么！”
<br/>　　翠云捂着脸哭，替叶少阳不舍得那笔钱。
<br/>　　小三也没说什么，头也不回地说道：“杀了她男人。”
<br/>　　“我拿，我拿！”翠云哭着趴在叶少阳身前，掀起衣服，解开了一圈又一圈的麻布条，从里面拿出了三块黄澄澄的金条。
<br/>　　“嚯！还真有家伙！”小三将三条小黄鱼拿在手里，掂量了一下，是真金，于是随后扔给身后的兄弟，对二蛋道：“多亏了你。”
<br/>　　二蛋嘿嘿傻笑，磕头如捣蒜。
<br/>　　小三抬起手，拉了枪栓，对着他就是一枪，把脑袋当场就轰碎了。
<br/>　　“啊！”翠云再度尖叫起来。
<br/>　　那几个士兵也吓了一跳，问小三：“干什么杀了他？”
<br/>　　“不光要杀他，这里所有人都要杀光，他们毕竟已经看过我们的样子，也知道我们的来历。”
<br/>　　这句话说完，老万和陈毛子嗷嗷大哭起来，跪在地上磕头求饶。
<br/>　　小三盯着翠云的腰身，在那些麻布条解开之后，她修长的腰身显露出来。
<br/>　　小三道：“抬起头来。”
<br/>　　翠云不敢不从，只好把脑袋抬了起来。
<br/>　　因为淋了一场雨，她脸上用来刻意隐藏容貌的妆容早就花了，被雨水淋掉。
