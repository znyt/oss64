　　不出意外的在和BK战队的比赛中，雷霆战队以3:1的战绩取胜了对手，拿下了两分。
<br/>　　看到这个结果叶帝几人都非常的满意，毕竟雷霆战队也是华夏战队的一员，他们自然希望对方取得更好的成绩，这样的话也更能证明华夏赛区的崛起。
<br/>　　后面的一场比赛叶帝几人倒是并没有继续关注下去，虽然也都是强队，但是没必要一场不落的全都看完，其实如果这一场比赛不是雷霆战队的话，他们也不会守在电视前的。
<br/>　　回到训练室后，众人依旧是在那里抓紧的训练着，而在他们训练的时候，无数的网友则是期待起了明天的荣耀蓝白的对决。
<br/>　　bo1的比赛和bo5的比赛完全是不同的两种赛制，就好比ST战队来说，他们在bo1的比赛中也输过，而且还经常输给一些中等的队伍，当然了这说的是其他的赛事，并不是夏季赛。
<br/>　　而在bo3或者是bo5的比赛中，ST在夏季赛却是一场比赛都没有输，所以说ST更擅长的其实是这种长时间的战斗。
<br/>　　也许bo1的比赛并不足以让他们找到手感，这也是为什么常常会在bo5的比赛中看到ST先输掉两场比赛，然后打出让二追三的奇迹。
<br/>　　也许在这种长时间的比赛中，他们就是需要前两场比赛来找手感呢。
<br/>　　所以对于明天的比赛，叶帝他们也不好说就一定能够取胜对手，毕竟到了这个层次，大家就算是有实力上的差距，那么差距也不会太大的，更不用说其实他们的战队也有短板呢。
<br/>　　一天的时间眨眼便是过去，在无数观众的期待下，在无数国内职业战队的瞩目下荣耀蓝白的比赛终于是到来。
<br/>　　吃过午饭后，蒋念一行人便是先行离去，蓝队的比赛是在白队之前进行的，而且因为是bo5赛制，所以叶帝等人并不着急。
<br/>　　加上这个时间段他们还需要在磨合一下手感，所以自然不能去的太早。
<br/>　　比赛还没开始，微博还有直播间里无数的观众便是已经疯狂的议论了起来，第一场比赛是荣耀蓝队和雷霆战队的比赛，都是华夏的战队。
<br/>　　对于双方的粉丝来说这还好一点，他们都是各自支持着自己的战队，难就难在那些中立的粉丝了，都是国内的，这个时候他们也不知道该支持谁。
<br/>　　“这么早就遇到了啊，我都有点不知道该支持谁了。”
<br/>　　“是啊，如果蓝队3:0雷霆战队，那样的话虽然蓝队会加三分，但是雷霆战队同样会减掉三分啊，这样的话对于雷霆来说真的有点不好啊。”
<br/>　　“就算是3：2同样也不怎么好啊，这样的话赢下来的那个队伍积分拉不开，说不定到时候就取得不了什么好名次了呢。”
<br/>　　“总之加油吧，具体还是要看双方的对决，好在雷霆战队昨天取得了两分，而且之前还是以小组第一的名次晋级的，现在他们手中有四分，就算是扣三分他们的积分也还是正的，不算丢人。”
<br/>　　“谁赢谁输还不一定呢，蓝队还不一定赢呢，说不定他们就阴沟里翻船被雷霆战队给拿下了呢。”
<br/>　　无数的观众在那里议论着，而就在他们的议论声中，双方的比赛终于是正式开始。
<br/>　　不得不说雷霆战队经过世界赛的洗礼，实力真的是又提升了一个档次，在和蓝队对决的时候，居然在第一局就成功击败了蓝队。
<br/>　　陆伟一手龙王玩的神乎其神，几次游走下来，直接将蓝队的核心下路给搞崩了。
<br/>　　“看来小伟这一阵子没少练习龙王啊，对于这个英雄理解的挺深的。”叶帝感叹的说着。
<br/>　　龙王本身就不是一个吃操作的英雄，所以几天的时间陆伟练会这个英雄，叶帝倒是并不稀奇，稀奇的只是对方对于这个英雄的理解。
<br/>　　毕竟中路游走是需要大局观的，什么时间去哪里，什么时候叫着自家的打野一起去gank，这些都很有说道的，而这一局陆伟和小刀两人的配合，确实是让叶帝比较吃惊的。
<br/>　　不过接下来在蓝队将雷霆战队的龙王针对下来之后，蓝队火力重新全开，让一追三，并没有给雷霆战队继续赢下去的机会。
<br/>　　“让我们恭喜荣耀蓝队获得本场比赛的胜利，这一场比赛赢下来，他们获得2个积分，加上原有的积分，现在他们已经是4分在手，而雷霆战队这边则是要扣掉两个积分，现在他们还有2分在手。”解说席上米粒开口说道。
<br/>　　“现在上半区是ST和KA两支战队各积3分，并列第一，昨天最后一场比赛KA战队3:2取胜了QM战队，加上之前KA战队是小组第一出现的，所以此刻他们手中有三个积分，而下半区这一场比赛结束过后，荣耀蓝队则是以4分的成绩暂列第一，当然了比赛才刚刚开始，这一切都是暂时的，稍后让我们期待荣耀白队和eP两支战队的精彩对决。”
<br/>　　随着林雅楠的开口，直播画面进入了短暂的广告画面。
<br/>　　而这个时候叶帝他们已经是到达了现场，在蒋念他们和雷霆战队打第三场比赛的时候，叶帝他们就已经动身了，来到现场后还完整的观看完了双方的第四场对决。
<br/>　　随着两支战队的比赛结束，叶帝等人便是向着舞台上方走来，等到他们走到台阶的时候，正巧碰到了下来的蒋念还有陆伟他们。
<br/>　　“比赛加油啊，我们在场下为你们打气。”笑着也叶帝拍了个手，蒋念便是走了下去。
<br/>　　这个时候叶帝他们还要进比赛室调试设备，热热手，所以他自然是不会耽搁叶帝几人的时间。
<br/>　　接下来秦霄还有陆伟小刀他们也都是如此，一个个都是和叶帝等人拍手，为他们加油。
<br/>　　等到叶帝等人身穿着红色的队服走到舞台上的时候，下方顿时传来了一阵热烈的欢呼声。
<br/>　　朝着台下的众人挥了挥手，叶帝等人便是向着比赛室内走去，进入比赛室的时候叶帝还看了一眼另外一边上台的eP战队等人，笑着和他们打了个招呼。
<br/>　　之前在十王者战的时候，他们就已经和emp比较熟悉了，大家私下里也经常的联系，所以几人的感情还是比较好的。
