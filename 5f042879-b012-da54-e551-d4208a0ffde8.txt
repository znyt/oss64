　　经过三个多月的飞行，苏莫来到了时空之城所在的极空城。
<br/>　　极空城还是往日的模样，非常的繁华，时空之城所在的时门和空门，依旧悬浮于半空之上。
<br/>　　下方地面上，聚集了密密麻麻的人，少数也有十几万之众。
<br/>　　时门和空门，均是九天开启一次，很显然，这些人均是在等待时门和空门的开启。
<br/>　　苏莫稍一询问了一番，便得知，距离时门开启，还有三天的时间。
<br/>　　随后，他也没有离开，走到一处无人注意的角落之处，静静的等待了起来。
<br/>　　这里强者极多，不乏深不可测的强者，他也是不得不小心。
<br/>　　“嗯？”
<br/>　　就在此刻，苏莫的眼角余光，发现了一位熟识之人，来到了此地。
<br/>　　此人不是别人，正是之前被苏莫击伤逃走的许焱。
<br/>　　“他怎么在这里？”苏莫心中一惊，眉头忍不住一皱，因为他看到，许焱身边还有一位青年，貌似实力很强。
<br/>　　毫不犹豫，苏莫催动运转天幻神丹，在外界之人的眼中，他的相貌和气息，再次改变。
<br/>　　改变了容貌和气息，苏莫便自顾自的盘膝而坐，静静的打坐了起来。
<br/>　　人德宫主说过，除了神王境的强者，没有人能看穿他的真实面目，所以他毫不担心。
<br/>　　……
<br/>　　许焱和慕容雷利两人，伫立在人群之中，目光环视众人，神念弥漫八方，在寻找苏莫。
<br/>　　他们两人，随着苏莫残留的气息，一路从神剑山跟到了极空城。
<br/>　　苏莫在极空中停下，让他们稍稍的松了口气。
<br/>　　不过，查看了片刻之后，两人并没有找到苏莫。
<br/>　　“慕容师兄，此地人太多了，各种气息杂乱无比，无法找到秦陨！”许焱眉头紧皱，他完全感知不到苏莫的存在。
<br/>　　慕容雷利阴沉着脸，他同样如此，此地不仅人多，各种强大的气息互相交织在一起，很难感知到秦陨的所在。
<br/>　　少倾，慕容雷利身形腾空而起，飞上了半空之中。
<br/>　　他强大的神念，笼罩全场，目光更是仔细的扫视下方的人群。
<br/>　　但是，无论是神念，还是靠目力，他都无法找到秦陨。
<br/>　　即便是他的神念，将整个极空城笼罩，也没有发现秦陨的踪迹。
<br/>　　“怎么可能？”慕容雷利的脸色黑了下来，根据那秦陨残留的气息，对方明明是来到了这极空城，为何找不到呢？
<br/>　　而现在，时门和空门并没有开启，对方也不可能进入时空之城。
<br/>　　“慕容师兄，我们怎么办？”许焱来到了慕容雷利的神身前，面色难看的问道，看对方的脸色，他也知道对方也没有找到。
<br/>　　“这不是慕容雷利吗？”
<br/>　　“是他，终极神王殿的万古天骄！”
<br/>　　“传闻慕容雷利雷系天赋无比逆天！”
<br/>　　很多人认出了半空中的慕容雷利，纷纷投去了惊讶的目光。
<br/>　　“哈哈哈！慕容兄来到时空界域，居然不事先通知一声！”
<br/>　　就在此刻，一声爽朗的笑声突然响起，只见不远处的人群，哗啦啦的分开了，一群人大步的走了过来。
<br/>　　这群人，足有十数人，个个气息强大，为首一人，是一位身穿华服，面容俊逸的年轻男子。
<br/>　　“原来是公孙兄！”
<br/>　　见到这位华服青年，慕容雷利不敢怠慢，立刻微微抱拳，并和许焱一起，降落了在了对方身前。
<br/>　　这位华服青年，名为公孙嘉，乃是时空界域的霸主势力永恒神宫的弟子。
<br/>　　不仅如此，公孙嘉在永恒神宫地位非凡，爷爷正是永恒神宫的宫主。
<br/>　　而公孙嘉的天赋，同样不凡，位列神域万古天骄之一。
<br/>　　“慕容兄，时空之城开启，还要三日，要不，我们去小酌几杯如何？”
<br/>　　所谓天才之间，惺惺相惜，公孙嘉面带微笑，向慕容雷利发出了邀请。
<br/>　　“这……！”
<br/>　　慕容雷利略一迟疑，随即向公孙嘉传音道：“公孙兄，实不相瞒，我在找一个仇人，此人刚刚进入了极空城，但是却莫名其妙的消失了！”
<br/>　　因为担心苏莫就藏在附近，所以，慕容雷利用意念传音与对方交流。
<br/>　　“哦？还有这种事？”公孙嘉闻言，顿时一怔。
<br/>　　“公孙兄，此人应该就在附近，我怀疑其隐藏了起来，不知公孙兄能否帮助？”慕容雷利传音道，对方可是时空界域的地头蛇，如果能得到对方的帮助，那就再好不过了。
<br/>　　“对方是什么身份？”公孙嘉皱着眉头传音问道，他可不会因为认识慕容雷利，就贸然出手帮助。
<br/>　　毕竟，要是碰上了硬茬子，那就得不偿失了。
<br/>　　“此人名为秦陨，乃是我终极神王殿的一个逆徒，残杀同门！”慕容雷利传音道。
<br/>　　“原来如此！”
<br/>　　公孙嘉闻言恍然，略一沉吟，传音道：“这样吧慕容兄，我命人封锁极空城，所有出城之人都要经过严格盘查，这样你就不担心此人逃走了！”
<br/>　　公孙嘉没有说帮助慕容雷利寻找，只是封锁极空城，毕竟两人关系并不深。
<br/>　　“如此甚好！”慕容雷利立刻向对方抱拳，对方虽然不帮忙寻找，但是只要封锁了极空城就足够了。
<br/>　　“嗯！”
<br/>　　公孙嘉微微点头，随即转头向身边的下属传音，便将事情吩咐了下去。
<br/>　　永恒神宫掌控整个时空界域，公孙嘉想封锁一城之地，实在是太容易了。
<br/>　　随即，那名下属火速的离开了，去通知极空城的城主，封锁极空城。
<br/>　　而后，公孙嘉和慕容雷利两人，又随便闲聊了几句之后，便一起离开了。
<br/>　　望着慕容雷利、许焱和公孙嘉等人离去，苏莫眉头一皱，他不知道几人说了什么，但是心中却也升起不好的预感。
<br/>　　他虽然不认识慕容雷利，但心中也清楚，对方是和许焱一起来找他的。
<br/>　　不过，他也没有太过担心，只要没有神王境的强者插手，他就不用担心。
<br/>　　神王境的强者，屈指可数，一个强大的界域，也就三五人而已。
<br/>　　少倾，苏莫不再多想，继续打坐，等待时空之城开启。
<br/>　　而慕容雷利和许焱两人，与公孙嘉一起离开之后，便一边闲聊，一边去了城主府。
<br/>　　极空城在城主的命令之下已经被封锁，上空有无形的禁空阵法，出城要经过严密的盘查。
<br/>　　许焱向极空城城主，反应了苏莫的具体容貌和气息，已经修炼的道统等等。
<br/>　　只要发现和苏莫类似之人，就会被立刻拿下。
<br/>　　一切安排妥当之后，慕容雷利和公孙嘉等人，进入了一座豪华的酒楼之中。
<br/>　　“慕容兄，既然来了，何不去时空之城中？”酒楼的豪华包间之中，公孙嘉一边饮酒，一边笑着问道。
<br/>　　“抓到那秦陨更重要！”慕容雷利叹了口气，他事先没有想到，抓住秦陨居然如此的麻烦。
<br/>　　“呵呵，你无需担心，听你们之言，此人修为不过二品，虽然战力堪比三品，但不可能逃出极空城！”
<br/>　　公孙嘉摆了摆手，不在意的道：“还有三日，时空之城便开启了，不如我们去时门之中比试一番如何？”
<br/>　　“比试？”慕容雷利闻言一怔。
<br/>　　“当然，并不是切磋实力，时门之中三十三重时光神殿，看看谁能走的更远！”公孙嘉补充道，他的修为和慕容雷利相当，想要看看谁的悟性更高。
<br/>　　“好吧，希望出来之后，那秦陨已经被擒拿！”慕容雷利无奈的点了点头，既然秦陨逃不走，他也不急于一时。
<br/>　　况且，时空之城的确是宝地。
<br/>　　“那好，到时候我们二人，一起进入时门。”公孙嘉朗笑一声，向慕容雷利举杯。
<br/>　　随即，两人碰杯，一饮而尽。
