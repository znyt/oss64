　　炼器师公会的建筑，形似塔楼，足有十几层之高。
<br/>　　高瘦炼器师带着苏莫，很快便从一楼来到了二楼，而后继续向三楼走去。
<br/>　　二楼同样是一个大殿，同样有大量的水晶柜台，也有少量的武者在挑选兵器。
<br/>　　苏莫只是看了一眼便心中震惊，因为他发现那些柜台内的宝器，居然个个都疑似尊器，强大无比。
<br/>　　每一件宝器都流光异彩，显得气势逼人。
<br/>　　“这些宝器不知道都是什么价格？”苏莫心中暗道，这些售卖的宝器并没有标注价格，但想必不会便宜。
<br/>　　很快，苏莫和秦不死跟着高瘦炼器师来到了三楼。
<br/>　　三楼不再是大殿，而是一个长长的走道，寂静无声，两边布满了一个个房间。
<br/>　　高瘦炼器师来到三楼之后，脸色郑重了起来，走到第一个房间门前，而后深深的吸了口气，才敲响了房门。
<br/>　　咚！咚！咚！
<br/>　　沉闷的敲门声响起，高瘦炼器师敲的很轻，面色非常的恭敬。
<br/>　　“进来吧！”房间内传来浑厚的男声。
<br/>　　随即，高瘦炼器师对苏莫示意，便轻轻推开房门，走进了房间中。
<br/>　　苏莫和秦不死立刻跟了进去。
<br/>　　房间很大，其内仿佛是一个小型的宫殿，客厅和练功房、炼器室等应有尽有。
<br/>　　客厅主座之上，端坐着一名面容黝黑的中年人，中年人身穿灰白色长衫，长衫的胸口处还绣有两个银色的标志。
<br/>　　这两个标志分别是一个炼器熔炉，和一个古朴的‘尊’字
<br/>　　苏莫见此心中一震，这个‘尊’字是表明此人是一名武尊境的尊者？还是说此人是尊级炼器师？
<br/>　　这里是炼器师公会，想必必是后者，此人是一名尊级炼器师。
<br/>　　“拜见柴管事！”高瘦炼器师走进客厅，立刻向中年人抱拳行礼。
<br/>　　苏莫和秦不死不敢怠慢，同样抱拳行礼。
<br/>　　“卢云，何事？”柴管事目光扫了一眼苏莫和秦不死，随即向高瘦炼器师问道。
<br/>　　高瘦炼器师的名讳，便是叫卢云。
<br/>　　柴管事的声音威严无比，身上虽没有气息显露，但自有一种上位者的威严。
<br/>　　“禀柴管事，这位说自己是帝级炼器师，要加入我们炼器师公会！”高瘦炼器师卢云面色恭敬，一指身边的苏莫。
<br/>　　“哦？”柴管事闻言，打量了苏莫一番，顿时眉头微微皱了起来。
<br/>　　这么年轻的帝级炼器师？
<br/>　　卢云见此，顿时心中咯噔一声，急忙道：“柴管事，我也不太相信他是帝级炼器师，但他坚称自己是帝级炼器师，我也不好把他轰出去，便将他带来见您了！”
<br/>　　卢云说的有板有眼，但若不是苏莫给了他好处，他怎么可能会为苏莫引荐。
<br/>　　他现在这样一说，若是待会苏莫出丑，柴管事也不会怪罪他了。
<br/>　　柴管事闻言微微颌首，随即面带惊疑的向苏莫问道：“阁下真的是帝级炼器师？”
<br/>　　他的确是有些不太相信，苏莫会是帝级炼器师，因为他能一眼看出苏莫的年纪，不超过三十岁。
<br/>　　不超过三十岁的年纪，成为帝级炼器师，这种炼器天赋实在非常骇人。
<br/>　　整个浩瀚的天荒星域，这种天赋的炼器天才，都是屈指可数。
<br/>　　柴管事作为炼器师公会此处分会的管事，负责考核前来加入的炼器师，有些炼器水平不行，却夸大其词的人，他见的多了！
<br/>　　“正是！”苏莫郑重的点了点头，脸不红心不跳。
<br/>　　卢云见苏莫一脸坦然的模样，心中徘腹不已，这特么太会装了吧？
<br/>　　你现在这么会装，看你等会怎么下台！
<br/>　　卢云压根不信苏莫是帝级炼器师，不然的话一开始也不会想做学徒了！
<br/>　　“那好，只要你能炼制出一种帝级宝器，便能通过考核，成为炼器师公会的炼器师。”
<br/>　　柴管事点头，随即长身而起，道：“老夫带你去考核炼器室。”
<br/>　　不管苏莫所说是否属实，只要一试便能明了，考核做不得假。
<br/>　　言罢，柴管事大步而行，便准备向考核炼器室而去。
<br/>　　卢云面带笑意，转身跟随柴管事向外走去，他现在是坐等苏莫出丑了！
<br/>　　“柴管事且慢！”就在此时，苏莫却是突然出声，制止了准备出去的柴管事。
<br/>　　“何事？”柴管事闻言，顿时停下了脚步，疑惑的问道。
<br/>　　苏莫咬了咬牙，向柴管事抱拳道：“柴管事，能不能过几日再行考核？”
<br/>　　他现在根本炼制不出帝级宝器，只能想办法拖延时间了。
<br/>　　“为何？”柴管事脸色沉了下来，他现在已经有些怀疑苏莫是来拿他消遣的了。
<br/>　　过几日再行考核？
<br/>　　是炼器水平达不到，不敢去接受考核了吧！
<br/>　　柴管事心中微怒了起来，若真是如此，他不会客气。
<br/>　　这种事情若是不严惩，不足以震慑四方，不然什么歪瓜裂枣都来考核，那他什么也不用做了！
<br/>　　卢云见此，差点忍不住笑出声来，不是吧？这么快就退缩了？连去考核的勇气都没有？
<br/>　　若不是柴管事在此，卢云会大笑三声，然后狠狠的羞辱一番苏莫！
<br/>　　就连一直在一旁默不作声的秦不死，都是忍不住老脸一红，脸皮不停的抽搐了起来。
<br/>　　这也太丢人了吧！
<br/>　　苏莫深吸一口气，心中组织了一番语言，满嘴胡诌道：“晚辈在来公会的路上，遭到歹人袭击，精神力大损，现在怕是难以炼制出帝级宝器！”
<br/>　　“精神了大损？”柴管事闻言眉头一皱，心中无语的摇了摇头，炼制不出帝级宝器，居然连这种借口都编出来了!
<br/>　　不过，这倒的确是个好借口，让他不好追究戏弄之责。
<br/>　　“既然如此，你走吧！老夫也不为难你了！”柴管事挥了挥手，示意苏莫离开。
<br/>　　苏莫当然不会就此离去，他立刻向柴管事抱拳，道：“柴管事，晚辈能否在公会中暂住几日，很快就能恢复精神力，到时候就可以考核了！”
<br/>　　苏莫现在的打算，就是先在公会中住下来，一则可以暂时躲避巫族两位尊者的杀机，二来可以抓紧时间学习炼器之术。
<br/>　　只要能有个把月的时间，他还是有希望炼制出帝级宝器的。
<br/>　　只要能炼成帝级宝器，就能成为公会的炼器师，他和秦不死就有了靠山，性命就可无忧了。
<br/>　　那两名巫族尊者，再想杀他，也会顾及他背后的炼器师公会吧！
