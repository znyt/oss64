　　特种部队所执行的任务是多样性的，在行动中，他们可能会遇到各种各样难以预料的情况，可是不管遇到什么样的情况都必须要保持冷静，即便是面对至亲之人，也要保持冷静。
<br/>　　因为行动中的冲动，不仅有可能会导致行动的失败，还有可能会使身边的队友陷入到危险中，甚至是全军覆没。
<br/>　　罗昊能理解慕容飞现在的内心状态，因为当年罗昊自己也曾经历过。当年为了追击费路南，过于深入敌穴，导致蓝若雪被俘，当时包括姜伟在内绝大部分人要求继续追击，把蓝若雪救回来，可罗昊最终还是保持了作为指挥官的冷静，命令小队撤退，不然的话，很有可能会导致小队全军覆没。
<br/>　　罗昊的决定保住了小队其他队员，但是却失去了蓝若雪，也最终导致了他跟姜伟之间的死结。
<br/>　　如果当时罗昊没有做出让小队撤退的决定，他不可能留在龙隐部队，也不可能成为现在的战术中队队长，即便他的师父是赵天宇也改变不了什么。
<br/>　　本来罗昊这次是不建议慕容飞参与这次行动，可是综合了各种考虑，罗昊还是让慕容飞参加了这次行动；如果慕容飞在这次行动中不能保持，罗昊一定会让慕容飞离开战术中队。
<br/>　　面对自己的至亲之人遇到危险无法保持冷静，慕容飞就已经失去了作为一个狙击手和战术中队队员的资格！
<br/>　　罗昊的话，让周围其他战术B组队员都沉默了下来，他们谁都不希望事情发展到这一步。
<br/>　　“走！”
<br/>　　罗昊抬起手向前一挥，示意继续前进。
<br/>　　……
<br/>　　“A组就位。”
<br/>　　齐凡带着战术A组抵达村子外围后，观察着村子里的情况，通过耳麦对罗昊说道：“暗杀韦斯利失败，布克尔特应该是有所察觉，村子里的守备情况严密的了很多。”
<br/>　　“我发现了。”
<br/>　　罗昊放下手中的MC迷彩紧凑型HK416自动步枪，按住耳麦说道：“布克尔特这家伙到底是专业的，而且他手下又有这么多从特种部队退役下来的老兵，他如果连这点警觉性都没有的话，他也不可能在南美洲有这么大的势力。C组，你们那边情况怎么样？”
<br/>　　“路上遇到了一组埋地雷的武装分子，解决掉了他们，耽搁了一些时间。”梅庆伟回答道。
<br/>　　“没给他们留‘礼物’？”罗昊通过耳麦对梅庆伟问道。
<br/>　　“必须的。”梅庆伟回答道：“留了一份‘大礼’给他们。”
<br/>　　“等待行动信号！”
<br/>　　罗昊向战术A组和战术C组发布了等待的命令后，通过手机向比洛多发送了一条短信。
<br/>　　嗡！嗡！
<br/>　　比洛多放在办公桌上的手机轻轻震动了两下，比洛多拿起手机看了眼上面罗昊发送给自己的短信，对自己的副官说道：“新闻发布会可以开始了。”
<br/>　　“是！”
<br/>　　副官把军帽递给比洛多，对着比洛多敬了个礼，转身走出了房间。
<br/>　　比洛多站在镜子前，整理了一下自己的仪表，走出了房间，开始召开新闻发布会。
<br/>　　新闻发布会是直播的，比洛多在新闻发布会上直接公布了巴坦伦在秘密别墅包养情人的事情，以及公开了巴坦伦跟他情人上床的视频，在新闻发布会现场顿时引起了巨大的轰动。
<br/>　　“妈的！”
<br/>　　秘密别墅中，巴坦伦将自己手中的遥控器仍在挂在墙壁的液晶电视上，把液晶电视砸裂开来，遥控器也被砸得粉碎，浑身的肥肉不断的抖动着，狠声说道：“比洛多怎么会有我们在这里的视频，他怎么可能有我们在这里的视频！？”
<br/>　　巴坦伦这话像是在问他自己，也像是在问坐在沙发上的詹弗妮。
<br/>　　“比洛多这混蛋一定是雇佣了雇佣兵来拍摄了我的证据，他想要破坏我的大选，做梦！”巴坦伦对着站在客厅中的警卫队长吩咐道：“立刻准备直升机，我要去新闻发布会现场！想要破坏我的大选，把我拉下来自己当总统，做梦！”
<br/>　　丛林中，天蛇通过无人机看到巴坦伦行色匆匆的走出别墅，朝着别墅直升机停机坪中的直升机走去，把无人机重新又收了回来，对莱昂说道：“这个死胖子果然是要坐直升机。”

<br/>　　布谷退出弹壳，用手接住，把弹壳塞进口袋中，对着莱昂做了一个“oK”的手势。
