　　“亏得你帮忙，我才这么容易击杀你家太上大长老。”
<br/>　　萧逸看着面前的火牙宗主，狞笑一声。
<br/>　　“你…”火牙宗主眼眸一睁，瞪得浑圆。
<br/>　　他自是看不到萧逸面具之下的脸色，但单听这狞笑声下的阴冷之音，便可知面具之下，必是一张如恶魔般狞笑的面容。
<br/>　　对，从始至终，萧逸都没有说过他的手套是中品巅峰圣器。
<br/>　　说出‘中品巅峰圣器’六个字的，是火牙宗主。
<br/>　　事实上，星幻手套并非是正儿八经的上品圣器。
<br/>　　原本的星幻手套，只是中品巅峰圣器；后残损，沦落为下品圣器，流落至星换城。
<br/>　　萧逸得之，渐渐将其恢复至巅峰状态。
<br/>　　后来，在星幻瓶的帮助下，星幻手套直接突破了桎梏，达到了上品圣器的阶品。
<br/>　　那星幻瓶，本就是星光洞府之内十八把上品圣器之一；自然的，星幻瓶内的星光之力，足够强猛。
<br/>　　简单来说，星幻手套，原本只是中品巅峰圣器的材质，只是靠着强猛的星光力量，才冲破了限制，达到上品层次。
<br/>　　故它不是正儿八经的上品圣器。
<br/>　　火牙宗主一时眼拙，便也误以为是中品巅峰圣器。
<br/>　　那老者，也就是太上长老，也因此没有细细感知，下意识地认为是中品巅峰圣器。
<br/>　　这才有了最后的交手时，萧逸忽然爆发，直接取了他性命。
<br/>　　若非如此，萧逸即便能赢，恐怕最后也要麻烦许多。
<br/>　　另外一点，星幻手套，虽然不是正儿八经的上品圣器，但毕竟是已经位列上品阶品。
<br/>　　同时，这一手套，对他控火能力上的增幅，绝不比其他上品圣器差。
<br/>　　啪…
<br/>　　这时，火牙宗主的身体，已无力跌落。
<br/>　　他已然身死，身上没有半滴鲜血流出。
<br/>　　身上鲜血，早便在萧逸那穿身的火焰拳头上，被焚烧殆尽。
<br/>　　他的尸体，已倒下，但那双瞪大的眼睛，一直合不上，显然是死不瞑目。
<br/>　　或许，他死前的最后一个念头，便是若他没有‘误导’了太上长老，太上长老不会那么轻易死去。
<br/>　　那今日，他也不会死去，死的，会是紫炎易霄。
<br/>　　当然，这也只是他的想法。
<br/>　　萧逸的目光，此时看向那十数邪修长老。
<br/>　　这些层次达到9990道和9991道的邪修长老，原本有数十人。
<br/>　　但在山腹之内时，便也被萧逸杀了过半，此刻只剩十数。
<br/>　　这十数邪修长老，注意到萧逸的目光，霎时面露慌色。
<br/>　　萧逸杀火牙宗主，只是一瞬间的事。
<br/>　　这些邪修长老，甚至还未来得及反应。
<br/>　　嗖…嗖…嗖…
<br/>　　萧逸的身影，在空气中留下数道残影。
<br/>　　待得残影消失，这十数邪修长老胸膛上，同时出现了一道火焰印记。
<br/>　　萧逸指尖轻扬，一股爆响从这十数邪修长老身上爆发。
<br/>　　爆响声落下，十数邪修长老，轰然倒地。
<br/>　　死状，与火牙宗主一模一样，身上鲜血早已被焚烧殆尽。
<br/>　　“还剩你们。”萧逸的目光，看向火鸦大阵屏障边缘的过万邪修。
<br/>　　这过万邪修，原本在维持着大阵，困住了所有武者。
<br/>　　他们，同样来不及反应。
<br/>　　萧逸的动作太快了。
<br/>　　在他们眼中，数秒前，太上长老还在与易霄战斗着。
<br/>　　可数秒后，一切忽然翻天覆地般变化。
<br/>　　太上长老死了，宗主死了，连来自各地的邪修长老也死了。
<br/>　　“不好，快跑。”忽然，一个邪修率先反应过来，惊呼了一声，连忙御空飞离。
<br/>　　几乎是这个邪修离开的瞬间，其余邪修，也一并反应了过来。
<br/>　　过万邪修，纷纷御空飞离，再也未管火鸦大阵。
<br/>　　偌大个火鸦大阵，瞬间消散。
<br/>　　“跑？”萧逸狞笑一声。
<br/>　　轰…一股天地武道之力，顷刻蔓延。
<br/>　　嘶…嘶…嘶…
<br/>　　也几乎是这一瞬间，才刚刚御空飞起的一众邪修，忽然化作一具具干尸，从高空跌落。
<br/>　　过万邪修，同时从高空跌落，那是怎样的场景？
<br/>　　宛若天上下‘人雨’？
<br/>　　若是这些跌落的身影，是一具具干尸呢？
<br/>　　“这…”一众武者，呆愣在地。
<br/>　　他们原本沉浸在火牙宗主等人身死，还有火鸦大阵消散的惊喜中。
<br/>　　可现在，这一具具跌落的干尸，却让他们心头一阵胆寒。
<br/>　　那个站在无数干尸中间的淡漠身影，这一刻，成了所有人的焦点。
<br/>　　这个戴着面具，看不清面容的年轻人，这一刻，宛若一个狰狞的恶魔。
<br/>　　萧逸，看着周遭尸体，甚至没有半分皱眉。
<br/>　　对于邪修，对于邪君府，萧逸不会有半分手软。
<br/>　　有些事，萧逸不愿搭理，也懒得麻烦。
<br/>　　但有些事，触及逆鳞，那便是不死不休。
<br/>　　萧逸那张俊朗的面容，会慵懒，会冷酷，也会冷漠。
<br/>　　但在极少极少的情况下，亦会狰狞如魔，但那时，也便是敌人连后悔的机会都没有。
<br/>　　那块逆鳞，邪君府从胆敢触碰的一刻起，便注定了萧逸与他们的不死不休。
<br/>　　嗖…
<br/>　　萧逸大手一挥，收走了所有邪修尸体的乾坤戒。
<br/>　　恰在此时，萧逸的手臂，忽然停在半空。
<br/>　　眉头，蓦地一皱。
<br/>　　冰冷的目光，转移到某个方向。
<br/>　　那里，一道‘黑袍’虚影，不知何时起出现。
<br/>　　呼…
<br/>　　整个火牙山峰，忽然在这一瞬间，阴风大作。
<br/>　　一众武者，在这阴风之下，实力稍弱者，竟是直接站不稳。
<br/>　　“又是你？”萧逸眉头一皱。
<br/>　　这个黑袍虚影，正是前些天在三风地域最后遇到的那道虚影。
<br/>　　“这句话，应该由我来说。”黑袍虚影，语气异常淡漠，“又是你，紫炎易霄。”
<br/>　　黑袍虚影一字一顿，阴冷的眸子，扫视着周遭。
<br/>　　当他看着火牙宗太上长老、宗主，以及各地邪修长老的干瘪尸体时，眼中阴冷，愈发惊人。
<br/>　　最终，目光定格在萧逸身上。
<br/>　　“一月时间，毁我邪君府过百分部，击杀邪修数以万计。”
<br/>　　“今日，又毁火牙分部，焚毁所有武道冰晶，杀尽在场所有邪修。”
<br/>　　“紫炎易霄，你到底想做什么？”黑袍虚影，咬着牙，“真要我邪君府追杀你至亡命天涯不成？”
<br/>　　“追杀？呵。”萧逸淡漠一笑，“我等着。”
<br/>　　“至于我要做什么？想听？”
<br/>　　黑袍虚影闻言，双眼一眯。
<br/>　　萧逸的语气，霎时狰狞，“我要这中域，一万三千地域，再无邪修。”
<br/>　　“有本事你们便躲起来，胆敢冒头，易某见一个杀一个。”
<br/>　　冰冷的话语，清楚地传出。
<br/>　　周遭武者，霎时身躯一震。
<br/>　　黑袍虚影，则眼眸一冷。
<br/>　　“好，很好，易霄，记住你今日的话。”黑袍虚影，似乎在压抑着自己的怒气。
<br/>　　“记住？不。”萧逸淡漠地摇了摇头，“我会让这句话成为事实。”
<br/>　　“滚吧。”
<br/>　　萧逸大手一挥，一道紫炎打出，直接将这道武道虚影轰散。
