
    中年女子走后，王昆仑给王惊蛰倒着茶水，说道“令歌跟我说的事，我只能说帮你过问下，毕竟我姓王而不是姓孔，在孔府我是因为这个女人才留在这里的，而我也无权怼孔府的事指手画脚，特别是和墨家的冲突上，这是可以关乎到孔府利益的问题，所以你得有个心理准备，成或者不成，都是说不准的”<br/>　　<br/>　　    王惊蛰敬了王昆仑一杯茶，点头说道“本来我也没有把孔府算在其中，我能来是因为令歌的提醒，没有他说话，我也不会来，所以孔府能出人那是好事，不出，就当没有这回事，谈不上失落”<br/>　　<br/>　　    “心智不错，跟你姐姐都算一类人”王昆仑点了点头，随后跟王惊蛰喝了会茶之后，他就起身说道“我去找个人过来，剩下的问题你跟他谈”<br/>　　<br/>　　    “唉，好叻，谢谢王叔”王惊蛰拱手说道。<br/>　　<br/>　　    王昆仑离去后，王惊蛰自己喝着茶，没过多久他就带了个四十来岁的中年进到凉亭里，这人长得温文尔雅一脸的书生相，看见王惊蛰站起来后就示意他坐下，开门见山的跟他说了起来。<br/>　　<br/>　　    “儒家和墨家虽然之间的冲突有千年多了，但是到如今这个社会，基本恩怨都已经放下了，为啥现在儒墨的思想谁还会拿出来大谈特谈啊，老祖宗们的恩怨到了现在已经淡很多了，我们也不是非得要对人家喊打喊杀，没完没了的了，和谐社会平稳发展，谁还能针对谁一直到死么，呵呵，法治年代跟以前不一样了”<br/>　　<br/>　　    王惊蛰瞥了眼王昆仑，他就低头品着茶水，没有任何的反应，似乎真就只是起了个牵线搭桥的作用。<br/>　　<br/>　　    王惊蛰点了点头，说道“您说的很有道理，也确实是这么回事，以前的恩怨是不死不休，现在则可能是得过且过了”<br/>　　<br/>　　    王惊蛰的谈判手法几乎等同于一个小学生在和大学生辩论的程度，引经据典，挑明厉害关系等等这些手段，他根本就没什么概念，在他的理念中就是，你能干就干，干不了就拉倒，劝什么劝啊<br/>　　<br/>　　    三人喝着茶，一时间寂静无声。<br/>　　<br/>　　    一壶茶喝完之后，那孔家中年就站了起来，似乎刚要说告辞，王昆仑忽然抬起头说道“德荣，你今天不是没事么，坐下在喝一会茶呗”<br/>　　<br/>　　    “什么我今天没事啊，我那边一堆事要处理呢”<br/>　　<br/>　　    “哎呀，孔家八百个人，啥事都得要你亲自出面啊，你看把你给厉害的，难得偷得浮生半日闲，咱俩也很久没见了，一起聊会天”<br/>　　<br/>　　    孔德荣顿时无语的说道“大哥，昨晚咱俩还喝到十二点多呢，这就是你说的许久不见啊不是，你什么意思啊，话里有话吧”<br/>　　<br/>　　    王昆仑点着王惊蛰说道“小朋友千里迢迢的来一回，你三两句话就给人打发走了，德容啊，孔府千年世家的风范可在你这没有体现出来啊”<br/>　　<br/>　　    “你鼓动我上去”孔德荣皱眉说道。<br/>　　<br/>　　    王昆仑干咳了一声，飘忽着眼神说道“那，要不我去也行最近正好欠活动筋骨，我走一走也可以”<br/>　　<br/>　　    孔德荣手指点着桌子，语气严肃的说道“昆仑，谁都知道你在孔家，你去就是等于我们孔府出面了，你别闹行么”<br/>　　<br/>　　    “谁跟你闹了，我是从朋友的角度出发，拔刀相助”<br/>　　<br/>　　    “不是，你这不是胡搅蛮缠么，我都说了咱们出人不合适，至少得有四五十年了吧，孔家从来都没和墨家有过任何的冲突和接触，你忽然来这么一下子，不是把已经平息下去的矛盾又给掀起来了么”<br/>　　<br/>　　    王昆仑仰着脑袋，眯了眯眼睛说道“德容，我就问问你，如果啊，假如有这么一个机会摆在墨家面前，那就是孔府已经到了苟延残喘的地步，只需要人退一下就进入万劫不复的境地，你说墨家会不会当这个推手，你别说什么不可能有这种事，我只是跟你打个比方”<br/>　　<br/>　　    孔德荣深深的吸了口气，点头说道“不会错过这个机会”<br/>　　<br/>　　    王昆仑两手一摊，说道“你明白就行了，剩下的我不多说”<br/>　　<br/>　　    王惊蛰忽然开口说道“我不需要孔府为我冲锋陷阵，只需要在关键时刻挺我一把就行，毕竟我也不想把人情欠大了，我来趟雷，你们垫后就可以”<br/>　　<br/>　　    孔德荣当即不解的问道“你怎么趟”<br/>　　<br/>　　    王惊蛰指了指桌上的茶壶笑道“茶水都凉了，要不换一壶”<br/>　　<br/>　　    一个多小时后，曲阜天色渐黑，王惊蛰才从孔府中出来。<br/>　　<br/>　　    王昆仑送他到门口，背着手轻声说道“孔府的事到此也就差不多了，剩下的你自己安排，如果还有什么需要的，你可以和我提，从向缺那里，还是从王胖子那，我都有情为你出面”<br/>　　<br/>　　    王晶和顿时摆手说道“谢了王叔，您能帮这个忙已经可以了，再有要求那就是我得寸进尺了”<br/>　　<br/>　　    王昆仑笑道“好，你既然这么说，那就是心里有数了，我不送你了，有空过来喝茶”<br/>　　<br/>　　    王惊蛰离开孔府之后也没急着去滕州，就在曲阜找个地方住了下来，这已经是他得到消息之后的第二天了。<br/>　　<br/>　　    唐梁山上，凉绣站在草庐外，琢磨不定的嘀咕道“王惊蛰来了又走，你这葫芦里到底卖的是什么药呢”<br/>　　<br/>　　    草庐内，茅小草托着腮帮子，静静地看着窗外，也轻声嘀咕道“我可不是败家的女人，不能让我男人白白的来救我一次，怎么着也得要弄点利息再回去吧”
