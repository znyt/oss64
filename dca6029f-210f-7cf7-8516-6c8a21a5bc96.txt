　　叶少阳陡然紧张起来，摆好了姿势，打算他一旦举枪之类的，就立刻冲过去弄死他。
<br/>　　只有他一个人，就算有枪，叶少阳自信弄死他也很轻松。
<br/>　　但是小三没有拔枪，为了表示自己的诚意，他甚至摊着两手走过来，叶少阳有点疑惑，只好站着不动。
<br/>　　“我把抢你的钱给你。”小三说着，把手插进口袋里，叶少阳立刻紧张了一下，不过小三摸出来的不是枪，而是一个布包，打开之后，里面是黄灿灿的几根金条。
<br/>　　“上次抢你的钱，还给你。”小三把布包扔给叶少阳，笑了笑道：“不打不相识。”
<br/>　　他笑得很温和，甚至让人有点如沐春风的感觉，如果不是见过他的另一面，叶少阳几乎会被他这副和善的模样给欺骗了。
<br/>　　能伪装成这样的笑容，足以证明此人城府极深。
<br/>　　不过给钱总是好事，而且本来就是自己的钱。叶少阳把金条收起来，问道：“什么意思？”
<br/>　　小三抱拳说道：“刚你对付女尸的手段，我躲在边上偷偷看到了，所以一路追过来……你是法师吧。”
<br/>　　“怎么了？”叶少阳不想跟他说什么，但一时揣摩不透他的想法，只好先敷衍着。
<br/>　　小三笑了笑。“我哥哥就是法师，五台山的一个弟子，很厉害，所以我是相信法术的，之前我也看到了，再想起之前在破庙里的事，我那几个手下，都是你杀的？”
<br/>　　叶少阳微微一怔，沉声道：“想报仇？”
<br/>　　“别误会，我如果是想报仇，之前在你背后放冷枪就可以了……我想跟你谈一笔买卖。你听说过大清宝藏吗？”
<br/>　　大清宝藏？
<br/>　　叶少阳摇了摇头，自己那个时代，离大清国都一百多年了，就算有什么传说也早就烟消云散了，于是摇了摇头。
<br/>　　小三听说他不知道，也并不意外，说道：“你不知道，也很正常，这件事本来就没多少人知道。”
<br/>　　叶少阳反问道：“难道你知道？”
<br/>　　小三笑道：“我要是不知道，也不会来问你了。”说完看了栅栏那边一眼，因为是夜晚，隔着树林依然能看见那边火光缭绕，有人打火把在走来走去。
<br/>　　“我时间不多了，长话短说，我需要一个法师，懂分金定穴术的，来帮我探知大慕的位置，里面的宝藏，苏皇之不尽，如果能得到，任何事情都有可能。”
<br/>　　叶少阳震惊地看着他。“你认真的？”
<br/>　　“当然，不然我怎么会追上你说这个。”
<br/>　　“你哥哥不就是法师吗，为什么找我？”
<br/>　　“我哥哥死了，他去找古墓，找到了，但是没能出来。”
<br/>　　叶少阳耸了耸肩，“为什么不找别人？”
<br/>　　“我以前觉得，以我的力量，不可能找到宝藏，最重要的是……我以前对宝藏并不很感兴趣，因为……你之前也算见识过我的谋略，凭我的能力，想过好日子并不难。”
<br/>　　叶少阳没有表示，不过内心其实是赞同的，何止过好日子，这种人，如果给他机会，只要中途不出意外死了，可以说绝对是个枭雄一样的人物。
<br/>　　小三接着说道：“以前我没有野心，但是现在，我有了。所以我对宝藏有了兴趣，而你是我遇到第一个法师，你有实力，眨眼之间杀了我几个手下，而且毫不手软……”
<br/>　　“如果当时有机会，我也想杀你。”叶少阳冷冷说道。
<br/>　　小三咧嘴笑起来，“一将功成万骨枯，我这么做，也只是为了保护自己……我没时间跟你说了，你有没有兴趣合作，宝藏，我一个人拿不完，就算是百分之一给你，都够你十辈子花了。”
<br/>　　叶少阳冷笑一声，道：“我如果拒绝呢，你想杀我？”
<br/>　　“哈哈！”小三大笑起来，光是看的话，笑的非常通达豪爽。“这绝对不会，猫有猫路，狗有狗路，你我走的不是一条路，犯不着互相残杀。”小三对他拱了拱手，“你慢慢考虑清楚，可以来找我，明天起，我就是二龙山的寨主，不过我在这呆不久，但你想找我，可以的。”
<br/>　　二龙山的寨主……
<br/>　　叶少阳想到他之前带领几个手下悄悄摸去找鬼见愁的事，原来是想趁着婚宴，大伙都喝醉的时候，发动政/变，取代鬼见愁的地位，不过对他这种人来说，当一个土匪寨的寨主，真的只是一个起点而已。
<br/>　　叶少阳也没兴趣找他，所以也没问他怎么联系，有什么手机号微信QQ之类的，当然了，这个时代也没有。他在想的是，自己要不要杀了他……
<br/>　　这种枭雄，一旦得势，将来会很危险，或许会有很多人为他而死。不过……自己好像管的有点宽了，自己只是个法师，没有权力决定别人的生死。
<br/>　　再说这不是自己的时代，自己一个外来者，在这个世界能够降妖除魔，已经超过自己的职责了，别的这些事情，自己真的管不着。
<br/>　　而且凡是大是大非，一切都是命数，如果面前这个家伙真的能在人间掀起一波风浪，也是注定的事，而且对自己这个百年后的人来说，这一切都是历史的尘埃。
<br/>　　还在叶少阳分神的工夫，小三已经踩着叶少阳之前堆的石头，翻身爬上栅栏，冲叶少阳笑了笑，“下次见面，我叫三爷。”说完，跳进了山寨里。
<br/>　　叶少阳站着望了一会，转身下山去了。
<br/>　　这里的事情已经结束，他要去做自己的事了。
<br/>　　小三穿过树林，来到叶少阳之前埋伏女尸的那个粪坑前。已经有很多土匪聚集在附近，朝粪坑里围观着。
<br/>　　他的一个手下见他走过来，立刻说道：“报告三爷，那僵尸已经死了。”
<br/>　　“中了我表弟的法术，当然是死了。”小三笑起来，望着面面相觑的众人，说道：“之前的事情，想必大家也都看到了，那位法师，是我的表弟，是我请来的，因为我一早就看出，这新娘子是个死人……也就是僵尸，但是跟大当家说，他却不信……大当家的脾气，你们也是知道的，于是我只好找我表弟前来，我表弟是个道士，我让他埋伏在这里。
