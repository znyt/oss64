　　“猎妖师王虎，你败了。”萧逸一剑指着他，淡淡地说道。
<br/>　　前面，是一个八人的小型队伍。
<br/>　　为首的便是猎妖师王虎，先天七重修为，在这次的考核中名气不小。
<br/>　　“是王某技不如人，分数可以给你们，但是，请放我们离开。”王虎面露哀求之色。
<br/>　　“可以。”萧逸点点头，道，“分数交出来，你们安然离去。”
<br/>　　“谢谢。”王虎感激地说了一声，与同伴悉数上交分数。
<br/>　　.....
<br/>　　“秃鹰许华，你败了。”
<br/>　　秃鹰许华，先天七重武者，带领着一支小型队伍。
<br/>　　“该死，没想到你们中竟然有飞禽武者，老子飞起来都没逃掉，算我倒霉。”
<br/>　　“分数给你们，只是....”
<br/>　　萧逸打断道，“交出分数，你们走吧。”
<br/>　　“谢谢。”秃鹰许华道了一谢。
<br/>　　....
<br/>　　“花剑何健，你败了。”萧逸淡淡地说道。
<br/>　　“哼，论剑术，在同龄人中，我从未一败，留下你的名字。”
<br/>　　何健，剑术高手，绰号花剑，先天八重修为，北山榜17，同样和几个志同道合的人组成了一支小型队伍。
<br/>　　“萧逸。”萧逸淡淡地说道。
<br/>　　取了分数，一行人离去。
<br/>　　....
<br/>　　“遁地鼠，何苍，认输吧。”这次说话的是铁牛。
<br/>　　“呼...呼....”何苍脑袋晕晕地，道，“你们厉害，老子的遁地鼠武魂逃到地下了，还是被你这头大蛮牛震出来了。”
<br/>　　铁牛憨笑一声，撼地神牛，岂能小觑。
<br/>　　“留下分数，走吧。”
<br/>　　.....
<br/>　　半月后，众人经历了大大小小数十场的战斗，无论是其他团伙的围攻，还是其他小型队伍的找茬，皆是完胜。
<br/>　　一行五人，一路横扫，渐渐地让得其他武者闻风丧胆。
<br/>　　敢招惹他们的，也就少数几个大型团伙了。
<br/>　　今日，森林某地。
<br/>　　萧逸席地而坐，四面是林劲等四人为他护法。
<br/>　　林劲沉声道，“萧逸兄弟就要突破了，千万不能让人打扰，否则后果很严重。”
<br/>　　“我知道。”铁牛认真地点了点头，憨厚的眼神变得无比凝重，仔细地盯着四周。
<br/>　　半月战斗，让萧逸今日迎来了修为上的突破。
<br/>　　不多时，嘭的一声，萧逸身上爆发出一股气势。
<br/>　　“突破了。”萧逸微微睁开了眼睛，脸上一喜。
<br/>　　距离他突破先天三重到现在，已经有一段时间了。
<br/>　　加上这半个月的战斗和修炼，终于让他突破到先天四重。
<br/>　　不过，他并未停下修炼，而是拿出了金翼丹。
<br/>　　他要借着丹药，一举突破到先天五重。
<br/>　　萧逸刚准备吞服，忽然想起来半月前林劲和铁牛说的下蛋之事，顿时皱了皱眉头。
<br/>　　半晌，他感觉到金翼丹上涌来一股武魂力量，顿时恍然。
<br/>　　这金翼丹，是金翼雕武魂衍生出来的特效，自然也算武魂力量。
<br/>　　既然如此，那便好办了。
<br/>　　趁着林劲四人背对着自己，萧逸双眸猛地一变，左眼漆黑如墨，右眼炙热如火。
<br/>　　一个虚无的漩涡凭空而现，将金翼丹吞噬，而后绞杀成一缕缕精纯力量，最后由漩涡融合掉。
<br/>　　下一秒，双目恢复正常，萧逸身上再次气势爆发，彻底突破到先天五重。
<br/>　　当萧逸站起来时，林劲四人才回过头来，问道，“行了吗？”
<br/>　　“可以了。”萧逸点点头。
<br/>　　四人围拢过来。
<br/>　　“啧啧。”铁牛说道，“萧逸兄弟你先天三重时就这么厉害了，如今先天五重，战力岂不是爆表，那顾长风要是再遇到你，死定了。”
<br/>　　萧逸淡淡一笑，脸上自信无比。
<br/>　　“切，这算个屁阿。”秦飞扬不屑地说道，“老子才是这里修为最高，实力最强之人好吧，你们当我先天八重的修为假的阿。”
<br/>　　这家伙总爱耍帅出风头，神经大条罢了，倒没有恶意。
<br/>　　但，如果让他知道，萧逸不过是修炼了数个月就已经先天五重的话，必然会惊骇得无以复加。
<br/>　　以萧逸的修炼速度，突破洞玄境，指日可待。
<br/>　　这时，林劲忽然说道，“话说，这半月时间，我们打败了不少团伙，也得到了不少分数，这三百里森林范围内，我们的名声可响亮了。”
<br/>　　铁牛瓮声道，“可也是如此，现在其他人看到我们，远远就跑了。”
<br/>　　萧逸思索了以下，道，“如今，在森林内的人，应该不足一半了吧。”
<br/>　　“是的。”秦飞扬道，“昨日晚上，我趁着夜色，在森林高空飞翔，大致数了下，顶多剩下三千人。”
<br/>　　“其中，那十几个团伙占了两千多，其余结伴而行的小型队伍，加起来不足一千人。”
<br/>　　柳嫣然惊道，“二十天前，参加考核的足有万余人，现在之剩下三千人了？这第一个考核的淘汰数也太高了吧。”
<br/>　　萧逸摇摇头，道，“你也不想想，这二十天来，那十几个团伙掠夺分数有多疯狂。那些家伙遇到独行武者或者小型队伍，抢了分数不说，动辄杀人，要么就是强迫他们使用禁制，自动认输，离开森林。”
<br/>　　“到了如今，人数自然锐减。”
<br/>　　柳嫣然道，“我只是觉得那些团伙太过分了，参加考核，争夺分数无可厚非。可是动辄杀人，强迫其他武者认输，让得其他人再无机会，过分了。”
<br/>　　萧逸淡淡道，“那十几个团伙，都是奔着前十奖励去的；而最后通过考核的，只有五百人的名额。”
<br/>　　“也就是说，他们最后也会火拼，出现一场场大型战斗。”
<br/>　　“所以，任何一个团伙都希望森林里的武者人数越少越好，也绝不会给他们打败的武者任何机会。”
<br/>　　“否则，那些被打败过的武者，若是为了报复自己的团伙，加入到别的团伙，让别的团伙势力增加，会对最后的火拼时自己团伙的战斗很不利。”
<br/>　　萧逸大致分析了一下，柳嫣然顿时明白过来。
<br/>　　就像顾长风的团伙，若他打败的小型队伍，为了报复他，全都投奔到别的团伙，那么最后几天火拼时，他就麻烦了。
<br/>　　这时，柳嫣然说道，“距离考核结束，还有十天。”
<br/>　　“那些大型团伙，一个个都在做着最后战斗的准备。”
<br/>　　“而那些还留在森林内的小型队伍，都是些实力强劲，身上有分数的。为防意外，都躲了起来，同样等待最后几天的战斗。”
<br/>　　“我们也得商量下一步怎么办了。”
<br/>　　众人思索起来。
<br/>　　林劲说道，“如今，能对我们造成威胁的，只有少数几个大型团伙。顾长风那支算一个，赵不群那支算一个。”
<br/>　　赵不群，先天九重修为，北山榜14。
<br/>　　“除此之外，我们没什么好担心的了。”林劲说道。
<br/>　　“不对。”秦飞扬忽然脸色骤变，“还有一人，我想起来，昨晚飞行时，在森林里见到了一个独行武者。”
<br/>　　“独行武者，现在森林内还有独行武者？”众人一惊。
<br/>　　“不错。”秦飞扬沉声道，“此人，你们肯定听说过，雷战。”
<br/>　　“雷战？”除了萧逸外，众人脸色一变。
<br/>　　“难道是东荒十八城的那个狠人？”
<br/>　　“不错，就是他。”秦飞扬道，“昨晚看到他时，正好顾长风带着数百个狗腿子去找寻其他武者，没想到遇到了他。”
<br/>　　“那顾长风，想都没想，立刻就带着几百号人飞也似的逃跑了，连战斗都不敢。”
<br/>　　“遭了，那个家伙也来了吗。”林劲脸色肃然，道，“东荒十八城，位于北山郡最东方的十八座最荒野的大城。”
<br/>　　“那里妖兽横行，各种势力错综复杂，混乱得很，因此狠人极多，这雷战就是其中之一。”
<br/>　　“年纪轻轻，已是三级猎妖师，完成了诸多四级任务，出道至今，未曾一败。被誉为东荒十八城百年一遇的天才猎妖师。”
<br/>　　“本身修为又强得可怕，北山榜排名13。但排名14的赵不群，曾经在他手上十招都接不过。”
<br/>　　秦飞扬一向自大，此刻也不得不说道，“论天赋，整个北山郡也就寥寥几人能胜过他。而若是加上天才猎妖师这一层身份的话，恐怕只有一人能压得住他。”
<br/>　　“谁？”柳嫣然疑惑地问道。
<br/>　　“紫炎。”秦飞扬和林劲同时说道。
<br/>　　柳嫣然美眸一亮，道，“那个天才猎妖师，天才炼药师？短短三月，横扫数十城山匪，胜无忧公子，连败慕容胤、方鹤以及裂天剑派药堂首席弟子叶铭...”
<br/>　　“不错，就是他。”秦飞扬、林劲、铁牛三人同时说道。
<br/>　　“紫炎易霄。”
