　　哪怕是青云志，万蝠古窟中，滴血洞口的机关都比石门玄妙百倍不止。
<br/>　　当然，既然有意将洞府建立在此处，洞府的主人自然期待有缘人前来，设定恐怖的阵法或者机关术，不是有毛病么。
<br/>　　慕清霄的视线打量着周围的石壁，石壁上有着一些隐约的刻痕，可能是随着时间的流逝，刻痕也有些模糊，却也能看出其上是一些人影。
<br/>　　片刻，蹲下身来，石门下方的地面上有一块小小的凸起，微微一按，山洞中就响起一阵轰鸣。
<br/>　　望着缓缓上移的石门，慕清霄微微一笑，小医仙却撅着小嘴，将小脑袋瞥向一边。
<br/>　　随着石门的上移，其中光线明亮，将附近的黑暗全都驱逐一空。
<br/>　　望着光明大显的石门内部，小医仙美眸中有些兴奋，慕清霄道：“进去吧。”
<br/>　　话语刚落，慕清霄就慢慢想内部走去，小医仙紧随其后。
<br/>　　石门内部，是一间巨大的石室，石室看上去有些简朴与空旷，墙壁上面，镶嵌着照明所用的月光石，使得偌大的石室明亮无比。
<br/>　　在石室的中央位置，有一张座椅，一具枯骨坐立其上，深陷地骷髅头，掉落在惨白地大腿骨处，这幅画面，在这寂静的氛围中，显得有些阴森。
<br/>　　在座椅前方，摆设着一张宽长的青石台，在青石台上，整齐的摆放着三个盒子。
<br/>　　另外，在石室的三个角落中，堆放着不少金灿灿的金币与其他珍贵的财物，金币数量颇为庞大，估计有几十万的样子。
<br/>　　对于普通修炼者而言，这绝对是一笔巨款，但慕清霄想要的并不是金钱。
<br/>　　而且，洞府的主人将金币随意堆放在角落，看样子也不是很看重。
<br/>　　目光从金灿灿的黄金上移开，慕清霄的目光，停留在了最后一处角落，嘴角牵起一丝淡淡的笑意。
<br/>　　在最后的角落中，是一处用泥土堆起的小花坛，花坛里种植着各种各样的花草，一股异香，缭绕其中。
<br/>　　望着这些花草，小医仙快步走了上去，慕清霄则跟在她的身后。
<br/>　　慕清霄身为炼药师，自然清楚认得这些花草，论其价值，这些看似普通的花草，实在比金币贵重的多。
<br/>　　“四色叶，龙阳果，冰灵焰草......”
<br/>　　小医仙美眸呆呆的望着这片小小的花坛，一个个代表着珍惜难寻的高级药材名字，自她红润的小嘴中吐露而出。
<br/>　　望着其中的灵药，慕清霄随手摘下其中的‘龙阳果’，道：“我需要这个，剩下的就都送给你吧。”
<br/>　　闻言，小医仙愣了愣，美眸望着慕清霄，道：“这可都是高级灵药，你拿的龙阳果，是其中价值最低的，你真的打算全部给我？”
<br/>　　慕清霄点了点头，龙阳果正好用来练习三品丹药的炼制，至于其他灵草，价值虽高，但对他来说根本毫无用处。
<br/>　　见慕清霄同意，小医仙美眸中满是兴奋，本想着洞府内的东西要与他平分，却没想到后者就取了几枚龙阳果。
<br/>　　“谢谢。”
<br/>　　道了一声谢，小医仙就拿出准备好的铁铲，开始小心翼翼的将花坛中珍惜药草挖掘了出来，随后盛放进温润的玉瓶中。
<br/>　　慕清霄就站在一边望着，见她小心翼翼的可爱模样，神骏的脸颊上忍不住露出一丝温柔的笑容，他记得小医仙也是剧情女主吧？
<br/>　　几分钟后，将珍惜药草全部收起来后，小医仙抬起头来，却见慕清霄站在一旁，面带温柔的望着她，俏脸一红，道：“我们去别处看看吧。”
<br/>　　随后，两人来到石桌面前，小医仙看了一眼上面的金属锁，完全没有生锈的痕迹，道：“强硬打开，肯定是不行的。”
<br/>　　慕清霄指了指石桌后面的枯骨，小医仙美眸一亮，只见骷髅手掌中攥着三把黑色钥匙，正悬在骨头上。
<br/>　　小医仙貌似有些不敢碰骷髅，转过头来，美眸扑闪扑闪的望着他。
<br/>　　慕清霄走上前去，直接将钥匙扯了出来，伴随着一声断裂声，钥匙连带着一根枯骨，一把带了下来。
<br/>　　望着手中断裂的骨臂，慕清霄仔细一看，果然上面有一道裂痕，在骨骼的缝隙中，隐藏着一卷小巧的卷轴。
<br/>　　望着若隐若现的古朴卷轴，在小医仙好奇的目光中，两根手指夹了出来。
<br/>　　小医仙问道：“这是什么东西，怎么会藏在骨骼中？”
<br/>　　上下打量一下古朴得有些泛黄的卷轴，慕清霄随手解开卷轴，也不在意探过小脑袋的小医仙。
<br/>　　手中的是一张不知用和材料制作而成的皮纸，略微泛黄地皮纸上，绘着一些看上去没有丝毫规律的纹路。
<br/>　　慕清霄视线在皮纸上扫了扫，道：“这是一张残缺的地图。”
<br/>　　“残缺的地图，难道是藏宝图？”小医仙美眸中满是兴奋，洞府前人藏在骨骼中的藏宝图，珍贵程度可想而知。
<br/>　　不理会兴奋的小医仙，慕清霄将皮纸完全摊开，目光落在皮纸角落有些模糊的图案上，图案貌似是一朵莲花状的东西。
<br/>　　或许是因为岁月的缘故，看上去隐隐有些泛黄，而且也有些模糊，不过倒也还能看清其大致所绘。
<br/>　　莲花呈黑色状，在莲花表面上，似乎粘附着一层薄薄的黑炎，认真的看上去，整朵莲花隐隐给人一种妖异地感觉。
<br/>　　仔细观察过图案后，慕清霄眼底闪过一丝惊喜，却面色平淡道：“果然没错。”
<br/>　　“你知道这是什么东西？”小医仙困惑道。
<br/>　　“知道，但不是现在的我能够接触的，我需要这份地图，你觉得怎么样？”
<br/>　　听到慕清霄征求她的意见，小医仙心间竟有些开心，昂了昂雪白细腻的小下巴，可爱道：“你想独吞藏宝图，念在你送我灵药的份上，地图就给你吧，反正是残缺的。”
<br/>　　见她露出这幅可爱的模样，慕清霄情不自禁的伸出右手，捏了捏她可爱的小鼻子，道：“若真是藏宝地图，等我集齐以后，带你一起去找。”
<br/>　　“我才不要跟一起呢，臭流氓。”
<br/>　　小医仙拍开他的手掌，脸颊微红，慎怒的瞥了他一眼，虽然嘴上这么说，但不知为何，芳心却加速跳动起来。
