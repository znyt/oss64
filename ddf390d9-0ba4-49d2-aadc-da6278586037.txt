        “你是想说明什么？”傅夜沉虽然不认为落以柔是在质疑他，但是他觉得或许她有看到一些他没太去注意过的数目。
<br/>
<br/>        “我能保证，我所经手的开销没问题。但是这笔开销，我没经手过，也不记得有过这笔开销，但是报销单上却又我的签名，不仅仅是我的，还有大哥你的签名。”落以柔慢条斯理地说，然后又翻到下一页资料，接着问道，“大哥，我记得，你的私人开销，从来不计入公司里的个人账户。”
<br/>
<br/>        “嗯？”
<br/>
<br/>        “难道大哥，你过去都是拿公款在泡妞吗？”落以柔略带打趣地说到这里，顿了顿后，又补充了一句，“虽然，公款其实也就是你的私款。”
<br/>
<br/>        毕竟，傅氏集团不是股份制公司，傅老太爷傅炎廷“退位”后，拿走了属于他的那部分薪酬所得，余下的傅氏集团旗下所有的资产，就已经全部移交给了傅夜沉。
<br/>
<br/>        “衣服、鞋子、香包、化妆品，全是国际高档限量版。大哥，你应该没有包养过任何女人吧？”落以柔觉得自己的猜测是对的，认识他至今，她还没见过他跟哪个女人有暧昧关系。
<br/>
<br/>        傅夜沉剑眉微蹙：“另请第三方财务公司来公司里查账，然后让公司的法务部做好打官司的准备。”
<br/>
<br/>        “好，我马上去办。”落以柔就知道傅夜沉会这么处理，所以她早就做好了准备，只等傅夜沉一声令下了。
<br/>
<br/>        傅夜沉下意识地问：“你怎么会突然注意到这件事情？”
<br/>
<br/>        “是师父交代的，让我没事多翻看一下财务部上交过来的电子账本，说不定能发现些什么。”落以柔抿了抿唇。
<br/>
<br/>        傅夜沉赞扬了一句：“名师出高徒！”
<br/>
<br/>        落以柔只是微微一笑，忽然想起了什么，连忙从衣兜里掏出一个小喷雾瓶，对着傅夜沉的身上喷了喷。
<br/>
<br/>        之前，她也有给他喷过，只不过是在他不在办公室里的时，喷在了他挂在落地衣架上的西装外套上。
<br/>
<br/>        “大哥，这是我自己养的薄荷草，再自制的薄荷水，可以让你中午一个人开车的时候提提神。”落以柔说完后，抬起手来指了指自己腕上的手表，暗示他该出门了。
<br/>
<br/>        “谢谢。”傅夜沉点了点头，转身疾步离开。
<br/>
<br/>        他一门心思只在童筱颖的身上，压根就没察觉到落以柔对他产生了另一种感情。
<br/>
<br/>        不知道，他的宝贝筱颖现在有没有起床？
<br/>
<br/>        他的私宅地处比较偏远，而且就他一户住那里，所以，过了市区和别墅区之后的那条路，几乎没有计程车会经过。
<br/>
<br/>        可偏偏在他回去的路上，迎面遇上了一辆计程车。
<br/>
<br/>        傅夜沉的脑海里忽然闪过童筱颖的倩影，这几乎是一瞬间的事情，他脚踩刹车，减速后立马打转了方向盘。
<br/>
<br/>        他超速追上那辆计程车后，用力按着方向盘上的喇叭键鸣笛，示意计程车停车，然而，这辆计程车丝毫没有要减速停车的意思。
<br/>
<br/>        傅夜沉被逼无奈，一脚油门踩到底，加速后找准了时机，直接横停在了马路上。
<br/>
<br/>        那辆计程车不得不一个急刹，车头差点撞上他的车身，在仅剩下一米的距离左右，稳稳地停住。
<br/>
<br/>        “喂！你有病啊！想找死啊！”司机落下车窗后，大大咧咧地谩骂。
<br/>
<br/>        傅夜沉淡定地从驾驶座上下来，无视掉计程车司机的存在，径直走到计程车后座旁，抬起骨节分明的手来，扣响了车门玻璃。
<br/>
<br/>        计程车内，童筱颖深吸了口气，落下车窗，一脸讪笑着仰面看着傅夜沉：“大哥，好巧啊！”
<br/>
<br/>        “宝贝，你这是去哪儿？我送你。”傅夜沉黑眸微眯，淡淡的语气从薄唇间飘出。
<br/>
<br/>        童筱颖笑得有些嘴角僵硬，转而是苦笑：“我只是想去……想去吃火锅而已。”
<br/>
<br/>        “下车！”傅夜沉偏了偏头。
<br/>
<br/>        童筱颖眨了眨眼睛，双手紧紧地拉住门上的手柄：“我看不用了吧！”
<br/>
<br/>        “司机，麻烦你开下车锁，谢谢。”傅夜沉随之掏出皮夹，从里面随手抽了几张百元大钞出来，伸到了驾驶座上落下车窗的司机眼前。
<br/>
<br/>        司机脸上刚刚那凶神恶煞的表情立马收敛，两眼盯着钞票直勾勾地发亮。
<br/>
<br/>        他抬起手来接过钱后，立即按了开锁键。
<br/>
<br/>        “咔嚓”一声。
<br/>
<br/>        傅夜沉利索地拉开了车门后，一手挡住车顶门框，一手抓住了童筱颖的手腕，将她从计程车内给拉了出来。
<br/>
<br/>        又是“乓”地一声，车门关上后，司机非常识趣地重新发动引擎，开车扬长而去。
