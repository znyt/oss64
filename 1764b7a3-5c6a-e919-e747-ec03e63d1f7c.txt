<br/>　　多亏了官人撕下灵符，救了小女子，小女子感激不清，还请各位官人做主。”

<br/>　　“我去你娘的！”三当家一只手指着女尸，“你们他么的当老子是瞎子，这特么是死人？”
<br/>　　叶少阳知道解释不清，对陈三摇了摇头。

<br/>　　女尸先是愣了一下，随口怯怯地说道：“有白面馒头吃吗！”
<br/>　　“哈哈哈！”几个土匪一起大笑起来。
<br/>　　“馒头，我说妹子，当了我们压寨夫人，别说馒头，每天鸡鸭鱼肉供着，要吃什么没有！”
<br/>　　女尸一听，犹豫了一下，咬牙说道：“好，我做！”
<br/>　　套路！一切都是套路！
<br/>　　叶少阳在心中叹息，这女鬼很聪明，一开始就说自己是流浪被拐卖的，眼下这时节，流民遍地，她这么说根本不会有人怀疑，现在又特意提起白面馒头，也只会让这些傻比土匪们觉得她是饿怕了，乱世之中，为了一口饭出卖自己的人不少，这女鬼为了一口吃的去当土匪窝的压寨夫人，最多会让人觉得勇敢，不会怀疑什么，何况这些傻比土匪也会想，这女子很聪明，知道自己被他们盯上，就算不从也得从了，只能说是识时务，根本不会怀疑她的动机有什么问题。
