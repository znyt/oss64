　　小青说道：“不是我说，阴司这是想让我们当炮灰吗？”
<br/>　　徐文长道：“实话说吧，空界那帮人对与橙子公主的作法，虽然得罪的是天子殿，但也让整个阴司愤慨不已，不太想过问空界的事，但空界万一真的被尸族完全攻陷，与太阴山夹攻阴司，后果不堪设想，阴司还希望用空界来牵制尸族和太阴山……只是现在大帝不放话，几个大佬又不想动用自己的力量，而你们阴阳司，本来就是负责阴阳两界的事宜，因此只有找你们出面。
<br/>　　这是其一，其二，谁都知道，你们阴阳司幕后主事是叶少阳，空界之战，也与他有莫大关系，你们去参展，也算是分辨因果，再合适不过，阴司可以给你们提供除了兵力之外的各种帮助。
<br/>　　其三，从私人角度来说，转世鬼童……是你们的主母，你们难道不想救她？驱逐尸族，就有机会救她。你们作为叶少阳的门人，这件事难道不应该代他去做？”
<br/>　　瓜瓜等人互相看了看，杨宫梓说道：“这是阴司对冷玉的态度吗？”
<br/>　　徐文长摆了摆手，“阴司绝不允许转世鬼童在世间存在，但转世鬼童现在尸族之中，阴司并没有办法。”
<br/>　　瓜瓜道：“假如我们救出主母，阴司一定会来要人的吧？”
<br/>　　徐文长道：“这个，我不敢做任何保证，但是有一点我可以透***司宁愿转世鬼童在你们手中，也绝不想让她跟尸族在一起……”
<br/>　　大伙有些不解地望着他，徐文长接着往下说道：“跟你们在一起，阴司可以随时掌控，这是其一，其二，转世鬼童是无极鬼王一缕神识历经轮回投生而成，她在人间，一定程度上能代表无极鬼王的意志……她现在呆在尸族，阴司很担心，万一她被尸族收买，太阴山与尸族结盟，后果不堪设想！”
<br/>　　原来……是为了这个原因。
<br/>　　小青说道：“那我们应该怎么去做，老大不在，军师也不在，没有拿主意的人了。”
<br/>　　杨宫梓说道：“如果你们信任我的话，我跟你们一起去空界，你们听我指挥。”
<br/>　　大伙转头互相看了看，都没有意见，风之谷跟捉鬼联盟，本来就是兄弟组织，更不要说杨宫梓之前也在叶少阳身边呆过一阵子，是他绝对信任的人。而且杨宫梓生性聪颖，又非常稳重，自己本身也是混沌界的主宰，她来暂时做风之谷的领头人，是最合适不过。
<br/>　　杨宫梓一个个望过去，说道：“我们不一定能够把冷玉抢回来，但我们都坚信，少阳一定是能够回来的，在他回来之前，我们尽力去跟尸族周旋，做好一起铺垫，等少阳他们回来，再一起战斗！”
<br/>　　一番话，提升了所有人的信心，大伙都坚定地点了点头。
<br/>　　“我也回来了！”小马举起手，“为了小叶子和和尚他们，没得说，就是干！”
<br/>　　大伙开始商量前往空界的计划。
<br/>　　道风没有参与，走到徐文长面前，说道：“阴司对少阳失踪这件事，有什么看法，我不要听官方的说法。”
<br/>　　徐文长苦笑起来，沉默片刻，说道：“几个大佬碰了面，讨论了一下，实话说，这件事出去所有人的预料，叶少阳的名字，而今已经从生死簿上消失，从某种方面来说，也算是跳出了六道轮回，一切演算对他都失去作用。阴司现在也很紧张，十分重视，但是没办法，没人能找到徐福。”
<br/>　　道风迟疑了一下，问道：“东皇钟也随少阳一起去了异界？”
<br/>　　“想必是的。在人间完全没了踪迹。”
<br/>　　“那依你之见，空界能守得住吗？”
<br/>　　徐文长摇了摇头，“尸族隐忍千年，一遭进犯，自然是全力而为，最好的结果，也就是守住半壁江山。现在阴司最担心的，就是尸族与太阴山联手，夹攻人间和阴司。但愿是大家多虑了。”
<br/>　　道风望着窗外的夜色，缓缓说道：“要变天了吗？”
<br/>　　徐文长叹了口气，没有作声。
<br/>　　没人知道，道风内心比所有人都担心叶少阳的安危。
<br/>　　以前在这个世界，少阳有一干朋友，还有他，每每总能在最关键的时候给与必要的帮助，但现在……身在异界的他，身边只有寥寥几个帮手，最要命的是，这几个帮手说不定跟叶少阳还不在一个地方。
<br/>　　假如叶少阳是自己一个人，别的危机就不说了，单单一个李浩然，就不是他能够对付得了的。
<br/>　　沉吟半晌，道风突然问道：“有没有办法，跟叶少阳取的某种联系，哪怕是单方面的也可以。”
<br/>　　徐文长不假思索地摇摇头：“这不可能。完全是两个不同的空间。”
<br/>　　“他去的世界，跟我们这个世界，应该是在同一条时间线上，如果他在之前改变了什么事情，我们的世界会不会做出改变，就像蝴蝶效应？”
<br/>　　徐文长道：“我不太明白。”
<br/>　　道风道：“假如他回到几百年前，杀了他自己的祖先，那么，这个世界还有他吗？”
<br/>　　“那当然没有。”徐文长以正常的逻辑回答道。
<br/>　　“有吗？没有他的祖先，他这一脉亲缘就绝后了，连他父亲都没有，怎么会有他？”
<br/>　　徐文长怔了怔，喃喃说道：“那这么说……肯定是没有了，不过……不对啊，既然他都不存在了，那么是谁杀死了他祖先呢，你懂我的意思吧？自己死了，就绝后了，又哪里来的后代去杀死自己？这……”
<br/>　　道风道：“可是他已经存在了。如果祖先死了，他的存在，又算是怎么回事？”
<br/>　　徐文长两条眉毛快拧成麻花了，怔怔地想了许久，喃喃说道：“我说……你这些问题，我从来没有想过，我不知道，我也没回到过去，你这些早就超过我的理解……估计也没人知道。”
<br/>　　道风没有说什么，他问这些问题，本来就没指望徐文长能够回答，只是想找个人探讨，顺便帮助自己理清头绪。谢雨晴听见他们的对话，上前说道：“道风，你问的这些，属于平行宇宙的范畴了，人回到过去，如果改变历史，改变的也只是那一条时间线的历史，我们这个世界一切都已经发生了，是无法被改变的。”
