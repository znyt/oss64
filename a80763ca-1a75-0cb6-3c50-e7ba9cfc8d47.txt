
<br/>　　<br/>　　    “噗通”那人身子突兀的就砸进了忘川河中，在掉落的一刹那脸上惊恐，绝望和不甘还有悔恨的表情全都掺杂在了一起，可能他也觉得自己率先出手有些鲁莽了。<br/>　　<br/>　　    这人掉入河中之后，忘川河里的亡魂厉鬼全都在瞬间就朝他冲了过去，他的身子渐渐的没入到水中，仿佛是被下面什么东西给生拉下去的一样，连一点挣扎的机会都没有，就彻底被没影了。<br/>　　<br/>　　    岸边的人陡然间寂静下来，除了水中厉鬼嘶吼的动静，谁也没有再身先士卒了，这就是明摆着的，先下手的那个人肯定立马就会成为其他人的眼中钉，这场面就略微有点尴尬了，彼岸花就在那里，可却没人敢动了。<br/>　　<br/>　　    “你想要的话，恐怕有点难了，正一教的人在旁边虎视眈眈的盯着呢，他们不动手其他人恐怕没人敢先下手”茅小草皱眉说道。<br/>　　<br/>　　    王惊蛰摇了摇头，说道：“我的不急，先可你来吧”<br/>　　<br/>　　    茅小草张了张嘴，最终有点纠结的说道：“彼岸花都是伴生的，白色的这一朵出现了，红色那一朵肯定也就离得不远了，让他们先在这里狗咬狗吧”<br/>　　<br/>　　    “走了！”王惊蛰伸手一拉茅小草的手腕，两人从人群后面快速的朝着前方找去。<br/>　　<br/>　　    这边的局面僵持了片刻，姜冰啸忽然从身后拔出一把桃木剑，秋成子见状也抽出长剑和他站在一旁，姜冰啸眼神在人群里扫了一圈后，说道：“各位道友，正一姜冰啸有礼了，这朵彼岸花对我们正一很是重要，还请各位能行个方便，正一教上下必定铭记，这个人情以后在外我们可以想办法还过来，还请几位能割爱，行么？”<br/>　　<br/>　　    姜冰啸的话音刚落，有些人就开始嘀咕起来：“正一教首席大弟子？据说，是近年来唯一一个能有望晋级天师的，正一教多少年都不出这种人了”<br/>　　<br/>　　    陈三岁也在人群里，不过他的眼神却没盯着那朵彼岸花，他眯着眼睛一直在搜寻着忘川河里的亡魂，一路走来他急迫的寻觅着让那个让他至死不渝的一道魂魄。<br/>　　<br/>　　    忽然之间，陈三岁发现在对面河岸下，水中飘荡着一个半边脸都塌陷了，身体一侧残缺不堪的亡魂，和其他水中亡魂不同的是，这一道显得似乎非常柔弱，凄哀的孤零零的躲在了一边。<br/>　　<br/>　　    陈三岁顿时眼中直发光，他长吐了口气，轻声呼唤道：“诗远······”<br/>　　<br/>　　    陈三岁一出声，四周就有人看了过来，他急促的朝前走去，眼中瞬间就湿润了：“诗远，我来了，我接你回去，这次我再也不会让你离开我了”<br/>　　<br/>　　    对面河岸下的诗远见到陈三岁后，死寂的目光突然焕发出神采，她艰难的伸出一只手仿佛想要抓过来一样，陈三岁顿时急迫的冲到了岸边。<br/>　　<br/>　　    姜冰啸见状，皱眉说道：“道友，你是不给我·····”<br/>　　<br/>　　    “一边呆着去！”陈三岁回头呵斥了一句，说道：“你抢你们的东西跟我无关，我也不敢兴趣，但谁要是敢拦着我，别说我没提醒你，这忘川河里不缺你们这些三瓜两枣的”<br/>　　<br/>　　    姜冰啸被怼了一句，冷着脸也没吭声，对方似乎不是奔着那彼岸花来的，他此时也没必要跟他计较。<br/>　　<br/>　　    陈三岁眼看着诗远从那边飘过来后，沉沉的吐了口气，身子略微一蹲人随即拔地而起落在了忘川河面那块凸起的礁石上，伸出手说道：“待会，你附在我身上，我带你离开阴曹地府······”<br/>　　<br/>　　    被禁锢在忘川河里的亡魂，是没有办法凭借自己出去的，除非能够鬼上身到人的身上，所以陈三岁一过来河中的亡魂厉鬼全都沸腾了，挣扎着想要能够给他上了。<br/>　　<br/>　　    岸上的人见状，发现陈三岁确实无意那朵彼岸花，就渐渐的放下了心，但谁也没有想到的是，陈三岁在伸出手的时候，脚尖突然踢了一脚那朵白色的彼岸花，花枝顿时折断，随即就飘向了忘川河。<br/>　　<br/>　　    他这么一干给其他人都吓了一跳，第一个念头就是：“这逼太不是人了”<br/>　　<br/>　　    陈三岁这么干，那确实是相当缺德的了，他这一脚无疑是把所有人的目光都给吸引了过去，彼岸花飘向忘川河，想要的人自然得想方设法的去捞到手了。<br/>　　<br/>　　    “混蛋！”姜冰啸咬牙骂了一嘴。<br/>　　<br/>　　    那朵彼岸花落在河里之后就飘走了，岸上的人不得不追了过去，顿时就把忘川河中的亡魂厉鬼全都给吸引了过去。<br/>　　<br/>　　    瞬间，那块礁石的附近就空了不少，那叫诗远的亡魂艰难的伸出一只手搭在了陈三岁递出的胳膊上。<br/>　　<br/>　　    “唰”诗远顿时上身，附在了陈三岁的身上，他眼神木然一呆，但很快就恢复如常了。<br/>　　<br/>　　    鬼上身了，如果这鬼没想着祸害你的话，其实没啥影响，一人一鬼和平共处呗。<br/>　　<br/>　　    再说另外一头，那朵彼岸花飘在忘川河上，顺着水流往下飘去的时候，就把人全都给吸引走了，但尴尬的是谁也不敢跳入河中去捞起那朵花，忘川河里数不尽的亡魂厉鬼能瞬间给你都挤爆了，刚才掉下去那人就是活生生的例子。<br/>　　<br/>　　    但更让人无语的是王惊蛰和茅小草，此时已经找到了那朵伴生的彼岸花。<br/>　　<br/>　　    一朵红的异常鲜艳的小花，盛开在了岸边，不时的被河水冲刷着，花瓣上还带着水滴，飘着阴气，显得异常晶莹剔透，有种诡异的美感。<br/>　　<br/>　　    两人找到这朵彼岸花的时候，所有人的目光还集中在上方，没有人留意到这里，但那朵被陈三岁踢走的彼岸花落在河水里飘过来的时候，就把人全都给带了过来。<br/>　　<br/>　　    “尼玛······”王惊蛰看着和自己相隔不到十来米的彼岸花，还有过来的一伙人，顿时懵了。<br/>　　<br/>　　    想要悄然采花而走，这明显是不可能了。
