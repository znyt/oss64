　　“搜魂！”
<br/>　　高空之上，一道淡漠的声音响起。
<br/>　　紧接着，罗睺就感受到自己的神魂之中传来一阵阵生不如死的痛苦，可惜，他已经被封住了，不能喊出声来。
<br/>　　渐渐地，他的双眼开始变得模糊，脸上的表情和眼神也慢慢变得僵硬、呆滞。
<br/>　　不过，元始可没有管这些，他将所有的注意力都集中在罗睺残魂的记忆之中。将他的一些重要信息一一提取出来。
<br/>　　一代魔祖的记忆，十分珍贵，不说其他，就说他对大道的感悟，便是异常珍贵，甚至比大多数先天至宝还要珍贵。
<br/>　　毕竟先天至宝可以找，但对于大道的领悟，这可不是那么领悟的，现如今有一个现成的，这对于元始来说，只要将他的领悟完全吸收，他完全可以节约很多时间。
<br/>　　没有什么比这更好的了，对比这个，先天至宝突然间变得不是那么重要了。
<br/>　　良久，元始才将罗睺的记忆完全搜索完毕，同时，召唤出灵魂之火，将之完全燃尽，化为自己神魂的养料。
<br/>　　在罗睺残魂被灭的同时，天上降下一道功德，虽然不多，但元始还是收下了，自己不用，以后用来炼功德灵宝还是不错的。
<br/>　　之后，元始接着闭关，开始整理罗睺残魂之中的收获。
<br/>　　百年之后，元始终将罗睺残魂之中的记忆吸收完毕，最主要的是，他的记忆并不是十分完整，仅仅只有罗睺认为重要的信息，而且，混沌魔神的信息也只有小部分，看来他还没有完全觉醒。
<br/>　　在罗睺的记忆之中，最重要的就是他对于大道的感悟以及对于魔道的领悟，罗睺为魔道之祖，这可不是说说而已。
<br/>　　魔道最为一种排名前二十的大道，威力自然不俗，况且其中还有各种诡秘，不容多说。而且除了魔道，他对于其余法则，也大有涉及，这对于元始的元始大道十分有利。
<br/>　　除了大道感悟外，魔祖罗睺重要的信息还有他修炼的功法——《万劫魔典》！
<br/>　　让元始感到意外的是，这《万劫魔典》竟然是一部修炼劫运之气的魔典。
<br/>　　劫运，天地之间无处不充斥着劫运之气，杀戮是劫，情关是劫，道心是劫，修道是劫，雷电是劫，风火是劫……
<br/>　　天有天劫，地有地劫，人有人劫……
<br/>　　天地万物，无处没有劫！
<br/>　　修道，便意味着一生要与劫对抗。
<br/>　　天地有阴阳，能量也有阴阳。若说天地灵气是阳，那么，天地劫气就是阴。
<br/>　　阳性能量能够为人吸收修炼，那么阴性能量为何不能为人吸收修炼？这当然能，只是这劫气之中充斥着一种邪恶属性，非有特殊功法，不能修炼，否则会被劫气反噬成魔。
<br/>　　故而，这样能够通过吸收劫气来修炼的功法少之又少。
<br/>　　天地之间的灵气由无数修士在吸收，而劫气能吸收者寥寥无几，就这样，劫气一天天积累增多，直到增多到一个极限，之后再爆发出来，形成天地大劫！
<br/>　　这《万劫魔典》就是能够吸收这些天地劫运的魔功，天地之间如此多的劫气，而且只要是杀戮，就会有劫气产生，洪荒之中的劫气每天都在增多，若是一人吸收这些劫气，修炼不快才怪。
<br/>　　《万劫魔典》共分难劫，灾劫，大劫，天劫，量劫，无量劫，无量量劫七重功法，每进一重，修为突破，实力大增。直到第六重无量劫，便能突破混元，成功突破第七重无量量劫之后，甚至能够突破混元后期境。
<br/>　　当然，这是不是真的还有待考证，以元始的眼光来看，这八九是真的了，无量量劫为天地宇宙复演混沌，天地寂灭，万物归虚，能够度过这一劫，定是混元后期无疑。
<br/>　　从这来说，这《万劫魔典》不失为一部最为顶级的天经。
<br/>　　不是什么都可以被称为天经的，只有那些能够修炼到准圣甚至混元的功法，才能被成为天经、玄功。
<br/>　　只是，任何事物都有好有坏，这《万劫魔典》一样如此。
<br/>　　首先，修炼《万劫魔典》必须要没有修炼过任何功法的人修炼，除此之外，还可以自废修为来修炼。就此一点，就可以难住一半以上的人。
<br/>　　第二，修炼此功法者，必须要有金仙的肉身强度。就此一点，又可以刷去大多数人，只剩下寥寥无几的几个天赋异禀之人。
<br/>　　没有修炼过功法，就会有堪比金仙的肉身，这怎么可能？就此一点，就会令无数人为之绝望。
<br/>　　第三点，也就是最后最为重要的一点，要有强大的心境修为与元神修为，否则，遭功法反噬，轻则修为尽废，化身成魔，屠戮苍生，重则身死道消，永生不得轮回！
<br/>　　就第三点，再次刷下无数人，天地之间，能够修炼此功法之人，应该就没有了。
<br/>　　故而，一般人就算得到功法，也不能修炼。但，这还难不倒元始，元始如今虽然不能修炼此功法，但不代表着他以后没有办法修炼。
<br/>　　此等功法，就连他都眼热无比。
<br/>　　不过，此事不急，元始并没有现在修炼的想法，这种事，以后再说。
<br/>　　罗睺最后一种值得元始注意的就是罗睺的神通——万道森罗！
<br/>　　万道森罗，可以自由吸取各种不同类型甚至相互排斥的功法神通，要诀八大门道：“天、地、奇、正、阴、阳、生、灭”。
<br/>　　万道森罗包容天地之力，能整合为一股灭世的力量，毁天灭地，覆灭乾坤。
<br/>　　万道森罗以念为刃，驾驭人心。每人所学的都各有不同，这样要看修炼之人所能融合的神通功法了。融合的越多，神通自然就越强。
<br/>　　这与元始所创的诛神印十分相似，他的诛神印也是以诸多神通汇聚在一起，经过奇异融合形成一杆诛神绝枪，弑神诛仙！
<br/>　　而魔祖罗睺的万道森罗就有混洞之道、寂灭之道、玄阴之道、屠天魔道、奇诡之道等诸多神通。
<br/>　　以万道森罗演化出的神通混洞之道，演虚空为混洞，吞噬万物。
<br/>　　寂灭之道，与元始的寒冰黑暗寂灭一道不同，罗睺领悟的是魔道寂灭，以劫魔之气，寂灭万物。
<br/>　　至阴之道，以诛仙四剑演化至阴之道，阴气逼人，剑气刺骨，诛仙绝神。
<br/>　　屠天魔道，以魔道化霸刀，斩天灭地，霸道绝伦。
<br/>　　奇诡之道，使对手产生幻觉以及迷惑的效果，平凡之物在其功力催动之下也能幻化出各种形态各异的凶兽攻击敌人。
<br/>　　这对元始来说不算什么，不就是一点幻之法则吗，还没有他的梦幻法则厉害呢。若是梦幻法则再加上灵魂法则，那就更厉害了，一念之间，就可以让敌人沉沦梦境，于梦境之中死亡，无声无息，诡异之极。
<br/>　　罗睺记忆之中最后一些有用信息，便是罗睺与人的战斗经历了，不过元始只是随便看看就过了，对他来说，最重要的还是以上三种。
<br/>　　花费五百年时间，元始才彻底将罗睺的大道感悟化为自己所有，道行更进一步，虽然修为不变，但其实力却增加不少，可战混元。
<br/>　　之后，元始宣布出关。
