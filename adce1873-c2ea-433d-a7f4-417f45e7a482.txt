　　从外面闯进来的是苏猛，这几天因为人手不足的关系，他已经被调回了情报科。
<br/>　　只见这家伙闯进来之后，风风火火的就来到了李承乾身边，张嘴刚想说些什么，忽的又把嘴闭上，顿了顿才开口道：“陛下，这里说话有些不大方便，请您移步！”
<br/>　　“什么事情？”李承乾见到苏猛的时候，心里便隐隐有了一种不好的预感，来到监牢的一个无人的角落之后，急声问道。
<br/>　　“陛下，太上皇那边的情况似乎有些不大对头，我们已经有近一个月没有收到太上皇的消息了，而且就我们所知，兵部也已经有两个多月没有收到南海舰队尉迟小将军的消息。”苏猛的声音很低，如果不仔细去听根本就不知道他在说什么。
<br/>　　但李承乾还是将一切都清楚的听在耳中，沉默片刻说道：“海上风急浪大，有没有可能是因为这样所以才耽搁了消息的传递？”
<br/>　　“陛下，这种可能不是没有，只是……只是太上皇的舰队所带的补给最多只够用到这个月的月底，按说他们现在应该是在往回赶的路上，在这个距离内就算是有什么风浪一时间消息送不回来，但是也绝不可能断绝消息达一月之久。”苏猛冷静的分析道。
<br/>　　这下，李承乾彻底深默了，良久之后，狠狠一挥手：“走，回宫！另外，让李佑、李震火速进宫！”
<br/>　　“是！”苏猛得了命令如飞一般离开，而李承乾则是带着小白与夜魅急匆匆的离开了大理寺，将袁守诚、秦英等一群道门之人丢在了大牢之中。
<br/>　　可是就在李承乾离开监牢之后，牢房内的气氛却显然更加紧张起来，袁天罡不知从身上什么位置摸出一只颜色金黄的龟甲以及数枚铜钱，一股脑的塞进龟甲之后便上上下下的摇了起来。
<br/>　　“哗啦”一声，经过数次摇动之后，铜钱撒了一地。
<br/>　　“怎么样？”旁边的牢房里面，袁守诚，李淳风齐齐凑了上来。
<br/>　　袁天罡面色铁青，双眼死死盯着地面，喃喃自语道：“纯，纯阳”
<br/>　　袁天罡的声音虽小，但还是被李淳风听了过去，只见他的脸色也变的十分故怪，同时着了魔一般自语道：“纯阳？怎么会是纯阳？怎么会这样！”
<br/>　　另一边袁守诚则扶着栏杆坐了下去，带着无限的失望，喃喃说道：“天意使然，强求不得，罢了，罢了！”
<br/>　　不管是李淳风还是袁守诚，他们对算命之法都有所涉猎，都知道纯阳卦象所代表的含议表示天机未显，也就是说他们要算的东西，还没有结果，无法测出吉凶。
<br/>　　水月显然也是知道那卦象的意思，但她显然并没有放弃的打算，见到诚守诚颓废的样子，不由大声说道：“道兄，难道你们就打算这样放弃了？万千黎民百姓，难道你们就要弃之不顾？”
<br/>　　“顾不过来了，当年陛下根本不会相信我们所说的一切，刚刚秦师伯已经对他说了那个传说的一部分，可是他的表现你也看到了，那种不屑的目光难道你还不明白代表着什么么？”袁守诚失望的说道。
<br/>　　“可是不管怎样也要再试一下，如果不试试又怎么能知道那一位到底是怎么想的。”水月似乎还没有死心，扶着栏杆的手用力捏着，就好像要把那胳膊粗的栏杆握断一样。
<br/>　　袁天罡这个时候已经从刚刚的恍惚中回过神来，面对水月苦笑说道：“没有用的，现如今很多事情已经开始发生改变，明明是应死之人，但却依旧活动于阳间，本应活于阳间之人却已经早早离世，现在的天机已经不是我们可以测得了的了，强测定会受极大的天谴！”
<br/>　　水月将目光转向袁天罡，良久之后终于还是低下头，长叹一声不再说话。
<br/>　　她知道袁天罡所说的天谴并不是指他自己，而是指其它很多方面，比如一些意想不到的天灾，如水患、干旱等等。想到些她也不敢让袁天罡强行再去推算，同样也没有再提将一切事由告知李承乾。
<br/>　　事实上，如果他们在刚刚李承乾在时候进行这样一番对话的话，李承乾未必不会接受他们的解释。
<br/>　　毕竟知道历史的李承乾很清楚，大唐眼下已经走上了一条与历史完全不同的道路，不说国土面积扩大了多少倍，有了多少新鲜事物诞生。
<br/>　　单就说他李承乾自己，其实早在贞观十九年的时候他就应该已经死了，而且与他一起死的还有赵节、杜荷、候君集。
<br/>　　可是现在看看，不但李承乾自己当了皇帝，杜荷、越节、候君集也没有参与谋反，依旧活的好好的，甚至就连早就应该已经死了的秦琼也还活着，更不要说还有长孙皇后这位千古贤后。
<br/>　　而那些该活着的却早早的便已经死去，比如泉盖苏文，比如真珠夷男。
<br/>　　就好像我们拿着一直光笔，当我们按动电源的时候，会有一条射线延伸出去。
<br/>　　以前的大唐就像是这条线上的一个点，这条线直接穿过这个‘点’不会发生任何改变，还会一直向前。
<br/>　　可是现在的大唐就好像是一面镜子，当这条射线经过大唐的时候发生了折射，那么光线后面所经过的一切都发生了改变，未来会是什么样子谁都无法再推测。
<br/>　　所以当袁天罡发现这一切的时候便向袁守诚做了说明，而后又惊动了秦英，最后水月也收到了消息。
<br/>　　也正是因为这样，袁天罡等人才想着将一切事情都修复，让一切再重新走回正轨，哪怕不能回到原来的轨迹，至少也要差不多才行。
<br/>　　而想要修复这一切，就只能把破坏原本轨迹的‘镜子’打破，也就是杀掉那些原本已经应该死去的人——李承乾！
<br/>　　只是杀死皇帝的事情实在是太大，袁天罡等人谁也背不起这么大的责任，故而水月便想到了自己的徒弟，让自己的徒弟接近长乐，用一些似是而非的理由来蛊惑她向李承乾发动袭击。
