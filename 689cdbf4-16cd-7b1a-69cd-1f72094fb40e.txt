<br/>　　“叛徒！”雪魔一眼看到瓜瓜，眼中立刻现出杀机。
<br/>　　瓜瓜起初见到他，有点怕他似的，躲躲闪闪，听见他这么说，反而释然了，冷笑道：“你知道也好，正好可以帮我带个话，我已经脱离太阴山，加入捉鬼联盟了。”
<br/>　　“殿下，你会害死自己的。”雪魔脸上带着一抹笑意，“虽然不是今天。”
<br/>　　瓜瓜没有理会，跟大伙一起来到叶少阳身边，一群人一起迟疑地看着他。
<br/>　　“这位雪魔大大，要帮我挡住他们的偷袭，你们可以不用冒险了。先回人间再说。”
<br/>　　大伙一听，顿时更是懵逼。叶少阳眼看着那些从山谷里追出来的摇光仙子等人，目光扫了一遍，道：“道风而他们呢？”
<br/>　　几个人面面相觑。小白说道：“道风大大疯了，见人杀人，已经杀了好些人了。现在被几个厉害的围住，不过好像也没什么。”
<br/>　　这……
<br/>　　小青道：“是，黎山老母败了，暂时没法恢复的，现在对他有威胁的只有一个李浩然了，不过也被宫梓姐姐拖住了。”
<br/>　　叶少阳道：“好吧，那不管他们了，我们走。”
<br/>　　“叶少阳！”
<br/>　　叶少阳话音刚落，对面摇光仙子的声音立刻响起。她从面前一众尸兵、鬼兵身上扫过去，冲叶少阳冷笑起来。
<br/>　　“果不其然，原来你早就投靠太阴山了，怪不得你要一直保护鬼童了，原来她不是你的情人，而是你的主子。
<br/>　　”
<br/>　　这大帽子扣的，叶少阳本想反驳一下，结果雪魔接过话头，说道：“不错，叶天师早就是我太阴山的客卿，今日我特来接应，只怕却是需要一场恶战了。”
<br/>　　叶少阳怔怔地朝雪魔看了一眼，他很明白，雪魔这么说，是想拉自己下水，就算自己将来不投奔太阴山，在人间和空界也会成为人人喊打的那种，基本上没的混。
<br/>　　不过叶少阳放弃了申辩，像是眼前这种情况，根本就解释不清楚，干脆对雪魔说道：“那你这个下属还不掩护我，还等什么！”
<br/>　　没等雪魔开口，对方摇光仙子等人先冲了过来，与那些鬼兵、尸兵厮杀在一起。
<br/>　　叶少阳转身对大伙说道：“跑！只要出了星宿海的灵力范围，立刻回人间，回人间再说！”
<br/>　　叶少阳说完，对小九使了个眼色，低声道：“帮我抓住小玉。”
<br/>　　小九猛然一跃，朝芮冷玉扑过去，芮冷玉立刻警觉地举剑抵挡，她自从觉醒之后，体内煞气爆发，修为也提升了好几分，不过因为她神识模糊，战斗意识薄弱，再说绝对实力比小九还是差不少，很快就被小九擒住，叶少阳立刻上前，划破指尖，用血在她印堂上飞快地写了个定魂符。
<br/>　　魂魄一旦被定住，自己怎么样都是解不开的。
<br/>　　叶少阳把芮冷玉扛在肩上，小九立刻坐下来，想让叶少阳爬上去。叶少阳却把芮冷玉放在她身上，说道：“你们先带她走，回我家等着我。”
<br/>　　众人大惊，忙问他要去干什么。
<br/>　　“我去帮道风。”叶少阳坚定地说道。
<br/>　　大伙呆住，随即明白了叶少阳的想法，雪琪怔怔地说道：“你想清楚，那里面很危险。”
<br/>　　“就是危险，我才要去，小玉已经获救了。我不能丢下道风和宫梓。”
<br/>　　“我跟你一起！”
<br/>　　“我们也去！”
<br/>　　大家纷纷报名。
<br/>　　小青说道：“老大你别阻止我们，你是为你师兄，我们是为你，都是一样的。”
<br/>　　叶少阳道：“如果是帮忙，我一定会让你们去，但是这次不行，人多了，虽然实力更强，但是目标也增加了，万一救出道风，但是你们中有人出意外，让我怎么办？再说人多了分心，我的实力，自保应该还是可以的。”
<br/>　　大伙还想说什么，叶少阳摆了摆手，说道：“我是老大，这次你们听我的，你们还有任务，就是保护小玉回到人间，这一路上说不定还会有援兵什么的，多加小心，就这样，快走！”
<br/>　　叶少阳说完，看了小九一眼，专门嘱咐了一声：“小心。”
<br/>　　小九盯着他深情地凝视片刻，一纵身，背着芮冷玉朝树林里窜了过去，别人也都陆续跟上。
<br/>　　叶少阳深吸一口气，纵身朝山谷冲了出去，虽然肋下被芮冷玉刺的伤口有些疼，一动就会牵扯，不过还在可控制的范围。
<br/>　　“师兄，我来了！”叶少阳眼望着山谷，喃喃说道。
<br/>　　“什么意思！”
<br/>　　雪魔见叶少阳又奔了回来，吃惊问道。叶少阳没回答他，雪魔也没工夫再问了。他已经跟空界联盟里两个强者交上了手，根本无暇再顾忌其他。

<br/>　　叶少阳看了一眼，巧得很，就是那一对跟自己打过交道的梨山弟子。
<br/>　　两人见到叶少阳，也有种仇人见面的感觉，立刻放开了上古邪神，奔着叶少阳过来。
<br/>　　上古邪神看了叶少阳一眼，飞奔进山谷深处，八成是去帮道风去了。
<br/>　　“你居然又回来了！”
<br/>　　两人冲叶少阳冷笑。然后道姑说道：“师兄我来吧。”
