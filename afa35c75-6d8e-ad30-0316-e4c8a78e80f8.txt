　　寒冰之髓，极其珍贵。
<br/>　　但其实它的最大用途，还是用于武者突破大境界。
<br/>　　特别是对于修炼寒冰一道的武者而言。
<br/>　　小小一滴乳白色液体内，便蕴含了惊人的寒冰力量。
<br/>　　这种恐怖的爆发，让得武者在冲击下一大境界时，难度骤减。
<br/>　　加之，它还有着凝神静气，稳固心神，疗伤强体等诸多神效。
<br/>　　这些才是它珍贵无比的原因。
<br/>　　像萧逸这般只吸收力量填充气泉，用于突破小境界，倒是显得相当奢侈且浪费了。
<br/>　　换句话说。
<br/>　　若萧逸将等量等价的寒冰之髓，换成等量等价的无杂质丹药的话，效果会大得多。
<br/>　　只是，萧逸现今没那个时间，也没兴趣浪费那个时间。
<br/>　　言归正传。
<br/>　　萧逸内视一番气泉。
<br/>　　此刻的气泉内，液态元力高度，已满。
<br/>　　且已经大量溢出。
<br/>　　金色的完美力量，散发着炽热与冰冷两种气息，在气泉之外的小世界中流转不停。
<br/>　　极寒、极热两股液态元力，泾渭分明，各占一半，绚烂至极。
<br/>　　不错，元力已经溢出气泉了。
<br/>　　之前各个境界的修炼，都是填充气泉。
<br/>　　而天元境，来到天元七重之后，修炼便产生了变化。
<br/>　　修炼，变为了填充小世界。
<br/>　　这个变化，为的是下一个境界，也就是极境所做的准备。
<br/>　　作为武神之下第一境界，极境，极难达到。
<br/>　　无数年来，无数天才武者，皆在这一步之前而却步。
<br/>　　这一步，难倒了、挡住了无数天骄。
<br/>　　武者一旦跨过这一步，便是天壤之别，如鱼跃龙门，一飞冲天。
<br/>　　言归正传。
<br/>　　天元境后期的修炼，元力溢出气泉，流淌于小世界中。
<br/>　　这些元力，将支撑起整个小世界，增加小世界的强度。
<br/>　　武者，从踏入洞玄境后，便有了‘小世界’，以及其中的‘气泉’，从此脱凡入道，手段颇多。
<br/>　　气泉，是储存真气、元力的地方。
<br/>　　小世界，则是与真实世界产生共鸣，从而让武者获得各种手段。
<br/>　　御空飞行、灵气禁制，用途各异的屏障等等。
<br/>　　小世界越强，与真实世界的共鸣便越剧烈，手段也就更强了。
<br/>　　武者修炼，随着境界不断突破，气泉不会有任何变化，但小世界却不停地在扩张，变强。
<br/>　　这也让得武者的手段，不断强大。
<br/>　　这就是所谓境界上的差距，所带来的实力差距之一。
<br/>　　而到了天元境后期，元力填充小世界。
<br/>　　将让得小世界的强度，陡然间飙升。
<br/>　　故，小世界与真实世界的共鸣，将异常强烈。
<br/>　　这，让得武者获得了新的手段。
<br/>　　感悟、乃至摄取天地间的武道力量。
<br/>　　天地武道，无穷无尽，剑道、刀道、寒冰一道、火之一道等等。
<br/>　　这一种种武道，充斥在整个世界中。
<br/>　　这，就是这个世界的规则力量。
<br/>　　一切，都因此而存在。
<br/>　　武者的一生，便是追求这些武道。
<br/>　　剑势，其实便是武道手段之一。
<br/>　　以自身武道，与天地武道产生共鸣，从而暂时借用天地武道之力，压迫敌人。
<br/>　　而天元境后期，将不再是借用。
<br/>　　而是真的吸入自身小世界，为己所用。
<br/>　　然，武道力量，飘渺、虚无。
<br/>　　这，便是天元境后期需要增加小世界强度的原因。
<br/>　　武者只有与真实世界产生足够强大的共鸣，才能吸摄这些飘渺虚无的武道力量。
<br/>　　曾有传言，武神的小世界，便与真实世界一般大小。
<br/>　　武神因此而掌控世界，凌驾天地之上。
<br/>　　武神能在小世界内做的一切，都能在真实世界中做到。
<br/>　　操控阴阳、打破轮回，无所不能。
<br/>　　更有传言，这个世界，本身就是武神的小世界。
<br/>　　当然了，两种传言，真假不定。
<br/>　　却也阐述了小世界的重要性。
<br/>　　萧逸现今，已停下了修炼。
<br/>　　他已经能从天地之间，感受到无数武道规则力量。
<br/>　　往日那些虚无缥缈的武道力量，仿佛变得可捉摸了起来。
<br/>　　不过，感知到归感知到。
<br/>　　要真正吸摄入小世界中，还是难度极大的。
<br/>　　这取决于武者本身对这些武道的掌控能力。
<br/>　　武者对这种武道的感悟越深，掌控力越高，便感知得越明显，越容易摄取。
<br/>　　反之，武者对某种武道的感悟很浅，便感知得极其模糊，更别说摄取了。
<br/>　　如今，萧逸感知最明显的，自然是霸道剑道，以及寒冰剑道。
<br/>　　“吸。”萧逸低喝一声。
<br/>　　天地间的这两种武道力量，顷刻间排山倒海般涌来。
<br/>　　武道力量，进入小世界，成为小世界所掌控之力量。
<br/>　　这些被吸收的武道力量，已真正成为自己的力量。
<br/>　　当初，不过是借用武道力量而成的剑势，威力已如此不凡。
<br/>　　真正由自己掌控并施展，又该强到什么地步呢？
<br/>　　半个时辰后，萧逸停下了吸收。
<br/>　　武道力量的吸收，存在着限制。
<br/>　　取决于武者自身修为境界，以及武道掌控力。
<br/>　　“寒冰裂天斩。”萧逸低喝一声。
<br/>　　暴雪剑上，一个缩小版的‘暴风雪’，陡然而生。
<br/>　　这次，除了耗费大量元力外，还融合了霸道剑道的武道力量。
<br/>　　轰的一声巨响。
<br/>　　一道恐怖剑气，轰出洞穴之外。
<br/>　　外头不断吹袭的凛冽罡风，瞬间被劈散。
<br/>　　那些连不夜宫主那个层次的武者都无法承受太久的凛冽罡风。
<br/>　　竟在这一剑下，不堪一击。
<br/>　　“好强。”萧逸双眼一眯。
<br/>　　这就是武道力量的强悍。
<br/>　　或者更准确来说，便是霸道剑道的武道威能。
<br/>　　萧逸收敛下思绪。
<br/>　　天元境后期的修炼，便是填充小世界，以及摄取武道力量。
<br/>　　待得小世界被元力完全填满。
<br/>　　一种武道力量，吸收程度达到一成。
<br/>　　那时，武者便能踏入极境。
<br/>　　那所谓的一成武道力量，指的是这种武道，真正完整中的一成。
<br/>　　比如说，霸道剑道。
<br/>　　整个天地，武道无数，剑道、刀道、棍道、寒冰一道、兽之一道等等。
<br/>　　剑道，只是这无数武道中的一种。
<br/>　　而剑道，又可分为大自在剑道、霸道剑道、狠辣剑道、轻柔剑道…等等无数种。
<br/>　　当然，各种武道间，有强弱之分。
<br/>　　萧逸要做的，便是将天地间那条完整的霸道剑道所蕴含的武道力量，吸收够一成。
<br/>　　那时，便可突破极境。
<br/>　　萧逸现今小世界内吸收的武道力量，大概便在半成左右，不足一成。
<br/>　　但，这些不成问题。
<br/>　　萧逸身怀裂天剑魔前辈的剑道传承，这些感悟和吸摄，对他而言轻而易举。
<br/>　　当然，顺带一提。
<br/>　　萧逸虽是从裂天剑魔前辈的武道传承中，获得霸道剑道，并参悟了10成以上。
<br/>　　可这10成，指的是裂天剑魔的全部剑道知识。
<br/>　　但裂天剑魔本身，并不代表完整的霸道剑道。
<br/>　　他也在一直修习并追求着霸道剑道，只不过他已经到达相当高深的地步罢了。
<br/>　　萧逸也不确定裂天剑魔前辈到底参悟了整条霸道剑道多少成。
<br/>　　但毫无疑问，这些武道知识，能让他在地极巅峰前，在武道力量摄取方面，一帆风顺。
<br/>　　反倒是填充小世界方面。
<br/>　　小世界是气泉的两倍。
<br/>　　且，洞玄境之后，一路修炼而来，气泉大小不变，小世界却随修为境界不断扩大。
<br/>　　萧逸要将小世界填充完整，所需的力量，简直无法想象。
<br/>　　极境，拦住了几乎所有绝世天骄。
<br/>　　可真正难住他们的，是天地武道力量的吸摄。
<br/>　　是对自身所修习武道的感悟，以及掌控。
<br/>　　反倒是填充小世界，对他们而言，只是轻而易举之事。
<br/>　　不少大势力，都有极界碑。
<br/>　　这些极界碑，可让后辈天骄，轻松参悟并获得极境的武道知识。
<br/>　　可是，参悟归参悟。
<br/>　　真正将这些武道知识，化为己用，受自己掌控，并达到可以摄取天地武道力量的地步，极其困难。
<br/>　　这对武者的资质、领悟力、天赋等等，有着极高的要求。
<br/>　　萧逸则恰恰相反。
<br/>　　庞大气泉之下，造就的庞大小世界，若要填满，成了他最大的阻难。
