　　第59章
<br/>　　蛤蟆岭下，夜幕里，有哭声时时响起。
<br/>　　“姆妈，我饿，什么时候吃饭？”一个孩子依偎在母亲怀里。
<br/>　　“睡吧，睡着了就不饿了。”
<br/>　　“呃……呃……”有一大汉哀吟。
<br/>　　“你一个男人哼唧什么，丢不丢人！”
<br/>　　“哥，腿……烂了……”
<br/>　　揭开褴褛的裤腿，小腿已经一片溃烂，散发着恶臭。
<br/>　　这些人最近日子过得不是很好，他们在沦落成奴隶后，每天不得不进行艰苦的劳动。如耕田，修城，挖堤，食不果腹，鞭子抽体。而也有些以前和柳家有仇的人，趁机痛打落水狗，跑过来踹两脚的行为也是可以理解的。
<br/>　　所以这九十几号人，身上或多或少都有些伤病。而夜里山林的环境无论如何也说不上好，饥饿，伤病，恐惧折磨着他们的心灵。
<br/>　　“我们为什么要忍受这些，为什么！”
<br/>　　有人终于喊出声来，像是询问这不公的苍天。可所有人都懂，他真正发难的是柳慕云。
<br/>　　这个白发苍苍的老人，代表家族最后的体面和荣耀。
<br/>　　柳慕云报以沉默，独自守着火堆。
<br/>　　这变相怂恿了众人，有纷纷议论声响起。
<br/>　　“如果在山上，他们会给我们一顿饱饭吧。”
<br/>　　“昨天吃得还是不错的。”
<br/>　　“那种叫辣椒的东西，城里也只有贵族才吃得到……”
<br/>　　“我没关系，我真的没关系，可我孩子才六岁……”
<br/>　　这些声音听在柳慕云耳中，他心底发出一声叹息：人心乱了。
<br/>　　这九十几号人，有柳家的血亲，更多的却是家里原先的仆人和奴役，他们也随着这次柳家出事被贬为奴隶。
<br/>　　如今在生死拷问下，这些人对柳家的忠诚正在慢慢减弱。而即便血亲之间，也不是其乐融融，你好我好大家好，互相之间勾心斗角不少。
<br/>　　有人想落草为寇，有人还想固守荣耀，意见不统，组织已经开始从内部瓦解。
<br/>　　现在这些人就算上了山，也是互相分成一波波小团体，必然被程大雷利用，无法像之前那样，占据山寨的主导权，和程大雷争权的事情更是想也不要想。
<br/>　　这也在你的计划之中么。
<br/>　　柳慕云猛然抬头，望向山上，心底浮现一丝寒意。
<br/>　　其实柳慕云不是不想留在蛤蟆岭，毕竟他无地可去，可是他也不想落草，因为固有的观念不允许他这么做。
<br/>　　那么，是否有一个两全其美的方法呢？
<br/>　　呃……这个问题，柳慕云没有去想。
<br/>　　他虽然饱读诗书，在柳家以渊博著称，但很显然没有太多智慧。
<br/>　　他只是把这个问题抛给程大雷，稍微展现自己在博弈学上的造诣，面对一个年轻人，展现自己老者的手腕：你还太年轻，是该懂点规矩，这里不是你说了算的。
<br/>　　然而程大雷没有同他玩博弈学，只是一脚将他踹在地上，恶狠狠的说：滚蛋，这里我定规矩。
<br/>　　屈辱，或者去死。
<br/>　　柳慕云面对程大雷砸出来的选择题，束手无策。
<br/>　　孟子云靠近柳慕云，低声道：“柳爷爷，我有个法子，不知道行不行得通。”
<br/>　　“说……”
<br/>　　“宝儿饿的不行，我们把他搁在寨门前，我不信程大雷不开门。只要他开门，就能把一些老弱送进去，留着强壮，咱们可以慢慢和他谈条件。”
<br/>　　柳慕云猛地抬头，像是第一次认识孟子云，二人以冰冷的目光对视，都从对方眼底感受到对彼此的恐惧。
<br/>　　柳慕云不得不承认，孟子云所说是个法子。程大雷摆出屈辱或死亡的选择题，孟子云的法子同样给程大雷一个选择。
<br/>　　道德么。
<br/>　　没有几个人能面对这样的拷问，可面对这个法子，柳慕云选择了沉默，良久，他才缓缓道出一句话。
<br/>　　“子曰：“志士仁人，无求生以害仁，有杀身以成仁。”
<br/>　　孟子云懂得柳慕云话中的意识，沉默告退，望着他的背影，柳慕云心底涌出深沉的寒冷。
<br/>　　内部的崩溃，比程大雷预估得来的更早。
<br/>　　在天没亮的时候，有人沿着下山的道路，小心谨慎的避过陷阱上山，双拳重重砸山寨大门。
<br/>　　“开门，我要上山，当山贼！”
<br/>　　他就是那个小腿溃烂的大汉，他的行为遭到鄙视。
<br/>　　“你竟然要做山贼，怎么去见柳家列祖列宗！”
<br/>　　“祖宗有灵，也无法瞑目在九泉！”
<br/>　　言语如刀，扎向汉子后背，他并没有回头，只是用拳头重重的砸着寨门，一声声的呼喊着。
<br/>　　山寨大门，没有未他打开，仿佛永远也不会打开。
<br/>　　如今这道门已是汉子生存的最后希望，它的紧闭似乎象征着死亡。大汉越来越恐惧，声音越来越大。
<br/>　　山下众人看着这一幕，他们讥讽这大汉，未必心中便不动摇。看着铁打的汉子如丧家狗一样摇尾乞怜，这事情仿佛就发生在他们自己身上，屈辱笼罩着每个人。
<br/>　　多么屈辱啊！
<br/>　　他们曾经被客客气气的请回山寨，悉心周到的安排好食宿，甚至还准备着干净衣服。
<br/>　　但是他们选择了拒绝，为了荣誉。
<br/>　　而如今，他跪在地上，想祈求天神一样叩门，只期望这门为他打开一条缝。
<br/>　　真屈辱啊！
<br/>　　可要屈辱到什么份上，这扇门才会为他开启，跪下还不够么，叩拜还不够么，哭喊还不够吗？我已经放下我的荣耀，放下我为人的尊严，卑微的趴在地上祈求您的怜悯，难道这样还不够嘛？
<br/>　　不够！
<br/>　　人可以有自己的荣耀，但你要有准备为这荣耀去死，没有人有责任为你的荣耀买单，除了你自己。
<br/>　　放下你的荣耀，尊严，为人信仰的一切，然后，这里可以给你活着，只有活着。
<br/>　　那些为荣耀去死的人我们尊重他，但这里没有为他们准备埋骨之地。死，也给我死远一些。
<br/>　　这扇紧闭的门，就代表程大雷的回答。
<br/>　　他自然听到外面的声音，翻身睡觉，只是用被子蒙住脸。
<br/>　　那句话怎么说来着：捧着敬着你不会说人话，非得狠狠上去踹上一脚，你才明白自己叫什么名字。
<br/>　　正如你每天给他一吊钱，一天不给，他就会恨你。
<br/>　　你每天给他一个耳光，一天不打，他就会爱你。
<br/>　　还有一句话这么说的：讨吃不如摔碗，张口不如板脸。
<br/>　　
<br/>(本章完)
