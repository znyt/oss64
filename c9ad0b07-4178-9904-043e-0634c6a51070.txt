　　曲波就站在慕寒身边，立刻点了点头。
<br/>　　清木冷哼一声：“就凭你们两个，怕是证据不足吧？”
<br/>　　龙阳真人喝道：“清木，你说这话什么意思，我知道你一直觉得自己本事好，前任掌教又没儿子，如果没有前任掌教的遗言，你最有可能继承大统，而今掌教换人了，你心中不舒服。”
<br/>　　清木拱手道：“不敢，不过，师兄你这话说得也没错，师父没儿子，新任掌教肯定是要在我们这些弟子中选出来，想法我是有的，但不管是不是我，至少要让大家心服口服，要大家说了算，你们现在突然拉来一个陌生人，就说他是掌教继承人，这算个什么，你也说龙虎山千秋万代，掌门人事关重大，我们都是龙虎山弟子，这件事谁也不敢马虎，大伙说对不对？”
<br/>　　本来，关于掌门人选，跟在场绝大多数都没关系，他们自己也当不上，也无所谓谁当，但清木毕竟是熟人，而这个慕寒是个之前从没见过也没听说过的陌生人，让他来当掌门，大伙当然不服，当下听了清木这番有煽动性的话语，纷纷符合起来。
<br/>　　龙阳真人怒道：“清木，连掌教的遗言你都怀疑吗？”
<br/>　　“我不怀疑，我只是怀疑你们两个，毕竟遗言大家都没听到，你有本事将掌教请来亲口告诉我们，就算掌教找一条狗来接任，我们都没话说，大伙说呢！”
<br/>　　众弟子在他煽动之下，纷纷站出来支持。
<br/>　　龙阳真人咬牙道：“清木，你是故意的是吧，你明知道掌教魂魄已经去轮回，上哪儿去找他老人家去！”
<br/>　　清木冷哼，“那就没办法了，重新选掌教吧。”
<br/>　　龙阳真人朝左侧看去，在道场左边，还坐了一排五个道士，大部分都是七老八十的老者，他们是龙虎山各堂的执事，都是跟张无生平辈的老人，实力虽然不出众，在法术界也声明不高，但一向兢兢业业，在龙虎山内部德高望重，说话很有分量。
<br/>　　“几位师叔师伯，你们是怎么说？”龙阳真人这是感觉自己罩不住了，只能找他们出面。
<br/>　　几人互相看了看，其中一个白胡子老者说道：“龙阳，掌门师兄的遗言，我们是信的，但为龙虎山千年计，这新任掌教，必须让大伙满意，否则将来别谈复兴，能不能维持住名声都是一回事。”
<br/>　　龙阳真人心中咯噔了一下，暗想这几个家伙真是够鬼的，之前接慕寒来之前，他就跟这几位通过气，当时他们都没什么意见，表示一切遵照张无生遗言，今天当着大伙的面，却又说出这番模棱两可的话，一时间有些无措，思索怎样回应。
<br/>　　曲波更是不敢作声，他本来地位就不高，只是作为遗言的传达者，才有幸站在台上，只要一开口，立刻就会成为众矢之的，只能紧张地看着龙阳真人。
<br/>　　“敢问这位师叔，什么才叫让大伙满意？”
<br/>　　慕寒突然开口，声音带着一股磁性，但语调平静，不带一丝情绪，给人一种深不可测的感觉。
<br/>　　老道士看了他一眼，眼珠子转了转，想了一下说道：“龙虎山，一向与茅山并蒂对比，我龙虎山弟子众多，声势浩大，在法术界的影响，一向在茅山之上，茅山数百年来，内门弟子只有一个，最多也才两个，如今，他叶少阳却成为天选之人，做出了一连串惊天动地的大事，连轩辕山都敢硬碰……
<br/>　　最近法术界发生的一系列大事，哪一桩不是跟以叶少阳为首的？我龙虎山名为道门魁首，却充作附庸，以前，道渊师叔在，还能震住场面，后道渊祖师为法术公会算计而死，如今掌门师兄也间接为他们所害，我龙虎山后继无人，新任掌教，须得有力挽狂澜之力，才能带领山门崛起、报的血海深仇，若是一介平庸之辈，便是掌门师兄遗言在，亦是无法遵守。”
<br/>　　这番话有理有据，在场的龙虎山弟子都不住地点头，感慨龙虎山多灾多难，山河日下。
<br/>　　慕寒耐心听完，微微一笑，说道：“师叔这意思，我是听懂了，新任掌教，必须有力挽狂澜的手段……但不知这手段，说的是哪方面？”
<br/>　　老道士道：“诸多方面，缺一不可，然则别的方面，须得日久才见水平，个人实力，是为首要。”
<br/>　　清木几人立刻附和：“没错，我龙虎山历代掌教，都是道门大能，实力太弱，如何当此大任！”
<br/>　　慕寒转头看他：“你实力如何？”
<br/>　　清木拱手道：“我学艺不精，不久之前，刚迈过天师门槛！”言语之间却带着一股傲气。
<br/>　　天师，在法术界已经很罕见了，关键是他才三十出头，便能当上天师，未来只要没大的意外，苦修个十年八年的，地仙也不是没指望。因此，他是有理由感到骄傲的。
<br/>　　“天师……还不错。”慕寒缓缓走下台阶，道：“你是龙虎山二代弟子中最厉害的？”
<br/>　　清木看了看左右，谦虚地说道：“我们这几位入室弟子，实力都差不多。”
<br/>　　随即反问道：“慕寒师弟，你这是要同我斗法？”
<br/>　　“既然别的才华一时间无法施展，那就直接一点，斗法一场，若诸位师兄弟中有人能胜过我，我立刻走人，此生不入龙虎山。”说到这，慕寒转头朝那几个执事看去，“几位师伯师叔，若有想法，也可一战。”
<br/>　　此言一出，全场皆惊。
<br/>　　要说挑战清木，还没什么，但这几个老家伙虽然资质不是特别高，好歹年龄大，法术界没有“拳怕少壮”这个说法，年纪大，同等天赋之下，罡气积累的一定比年轻人多，张无生以前一枝独秀，他一死，这几个老家伙便是山上实力最强的，年轻人挑战他们，简直自取其辱。
<br/>　　几个老道士互相看了看，都不要动手，一来他们早就过了置气的年纪，二来慕寒既然敢挑战，必然是有一定把握，以他们的年纪辈分，打赢后辈不光彩，打输了那就太丢人了，于是其中一人道：“我们这几个老家伙就算了，掌教之位，本来就是你们年轻人的，清木，你与慕寒师侄较量一番吧，点到为止。”
<br/>(本章完)
