　　“你必败！”苏莫淡淡的回了一句，他同样信心十足。
<br/>　　两人互相凝视，火药味十足。
<br/>　　对于两人而言，少司之位，均是志在必得。
<br/>　　苏莫对于少司之位，已经谋划许久，只有成为少司，他才有救宏青璇的机会，只有成为少司，他才能在巫族之内为所欲为。
<br/>　　而巫人王对于少司之位，已经等了太久了，上一届少司选拔他败给了巫凌，这一次是最后的机会，他无论如何也要拿下。
<br/>　　“敢打赌吗？”巫人王见苏莫如此自信，嘴角顿时勾起了一丝森然的笑意。
<br/>　　“来者不拒！”苏莫面色淡淡，他极为自信，无论对方要赌什么，他都一并接了。
<br/>　　“一万块下品灵晶，敢吗？”巫人王沉声说道，他开的赌注非常的大，一万块下品灵晶对于武帝境武者而言，已经是一笔巨富了。
<br/>　　尤其是对苏莫这种低阶武帝而言，可以说就算是全部的身家，也不一定有一万下品灵晶。
<br/>　　“一言为定！”苏莫郑重的点了点头，丝毫无惧的接下了对方的打赌，虽然他没有一万下品灵晶但他有必胜的把握。
<br/>　　众人望着火药味十足的两人，心中更加的期待了，两人均是自信十足，难道都自信天剑诀达到了大成境界！
<br/>　　大祭司目光流转，望了巫人王片刻，又瞥了眼苏莫，眸中闪过一丝莫名的光芒。
<br/>　　他已经知晓结果，接下来两人的施展，只不过是走个过场而已。
<br/>　　“你们两人同时向老夫出手吧！”
<br/>　　大祭司开口了，随即他袖袍左右一挥，两股气浪席卷而出，顿时将苏莫和巫人王卷了出去，分别卷到了圣巫广场的两端，相隔近四百里。
<br/>　　两人漂浮在百丈高空之中，目光依旧在互相凝视。
<br/>　　下一刻，两人同时动了！
<br/>　　“起！”
<br/>　　“天剑诀！”
<br/>　　两人几乎同时大喝一声，纷纷施展了天剑诀，刹那之间，整个圣巫广场风云变换。
<br/>　　一切细小的杂物，通通飞了起来，而且由于两人都位于广场的边缘，施展的外围影响到了广场之处。
<br/>　　只见巫人王周身方圆两百里之内，大量碎石、落叶，甚至是很多建筑上的瓦砾全部被调动了起来。
<br/>　　无数的杂物，通体闪烁着晶莹的光芒，爆发出惊人的剑气。
<br/>　　而后，这些杂物化为了一只只利剑，爆射长空，洞穿空间，裹挟浩大的威势，向大祭司袭杀而去。
<br/>　　巫人王的天剑诀相当强大，远超之前的任何人几十倍都不止，甚至比巫玛的天剑诀都要强大十倍。
<br/>　　当然，这其中有一部分原因，是因为巫人王的修为较高，对于天剑诀的威力，自然加成极大。
<br/>　　但若是抛却这种因素，巫人王的天剑诀水平，也是超越了巫玛。
<br/>　　但是，这一刻，却是没有人关注巫人王，因为所有人都被苏莫吸引了。
<br/>　　只见另一边，仿佛是大地失去了引力一般，方圆三、四百里之内，无数的物体飘飞而起。
<br/>　　碎石、落叶、木屑，店铺的牌匾、瓦砾等等，甚至一些脆弱的建筑都四分五裂。
<br/>　　数之不尽的物体飞到了高空，铺天盖地，发出璀璨的剑光。
<br/>　　“去！”
<br/>　　苏莫一指点出，无数的物体光芒一闪，向大祭司爆.射而去。
<br/>　　数之不尽的物体，化为了数之不尽的利剑，仿若一阵浩大的剑雨，剑威冲天，气势磅礴，形成了剑之风暴，笼罩了一方空间。
<br/>　　苏莫的修为比之巫人王低的多，但天剑诀的威势，却是强大了不止一筹，孰强孰弱一目了然。
<br/>　　众人见此心惊不已，巫尘的天剑诀威势，居然比巫人王还要强大！
<br/>　　巫尘悟性究竟有多高？能在三天之内，将天剑诀修炼到如此地步，简直是将巫穹、巫天御的人甩出了十万八千里。
<br/>　　这意味着什么？
<br/>　　意味着巫人王要败了，巫尘获胜了！
<br/>　　少司之位，很可能要被巫尘得去了。
<br/>　　“呵呵！都不错，都达到了大成境界！”
<br/>　　大祭司的口中传出一声轻笑，随即一挥袖袍，巫人王和苏莫的攻击，瞬息间便全部烟消云散。
<br/>　　现场平静了下来，但是巫人王的脸色却是黑了下来，简直是黑如锅底。
<br/>　　他居然败了，这怎么可能？
<br/>　　巫人王心中咆哮，巫尘区区武帝境三重修为，怎么可能有这么强的悟性？怎么可能天剑诀修炼得如此高深？
<br/>　　这简直不可能！
<br/>　　巫人王不怕输灵晶，再多的灵晶对于他来说，都只是身外之物。
<br/>　　他怕的是因此失去少司之位。
<br/>　　现场鸦雀无声，所有的人都望着大祭司，要看大祭司如何决断！
<br/>　　巫尘和巫人王两人，同时通过了三关考核，按照规矩的话，两人都能成为少司。
<br/>　　但是，每一届少司选拔，都只会选取一人。
<br/>　　巫尘在第二关之时夺了第一名，这第三关又夺了第一名，其表现明显超越了巫人王。
<br/>　　所以，接下来巫尘很可能被直接选为少司。
<br/>　　苏莫面色平静，目光同样凝视和大祭司，他的表现超越了所有人，他相信大祭司若是没有私心的话，就一定会选择他。
<br/>　　看台之上，大祭司面现沉吟之色，随即转头看向身边的四祭司，两人相视一眼，而后四祭司点了点头。
<br/>　　显然，两位祭司在无声的交换心中的意见。
<br/>　　片刻之后，大祭司目光转动，望向了苏莫。
<br/>　　“巫尘天赋超绝，成绩优异，经老夫和四祭司决断，成为最终的胜出者，赐予我族少司之位！”
<br/>　　大祭司的声音洪亮异常，如同天地雷音，传遍了圣巫广场，传遍了天冥圣城，传遍了周围的九大卫城，向八方扩撒而去。
<br/>　　圣巫广场一片死寂，整个巫族的十大核心城池均是安静了下来，所有人都听到了大祭司的声音。
<br/>　　大祭司，是在向所有族人宣布，少司选拔最终的结果。
<br/>　　广场上的众人均是凝望着苏莫，目中露出羡慕、甚至是嫉妒之色，少司之位，居然被巫尘得去了，这是选拔之前无人能猜测到的结果。
<br/>　　但是有几人，却是怎么也高兴不起来，他们简直不敢相信这铁一般的事实。
<br/>　　这几人就是巫天御、巫穹以及长老巫源三人。
<br/>　　三人和巫尘都有过节，甚至是仇怨，巫尘现在成为了少司，地位将超越他们，直接压在了他们的头上。
<br/>　　即便巫源身为族中普通的长老，面对族中少司，无论是身份还是地位，都明显的低了一头。
<br/>　　所以，三人简直不愿意接受这个事实！
<br/>　　但是事实摆在这里，无论三人接受与否，都改变不了这一切。
<br/>　　“我——不——服！”
<br/>　　就在此刻，蓦地，低沉的吼声突然响起，声音中充满了不甘与愤怒。
<br/>　　是巫人王开口了！
