　　相比于已经开始降霜的倭岛，天竺的气候要好的多，虽然已经是八月底，但因为位置比较靠南的关系，这里依旧没有任何一丝的凉意。
<br/>　　尉迟宝林光着膀子坐在军帐里面，所有的窗子都已经被打开，但还是觉得有些烦躁。
<br/>　　程家那两个活宝回去成亲去了，军营里冷清了许多，没人与他拌嘴反倒让尉迟宝林觉得有些无聊。
<br/>　　如果能再打几仗就好了，至少打起来之后不会那么无聊，这黑铁塔一样的汉子如此想着。
<br/>　　“尉迟将军，能不能，能不能再给我们拨下来一批人？橡胶树种的培育基本上已经完成了，我们需要大量的人手来进行栽种。”
<br/>　　卡皮尔坐在大帐里面，他并没觉得这里的环境有什么不好，毕竟他就是本地人，早就已经习惯了这样的气候条件。
<br/>　　“要人要人，这段时间你都要了多少人了？五百了吧？你还想怎么样？”尉迟宝林心里烦闷，自然不会对卡皮尔有什么好态度。
<br/>　　不过好在卡皮尔并不在乎这些，咂了咂嘴继续说道：“将军，这一次我们不需要军士，只要普通百姓就好，不过需要一些有种植经验的，最好是常年务农的。”
<br/>　　“那不如让你当城主好了，这样你岂不是可以管更多的人。”
<br/>　　“不敢，不敢！”卡皮尔连连摆手，尴尬的说道。
<br/>　　他可以来提要求，但是若让他去管理一个城，这事儿他可不敢干，天知道被大唐那位太子知道了，自己的脑袋还能不能保得住。
<br/>　　毕竟今时不同往日，他的身份也不一般，如果让他在大唐当个县令什么的他还敢接受，在天竺这地方当城主，再加上他天竺人的身份，一个不好让人误会，那就是掉脑袋的大罪。
<br/>　　“行吧，看你可怜巴唧的，人我可以给你，不过若是这些人出了问题，可别我唯你是问。”尉迟宝林刚刚也只是无聊逗这个在大唐有公爵之位的天竺人开心，现在逗的差不多了便点了点头答应了他的要求。
<br/>　　本来嘛，卡皮尔过来种树那可是奉了李承乾的命令，就算是看在那位小爷的面子上，尉迟也不会不答应卡皮尔的要求。
<br/>　　……
<br/>　　从尉迟宝林的大帐中出来，拿着一份调人的手令，卡皮尔登上了回程的马车，一路无话回了那个不知名的小村。
<br/>　　黑胖子依旧在树林中忙碌着，他现在已经是有身份的人了，但却依旧要干和以前相同的工作，虽然有些不大合心意，但也算免强能够接受，毕竟能够活着也是一件让人十分愉快的事情。
<br/>　　“老黑，老黑！”远远的，回到小村的卡皮尔对刚刚从林子里出来的黑胖子挥了挥手。
<br/>　　“公爵大人，您回来啦？”黑胖子见是卡皮尔叫自己，立刻迎了上去。
<br/>　　这家伙因为长的太黑了，卡皮尔干脆也不叫他的名字，反而直接叫他老黑。
<br/>　　不过这样也好，显得亲切，所以黑胖子并不反对这样的叫法。
<br/>　　“回来了，这下我们应该很快就可以大唐了。”卡皮尔兴奋的说着，然后拿出了得自尉迟宝林的手令：“看看，手令拿到了，只要调集了人手，大面积试种成功，我们就可以带上种子回去大唐，再也不回这个破地方了。”
<br/>　　“嘿嘿，那敢情好，公爵大人，不瞒您说，这里我早就待够了，还是大唐好啊，什么都不缺，不像这里吃东西连个调料都凑不齐。”黑胖子赔着笑脸附和着卡皮尔。
<br/>　　“行了，不多说了，跟我走，我们要人去。”卡皮尔似乎还没有从兴奋中回过神来。
<br/>　　“公爵大人，那个，有，有件事。”黑胖子见卡皮尔要走，再次叫住了他。
<br/>　　“什么？”
<br/>　　“程大将军到了。”黑胖子说道。
<br/>　　“程……”卡皮尔脑子里闪过一对漂亮的大门牙，人便是一个哆嗦。
<br/>　　“您要不要见见？”黑胖子看着有些犹豫的卡皮尔问道。
<br/>　　那程大将军实在是有些难缠，卡皮尔不在的这几天里，黑胖子在他的面前总是提心吊胆，生怕一不小心被他给吃了。
<br/>　　卡皮尔暗中祈祷了一下，然后硬起头皮说道：“见，程大将军可是太子殿下的丈人，必须要见。”
<br/>　　……
<br/>　　海边，一座明显是刚刚修好的码头上，卡皮尔和黑胖子坐上了一艘小船，随着小船离开海岸，向着不远处的未知而去。
<br/>　　时间不大，小船已经靠到了停在大海中的战船边上，顺着上面挂下来的绳梯，两人哆哆嗦嗦爬了上去。
<br/>　　虽然说这不是黑胖子第一次爬了，但不知为什么，他还是很担心这绳子会不会断了，自己会不会掉进海里，要知道他可是不是游泳的，而且这片海域听说还有鲨鱼存在。
<br/>　　战船之上，已经习惯了海上颠簸的程老魔头正光着膀子吃着传说中的生鱼片，也就是鱼脍，刚刚从海里钓上来的鱼很新鲜，除了老程的吃法让人有些不敢苟同之外，倒也可以算得上别有一番趣味。
<br/>　　什么？你说老程的吃法有什么特殊？
<br/>　　不怕跟大家说，这货竟然是抱着鱼在啃，啃一大口然后往嘴里倒一口调料。
<br/>　　如此特殊的吃法，简直能吓死个人，尤其是刚刚上船的卡皮尔两人更是看的头皮发麻，只觉得被着老魔头抱着的就是自己，那老魔每啃一下都让两人不由自主的哆嗦一下。
<br/>　　好半晌之后，老程将一条鱼啃的差不多了，随手丢到一边，用手往嘴上一抹：“喂，我说你们两个怎么回事？来找本将军为何不说话！”
<br/>　　“程，程大将军，我们两个就是来拜见一下。”卡皮尔本来是说准备了一肚子的说词的，可是看到老程如此彪悍的进食方式，啥说话都给吓没了，现在能站在这里已经很不容易了。
<br/>　　“哦。那就是你们两个都没啥事儿，是吧？”程老魔一点吓到人的自觉都没有，随意的又从边上盆里抓起一条鱼来，‘吭哧’就是一口，然后又往嘴里倒了一口调料，嚼了一会儿吐出无数片鱼鳞。
