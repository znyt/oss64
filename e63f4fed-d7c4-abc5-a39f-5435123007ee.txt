　　那族人看了眼周遭的子弟，脸色中有些迟疑与凝重。
<br/>　　四长老会意，道，“我们进议事厅说。”
<br/>　　萧逸本想离开，却被六长老一并拉了进议事厅。
<br/>　　萧家在陨星山脉的族人，一部分开采紫云铁，一部分则是作为探子被派遣出去。
<br/>　　这些探子的职责是到山脉的更深处开路，一来是看能不能遇到天材地宝，并收集起来，二来则是看看有没有其他的紫云铁矿脉。
<br/>　　慕容家和江家也会有这样的探子。
<br/>　　陨星山脉非常辽阔，覆盖的范围比十个紫云城还要大，有很多地方是人类武者没有踏足过的。
<br/>　　而这个族人，是萧家的一位执事，后天七重武者，也是探路队的队长，名为萧景。
<br/>　　“萧景执事，何时如此慎重？”四长老问道。
<br/>　　萧家拱了拱手，对两位长老行了一礼，这才禀报道，“这一个月我们探路队一直在我们萧家矿脉以东的方向探路。”
<br/>　　“就在前日，我们在距离此地大概百里之处，发现了一处隐秘的山林，林中竟有一株...”
<br/>　　萧景顿了顿，沉声道，“妖血炎心果。”
<br/>　　“妖血炎心果？”四长老和六长老同时惊呼一声。
<br/>　　就连萧逸也是眼前一亮。
<br/>　　大长老交给他的炼药心得中，记载了许多珍稀的天材地宝。
<br/>　　妖血炎心果正是其中之一，而且位属三品，更是三品中的极品。
<br/>　　比之同为三品天材地宝的培源朱果珍贵十倍都不止。
<br/>　　妖血炎心果，可用于炼丹，可直接服用。武者服之，有焚炼经脉之效，那庞大的药力，更是能助武者轻松增加修为，乃是不可多得的灵果。
<br/>　　只可惜，妖血炎心果的生长范围很隐蔽，数量也极少。
<br/>　　紫云城内，数十年没有出现过这么珍贵的天材地宝了。
<br/>　　“那灵果还有多久成熟？”四长老面露急切之色，立刻追问了一声。
<br/>　　萧景思索了一下，回答道，“据我的观察，最多不会超过三天。”
<br/>　　“还有三天？”四长老顿时一急，道，“那你还跑回来干嘛，在那守着阿，要是被其他妖兽吞了怎么办？”
<br/>　　萧景连忙道，“四长老放心，其他探路队的队员正在那里看守。而且我回来前已经四处探查过，并没有其他妖兽的踪迹。”
<br/>　　一旁的六长老对四长老翻了个白眼，道，“萧景出了名的谨慎，你急个屁阿。”
<br/>　　“废话。”四长老反驳道，“那可是妖血炎心果阿，谁不眼馋。”
<br/>　　“萧景。”四长老看向萧景执事，问道，“你特地赶回来，是想要援手吧。”
<br/>　　“是的。”萧景道，“那里毕竟与我们萧家营地有百多里的距离，我怕在回来的路上有意外。”
<br/>　　“嗯。”四长老点点头，道，“妖血炎心果乃是重宝，确实该谨慎些。萧景，那你便在家族这些子弟中跳些人手吧。”
<br/>　　“阿？在家族子弟里跳？”萧景愣了愣，惊疑道，“长老你们不去吗？要不然，家族的其他执事也行。”
<br/>　　萧景一向谨慎，他本想让四长老或者六长老一同前往的，有先天武者在，必可保万无一失。
<br/>　　再不济，家族里的后天武者随同，也能让回来的路上把握高些。
<br/>　　可是一群家族子弟，充其量不过凡境九重，要是真的遇上危险，他们根本帮不上忙。
<br/>　　四长老摇摇头，沉声道，“我们三家的高层武者，彼此都在互相注意，一旦我们有异动，反倒会引起另外两家长老的注意。再者，矿脉这边更为重要，我和六长老不能轻易离开。”
<br/>　　“其他执事也一样，在矿脉这边脱不了身的。所以，你只能在家族子弟里选。”
<br/>　　六长老也说道，“妖血炎心果虽然珍贵，但对我们先天的效果很有限。就算拿回来，也是给年轻子弟服用的。倒不如借这个机会，让他们去见识一番。”
<br/>　　“妖血炎心果安全拿回来了，算他们护送有功，灵果也给他们。若失败了，也只能怪他们没有那个机缘。”
<br/>　　两个长老你一言我一语，但目光都看向了萧逸。
<br/>　　萧逸苦笑一声，道，“这趟苦差，你们打算让我去吧。”
<br/>　　“不错。”四长老笑道，“回来的路上虽然有危险，但也会太严重。充其量就是妖兽的袭击，或者一些不开眼的猎妖队找麻烦。”
<br/>　　“我可听说了，你昨日一拳便打得慕容轩吐血，以一己之力就把慕容家一百多号子弟全绑了。这证明你的实力起码在后天二重乃至三重以上，应付那些麻烦应该没有问题。”
<br/>　　四长老满是放心的脸色。
<br/>　　“得。”萧逸点点头，问道，“何时出发。”
<br/>　　四长老道，“现在就出发吧，顺便把萧壮和萧子木也带上。现在家族的年轻子弟里，也就这么几个没有伤势的。”
<br/>　　“他们两人也是凡境九重了，这次出去闯一番，若能突破到后天最好。若是不能突破，增长见识也是好的。”
<br/>　　“好。”萧逸应答一声，起身离开，准备去找萧壮和萧子木。
<br/>　　萧景看了眼萧逸的背影，暗道，“我才出外探路一月，没想到萧逸少家主已是如此天才。年纪轻轻，竟有那番实力力敌慕容家所有年轻子弟，果真厉害。”
<br/>　　萧景看着萧逸已经走远，刚想跟上，却被四长老一把拉住。
<br/>　　萧景一脸疑惑之色。
<br/>　　“萧景，给我听好了。”四长老忽然收起了之前的笑意，一脸凝重与严肃。
<br/>　　萧景见四长老如此严肃，先是一惊，而后立马认真聆听。
<br/>　　四长老低声道，“虽然我不认为回来的路上会有多大的危险，但以防万一，我还是要嘱咐你。此行，妖血炎心果事小，萧逸事大。若真的遇到不可抗的危险，宁愿扔了那果子也要保萧逸平安归来。”
<br/>　　萧景道，“四长老放心，这个我明白。少家主乃我们萧家第一天才，属下自然不会让他出事。”
<br/>　　“嗯。”四长老道摆摆手，道，“去吧。”
<br/>　　萧景离开，六长老挪揄道，“你担心萧逸，干嘛还放他出去？”
<br/>　　“笨蛋。”四长老骂道，“若不让他经历危险磨砺，如何成长。若真的困着他，那不叫保护他，而是在扼杀他的天赋。”
<br/>　　“炼药、武道，无论哪一样，他都足以让其他所谓的天才汗颜和仰望。毫无疑问，萧逸将是我们萧家下一代的所有希望，我自然要处处慎重。不过，此行有萧景在，我倒是很放心。”
<br/>　　.....
<br/>　　萧逸找了萧壮和萧子木后，另外找了两个凡境八重的子弟一同前行。
<br/>　　一行人加上萧景，往东面百里之外赶去。
<br/>　　萧逸万万想不到，此行，他将遇到自来到这个世界以来最严重的一次危机。
<br/>　　幽深的山脉，一眼看不到尽头的山路和野林，仿佛一个噬人的巨口，危险而让人毫无所觉。
