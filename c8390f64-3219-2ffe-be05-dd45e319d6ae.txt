        她和他除了在每个周末的那个下午，是独处在宿舍里，两人甚至没有任何话题，他也很少主动找她说话。
<br/>
<br/>        而他为什么要喜欢上她啊？！
<br/>
<br/>        他如果不喜欢她，该多好……
<br/>
<br/>        至少现在，他还活得好好的。
<br/>
<br/>        童筱颖“啪”地一声，合上了相册，吸了吸鼻子，拿出便利贴，写上了自己的电话号码，留给了马智杰的父亲。
<br/>
<br/>        “这是我的电话号码，马智杰什么时候出殡，马叔叔到时候给我打电话。我定会让这个女孩来见马智杰最后一面！”童筱颖心情沉重地说道。
<br/>
<br/>        马智杰的父亲接过便利贴，抹着眼泪，点了点头，然后送童筱颖出了门。
<br/>
<br/>        童筱颖顺手将相册也带走了。
<br/>
<br/>        暗恋一个人是什么滋味，期间的酸甜苦辣，是这本相册，让她彻底明白了他的心意。
<br/>
<br/>        而他对她的喜欢是有多深，她也终于能明白了。
<br/>
<br/>        他喜欢她，在死之前，他的心里脑海里全是她，就连是谁给了他致命的伤害，他都无所谓去计较了。
<br/>
<br/>        只因为他仅仅只留最后一口气时，还能再见到她而开心，甚至愿意用他最后一口气去珍惜与她独处时的每一秒。
<br/>
<br/>        这份沉重的爱，压得童筱颖喘不过气来，更加让她有了一份去为马智杰将真凶绳之以法的冲动。
<br/>
<br/>        何明旭，她童筱颖发誓，一定不会让他逍遥法外。
<br/>
<br/>        童筱颖说要回滨江城，她还真就只身回到了滨江城。
<br/>
<br/>        这事，傅夜沉知道，他不放心她，于是跟了过去。
<br/>
<br/>        苏漫雪被留在了傅家大庄园里。
<br/>
<br/>        童筱颖回到家后，给傅昼景打了电话，然而，傅昼景却告诉她，他又在出差的路上。
<br/>
<br/>        她知道这段时间，他很忙，而且忙得焦头烂额，所以，她很乖巧，在电话这头，不吵不闹，愿意安安静静地等他回家。
<br/>
<br/>        两人通话的时间不过五分钟，童筱颖挂了电话后，看着手机屏幕苦笑。
<br/>
<br/>        每一次，他都说，忙完之后，一定好好陪她。
<br/>
<br/>        但是，每一次，他刚忙完，仅陪了她一会儿，又要开始忙碌起来。
<br/>
<br/>        童筱颖有个时候会想，昼景的压力与负担，是不是她造成的？
<br/>
<br/>        因为，在昼景的心里，他一定是想着要养着她。
<br/>
<br/>        其实，童筱颖很想对傅昼景说，她不需要他来养，她也可以好好工作，努力赚钱。
<br/>
<br/>        但是，傅昼景偏偏又是大男子主义的性子。
<br/>
<br/>        思绪从傅昼景的身上回来后，童筱颖开始一个人做饭、一个人吃饭，饭后，暮色降临，她提起包包，去了一趟“夜惑”。
<br/>
<br/>        童筱颖直接找上了费尔南，而费尔南也正为“夜惑”要被停业整顿的事情，急得焦头烂额。
<br/>
<br/>        费尔南本来没心思搭理童筱颖，却又在看到童筱颖后，突然想起了一个人，于是，在态度上，对童筱颖来了个三百六十度的大转弯。

<br/>        “小谢啊！你和那个连少，应该或多或少，都还有联系吧！”费尔南招呼小姐过来给童筱颖泡茶，并让童筱颖在沙发上坐下，有什么事儿，两人可以慢慢说。

<br/>        童筱颖在沙发上坐下后，双手提着包包放在双膝上，也不打算跟费尔南拐弯抹角，直接开门见山地答非所问：“我想找你要一些，‘老实药’，我知道，你这儿一定有！”

<br/>        她在“夜惑”里待的时间不算久，却也能看透一些事情。

<br/>        唐倩兮一走，十爷就把她赶出“夜惑”，多半也有这个因素在其中。
