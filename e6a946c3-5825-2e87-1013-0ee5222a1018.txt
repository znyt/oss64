
<br/>　　<br/>　　    当王惊蛰他们从金字塔中冲出来的时候，首先就感觉到了来自于百米深的水下带来的巨大压力，几乎险些都要把胸腔给挤裂了，深潜都是循序渐进的，由浅入深，不然直接就处于深潜水位的话，可能极易让人的内脏受到严重的危害。<br/>　　<br/>　　    不过，金字塔是在水下古城中，哪怕是突然从中冲出来，由于内外的压力并不是太离谱，所以他们尽管很难受，但还能坚持得住。<br/>　　<br/>　　    进入湖水中，在压力挤压身体的同时，他们的双眼也是不能视物的，只能任由身体漂浮在水中，然后划动双手尽量的往水面上升去。<br/>　　<br/>　　    这个时候他们四人无疑就是在赌博了，赌午桥和蛙人队伍能在这个关键时刻神兵天降的出现在他们面前。<br/>　　<br/>　　    可能是短短的几秒，也可能有一两分钟，这段时间成为了他们有生以来最难度过，也是最漫长的一段时间，几乎每一秒都让他们感觉到了，什么叫度日如年。<br/>　　<br/>　　    “唰！”正当王惊蛰闭着眼睛身体缓慢上升的时候，突然间就觉察到自己的身体被两双手给扶住了，紧接着一个呼吸塞就递到了他的嘴前，当他张开嘴巴咬住的时候，一口浓郁的氧气瞬间进入到胸腹中，顿时就缓解了他的缺氧和压力，紧接着脑袋上也没套上了潜水镜，片刻后王惊蛰睁开眼睛，被人背上了氧气瓶，他就看见自己旁边飘着菜刀文和午桥，林汶骐他们全都被蛙人给夹住了，也带上了潜水装备。<br/>　　<br/>　　    午桥拍了他的肩膀一下，然后摊开手询问他有没有什么状况，王惊蛰伸手比划了个OK的手势，由朝着水面的方向指了指，示意自己没有问题，于是几人就开始缓缓的朝着水面升去。<br/>　　<br/>　　    “哗啦”逐步上潜，一队人陆续从水面钻出，附近两艘船上的人看见他们钻出水面，顿时传来一声惊呼，被困于水下一天一夜的王惊蛰等人，到底还是死里逃生的出来了。<br/>　　<br/>　　    王令歌摘掉护目镜，仰着脑袋大口大口的喘息着，林汶骐舔了舔嘴唇，说道：“从来没有感觉，阳光，空气能让人这么舒坦啊，真他么不容易，重见天日了”<br/>　　<br/>　　    李菲儿掩面哭泣，抽泣了片刻后，朝着黄詹和自己的同学摆了摆手，示意自己没事。<br/>　　<br/>　　    片刻后，王惊蛰和林汶骐还有李菲儿，王令歌进入了减压舱中进行减压，当确定身体没有什么大碍之后，他们才又被送上了船。<br/>　　<br/>　　    到了一艘船上，死里逃生几乎崩溃的他们直接就躺在了甲板上，睁着眼睛看着头顶天空中的太阳，这才再一次的真实的感觉到了，什么叫做重见天日了。<br/>　　<br/>　　    长长的吐了口气，王惊蛰仰着脑袋，伸出两根手指说道：“给我来根烟，憋不住了”<br/>　　<br/>　　    黄九郎皱眉说道：“你刚刚从深水区域回来，最好还是先检查下身体吧”<br/>　　<br/>　　    “没事，我自己什么样我心里有数，不抽这根烟，难以平复我的心情……”<br/>　　<br/>　　    他们四个上了船后，稍微休息稳定了下，马上就有随船的医生开始检查几人的身体状况，血压，心率都查了一遍，简单的初步体检过后，他们除了身体有点虚弱外，其他的指标还算是有点正常，不过也得等上岸了后进行综合性检查，才能最后确定有没有问题。<br/>　　<br/>　　    黄詹有些急不可耐的问道：“你们到了哪里？是不是进入金字塔内部了？”<br/>　　<br/>　　    “对！”<br/>　　<br/>　　    “呼”黄詹吐了口气，激动的搓着手，说道：“说说看，这个金字塔里到底……”<br/>　　<br/>　　    “是滇国国主党羌的墓，墓中有殉葬的士兵，陪葬用品，还有一些关于滇国的描述”王惊蛰一边接受检查，一边简练的将他们在金字塔里的经历叙述了一遍，听的围拢在旁边的人一顿惊诧不已。<br/>　　<br/>　　    船只靠岸以后，他们就被送上了救护车，然后开往了最近的医院，在病房里分别被隔离开了，由于刚从两千多年前的古墓中出来，谁也不敢确定王惊蛰等人的身上会不会携带什么细菌，至少得先要隔离二十四小时之后，检查没事了才能接触隔离，同时也可以检查下身体。<br/>　　<br/>　　    在水下还没有上岸时，林汶骐就偷摸的将那四十九张巴掌大的人皮塞给了陈三岁，同时是示意他收好别漏出去。<br/>　　<br/>　　    一天过后，隔离和检查全都完事了，他们四人的体征没有任何问题，除了李菲儿受到了一些惊吓情绪稍微有些波动，其他三人一切如常。<br/>　　<br/>　　    所以，一天过后就又被送回到了抚仙湖旁的村子，而这次再回来，就是关于抚仙湖水下问题的交代了，并且这次交代问题，并不是那么简单的。<br/>　　<br/>　　    参与的人，除了他们四个当事人，黄九郎，黄詹和张生利外，还有个国安部门的负责人何平，和某办公室来的一个邱秘书，这次抚仙湖事件的前因后果是要被入档的。<br/>　　<br/>　　    村子里的临时指挥部里，屋子里坐了八九个人，王惊蛰，王令歌和林汶骐还有李菲并排坐在四张椅子上，对面是官方的人，甚至还有个专门负责记录的人，门外还有站岗的，而且在这之前他们几个也都被严密看管了起来，都没有机会去和外界联系，保密工作做得可谓是相当到位了。<br/>　　<br/>　　    王惊蛰倒是第一次碰见这种状况，不过他一点压力都没有，湖下发生的事就在他们四个肚子里，上下两片嘴唇，他们怎么说就怎么是了，外人尽管会有异议，可你也找不到啥证据。<br/>　　<br/>　　    这个相当于审查的会议，前半段全是由张生利和黄詹来介绍关于抚仙湖的状况，一直谈论到了发现水下古城，和王惊蛰等人消失为止。<br/>　　<br/>　　    两人说完了以后，黄九郎就看着王惊蛰，示意道：“接下来的事情，就得靠你们口述了，希望还能如实交代”<br/>　　<br/>　　    王令歌忽然插了一嘴，斜了着眼睛说道：“这种状况，不得应该是把我们四个全都隔离开，然后分别询问，免得我们串口供的么？”<br/>　　<br/>　　    黄九郎顿时无语了，就差没跟他们明说了，你们在水下一天一夜，有他么啥口供也早就串好了，隔离询问就跟脱裤子放屁一样，啥用没有。<br/>　　<br/>　　    黄九郎摆了摆手，说道：“不用那么麻烦，又不是审查什么案件你们也没犯法，就在这里说吧，就当是聊天了，不过还是希望你们能尽量详谈一下，别有什么遗漏的地方，就从你们几个消失的时候说起吧”
