　　“我用来干嘛，你不需要知道。你只要按照我的要求，准备好我所需要的东西。”齐凡看着比洛多说道。
<br/>　　齐凡所提出的要求，比洛多根本没有办法拒绝，只能选择答应。比洛多在一份命令上签下自己的名字后，交给齐凡：“拿着这张命令，你们可以得到你们想要的东西。”
<br/>　　接过比洛多递过来的纸，齐凡看了眼上面的内容，收起后对塔尼特说道：“塔尼特小姐，麻烦你再跟我们走一趟吧。”
<br/>　　“我已经按照你们的要求，给了你们命令，你们还想要怎么样！？”比洛多双手撑着桌子，对着齐凡质问道。
<br/>　　“因为你在我的眼中已经不具备诚信。”齐凡看了眼塔尼特，说道：“塔尼特小姐，我们走吧。”
<br/>　　“爸爸，你不用担心我。”塔尼特对脸上露出担忧之色的比洛多说道：“我去去就回来。”
<br/>　　“对于现在所发生的事情，我很抱歉。”
<br/>　　比洛多看着离开书房的齐凡和莱昂说道：“我没有选择，只能这么做！”
<br/>　　齐凡停住脚步，瞥了眼自己身后的比洛多，讥诮一笑，说道：“你不是没有选择，而是你已经做出了选择，在我们跟美国政府之间，你选择了后者，因为在你眼中跟后者合作能带给你的利益更多。”
<br/>　　“之前跟罗队长说好的价值两亿美金的军火合同依旧有效。”比洛多对齐凡说道。
<br/>　　“这样很好。”
<br/>　　齐凡丢下一句话，跟莱昂带着塔尼特离开了书房。比洛多还记得跟罗昊之间那笔总价值两亿美元的军火订单，那自然是再好不过，毕竟没有人会讨厌送上门来的钱。
<br/>　　比洛多讷讷的看着齐凡和莱昂带着塔尼特从书房中离开，最终还是没有能说出一句话来，这是比洛多这几天来第一次对自己做出的选择产生了换衣，只不过开弓没有回头箭，比洛多已经没有重新选择的机会。
<br/>　　有塔尼特还有比洛多亲笔签署的命令在手，齐凡和莱昂很快就得到了他们想要的东西，一架“支奴干”运输直升机和两辆防弹突击车。
<br/>　　把两辆防弹突击车驶入运输直升机上后，齐凡让塔尼特离开运输直升机，对着她说道：“塔尼特小姐，非常感谢你的帮助，你现在可以选择告诉你的父亲我们藏身在哪个破旧学校中，不过想要抓住我们的可能性几乎为零。”
<br/>　　“支奴干”运输机的螺旋桨卷起大量的尘土，一时间，机动部队营地上空尘土飞扬，遮天蔽日。
<br/>　　塔尼特看着升空离开的“支奴干”运输机，心中默然，她知道，齐凡他们还是不相信自己，以后也不会再相信自己。人与人之间的信任没了，就很难再救回来。
<br/>　　……
<br/>　　齐凡、莱昂、布谷、天蛇和玄龟返回那所旧学校后，把那两个被绑起来的警卫释放，然后所有人携带着武器弹药迅速从学校离开。这里已经被塔尼特得知，所以已经被暴露，既然已经暴露就不能再继续留在这里。
<br/>　　在齐凡、梅庆伟他们离开后不久，维尔纳特的“黑箭”小队就赶到了学校。
<br/>　　一架“小鸟”直升机悬停在教学楼前，直升机下方的探照灯打在教学楼上，但因为教学楼的窗户上贴着报纸的关系，雪白的光柱并不能照入教学楼内，看清楚教学楼内的情况。“小鸟”直升机上的火箭弹发射巢正对着教学楼，只要教学楼中有任何异常的情况，直升机上的火箭弹发射巢会把一整层楼层化成火海。
<br/>　　维尔纳特的“黑箭”小队冲进教学楼中进行搜查，而“黑箭”小队进入学校的行动全都被天蛇安装在教学楼中的监控探头给拍摄了下来，并且传到了天蛇的PDA上。
<br/>　　天蛇看着PDA上的影像，轻蔑一笑，说道：“真是狗改不了吃屎，比洛多还是选择了告密。不过也无所谓了，反正他们注定要扑空了。”
<br/>　　“意料之中的事情。”
<br/>　　齐凡坐在机舱中，把那些从水库中取出来并且经过了筛选的子弹压进战术背心的空弹匣中，把压满子弹的弹匣放在桌子上轻轻敲打了几下，说道：“少了一架‘支奴干’运输直升机和两辆防弹突击车，很容易会被发现，比洛多是隐瞒不住的，所以他选择了向维尔纳特告密，这样他就能彻底甩脱自己的责任。”
<br/>　　“可是比洛多是怎么知道我们在那所学校里的？”玄龟对齐凡问道。
<br/>　　“比洛多不知道，但是比洛多的女儿知道。”齐凡把压满子弹的弹匣塞进弹匣槽中，拉开枪栓，把子弹推入枪膛，然后关闭了保险，把枪放在一旁，说道：“从塔尼特跟我们接触，为她父亲说话就能知道，塔尼特跟她父亲的感情很深，如果比洛多去恳求塔尼特，塔尼特一定会说，因为塔尼特知道我们会离开那所学校，所以她也不担心我们会遇到危险。”
